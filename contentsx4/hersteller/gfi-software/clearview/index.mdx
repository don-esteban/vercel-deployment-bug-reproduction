## Lizenzierung

<ArrowList>

  GFI ClearView wird als **Abonnement** (Subscription) **pro Instanz** mit einer Laufzeit von einem Jahr nach der **Anzahl der zu überwachenden Nodes (Endpunkte), je nach Paket S, M oder L,** lizenziert.

  Während der Abonnement-Laufzeit erhalten Sie die folgenden Vorteile:

- Kostenlose Upgrades auf die aktuelle Version
- Bug-Fixes, Patches und Zwischen-Releases
- Technischer Support via Web-Chat, E-Mail und Telefon
- Zugriff auf digitale GFI Knowledge-Base und Foren

</ArrowList>

## Funktionen

<ArrowList>

  GFI ClearView ist eine einfach einzubindende Werkzeugsammlung (als VM/Instanz) mit einer benutzerfreundlichen Oberfläche zur Netzwerk-, Datenverkehrs- und Anwendungsüberwachung in Echtzeit, gekoppelt mit interaktiven Analysen und Reportingfunktionen.

  - Echtzeit-Monitoring
  - Analyse der Anwendungsleistung und Netzwerkaktivität
  - Kontrolle über Netzwerktraffic und Anwendungen
  - Berichte und Warnmeldungen (konfigurierbar)
  - Active Directory Integration
  - SIEM-System Integration
  - Predictive Recommendations
  - Flexibles Deployment
  - Bereitstellung in lokaler VM
  - Zentrales Dashboard

</ArrowList>
