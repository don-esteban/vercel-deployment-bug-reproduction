## Lizenzierung des GFI HelpDesk

  GFI HelpDesk ist eine **Abonnement Lizenz** und wird nach **Benutzer, bzw. IP-Anzahl** lizenziert wird. Ein Software Maintenance Agreement (SMA) ist für die Laufzeit der Lizenz enthalten. Für jeden Arbeitsplatz, der gesichert werden soll, wird eine Lizenz benötigt.

  Für eine Bestellung werden **mindestens 5 Lizenzen** benötigt, ab 10, bzw. 30 Lizenzen erhalten Sie eine bessere Rabattstaffel und günstigere Preise für die Einzellizenz.

  Während der Abonnement-Laufzeit erhalten Sie die folgenden Vorteile:

- Kostenlose Upgrades auf die aktuelle Version
- Bug-Fixes, Patches und Zwischen-Releases
- Technischer Support via Web-Chat, E-Mail und Telefon
- Zugriff auf digitale GFI Knowledge-Base und Foren
