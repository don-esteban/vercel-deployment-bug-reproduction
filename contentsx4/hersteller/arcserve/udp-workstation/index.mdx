## Lizenzierung

<ArrowList>

- Wird nach der **Anzahl der zu sichernden Systeme** lizenziert.
- Unterstützt nur Workstation OS.
- Mindestens ein Jahr **Enterprise Support** benötigt.
- Upgrade von Version 8.0, 7.0 und Version 6.5 Update 4 auf Anfrage
- Wartungsverlängerung oder Wiedereinstieg in die Wartung auf Anfrage.
- Höhere Lizenz-Packs auf Anfrage.
- Laufzeitanpassungen der Wartung sind bei einem Renewal möglich.

</ArrowList>



## Funktionen

<ArrowList>

- Image-basiertes Backup von Windows
- Bare-Metal-Recovery, einschließlich Unterstützung von UEFI und Secure Boot
- Web-basiertes zentrales Management
- Globale quellenbasierte Deduplikation
- WAN-Optimierung
- Virtual Standby auf vSphere und Hyper-V
- Basic-Tape-Backup (Migration von UDP- Wiederherstellungspunkten auf Band, Unterstützung einzelner Band-Laufwerke)
- Microsoft-SQL-Support, einschließlich CSVFS und Point-in-Time-Recovery (Nur SQL Express)

</ArrowList>
