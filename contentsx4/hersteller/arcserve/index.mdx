<Intro>

Arcserve Unified Data Protection (UDP) *protected by* [Sophos Intercept X](/hersteller/sophos/intercept-x-for-server/) bietet eine All-in-One-Lösung für Backup, Disaster-Recovery und Restore für alle Systeme, wie Workstations oder Cluster-Server.
Es gibt Kauflizenzen mit (mindestens einem Jahr) Maintenance oder Mietlizenzen (Subscriptions).
Aktuelle Version ist ArcServe UDP 9 - [Was ist neu?](/hersteller/arcserve/whatsnew_arcserve_udp9-de.pdf)



</Intro>

NEU in UDP Version 9

- Cloud-basierte Verwaltungskonsole: Flexible und sichere Verwaltung, Deduplizierungs-Datenspeicher in der Cloud über die Benutzeroberfläche verfügbar. Ohne Installation.
- Backups direkt in die Cloud-Objektspeicherung: ArcServe UDP Backups können direkt in die Cloud Storage gesichert werden.
- Verbesserte Unterstützung von Unternehmensanwendungen, wie [Oracle-DB](/hersteller/oracle/) Datenbank-Schutz und erweiterte Wiederherstellungsoptionen für [MS SQL-Server](/hersteller/microsoft/sql-server/)
- Unterstützung neuer Plattformen: [Microsoft Windows Server 2022](/hersteller/microsoft/windows-server/), [Microsoft Windows 11](/hersteller/microsoft/windows/), [VMware vSphere 8.0](/hersteller/vmware/vsphere/), [Red Hat Enterprise Linux 9.x](/hersteller/red-hat/enterprise-linux-server/), [SLES 15](/hersteller/suse/linux-enterprise-server/)
- Verbesserungen in Funktion, Sicherheit und Stabilität

## Arcserve UDP Portfolio

<ArrowList>

- [Arcserve UDP 9.0 Workstation](/hersteller/arcserve/udp-workstation/) <br />
  Schutz für Windows Workstations. Basisfunktionen zur Sicherung auf Bandlaufwerke (Basic Tape).
- [Arcserve UDP 9.0 Standard](/hersteller/arcserve/udp-standard/) <br />
  Schützt Windows und Linux Fileserver. Replikation für RPS Datastores, Basic Tape und hostbasierten VM Backups.
- [Arcserve UDP 9.0 Advanced](/hersteller/arcserve/udp-advanced/) <br />
  Zusätzlicher Schutz für Exchange Server. Enthält alle Komponenten der Standard Edition und unterstützt granulare Sicherung von Microsoft SQL und Exchange.
- [Arcserve UDP 9.0 Premium](/hersteller/arcserve/udp-premium/) <br />
  Erweiterte Bandsicherung und Replikation und Dateiebene. Enthält alle Komponenten von Advanced inkl. Premium Tape Funktionalität (CA Arcserve Backup mit allen Optionen und Agenten) und Replikation auf Dateiebene.
- [Arcserve UDP 9.0 Premium Plus](/hersteller/arcserve/udp-premium-plus/) <br />
  Verbesserte Hocherverfügbarkeitsfunktionen. Enthält alle Komponenten von Premium Plus inkl. Premium High Availability (CA Arcserve RHA). Nur noch für Bestandskunden verfügbar.
- [Arcserve UDP Backup for Microsoft 365](/hersteller/arcserve/udp-microsoft-office-365-backup/) <br />
  Sicherung und Schutz von Microsoft 365 Konten, Daten und Dokumenten (Exchange, Teams, SharePoint, OneDrive). Subscription.
- [Arcserve UDP Cloud Hybrid Secured by Sophos](/hersteller/arcserve/udp-cloud-hybrid-secured-by-sophos/) <br />
  Repliziert Ihre Daten oder Sicherungen automatisch von einem lokalen Arcserve UDP-Wiederherstellungspunktserver (RPS) auf einen entsprechende RPS in der Arcserve Cloud.

</ArrowList>

## Lizenzierung

Lizenziert wird allgemein nach Prozessorsockeln (Socket License), nach der verwalteten Speichermenge in TB (Managed Capacity) oder nach Betriebssystem-Instanzen. UDP Office 365 wird nach Usern lizenziert.

Wir geben Ihnen einen kurzen Überblick über die Lizenzierungsoptionen. Ausführliche Informationen finden Sie auf unser Seite [Arcserve Lizenzierung](/hersteller/arcserve/lizenzierung/).
