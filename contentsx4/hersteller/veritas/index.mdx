## Vergleich Backup Exec vs System Recovery

Veritas Backup Exec (BE) führt eine Sicherung auf Dateiebene durch.
BE ist einfacher zu verwalten und bietet ein hohes Maß an Flexibilität und Skalierbarkeit. Datensicherung auf Band wird unterstützt.

Veritas System Recovery (VSR) setzt auf Block- oder Sektor-basierte Sicherung. Auf diese Weise erfasst System Recovery den aktuellen Systemzustand und kann das System in den Zustand zurückversetzen, in dem es sich zum Zeitpunkt der Sicherung befand.

<br />

Die folgende Tabelle zeigt einige der wichtigsten Unterschiede zwischen Veritas Backup Exec und Veritas System Recovery:

| Funktion | Veritas Backup Exec | Veritas System Recovery |
| :--- | :--- | :--- |
| Hauptfunktion | Backup und Wiederherstellung | Disaster Recovery |
| Backup-Typ | Datei | Block/Sektor |
| Direkte Sicherung auf Band | ✓ | – |
| Unterstützung für komplexe Unternehmenssoftware | ✓ | – |
| Wiederherstellung auf unterschiedlicher Hardware | Schwierig | Einfach |
| Bare-Metal-Wiederherstellung | – | ✓ |
| Backup in die Cloud oder Remote-Server | ✓ | ✓ |
| Backup von Cloud- oder hybrider Daten | ✓ | – |
