<Intro>

  Mit der neuen BE Simple Lizenzierung erhalten Sie alle Backup Exec Funktionen mit einer *einfachen, einheitlichen Subscription* für alle IT-Umgebung und MS365 Cloud-Dienste.
  BE Simple ist ein One-For-All Programmpaket, besonders geeignet für kleine und mittlere Unternehmen (KMU).

  Aktuelle Version ist Backup Exec 22.

</Intro>

## Lizenzierung

  Ein Simple Core Pack enthält Lizenzen für 5 Instanzen und 10 Microsoft 365 User. Weitere Instanzen (oder 10 MS365 Benutzer) können mit Add On Packs hinzugefügt werden.

<ArrowList>

- Lizenzart: Abonnement (Subscription)
- Laufzeit: 1 Jahr (mindestens), weitere auf Anfrage
- **Core Pack** (Basis-Lizenz): 5 Instanzen *_und_* 10 Microsoft 365 Benutzern
  - Unter einer Instanz versteht Veritas jede Art von Workload (Prozess, Anwendung oder Datenbank, System, etc.).
  - Jede nicht benötigten Instanz kann für weitere 10 MS365 Benutzer eingesetzt werden. Bei "Nur-Benutzer-Lizenzierung" sind somit bis zu 60 Benutzer lizenziert.
  - Pro Backup Exec Server kann nur ein Simple Core Pack lizenziert werden.
- **Add On**: Erweitert **Core Pack** um eine (1) weitere Instanz *oder* 10 MS365 User
- Während der Laufzeit ist Essential Maintenance mit 7x5 Support enthalten.

</ArrowList>

<Info>

  *Wichtig:* Backup Exec Simple Lizenzen können nicht auf Servern installiert werden, auf denen Legacy Editionen, wie Bronze, Silver, Gold, Custom Editionen oder V-Ray-Lizenzen, installiert sind.
  Kunden mit älteren Lizenzen, die Microsoft 365 Benutzerdaten schützen möchten, müssen ein Backup Exec Simple Core Pack erwerben und einen separaten Backup Exec Server bereitstellen, der unabhängig von Servern mit älteren Lizenzen ausgeführt wird.

</Info>
