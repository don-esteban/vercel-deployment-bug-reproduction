<Intro>

  Veritas bietet mit Backup Exec Lösungen zur Datensicherung und Wiederherstellung (Disaster Recovery) für mittelständische Unternehmen. Unterstützt werden physische, virtuelle und Cloud-Deployments.
  *Backup Exec Simple Core Packs* bietet eine neue Lizenzierung mit einer einheitlichen Subscription für alle Umgebungen und mit M365 Unterstützung an.

  Aktuelle Version ist Backup Exec 22

</Intro>

<TOC />



## Backup Exec Editionen

<Info type="light">

<ArrowItem>


 **[Backup Exec Simple Core Pack](/hersteller/veritas/backup-exec/simple-core-pack/)**

  Die neuen Backup Exec Simple Core Packs enthalten den **Backup Exec Server mit allen Funktionen und Backup für Microsoft 365 Konten** in einem einfache Lizenzierungsmodell für alle Umgebungen.
  Lizenziert wird als Subscription nach Instanzen.

</ArrowItem>

</Info>


### Legacy Editions (Renewals only)

<Info type="light">

<ArrowItem>

  [Backup Exec Bronze, Silver und Gold Editions](/hersteller/veritas/backup-exec/editionen/)

  Backup Exec Legacy Editions (BE Server als Bundle) in der Übersicht.
  *Es werden nur noch Renewals angeboten.*

</ArrowItem>

<ArrowItem>

  [Backup Exec Server](/hersteller/veritas/backup-exec/server/)

  Backup Exec Server, erweiterbar.
  *Es werden nur noch Renewals angeboten.*

</ArrowItem>

</Info>


<ArrowList>

- [Backup Exec Agenten](/hersteller/veritas/backup-exec/agenten/)

  Mit Agenten können Sie die Backup Exec Server Edition erweitern und weitere physische oder virtuelle Systeme, sowie Datenbanken absichern. *(Renewals only)*

- [Backup Exec Optionen](/hersteller/veritas/backup-exec/optionen/)

  Mit Optionen erweitern Sie die Funktionalität der Server Edition: Beispielsweise die zentrale Verwaltung mehrerer Backup Exec Server oder das Hinzufügen weiterer Bandlaufwerke. *(Renewals only)*

- [Backup Exec V-Ray Edition](/hersteller/veritas/backup-exec/v-ray-edition/)

  Die Backup Exec V-Ray Edition ist eine Backup-Komplettlösung für virtuelle VMware und Microsoft Hyper-V Umgebungen lizenziert nach Prozessoren. Agenten für verbreitete Anwendungen sind im V-Ray Bundle enthalten, es kann aber auch der komplette Host gesichert werden. *(Renewals only)*

</ArrowList>
