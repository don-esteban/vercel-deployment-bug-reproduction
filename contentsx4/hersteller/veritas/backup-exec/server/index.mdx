<Intro>

  Veritas Backup Exec Server als Einzelprodukt (renewals only). Jeder Backup Exec Infrastruktur liegen eine oder mehrere Backup Exec Server Lizenzen zugrunde. Der Backup Exec Server, auch Medienserver genannt, bietet die zentrale Verwaltung für Ihre Backups und ist in allen Backup Exec Editionen bereits enthalten. Über den Medienserver werden die Daten im definierten Storage gesichert und/oder auf Band geschrieben.

</Intro>

Neulizenzen sind als [Backup Exec Simple Core Pack](/hersteller/veritas/backup-exec/simple-core-pack/) erhältlich. Darin sind sämtliche Funktionen inklusive.

</Info>

## Lizenzierung

Die Lizenzierung bei der Server Edition ist modular: Die Sicherung von Servern und Datenbanken erfolgt über [Agenten](#agenten), weitere Funktionalität kann über die [Server-Optionen](#optionen) hinzugefügt werden.


### Agenten

  Um unterschiedliche Datenquellen im Unternehmen zu sichern, benötigt man verschiedene Agenten. Für die Sicherung von Windows oder Linux Servern, die nicht als Medienserver genutzt werden, sind Agenten für die Sicherung nötig. Spezielle Datenbanken, wie beispielsweise Exchange, SQL oder Lotus Notes, benötigen für die Sicherung im laufenden Betrieb einen zusätzlichen Agenten, auch wenn der zugrunde liegende Server bereits mit Backup Exec gesichert wird. Für standortferne Server, beispielsweise in anderen Niederlassungen, benötigen Sie Remote-Agenten.

  [Zu den Backup Exec Agenten](/hersteller/veritas/backup-exec/agenten/) *(Renewals Only)*



### Optionen

  Zusätzlich gibt es für Backup Exec weitere Optionen, die bei Bedarf dazu lizenziert werden können. Diese haben wir Ihnen auf einer separaten Seite zu den Backup Exec Optionen aufgelistet.

  [Zu den Backup Exec Optionen](/hersteller/veritas/backup-exec/optionen/) *(Renewals Only)*



### Upgrades

  Veritas bietet Upgrades von der Einzellizenzierung und den älteren Editionen V-Ray und Capacity auf die neuen Backup Exec Bronze, Silver und Gold Bundles. *(nicht mehr möglich)*
