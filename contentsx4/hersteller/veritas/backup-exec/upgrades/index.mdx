  Veritas bietet Kunden mit Backup Exexc die Möglichkeit auf neuere Versionen zu wechseln.

</Intro>

## Upgrade

<Info>

Upgrades auf aktuelle Backup Exec Editionen sind teilweise möglich, aber von unterschiedlichen Faktoren abhängig und müssen individuell für Sie angefragt werden.

Rufen oder schreiben Sie mich an bei Fragen.

</Info>

### Lizenzierung

Beachten Sie bitte, dass verschiedene Lizenzmodelle nicht gemischt werden dürfen.
Beispielsweise darf mit [Backup Exec Simple](/hersteller/veritas/backup-exec/simple-core-pack/) kann kein anderer *Backup Exec* Lizenztyp installiert sein.


### Voraussetzung

Mögliche Upgrades von *Backup Exexc 21* auf die aktuelle Subscription-Version [Backup Exec Simple](/hersteller/veritas/backup-exec/simple-core-pack/) (derzeit v22.1) setzen eine noch unter Support laufenden Lizenz voraus.
Die Laufzeiten bestehender Supportverträge werden mit berücksichtigt.

Eine endgültige Zusage erfolgt erst nach Freigabe des Herstellers.
Bitte haben Sie dafür Verständnis.
