<Intro>

SUSE stellt im Enterprise Segment Linux-Distributionen für Unternehmen bereit, mit dem Fokus auf Stabilität, Sicherheit und lange Wartungszyklen. Neben den Linux Distributionen bietet SUSE auch Lösungen für Container- und Anwendungsbereitstellungstechnologie sowie Linux-Management- und Automatisierungs-Tools. Die Informationen richten sich insbesondere an IT-Einkäufer und Administratoren. Fordern Sie mit einem Klick ein kostenloses und unverbindliches Sofortangebot an.

</Intro>

## SUSE Linux Enterprise (SLE)

SUSE Enterprise Linux kann beispielsweise auch mit [Docker/Kubernetes Container](/hersteller/mirantis-docker/) verwendet werden.

<ArrowList>

- [SUSE Linux Enterprise Desktop (SLED)](/hersteller/suse/linux-enterprise-desktop/) <br />
Linux Distribution für den Desktop. Subscription. Optional mit SUSE Support.
- [SUSE Linux Enterprise Server (SLES)](/hersteller/suse/linux-enterprise-server/) <br />
Linux Server Distribution mit SUSE Support. Erweiterbar mit Add-Ons (z.B. High Availabilty (HA), Virtual Machine Driver Pack).
- [SUSE Linux Enterprise Real Time](/hersteller/suse/linux-enterprise-real-time/) <br />
SLES mit Echtzeitkernel. Speziell für zeitkritische Anwendungen. Verkürzt die Latenzzeiten bei der Prozessverarbeitung.
- [SUSE Linux Enterprise Micro](/hersteller/suse/rancher/) <br />
Linux Distribution für Edge Computing sowie containerisierte und virtualisierte Workloads
- [SUSE Linux Enterprise for SAP Application](/hersteller/suse/rancher/) <br />
Linuxdistribution für SAP HANA-, SAP NetWeaver- und SAP S/4HANA-Lösungen optimiert.

</ArrowList>

## SUSE Managementplattformen

<ArrowList>

- [SUSE Manager](/hersteller/suse/linux-manager/) <br />
Open Source-Infrastrukturmanagement für softwaredefinierte Infrastruktur. Benötigt SUSE Linux Enterprise Server.
- [SUSE Rancher](/hersteller/suse/rancher/) <br />
Kubernetes-Management-Plattform, Bereitstellung der Anwendungen sowie unternehmenskritischer Services und Workloads verwalten und Prozesse automatisieren. Nachfolger von SUSE CaaS Platform.
- [SUSE NeuVector](/hersteller/suse/neuvector/) <br />
Zero-Trust-Containersicherheitsplattform für Container-Management-Plattformen. Stand-Alone oder Integrierbar in Rancher.


</ArrowList>

## Was ist mit SUSE Studio passiert?

SUSE Studio heißt nun SUSE StudioExpress und ist kostenlos. Wie Sie Ihre SUSE Studio Templates in SUSE StudioExpress importieren können, erfahren Sie auf der [SUSE StudioExpress](https://studioexpress.opensuse.org/) Seite.

## openSUSE und Enterprise Testversion

SUSE Linux Enterprise Server basiert auf [openSUSE](https://www.opensuse.org/). Die kostenlose openSource Linuxdistribution enthält Basis Werkzeuge für Selbstkonfiguration des Linux Betriebssystems.

Falls Sie direkt SUSE Linux Enterprise testen wollen, bietet SUSE eine [kostenlose 60-tägige Testversionen](https://www.suse.com/products/?search_type=advanced) an. Die Migration zur Vollversion SUSE Linux Enterprise 15 wird seitens SUSE unterstützt, sodass es keiner Neueinrichtung und -konfiguration bedarf.
