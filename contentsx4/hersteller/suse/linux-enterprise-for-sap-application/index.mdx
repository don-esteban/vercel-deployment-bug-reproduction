<Intro>

SUSE Linux Enterprise for SAP Application ist für SAP HANA-, SAP NetWeaver- und SAP S/4HANA-Lösungen optimiert. Unterstützt TDI-Bereitstellungen (Tailored Datacenter Integration) von SAP HANA und Datenbankcontainern. Beinhaltet High Availability (HA) und Disaster Recovery (DR).

</Intro>

## Preise

Linux Enterprise for SAP Application ist mit [Priority Support](/hersteller/suse/lizenzierung/#priority) verfügbar. Lizenziert wird pro Sockets oder virtuellen Instanzen. Für verschiedene Kundengruppen wie _Bildungseinrichtungen_ gibt es Rabatte und Sonderkonditionen. Gerne prüfe ich, ob für Sie spezielle Rabatte gelten.

Integrierbar in [SUSE Manager](/hersteller/suse/linux-manager/).

### Physical or Virtual

Deckt zwei (virtuelle) Instanzen oder Sockets ab. Es spielt keine Rolle, wie viele CPU/vCPU verwendet werden.

<Products bkey="suse" tags={["enterprise-sap"]}/>

### Physical and Unlimited Virtual

Deckt alle Instanzen auf diesem Server ab. Es müssen hier alle physischen Sockets lizenziert werden. Nur inklusive Live Patching verfügbar.
