<Intro>

SUSE Manager, ein Linux Management- und Automatisierungstool für Open Source IT-Strukturen. Erweitertes Patch-Management, VM-Tuning. Bietet verbesserte Sicherheit und Compliance durch Bereitstellung von OpenSCAP-Inhalten. Unterstützung verschiedener Distributionen und Architekturen, z.B. Red Hat Enterprise Linux (RHEL) Server und auch Mirantis/Docker Container.

</Intro>

## Preise Manager Lifecycle Management

SUSE Manager Server ist nur mit Lifecycle Management (für die Verwaltung der Instanzen) erhältlich. Der Server benötigt die Basis Lizenz SUSE Manager Server. Jede verwaltete Instanz benötigt dann eine SUSE Manager Lifecycle Management Subscription. Benötigt als Voraussetzung einen [SUSE Linux Enterprise Server](/hersteller/suse/linux-enterprise-server/). Der Manager Server ist mit [Priority Support](/hersteller/suse/lizenzierung/#priority) verfügbar. Für zusätzliche Funktionen gibt es [SUSE Manager Monitoring](#monitoring-subscription). Für Kassensysteme bietet sich [SUSE Manager Retail Branch Server](#retail-branch-server-management) an.

Für verschiedene Kundengruppen wie _Bildungseinrichtungen_ gibt es Rabatte und Sonderkonditionen. Gerne prüfe ich, ob für Sie spezielle Rabatte gelten. Mehr dazu unter [Lizenzierung](/hersteller/suse/lizenzierung/).

### Physical or Virtual

Deckt zwei (virtuelle) Instanzen oder Sockets ab. Es spielt keine Rolle, wie viele CPU/vCPU verwendet werden.

### Physical and Unlimited Virtual

Deckt alle Instanzen auf diesem Server ab. Es müssen hier alle physischen Sockets lizenziert werden.

## Preise Add-ons

### Monitoring Subscription

Zur Überwachung von Speicher, CPU, Netzwerkauslastung und Festplatten. Detaillierte Berichte können erstellt werden, z.B. eine grafische Darstellung der Serverleistung. Jede _verwaltete_ Instanz benötigt eine SUSE Manager Monitoring Subscription.

<Products bkey="suse" tags={["manager-monitoring"]}/>


### Retail Branch Server Management

Speziell für _Kassensysteme_ in Supermärkten. SUSE Manager befindet sich in der Zentrale und verbindet sich mit dem Retail Branch Server. Der Retail Branch Server sorgt für die Verteilung auf die Kassensysteme.

<Products bkey="suse" tags={["manager-retail-branch"]}/>


### Virtualization Management

Zentrales Management für VMs über ein Web-Interface. Überwachung und Kontrolle der verschiedenen VMs. Auch Konfigurationen und Installationen von Patches, Paketen, Upgrades etc. möglich.

<Products bkey="suse" tags={["manager-virtualization"]}/>
