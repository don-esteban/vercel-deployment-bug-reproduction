<Intro>

SUSE Linux Enterprise Server Real Time (SLE RT) enthält Linux Enterprise Server mit Performance- und Entwicklungstools sowie einen Echtzeitkernel für zeitkritische Prozesse. Durch Virtualisierung werden Latenzzeiten reduziert und die Anwendungsleistung erhöht.

</Intro>

<TOC />

## Preise

Linux Enterprise Server Real Time (SLE RT) ist mit [Standard](/hersteller/suse/lizenzierung/#standard) oder [Priority Support](/hersteller/suse/lizenzierung/#priority) verfügbar. Lizenziert wird nach Sockets oder virtuellen Instanzen. Es gibt auch dazu buchbare [Add-Ons](#preise-add-ons). Für verschiedene Kundengruppen wie _Bildungseinrichtungen_ gibt es Rabatte und Sonderkonditionen. Gerne prüfe ich, ob für Sie spezielle Rabatte gelten.


### Physical or Virtual

Deckt zwei virtuelle Instanzen ab (Instanz-Paar). Es spielt keine Rolle, wie viele CPU/vCPU verwendet werden.

### Physical and Unlimited Virtual

Deckt alle Instanzen auf diesem Server ab. Es müssen hier alle physischen Sockets lizenziert werden. Nur inklusive Live Patching verfügbar.

## Preise Add-Ons

Inherited Subscription bedeutet, dass der Inhalt der SLE Subscription vererbt wird z.B. Support. Eine Lizenz deckt 1-2 physische Sockets ab. Inherited Virtual deckt die Anzahl an virtuellen Instanzen der SLE Subscription ab.

### High Availabilty (HA)

Für HA-Clustering zur Automatisierung von Wiederherstellung von Datenverlusten. Kann in physischen als auch in virtuellen Umgebungen implementiert werden.

<Products bkey="suse" tags={["addon-ha"]}/>


### Live Patching

Benötigt SUSE Linux Enterprise _Priority_. Kein Neustart des Systems nötig und reduziert so die Ausfallzeiten. Unterbrechungsfreier Systembetrieb während dem Update. Ab 10 Systemen empfehlen wir den [SUSE Manager](/hersteller/suse/linux-manager/) einzusetzen.

<Products bkey="suse" tags={["addon-live"]}/>


### Virtual Machine Driver Pack

Für 1-4 Virtual Machines oder Unlimited Virtual Machines. Enthält paravirtualisierte Festplatten-, Netzwerk- und Balloon-Treiber. Verlängert die Nutzungsdauer unternehmenseigener und älterer Anwendungen (speziell im Fall älterer Betriebssysteme).

<Products bkey="suse" tags={["addon-driver-pack"]}/>


### Workstation Extension

Eine Lizenz pro PC. Eine Instanzierung möglich. Enthält Anwendungen von SUSE Linux Enterprise Desktop, wie LibreOffice, E-Mail und Kalender.

<Products bkey="suse" tags={["addon-workstation"]}/>
