Auf diesen Seiten finden Sie Informationen zu Parallels Lizenzen und Preisen, für wen die Lösungen geeignet sind und wo Unterschiede sind.

## Produkte

### Remote Application Server (RAS)

<Info type="light">

<ArrowItem>

  [Parallels RAS](/hersteller/parallels/ras/) <br />

  Parallels VDI-Lösung (Virtuelle Desktop Infrastruktur) zur Ausführung von Applikationen und Desktops auf beliebigen Endgeräten. On-Premises-, hybrid und Cloud-Deployment.

</ArrowItem>

</Info>

### Desktop for Mac Pro Edition

<Info type="light">

<ArrowItem>

  [Parallels Desktop für Mac Pro Edition](/hersteller/parallels/desktop-mac-pro-edition/) <br />

  Parallels VM-Lösung zur Nutzung von Windows und Linux auf Geräten mit macOS.
  Konzipiert für einzelne Entwickler und Tester.

</ArrowItem>

</Info>


### Desktop for Mac Business Edition

<Info type="light">

<ArrowItem>

 [Parallels Desktop für Mac Business Edition](/hersteller/parallels/desktop-mac-business-edition/)  <br />

  Parallels VM-Lösung für größere Entwickler Teams, zur Nutzung von Windows und Linux auf Geräten mit macOS.
  Konzipiert für Entwickler-Teams. Einheitlicher Lizenzschlüssel.

</ArrowItem>

</Info>

### Desktop for ChromeOS


<Info type="light">

<ArrowItem>

  [Parallels Desktop für ChromeOS](/hersteller/parallels/desktop-chromebook-enterprise/) <br />

  Nutzung von Windows und ChromeOS auf ChromeBook Enterprise Geräten mit zusätzlichen Funktionen für zentrale Verwaltung und Sicherheit.


</ArrowItem>

</Info>
