## Lizenzierung

<ArrowList>

- Lizenziert wird pro Nutzer. Ab 26, bzw. 51 Benutzern erhalten Sie eine höhere Rabattstaffel und bessere Konditionen. 
- Als Subscription für 1 Jahr verfügbar. Andere Laufzeiten auf Anfrage.
- Kostenlose Updates und Support sind inbegriffen.
- Sie erhalten einen einzigen Lizenzschlüssel für alle Lizenzen.
- Nachkauf (auf den vorhanden Lizenzschlüssel) jederzeit möglich.

</ArrowList>


<Info>

Für einzelne Nutzer empfiehlt sich die [Parallels Desktop für Mac Pro Edition](/hersteller/parallels/desktop-mac-pro-edition/).

</Info>


## Funktionen

<ArrowList>

- Vollständige Unterstützung der Apple Silicon Chips
- Vollständige Unterstützung für macOS Sonoma 14 (und ältere ab Mojave 10.14)
- Mac Studio Nutzer können einer Windows-11-VM kann nun bis zu 128 GB RAM zuweisen 
- Bessere Windows 11 Unterstützung (auch mit TPM 2.0)
- Unterstützt Anmeldung bei Windows über die Touch ID
- Unterstützung von OpenGL bis Version 4.1
- Erweiterte Unterstützung für x86-basierte Windowsanwendungen auf Mac mit M-CPUs
- Nahtlose Integration von Windows Zwischenablage und Ordner in macOS
- Aktuelle Linuxversionen werden unterstützt (u.a. Ubuntu 23.04, RHEL9)
- Erweiterte Netzwerkkonfiguration und Verwaltung für VMs
- Single Sign-on/SAML für Mitarbeiter

</ArrowList>


<Info>

Version 18.x enthält als letzte Ausgabe Parallels Tools mit Unterstützung für Windows 8.1 und älter, da Microsoft die nötige Treibersignierung nicht weiter unterstützt.

</Info>


## Sonstiges

Die Virtualisierungslösung richtet sich an **Entwickler-Teams**, die mit Werkzeugen wie z.B. [Microsoft Visual Studio](/hersteller/microsoft/visual-studio/) arbeiten. Unterstützt werden ebenso Cloud-Services für Unternehmen. Kann auch in Chef, Ohai, Vagrant oder [Mirantis-Docker](/hersteller/mirantis-docker/) integriert werden.
