<Intro>

JetBrains PhpStorm bietet eine auf PHP zugeschnittene Entwicklungsumgebung und basiert auf IntelliJ IDEA.  Darüber hinaus unterstützt die IDE PHPTools sowie Webtechnologien (HTML5, CSS, Sass, Less, Stylus, CoffeeScript, TypeScript, Emmet, JavaScript).

</Intro>

## Preise

<ArrowList>

- Lizenziert nach Nutzern. Jährliche Subscriptions enthalten dauerhafte Lizenzen (Fallback-Lizenz)
- Ab 50 Lizenzen erhalten Sie optional den JetBrains Lizenz-Server und auf Anfrage optionalen Umstieg auf floating Lizenzen (Zugriffs-/Concurrent-Lizenzen)
- Siehe auch [Lizenzierung - Entwicklungsumgebungen (IDEs)](/hersteller/jetbrains/#kauf-und-lizenzierung)

</ArrowList>

<Products bkey="jetbrains" tags={["phpstorm"]}/>

<GetInTouch>

Upgrade-, (Mitbewerber-)umstieg und Renewal-Preise erhalten Sie auf Anfrage. Bei größeren Lizenzmengen und mehrjährigen Verlängerungen erhalten Sie mehr Rabatte. Gerne können Sie mich kontaktieren.

</GetInTouch>

## Funktionen


- Unterstützung von PHPTools, sowie Webtechnologien (JavaScript, XML, HTML, CSS)
- Unterstützung von Vagrant, Docker, Composer
- Unterstützt SQL Einbindung
- Smart PHP Code Editor
- Code-Refactoring und -Generierung
- Code Analyse während dem Arbeiten
- HTML/CSS/JavaScript Editor
- Xdebug oder Zend Debugger für Reports
- Versionskontrolle
- Unit Tests


_*Eine vollständige Funktionsübersicht finden Sie auf der [JetBrains PhpStorm Funktionsübersichts Seite](https://www.jetbrains.com/de-de/phpstorm/features/)._
