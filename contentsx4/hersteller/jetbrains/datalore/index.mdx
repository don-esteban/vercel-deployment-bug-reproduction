<Intro>

JetBrains Datalore ist eine Notebook-Plattform für Data-Science- und Analytics-Teams. Datalore Enterprise ermöglicht Unternehmen die individuelle Konfiguration von SSO, Umgebung und Computing-Ressourcen und die Installation von Datalore in eigenen Hosting-Diensten wie Kubernetes-Clustern.

</Intro>

## Funktionen

- Remote-Jupyter-Notebooks
- Unterstützung für Python
- R-Unterstützung	Über Plugin
- Scala-Unterstützung
- Unterstützung für Datenbanken
- SQL-Unterstützung
- Gemeinsame Echtzeit-Bearbeitung von Code
- Notebooks teilen
- Versionsverwaltung: Interne Versionierung
- Veröffentlichung von Berichten
- Terminal
- Rechenressourcen: Cloud-Systeme (CPUs und GPUs)


_*Eine vollständige Funktionsübersicht finden Sie auf der [JetBrains Datalore Funktionsübersichts Seite](https://www.jetbrains.com/datalore/features/)._
