<Intro>

JetBrains All Products Pack, auch als JetBrains Toolbox bekannt, bietet Zugriff auf alle JetBrains IDEs und Profiler.

</Intro>

<TOC />



## Preise

<ArrowList>

- Als **jährliche Nutzer-Abonnements** (Subscription) erhältlich. Jährliche Subscriptions enthalten dauerhafte Lizenzen
- Ab 50 Lizenzen erhalten Sie optional den JetBrains Lizenz-Server und auf Anfrage optionalen Umstieg auf floating Lizenzen (Zugriffe)
- Siehe auch [Lizenzierung - Entwicklungsumgebungen (IDEs)](/hersteller/jetbrains/#kauf-und-lizenzierung)

</ArrowList>

<Products bkey="jetbrains" tags={["all"]}/>

<GetInTouch>

Upgrade-, (Mitbewerber-)umstieg und Renewal-Preise erhalten Sie auf Anfrage. Bei größeren Lizenzmengen und mehrjährigen Verlängerungen erhalten Sie mehr Rabatte. Gerne können Sie mich kontaktieren.

</GetInTouch>

## Bestandteile

<ArrowList>

- [CLion](/hersteller/jetbrains/clion/)  <br />
Cross-Plattform IDE für C and C++.
- [DataGrip](/hersteller/jetbrains/datagrip/)  <br />
Vereint eine Vielzahl von SQL-Datenbanken (mit JDBC Anbindung).
- [DataSpell](/hersteller/jetbrains/dataspell/)  <br />
Interaktive Python-Konsole mit integrierten Ausgabemöglichkeiten. Für Jupyter Notebooks.
- [GoLand](/hersteller/jetbrains/goland/)  <br />
Entwicklungsumgebung für die Programmiersprache Go.
- [IntelliJ Idea Ultimate](/hersteller/jetbrains/intellij-idea-ultimate/)  <br />
Bietet Unterstützung bei der Entwicklung von Web und Enterprise Anwendungen mit Java und JVM.
- [PhpStorm](/hersteller/jetbrains/phpstorm/)  <br />
Schnelle PHP IDE mit Code Quality Analysis und Smart PHP Code Editor.
- [PyCharm](/hersteller/jetbrains/pycharm/)   <br />
Python IDE für professionelle Entwickler.
- [RubyMine](/hersteller/jetbrains/rubymine/) <br />
Entwicklungsumgebung für Ruby und Rails.
- [WebStorm](/hersteller/jetbrains/webstorm/) <br />
JavaScript IDE mit Unterstützung für Angular, React, Vue.js, Ionic, Cordova, React Native, Node.js, Meteor, Electron.
- [Rider](/hersteller/jetbrains/rider/) <br />
Entwickeln von .NET, ASP.NET, .NET Core, Xamarin oder Unity Anwendungen unter Windows, Mac und Linux.
- [ReSharper](/hersteller/jetbrains/resharper/) <br />
Bundle der beiden ReSharper Editionen für .NET und C++.
- [dotMemory](https://www.jetbrains.com/de-de/dotmemory/) <br />
Speicherauslastung in einer Vielzahl von .NET- und .NET Core-Anwendungen analysieren.
- [dotCover](https://www.jetbrains.com/de-de/dotcover/) <br />
.NET Unit-Test-Runner und Code-Coverage-Tool.
- [dotTrace](https://www.jetbrains.com/de-de/profiler/) <br />
.NET-Leistungsprofiler.

</ArrowList>
