<Intro>

JetBrains Rider, eine Cross-Plattform Entwicklungsumgebung für .NET, ASP.NET, .NET Core, Xamarin und Unity zur Erstellung von Anwendungen unter Windows, Mac und Linux. Bietet zusätzlich hilfreiche Editierwerkzeuge und Codevisualisierung. Basiert auf der IntelliJ Plattform und ReSharper.

Auch Teil des neuen Bundles [dotUltimate](/hersteller/jetbrains/dotultimate/), enthält auch bereits ReSharper (Ultimate).



</Intro>

## Preise

- Lizenziert nach Nutzern. Jährliche Subscriptions enthalten dauerhafte Lizenzen (Fallback-Lizenz)
- Ab 50 Lizenzen erhalten Sie optional den JetBrains Lizenz-Server und auf Anfrage optionalen Umstieg auf floating Lizenzen (Zugriffs-/Concurrent-Lizenzen)
- Siehe auch [Lizenzierung - Entwicklungsumgebungen (IDEs)](/hersteller/jetbrains/#kauf-und-lizenzierung)

<Products bkey="jetbrains" tags={["rider"]}/>

<GetInTouch>

Upgrade-, (Mitbewerber-)umstieg und Renewal-Preise erhalten Sie auf Anfrage. Bei größeren Lizenzmengen und mehrjährigen Verlängerungen erhalten Sie mehr Rabatte. Gerne können Sie mich kontaktieren.

</GetInTouch>

## Funktionen

- Entwicklen von .NET, ASP.NET, .NET Core, Xamarin oder Unity Anwendungen unter Windows, Mac oder Linux
- Unit Test Runner
- Code-Refactoring
- Code Analyse während dem Arbeiten
- Code-Editor für Syntax und Semantic Highlighting
- Decompiler
- Versionskontrolle mit Git, Subversion, Mercurial, Perforce und TFS
- Unit Tests
- Datenbanken und SQL Unterstützung
- Einstiegspaket mit beliebten Templates zur Vorlage

## All-in-One Lösung: dotUltimate

[dotUltimate](/hersteller/jetbrains/dotultimate/) enthält [ReSharper](/hersteller/jetbrains/resharper/) und auch [Rider](/hersteller/jetbrains/rider/) zusammen mit weiteren Tools zur .NET Entwicklung. Entwickler die die Plattform sowie die Visual Studio Plugins für .NET und C++ nutzen, soll somit u.a der Wechsel zwischen Rider und Visual Studio mit ReSharper erleichtert werden.
