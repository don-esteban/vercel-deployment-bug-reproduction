<Intro>

JetBrains dotUltimate ist die neue All-in-One Lösung bestehend aus ReSharper (inkl. ReSharper C++), JetBrains Rider, dotMemory, dotTrace, dotPeek und dotCover. Das Bundle löst ReSharper Ultimate (+Rider) ab und soll u.a den Wechsel zwischen Rider und Visual Studio mit ReSharper erleichtern. Die weiteren Tools für Memory und Performance Profiler, Unit Test Runner unterstützen die .NET Entwicklung.

Hinweis: [ReSharper](/hersteller/jetbrains/resharper/) oder [Rider](/hersteller/jetbrains/rider/) können Sie auch weiterhin einzeln kaufen.


</Intro>

## Funktionen

- Integrierte Unterstützung für VCS
- Lokaler Verlauf immer im Überblick
- SQL-Bearbeitungstools aus dem DataGrip-Team
- Zugriff auf dotTrace, dotMemory und dotCover
- Plugins für dotCover und dotTrace in Rider
- Entwicklen von .NET und C++ Anwendungen
- Unit Test Runner
- Code-Refactoring
- Code Analyse während dem Arbeiten
- Code-Editor für Syntax und Semantic Highlighting
- Decompiler
- Versionskontrolle mit Git, Subversion, Mercurial, Perforce und TFS
- Unit Tests
- Datenbanken und SQL Unterstützung
- Einstiegspaket mit beliebten Templates zur Vorlage
