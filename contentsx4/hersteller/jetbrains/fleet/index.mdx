<Intro>

JetBrains Fleet ist eine neu entwickelte, verteilte IDE-Architektur, die die IntelliJ Code Processing Engine mit einer vereinfachten Benutzeroberfläche kombiniert. Sie bietet eine intuitive, kontextsensitive Entwicklungsumgebung, in der Teams an gemeinsamen Projekten arbeiten können. Fleet befindet sich aktuell im Preview. Das Release Datum steht noch nicht fest.

</Intro>

<TOC />


## Lizenzierung und Preise

Fleet wird als Subscription angeboten. Preise stehen noch nicht fest. Die Kosten werden sich, laut JetBrains, im ähnlichen Umfang belaufen wie der anderen IDEs.

## Testversion

Aktuell können Sie JetBrains Fleet kostenlos testen. Die Testversion können Sie für Windows, macOS und Linux auf der [JetBrains seite Download Seite](https://www.jetbrains.com/de-de/fleet/download/#section=windows) über die Toolbox App herunterladen.

## Funktionen


- Für macOS, Windows oder Linux.
- Unterstützt Python, Java, JavaScript, C#, C++, PHP, TypeScript, Go, Kotlin und Rust. Weitere wie Ruby folgen.
- Erweiterbar über Plugins.
- Nutzen Sie Fleet mit Space, einer vorkonfigurierten, leistungsstarken virtuellen Maschine.
