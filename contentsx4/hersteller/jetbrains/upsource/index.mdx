<Intro>

JetBrains UpSource ist ein Code-Review Tool mit Funktionen eines Repository Browsers, welches die Projektanalyse in Teamarbeit unterstützt.

**UpSource ist zum 1. Februar 2022 abgekündigt. Der Nachfolger ist [JetBrains Space](/hersteller/jetbrains/space/).**

</Intro>

<TOC />



## Preise

<ArrowList>

- Lizenziert wird nach der Anzahl der Nutzer
- Mehr unter [Lizenzierung](/hersteller/jetbrains/#kauf-und-lizenzierung)

</ArrowList>

<Info>

UpSource ist seit dem 1. Februar 2022 kommerziell nicht mehr erhältlich. Der Nachfolger ist [JetBrains Space](/hersteller/jetbrains/space/). Technischer Support und kritische Sicherheitsupdates laufen noch bis zum 31. Januar 2023.

Mehr Informationen zur Abkündigung finden Sie im [JetBrains Blog](https://blog.jetbrains.com/upsource/2022/01/31/upsource-end-of-sales-announcement/).

</Info>



## Funktionen

- Unterstützt Java, Kotlin, PHP, JavaScript, and Python
- Code-Refactoring und -Generierung
- Code Analyse
- Run & Debug
- Unit Tests
- Code-Editor für Syntax und Semantic Highlighting
- Team Collaboration
- Autorisierungskontrolle durch verschiedene Autorisierungsmodule z.B. wie Projektzuweisung der Benutzer
- Versionskontrolle
