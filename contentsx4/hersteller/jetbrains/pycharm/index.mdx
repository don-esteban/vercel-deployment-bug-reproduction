<Intro>

JetBrains PyCharm bietet eine native Cross-Plattform für Python und basiert auf IntelliJ IDEA. Die IDE unterstützt auch wissenschaftliche Anwendungen und Web-Technologien (JavaScript, CoffeeScript, TypeScript, Cython, SQL, HTML/CSS, Template-Languages, AngularJS, Node.js).

</Intro>

## Preise

<ArrowList>

- Lizenziert nach Nutzern. Jährliche Subscriptions enthalten dauerhafte Lizenzen (Fallback-Lizenz)
- Ab 50 Lizenzen erhalten Sie optional den JetBrains Lizenz-Server und auf Anfrage optionalen Umstieg auf floating Lizenzen (Zugriffs-/Concurrent-Lizenzen)
- Siehe auch [Lizenzierung - Entwicklungsumgebungen (IDEs)](/hersteller/jetbrains/#kauf-und-lizenzierung)

</ArrowList>

<Products bkey="jetbrains" tags={["pycharm"]}/>

<GetInTouch>

Upgrade-, (Mitbewerber-)umstieg und Renewal-Preise erhalten Sie auf Anfrage. Bei größeren Lizenzmengen und mehrjährigen Verlängerungen erhalten Sie mehr Rabatte. Gerne können Sie mich kontaktieren.

</GetInTouch>

## Funktionen


- Neu: Unterstützung für lokale Jupyter-Notebooks
- Cross-Plattform-Entwicklung mit Python
- Unterstützung von Django, Flask, Google App Engine, Pyramid, und web2py, sowie Webtechnologien (JavaScript, CoffeeScript, TypeScript, Cython, SQL, HTML/CSS, template languages, AngularJS, Node.js und mehr)
- Unterstützung von wissenschaftlichen Anwendungen wie Anaconda, Numpy, Matplotlib
- Unterstützt [Oracle](/hersteller/oracle/), [SQL Server](/hersteller/microsoft/sql-server/), PostgreSQL, MySQL und andere Datenbanken
- Interaktive Python Konsole
- Smart Code Editor
- Live Edit Vorschau
- Code-Refactoring und -Generierung
- Code Analyse während dem Arbeiten
- Eingebautes Terminal und Docker
- Versionskontrolle sowie Deployment und Remote Development
- Unit Tests
- Individualisierbare Oberfläche
- Mehr als 50 Plugin Erweiterungen


_*Eine vollständige Funktionsübersicht finden Sie auf der [JetBrains PyCharm Funktionsübersichts Seite](https://www.jetbrains.com/de-de/pycharm/features/)._
