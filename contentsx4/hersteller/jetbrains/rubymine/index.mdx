<Intro>

JetBrains RubyMine bietet eine Entwicklungsumgebung für Ruby und Rails und basiert auf IntelliJ IDEA. Die IDE unterstützt auch Webtechnologien wie JavaScript und CoffeeScript, ERB und HAML, CSS, Sass und Less.


</Intro>

## Preise

<ArrowList>

- Lizenziert nach Nutzern. Jährliche Subscriptions enthalten dauerhafte Lizenzen (Fallback-Lizenz)
- Ab 50 Lizenzen erhalten Sie optional den JetBrains Lizenz-Server und auf Anfrage optionalen Umstieg auf floating Lizenzen (Zugriffs-/Concurrent-Lizenzen)
- Siehe auch [Lizenzierung - Entwicklungsumgebungen (IDEs)](/hersteller/jetbrains/#kauf-und-lizenzierung)

</ArrowList>

<Products bkey="jetbrains" tags={["rubymine"]}/>

<GetInTouch>

Upgrade-, (Mitbewerber-)umstieg und Renewal-Preise erhalten Sie auf Anfrage. Bei größeren Lizenzmengen und mehrjährigen Verlängerungen erhalten Sie mehr Rabatte. Gerne können Sie mich kontaktieren.

</GetInTouch>

## Funktionen

- Unterstützung von Webtechnologien wie JavaScript, CoffeeScript, TypeScript, HTML, CSS, Less oder Sass
- Unterstützt die volle Funktionalität von Datenbank Editoren und SQL
- Ruby Code Editor
- Code-Refactoring und -Generierung
- Code Analyse während dem Arbeiten
- Code-Editor für Syntax und Semantic Highlighting
- Run & Debug
- Versionskontrolle
- Eingebautes Terminal


_*Eine vollständige Funktionsübersicht finden Sie auf der [JetBrains RubyMine Funktionsübersichts Seite](https://www.jetbrains.com/ruby/features/)._
