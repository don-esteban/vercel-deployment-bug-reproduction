<Intro>

JetBrains GoLand bietet eine auf Go zugeschnittene Entwicklungsumgebung und basiert auf IntelliJ IDEA. Die IDE unterstützt GoTools sowie Webtechnologien (JavaScript, XML, HTML, CSS).

</Intro>

## Preise

<ArrowList>

- Lizenziert nach Nutzern. Jährliche Subscriptions enthalten dauerhafte Lizenzen (Fallback-Lizenz)
- Ab 50 Lizenzen erhalten Sie optional den JetBrains Lizenz-Server und auf Anfrage optionalen Umstieg auf floating Lizenzen (Zugriffs-/Concurrent-Lizenzen)
- Siehe auch [Lizenzierung - Entwicklungsumgebungen (IDEs)](/hersteller/jetbrains/#kauf-und-lizenzierung)

</ArrowList>

<GetInTouch>

Upgrade-, (Mitbewerber-)umstieg und Renewal-Preise erhalten Sie auf Anfrage. Bei größeren Lizenzmengen und mehrjährigen Verlängerungen erhalten Sie mehr Rabatte. Gerne können Sie mich kontaktieren.

</GetInTouch>

## Funktionen


- Unterstützung von GoTools, sowie Webtechnologien (JavaScript, XML, HTML, CSS)
- Unterstützt SQL Einbindung
- Smart Editor
- Code-Refactoring und -Generierung
- Code Analyse während dem Arbeiten
- Code-Editor für Syntax und Semantic Highlighting (z.B. Plan9), sowie Parameter Hints
- Testrunner Run & Debug
- QuickFixes wie hilfreiche Pop-Ups oder Fehleranalyse
- Versionskontrolle
- Eingebautes Terminal und Docker
- Mehr als 50 Plugin Erweiterungen


_*Eine vollständige Funktionsübersicht finden Sie auf der [JetBrains GoLand Funktionsübersichts Seite](https://www.jetbrains.com/de-de/go/features/)._
