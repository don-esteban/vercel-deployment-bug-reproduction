<Intro>

JetBrains ReSharper ist ein Plugin für Microsoft Visual Studio und erweitert dieses um zahlreiche Funktionen, um die Entwicklung von .NET-Programmen zu vereinfachen und zu beschleunigen. ReSharper bietet unter anderem eine verbesserte Navigation und Suche, Codeanalyse mit Verbesserungsvorschlägen oder automatische Generierung und Formatierung.

Mit Erscheinen der JetBrains Version 2020.2 enthält ReSharper auch Zugriff auf ReSharper C++ wie Code-Assistenz. Dafür ist ReSharper C++ nicht mehr einzeln erhältlich.

ReSharper ist auch Teil des neuen Bundles [dotUltimate](/hersteller/jetbrains/dotultimate/), enthält auch bereits ReSharper (Ultimate).

</Intro>

## Preise

<ArrowList>

- Lizenziert nach Nutzern. Jährliche Subscriptions enthalten dauerhafte Lizenzen (Fallback-Lizenz)
- Ab 50 Lizenzen erhalten Sie optional den JetBrains Lizenz-Server und auf Anfrage optionalen Umstieg auf floating Lizenzen (Zugriffs-/Concurrent-Lizenzen)
- Siehe auch [Lizenzierung - Entwicklungsumgebungen (IDEs)](/hersteller/jetbrains/#kauf-und-lizenzierung)

## Funktionen


- On-the-fly Code Analyse für C#, VB.NET, XAML, ASP.NET, JavaScript, TypeScript, CSS, HTML, and XML
- Code-Refactoring und -Generierung
- Code Analyse während dem Arbeiten
- Code-Editor für Syntax und Semantic Highlighting
- Neu: Code-Assistenz für Syntax und Semantic Highlighting
- Automatische Code Formatierung und Einhaltung von Standards
- Unit Test Runner
- Einstiegspaket mit beliebten Templates zur Vorlage
- Neu: Verbesserung der Navigation und Suche


## All-in-One Lösung: dotUltimate

[dotUltimate](/hersteller/jetbrains/dotultimate/) enthält [ReSharper](/hersteller/jetbrains/resharper/) und auch [Rider](/hersteller/jetbrains/rider/) zusammen mit weiteren Tools zur .NET Entwicklung. Entwickler die die Plattform sowie die Visual Studio Plugins für .NET und C++ nutzen, erhalten diese somit günstiger als beim Kauf der Einzelprodukte.
