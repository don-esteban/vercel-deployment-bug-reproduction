<Intro>

JetBrains WebStorm bietet eine auf Javascript zugeschnittene Entwicklungsumgebung und basiert auf IntelliJ IDEA. Zusätzlich unterstützt die IDE Webtechnologien wie HTML5, Node.js, Bootstrap, Angular/AngularJS, TypeScript, PhoneGap/Cordova, Dart, hauptsächlich zur Entwicklung mobiler Anwendungen.

</Intro>

## Preise

<ArrowList>

- Lizenziert nach Nutzern. Jährliche Subscriptions enthalten dauerhafte Lizenzen (Fallback-Lizenz)
- Ab 50 Lizenzen erhalten Sie optional den JetBrains Lizenz-Server und auf Anfrage optionalen Umstieg auf floating Lizenzen (Zugriffs-/Concurrent-Lizenzen)
- Siehe auch [Lizenzierung - Entwicklungsumgebungen (IDEs)](/hersteller/jetbrains/#kauf-und-lizenzierung)

</ArrowList>


<Products bkey="jetbrains" tags={["webstorm"]}/>

<GetInTouch>

Upgrade-, (Mitbewerber-)umstieg und Renewal-Preise erhalten Sie auf Anfrage. Bei größeren Lizenzmengen und mehrjährigen Verlängerungen erhalten Sie mehr Rabatte. Gerne können Sie mich kontaktieren.

</GetInTouch>

## Funktionen


- Unterstützung von Webtechnologien wie HTML5, Node.js, Bootstrap, Angular/AngularJS, TypeScript, PhoneGap/Cordova, Dart
- Build Tools für Grunt, Gulp oder npm Tasks
- Smart Editor
- Code-Refactoring und -Generierung
- Code Analyse während dem Arbeiten
- Code-Editor für Syntax und Semantic Highlighting
- Testrunner Run & Debug
- Versionskontrolle
- Unit Tests
- Individualisierbare Oberfläche
- Einstiegspaket mit beliebten Templates zur Vorlage
