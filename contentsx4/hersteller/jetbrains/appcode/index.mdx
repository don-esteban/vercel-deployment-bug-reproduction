<Intro>

JetBrains AppCode bietet eine Entwicklungsumgebung für Macs (iOS und macOS) zur Programmierung mit Swift, Objective-C, C++, sowie Unterstützung für Web-Entwicklung und Cross-Plattform. Appcode ist zum 14. Dezember abgekündigt.

</Intro>

<TOC />




## Preise

<ArrowList>

- Lizenziert nach Nutzern. Jährliche Subscriptions enthalten dauerhafte Lizenzen (Fallback-Lizenz)
- Ab 50 Lizenzen erhalten Sie optional den JetBrains Lizenz-Server und auf Anfrage optionalen Umstieg auf floating Lizenzen (Zugriffe)
- Siehe auch [Lizenzierung - Entwicklungsumgebungen (IDEs)](/hersteller/jetbrains/#kauf-und-lizenzierung)

</ArrowList>

<Info>

JetBrains bietet seit dem 14. Dezember 2022 keine neuen Abonnements oder Verlängerungen von AppCode-Lizenzen mehr an. <br />
Alle aktiven Abonnements erhalten eine Fallback-Lizenz. Technischer Support und kritische Sicherheitsupdates laufen noch bis zum 31. Dezember 2023. AppCode wurde ersatzlos gestrichen, einen Nachfolger gibt es nicht.

Siehe dazu [Blog-Beitrag bei Jetbrains](https://blog.jetbrains.com/appcode/2022/12/appcode-2022-3-release-and-end-of-sales-and-support/)

</Info>


## Funktionen

- Unterstützt Objective-C, C++ und Swift
- Unterstützt Webentwicklung mit JavaScript, XML, HTML, CSS and XPath
- Smart editor
- Code-Refactoring und -Generierung
- Code Analyse
- Run & Debug
- Unit Tests
- Versionskontrolle
- Xcode Unterstützung

_*Eine vollständige Funktionsübersicht finden Sie auf der [JetBrains AppCode Funktionsübersichts Seite](https://www.jetbrains.com/de-de/objc/features/)._
