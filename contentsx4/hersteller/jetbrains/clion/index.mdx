<Intro>

JetBrains CLion bietet eine native Cross-Plattform für die Entwicklung von C, C++, Kotlin/Native, Rust und Swift Programmen. Außerdem wird die CMake Programmiersprache, sowie Python und weitere Webtechnologien (JavaScript, XML, HTML, Markdown, etc) unterstützt.


</Intro>

## Preise

<ArrowList>

- Lizenziert nach Nutzern. Jährliche Subscriptions enthalten dauerhafte Lizenzen (Fallback-Lizenz)
- Ab 50 Lizenzen erhalten Sie optional den JetBrains Lizenz-Server und auf Anfrage optionalen Umstieg auf floating Lizenzen (Zugriffs-/Concurrent-Lizenzen)
- Siehe auch [Lizenzierung - Entwicklungsumgebungen (IDEs)](/hersteller/jetbrains/#kauf-und-lizenzierung)

</ArrowList>

<Products bkey="jetbrains" tags={["clion"]}/>

<GetInTouch>

Upgrade-, (Mitbewerber-)umstieg und Renewal-Preise erhalten Sie auf Anfrage. Bei größeren Lizenzmengen und mehrjährigen Verlängerungen erhalten Sie mehr Rabatte. Gerne können Sie mich kontaktieren.

</GetInTouch>

## Funktionen

- Native Cross-Plattform-Entwicklung mit C, C++
- Unterstützung von Kotlin/Native, Rust and Swift
- Unterstützung von CMake, Python und Webtechnologien (JavaScript, XML, HTML, Markdown)
- Smart editor
- Code-Refactoring und -Generierung
- Code Analyse während dem Arbeiten
- Run & Debug
- CMake Unterstützung
- Unit Tests
- Code-Dokumentation mit Doxygen
- Versionskontrolle
- Eingebautes Terminal und Vim Mode

_*Eine vollständige Funktionsübersicht finden Sie auf der [JetBrains CLion Funktionsübersichts Seite](https://www.jetbrains.com/de-de/clion/features/)._
