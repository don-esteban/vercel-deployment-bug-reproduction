<Intro>

  N-able Cove Data Protection (ehem. N-able Backup) ist eine umfassende Backup und Disater Recovery Lösung aus der Cloud. Bietet Backups für physische und virtuelle Server, Workstations und Microsoft 365 Konten.
  Mit einer einfachen Subscription erhalten Sie unbegrenzten Cloud-Speicherplatz für Ihre Backups, inklusive Archivierung.
  Das Backup lässt sich als Zusatzoption zu N-central oder RMM hinzubuchen, kann aber auch Stand-alone genutzt werden.

</Intro>

## Lizenzierung

<ArrowList>

  - N-able Produkte sind ausschließlich als **Subscription (Abonnement)** erhältlich.
  - Lizenziert wird nach **Geräten (Device)**, bzw **Konten** ([Microsoft 365](/hersteller/microsoft/microsoft-365/))
  - Beim Backup wird wird bei den Geräte zwischen **Server und Workstation** unterschieden
  - **24/7 Support** ist immer enthalten.
  - Ein Angebot über Wartungs-Verlängerungen erhalten Sie auf Anfrage.

</ArrowList>

## Funktionen

<ArrowList>

 -  Backup von physischen und virtuellen Servern
 -  Backup von Dokumenten und Dateien
 -  Backup für Microsoft 365
 -  Bare-Metal-Wiederherstellung
 -  Schnellere Sicherung und Wiederherstellung
 -  Archivierung ohne Aufpreis

</ArrowList>
