## Preise

GravityZone Security for Mobile Devices wird als Subscription angeboten. Die zentrale Verwaltungskonsole ist als SaaS Lösung und zur eigenen Installation enthalten. Lizenziert wird nach mobilen Geräten. Siehe [Bitdefender GravityZone Lizenzierung](/hersteller/bitdefender/gravityzone-lizenzierung/).
