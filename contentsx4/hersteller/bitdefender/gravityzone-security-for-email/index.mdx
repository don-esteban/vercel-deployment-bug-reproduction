## Unterstützte E-Mail Lösungen

*GravityZone Security for Email* ist ein Gateway in der Bitdefender Cloud und unterstützt alle gängigen E-Mail-Lösungen. Zum Schutz eingehender E-Mails wird der MX-Record angepasst. Zur Prüfung ausgehender E-Mails wird ein Smarthost eingetragen.
