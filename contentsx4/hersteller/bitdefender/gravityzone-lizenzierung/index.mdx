## Abonnement-Lizenzen (Subscriptions)

GravityZone ist ausschließlich als Subscription erhältlich. Verfügbare Laufzeiten sind:

<ArrowList>

- Subscription mit 1 Jahr Laufzeit
- Subscription mit 2 Jahre Laufzeit
- Subscription mit 3 Jahre Laufzeit

</ArrowList>

<Info>

**Hinweis:** Passen Sie beim Nachkauf Restlaufzeiten an. So haben Sie immer einen einheitlichen Renewal-Zeitpunkt. Gerne mache ich das für Sie.

</Info>

## Ermittlung benötigter Lizenzen

<Info>

**Faustregel:** Die Anzahl benötigter Lizenzen entspricht der Anzahl installierter Agenten. Allerdings gibt es Ausnahmen.

</Info>


### GravityZone Business Security Editionen

#### Lizenzierung nach Nodes

[GravityZone Business Security Bundles](/hersteller/bitdefender/#gravityzone-business-security) werden pro Node lizenziert. Nodes sind Endpoints wie:

<ArrowList>

- Physische und virtuelle Server
- Workstations und virtuelle Desktops
- Mobile Android und iOS-Geräte
- Container-Hosts, Docker-Hosts

</ArrowList>

Für jeden Node (Endpoint) benötigen Sie eine Lizenz. Auf jedem Endpoint wird ein Agent ausgebracht.

#### Mischen, kombinieren erweitern

Lizenzrechtlich sind nicht alle Kombinationen von GravityZone Produkten erlaubt.

<Info>

**Faustregel:** Erweitern (Add-ons) ist erlaubt, mischen (Bundles untereinander) und kombinieren (mit Enterprise Einzelprodukten) ist verboten.

</Info>

<ArrowList>

- Verschiedene Business Editionen dürfen Sie in einer Organisation nicht mischen. Nur eine Edition ist erlaubt. Wenn Sie für einen Teilbereich einen höheren Schutz benötigen, müssen Sie mit allen Nodes auf die höherwertige Edition wechseln.
- Business Editionen dürfen Sie ebenfalls nicht mit Enterprise Einzelprodukten kombinieren.
- Business Editionen können Sie mit allen Add-ons erweitern. Ausnahme ist [GravityZone Endpoint Detection and Response](/hersteller/bitdefender/gravityzone-endpoint-detection-and-response/). Wenn Sie EDR benötigen müssen Sie auf das Enterprise Bundle wechseln.



</ArrowList>

#### 150 Prozent Postfächer-Obergrenze

Mailboxen sind keine Nodes und müssen nicht mitgezählt werden. Allerdings gibt es eine Obergrenze im Verhältnis zur Gesamtzahl der Nodes.

<Info>

Enthalten sind bis zu 150 Prozent Mailboxen bezogen auf die lizenzierten Nodes.

</Info>

##### Beispiel

Nehmen wir an, Sie lizenzieren 100 Nodes der Business Security Premium Edition. 150 Mailboxen sind enthalten und müssen bei der Lizenzierung nicht weiter berücksichtigt werden. Haben Sie jedoch 180 Mailboxen, dann müssen Sie 20 Nodes zusätzlich kaufen. 150 Prozent von 120 (100 + 20) ist 180.

**Für die Lizenzierung von 99 Workstations und 1 Exchange Server und 180 Mailboxen benötigen Sie 120 Lizenzen**

#### 30/35 Prozent Server Obergrenze

Wie bei den Postfächern gibt es auch eine Obergrenze für den Server-Anteil (physisch/virtuell) an der Gesamtzahl der Nodes. Anders als bei Postfächern müssen Server jedoch mitgezählt werden.

<ArrowList>

- [GravityZone Business Security](/hersteller/bitdefender/gravityzone-business-security/): Server-Anteil maximal 30 Prozent aller Nodes
- [GravityZone Business Security Premium](/hersteller/bitdefender/gravityzone-business-security-premium/): Server-Anteil maximal 35 Prozent aller Nodes
- [GravityZone Business Security Enterprise](/hersteller/bitdefender/gravityzone-business-security-enterprise/): Server-Anteil maximal 35 Prozent aller Nodes

</ArrowList>

##### Beispiel

Angenommen, Sie haben 70 Workstations und 30 Server. Das sind zusammen 100 Nodes. Das ist mit [GravityZone Business Security](/hersteller/bitdefender/gravityzone-business-security/) gerade noch möglich. 30 Prozent von 100 ist 30. Wenn Sie jedoch 70 Workstations und 45 Server haben, dann müssten Sie 150 Nodes lizenzieren. 30 Prozent von 150 ist 45. In diesem Fall haben Sie aufgrund der 30-Prozent-Regel weniger echte Nodes (installierte Agents) als Lizenzen.

**Für die Lizenzierung von 70 Workstations und 45 Server benötigen Sie 150 Lizenzen**

### GravityZone Enterprise Security

GravityZone Enterprise Security sind Einzelprodukte, mit denen Sie eine vorhandene Security Lösung ergänzen oder à la carte eine eigene Cyber Security Lösung zusammenzustellen.

<ArrowList>

- [GravityZone Security for Workstations](/hersteller/bitdefender/gravityzone-security-for-workstations/) wird nach physischen Workstations und virtuellen Desktops lizenziert.
- [GravityZone Cloud and Server Security](/hersteller/bitdefender/gravityzone-cloud-and-server-security/) wird nach physischen und virtuellen Servern lizenziert.
- [GravityZone Security for Mobile Devices](/hersteller/bitdefender/gravityzone-security-for-mobile-devices/) wird nach Android und iOS-Geräten lizenziert.
- [GravityZone Security for Exchange Servers](/hersteller/bitdefender/gravityzone-security-for-exchange-servers/) wird nach Postfächern lizenziert. **Hinweis:** Endpoint-Schutz der Windows Server Instanz, in der Exchange ausgeführt wird, ist enthalten.
- [GravityZone Security for Linux Mail Servers](/hersteller/bitdefender/gravityzone-security-for-linux-mail-servers/) wird nach Postfächern lizenziert.
- [GravityZone EDR Cloud](/hersteller/bitdefender/gravityzone-edr-cloud/) wird nach Nodes lizenziert. Ein Node ist ein Endpoint, auf dem Sie den Agenten installieren.

</ArrowList>

<Info>

Einzelprodukte enthalten die zentrale Verwaltungskonsole *GravityZone Control Center*.

</Info>

## GravityZone Add-ons

Mit Add-ons können Sie Business-Bundles und Einzelprodukte erweitern.

<ArrowList>

- [GravityZone Security for Storage](/hersteller/bitdefender/gravityzone-security-for-storage/) wird nach NAS-Einheiten oder Fileservern lizenziert.
- [GravityZone Security for Containers](/hersteller/bitdefender/gravityzone-security-for-containers/) wird nach Docker Container Hosts
lizenziert.
- [GravityZone Security for Email](/hersteller/bitdefender/gravityzone-security-for-email/) wird nach Postfächern lizenziert.
- [GravityZone Patch Management](/hersteller/bitdefender/gravityzone-patch-management/) wird nach physischen und virtuellen Servern, Windows Workstations und virtuellen Desktops lizenziert.
- [GravityZone Full Disk Encryption (FDE)](/hersteller/bitdefender/gravityzone-full-disk-encryption/) wird nach verschlüsselten Workstations, Server und Notebooks (Windows BitLocker und macOS FileVault) lizenziert.
- [GravityZone Endpoint Detection and Response](/hersteller/bitdefender/gravityzone-endpoint-detection-and-response/) wird nach Nodes lizenziert. Ein Node ist ein Endpoint, auf dem Sie den Agenten installieren.

</ArrowList>

## Mindestbestellmengen

Es gibt Mindestbestellmengen. Sie sind produktabhängig und gehen aus den Produktbezeichnungen hervor. Die kleinste aufgeführte Preisstaffel entspricht der Mindestbestellmenge.

<Info>

**Hinweis:** Beim Nachkauf gilt keine Mindestbestellmenge.

</Info>

## Laufzeitanpassung

Subscription haben feste Laufzeiten. Ein Jahr, zwei Jahre oder drei Jahre. Wenn Sie während der Laufzeit einer bestehenden Subscription Lizenzen nachkaufen, kann ich für Sie die Laufzeiten anpassen. Sie haben dann zukünftig nur einen einzigen Renewal-Termin, egal wie oft und wann Sie Lizenzen nachkaufen.
