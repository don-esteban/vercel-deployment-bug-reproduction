<Intro>

GravityZone Endpoint Detection and Response ist ein Add-on für *GravityZone Enterprise Security Einzelprodukte*. Erkennt ungewöhnliches Verhalten auf der Grundlage der MITRE-Wissensbank und Bitdefenders eigener Forschung. Der Abruf unserer One-Click-Angebote dauert nur Sekunden. Hier finden Sie Preise und andere nützliche Informationen für Ihren Einkauf.

</Intro>
