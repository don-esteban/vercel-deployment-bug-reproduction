<Intro>

GravityZone Security for Linux Mail Servers ist abgekündigt. Security for Linux Mail Servers schützt Linux-Mail-Server wie Postfix, Sendmail (Milter), qmail, CommuniGate Pro, Courier und alle anderen SMTP-Mailsysteme über SMTP-Proxy-Modus. Der Abruf unserer One-Click-Angebote dauert nur Sekunden. Hier finden Sie Preise und andere nützliche Informationen für Ihren Einkauf.

</Intro>

<TOC />

## Preise

GravityZone Security for Linux Mail Servers wird als Subscription angeboten. Die zentrale Verwaltungskonsole zur eigenen Installation (On-Premises) ist enthalten. Lizenziert wird nach Postfächern. Siehe [Bitdefender GravityZone Lizenzierung](/hersteller/bitdefender/gravityzone-lizenzierung/).

<GetInTouch variant="Info">

GravityZone Security for Linux Mail Servers wurde abgekündigt. Für bestehende Lizenzen wird der Support bis zum Ende der Laufzeit fortgesetzt. Offizielles Nachfolgeprodukt seitens Bitdefender ist [GravityZone Security for Email](/hersteller/bitdefender/gravityzone-security-for-email/), das jedoch laut Funktionsumfang keine Linux Mail Server schützt. Bei Fragen oder Interesse an einem alternativen Produkt können Sie mich gerne kontaktieren.
