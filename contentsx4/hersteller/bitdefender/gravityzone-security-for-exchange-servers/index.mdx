<Intro>

GravityZone Security for Exchange Servers schützt Microsoft Exchange Server und die ausführenden physischen oder virtuellen Windows Server. Der Abruf unserer One-Click-Angebote dauert nur Sekunden. Hier finden Sie Preise und andere nützliche Informationen für Ihren Einkauf.

</Intro>

## Preise

GravityZone Security for Exchange Servers wird als Subscription angeboten. Die zentrale Verwaltungskonsole zur eigenen Installation (On-Premises) ist enthalten. Lizenziert wird nach Postfächern. Siehe [Bitdefender GravityZone Lizenzierung](/hersteller/bitdefender/gravityzone-lizenzierung/).

<Info>

Wir dürfen an dieser Stelle keine Preise veröffentlichen. Mit wenigen Mausklicks können Sie jedoch ein Sofort-Angebot anfordern. Das dauert nur wenige Sekunden.

</Info>
