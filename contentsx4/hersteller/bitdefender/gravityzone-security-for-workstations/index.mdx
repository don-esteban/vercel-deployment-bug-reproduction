## Preise

GravityZone Security for Workstations wird als Subscription angeboten. Die zentrale Verwaltungskonsole ist als SaaS Lösung und zur eigenen Installation enthalten. Lizenziert wird nach physischen oder virtuellen Workstations/Desktops. Siehe [Bitdefender GravityZone Lizenzierung](/hersteller/bitdefender/gravityzone-lizenzierung/).
