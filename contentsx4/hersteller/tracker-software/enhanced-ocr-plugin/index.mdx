<Intro>

  Das OCR Plugin erweitert die PDF-XChange Lösung mit einer neuen OCR-Engine, für eine schnellere und bessere Erkennung von Text und Bildern aus Scans. Mit Multi-Threading und zusätzlicher Automatisierung von (Bild-)Korrekturen erhalten Sie eine optimale Lösung für professionelle Optical Character Recognition (OCR).

  **Das Enhanced OCR Plugin ist mit Version 9 Bestandteil der [PDF-XChange Editor Plus](/hersteller/tracker-software/pdf-xchange/) und [PDF-XChange Pro](/hersteller/tracker-software/pdf-xchange-pro/) Bundles. Einzeln ist das Plugin nicht mehr verfügbar.**


</Intro>
