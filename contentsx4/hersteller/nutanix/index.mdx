## Vergleich Nutanix und VMware

Nutanix und [VMware](/hersteller/vmware/) sind zwei sehr bekannte Namen im Bereich der hyperkonvergenten Infrastruktur (HCI) und der Virtualisierung. VMware hat mit seinem Produkt [vSphere](/hersteller/vmware/vsphere/), das die Ausführung mehrerer Workloads auf demselben physischen Server ermöglicht, die IT-Landschaft durch Virtualisierung neu definiert. Nutanix HCI ist in der Hardware-Kompatibilitätsliste (HCL) von VMware aufgeführt und bietet zertifizierte Kompatibilität mit dem ESXi-Hypervisor von [VMware vSphere](/hersteller/vmware/vsphere/), wodurch das Management vereinfacht und eine Virtualisierungsplattform für VMware vSphere-Benutzer bereitgestellt wird.

Für Anwender, die nicht mit VMware arbeiten, bietet Nutanix eine Alternative zu VMware vSphere auf Basis des eigenen Hypervisors “Nutanix AHV” (Acropolis HyperVisor). Dieser Hypervisor ist Bestandteil aller Nutanix Subscriptions. Viele VMware-Kunden nutzen mit VMware ESXi auch Nutanix und sind daher mit AHV vertraut.

VMware hat sich auf die Bereitstellung von Virtualisierung konzentriert. Nutanix hat von Anfang an auf mehr Flexibilität in Bezug auf Hardware, Hypervisoren, verschiedene Cloud-Topologien und Workload-Management gesetzt. Nutanix ist eine vollständige Plattformarchitektur, die die Skalierung von Workloads von On-Premise über den Netzwerkrand bis hin zur Cloud ermöglicht, wobei die Kosteneffizienz stets im Vordergrund steht.

Durch die Bereitstellung von Anwendungen, Diensten und Daten über mehrere Endpunkte hinweg und die zentrale Verwaltung auf einer einzigen Plattform bietet Nutanix eine wichtige Alternative für VMware-Nutzer. Wechselwilligen VMware-Kunden werden verschiedene Migrationsstrategien angeboten.


| Merkmale | Nutanix | VMware |
|:---|:---|:---|
| **HCI** |  |  |
| Unterstützte Hypervisoren | VMware ESXi, Nutanix AHV, Microsoft Hyper-V | Nur VMware ESXi |
| Zusätzliche Hypervisor-Lizenzen erforderlich | AHV-Virtualisierung ohne zusätzliche Kosten enthalten | Virtualisierungslizenzen separat pro CPU erhältlich |
| Architektonische Konzeption | Moderne Web-Scale-Architektur mit HCI | Ältere Storage-Technologien werden in Software-definierte Lösungen umgewandelt |
| Native Dateispeicherdienste | Ja, über Nutanix Files | Ja, über File Services |
| Native Object Storage Services | Ja, über Nutanix Objects | Keine native Unterstützung. Produkte von Drittanbietern erforderlich |
| Automatisierte Datenbankverwaltung, Provisionierung und Sicherheit | Ja, über Nutanix Database Service | Nicht unterstützt |
| Automatisiertes maschinelles Lernen für Workload-basierte Ressourcenanomalien | Ja | Nein |
| Geringe Latenzzeiten für Anwendungen durch Datenlokalisierung | Ja | Begrenzt |
| **Hybrid Cloud** |  |  |

| Hibernate VMs zur Kostensenkung | Unterstützt | Nicht verfügbar |

### Nutanix Move

Nutanix Move ist eine Hypervisor-übergreifende Softwarelösung, die die Migration u.a. von VMware ESXi-Umgebungen zu Nutanix AHV unterstützt. Neben VMware ESXi werden weitere Umgebungen für eine Migration unterstützt, wie Nutanix AOS, Microsoft Hyper-V (Hyper-V zu AHV), Microsoft Azure oder auch Amazon Web Services.

Das Tool legt den Schwerpunkt auf minimale Ausfallzeiten bei Migrationen, was für Unternehmen äußerst wichtig ist. Das Programm ist auf die Produktions-Workloads virtueller Maschinen ausgerichtet und soll diese so schnell und effizient wie möglich übertragen und den Administrator soweit möglich entlasten.

Nutanix Move ist kostenlos über das Kundenportal erhältlich, auch wenn der Download etwas versteckt ist. Ein Zugang zum Kundenportal setzt eine Registrierung voraus.

Siehe zu Nutanix Move auch die Dokumentation zu [AHV-Migration](https://portal.nutanix.com/page/documents/solutions/details?targetId=TN-2072-AHV-Migration:nutanix-move.html) und [Nutanix Move User Guide](/hersteller/nutanix/nutanix-move.pdf)




## Support

Nutanix bietet eine Reihe von Supportleistungen für die Cloud Infrastructure:

- **Rund-um-die-Uhr-Support**: Nutanix bietet Kunden einen 24/7-Support.
- **Vorrangige Anruf- und Anfragebearbeitung**: Kunden haben Anspruch auf vorrangige Bearbeitung ihrer Anrufe und Anfragen.
- **Vor-Ort-Service am nächsten Werktag**: Nutanix bietet einen Vor-Ort-Service am nächsten Werktag.
- **Mission Critical Support**: Dieser Service bietet schnellere Antwortzeiten, direkten Kontakt zu Senior-Level-Ingenieuren und Vor-Ort-Service von bis zu 4 Stunden.
- **Automatisierung**: Nutanix Cloud Manager bietet eine End-to-End-Management-, Betriebs- und Automatisierungslösung für eine Vielzahl von Umgebungen.
- **Self-Service und Automatisierung**: Nutanix Database Service (NDB) reduziert die manuellen Aufgaben, die mit der Datenbankverwaltung verbunden sind, deutlich.
