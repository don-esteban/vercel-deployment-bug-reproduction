<Intro>

NXPowerLite für File Server komprimiert Dateien und Dokumente auf [Windows Servern](/hersteller/microsoft/windows-server/), EMC, NetApp und Novell Netware. Es können Regeln für den Zeitpunkt und spezifische Ordner für die Komprimierung erstellt werden.

In drei Editionen erhältlich: Single Server, Storage Plus, Enterprise.



</Intro>

<TOC />



## Lizenzierung

- Wird als Kauflizenz mit einem Jahr Support angeboten. Andere Laufzeiten erhalten Sie auf Anfrage.
- Lizenziert wird pro File-Server.
- **Single Server:** komprimiert lokale Dateien, enthält 5 TB Speicherplatz
- **Storage Plus:** komprimiert lokale Dateien, automatische Erkennung von unkomprimierten Dateien/Dokumenten, enthält 10 TB Speicherplatz
- **Enterprise:** komprimiert Dokumente auf Netzwerkfestplattenpeicher (SAN/NAS), automatische Erkennung von unkomprimierten Dateien/Dokumenten, keine Speicherplatz Begrenzung

**Der Single Server enthält nicht die automatische Erkennung von umkomprimierten Dateien.**

## Funktionen

- Komprimieren Sie mehrere Ordner - Komprimieren Sie Dateien auf jedem Speichergerät in derselben Windows-Domäne wie NXPowerLite. Fügen Sie einfach Laufwerke, Netzwerkfreigaben und Ordner hinzu, um sicherzustellen, dass die Software alle Ihre Dateien erreicht.
- Transparent - Dateien werden mit durch die einzigartige Komprimierungs-Engine reduziert, die das Format oder die Attribute nicht ändern. Es werden keine zusätzlichen Dateien erstellt und die Originale werden nicht verschoben.
- Dateiverfolgung - Komprimierte Dateien und nicht komprimierbare Dateien werden automatisch nachverfolgt und bei nachfolgenden Läufen übersprungen (nicht beim Single Server).
- Single Pass oder Loop - Stellen Sie die Software so ein, dass sie Ihre Dateien einmal durchläuft und stoppt. Oder stellen Sie es alternativ so ein, dass es sich wiederholt wiederholt und neue Dateien aufnimmt, sobald sie ankommen.
- Multithreading - Führt standardmäßig zwei Optimierungsprozesse parallel aus, kann aber optional erhöht werden um mehr Durchsatz zu erzielen.
- Festgelegten Regeln - Wählen Sie aus, welche Ordner und Dateien komprimiert werden sollen. Ausschluss nach bestimmten Attributen wie Datum der letzten Änderung, Ort und Erweiterung.
- Plan - Verwenden Sie den integrierten Planer oder steuern Sie den Dienst mit Hilfe des Windows-Taskplaners von anderen Ereignissen aus.
- Berichte - Erhalten Sie detaillierte Berichte über die durch das Komprimieren von Dateien erzielten Einsparungen.
