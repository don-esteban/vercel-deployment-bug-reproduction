<Intro>

NXPowerLite, eine Software-Lösung zur Komprimierung von Dateien, Grafiken und Dokumenten.	[Microsoft Office Dateien](/hersteller/microsoft/office/), wie z.B. [PowerPoint Präsentationen](/hersteller/microsoft/office/einzelprodukte/powerpoint-2021/) und [Word Dateien](/hersteller/microsoft/office/einzelprodukte/word-2021/), sowie PDFs, JPGs, PNGs und TIFFs können bis zu 95% komprimiert werden. Die Qualität des Bildes oder die Funktionen werden nicht beeinträchtigt.

## Lizenzierung

 NXPowerLite wird als Einzellizenz pro Server oder als Volumenlizenz pro User angeboten, je nach Produkt. Es gibt auch Kauflizenzen (perpetual License) mit Support.


### Einzellizenz

[NXPowerLite für File Server](/hersteller/neuxpower/nxpowerlite-file-server/) und	[NXPowerLite für Exchange Server](/hersteller/neuxpower/nxpowerlite-exchange/) wird als **Einzellizenz** verkauft. Lizenziert wird nach Servern.

Jede Einzellizenz erhält einen eigenen Lizenzschlüssel. Diese müssen manuell auf jeden Server installiert werden.



### Volumenlizenz

[NXPowerLite für Desktop](/hersteller/neuxpower/nxpowerlite-desktop/) wird als **Volumenlizenz** verkauft.

Volumenlizenzen sind ab einer Bestellmenge von 10 Lizenzen verfügbar. Mit einem einzigen Lizenzschlüssel können alle erworbenen Lizenzen freigeschaltet werden. Bei einem bestehenden Lizenzkey und einem aktiven Wartungsvertrag, können Nachbestellungen auf die ursprüngliche Lizenz registriert werden, Sie behalten also die ursprüngliche Rabattstaffel.


### Beispiel zum Kauf der Volumenlizenzen

**Ersteinkauf:** Beim Kauf von 30 Lizenzen erhalten Sie den Preis der Staffel 10-49 User und einen einzigen Volumenlizenzschüssel zur Freischaltung.

**Nachbestellung von 8 Lizenzen:** Bei der Bestellung von 20 zusätzlichen Lizenzen, erhalten Sie diese in der günstigeren Preisstaffel 50-99 User. Der bestehende Volumenlizenz-Key wird um die 20 zusätzlichen Lizenzen erweitert und besitzen nun insgesamt 50 Lizenzen.

**Weitere Nachbestellung von 3 Lizenzen:** Mit einem Volumenlizenzvertrag können Sie auch weniger als 10 Lizenzen kaufen. Die 10 Lizenzen erhalten Sie in der Staffel 50-99 User und der vorhandene Volumenlizenz-Key wird erweitert.

**Weitere Nachbestellung von 6 Lizenzen mit einem ausgelaufenen Wartungsvertrag:** Sie wollen nun 6 weitere Lizenzen auf den Volumenlizenzschüssel bestellen. Dies ist mit nur mit einem aktiven Wartungsvertrag möglich. Um wieder in die Wartung einzusteigen, benötigen Sie Upgrade Lizenzen. Erst dann können die Lizenzen wieder auf den Volumenlizenzschüssel gebucht werden. Hierzu kann ich Ihnen ein individuelles Angebot erstellen. Rufen Sie mich an.



## Support

Neulizenzen enthalten immer mindestens ein Jahr Wartung. Mit einem aktiven Wartungsvertrag haben Sie Anspruch auf kostenlose Updates, Patches sowie Upgrades auf die neuste Version. Dazu kommen die Vorteile wenn Sie [Volumenlizenzen](#beispiel-zum-kauf-der-volumenlizenzen) haben.
