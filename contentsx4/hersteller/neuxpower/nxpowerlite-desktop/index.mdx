## Funktionen

- Benutzerdefinierte Profile - Erstellen Sie benutzerdefinierte Einstellungsprofile für Ihren Schnellstart
- Explorer-Integration - Komprimieren Sie Dateien direkt aus Windows Explorer mit den Rechtsklick-Menüoptionen.
- Zip-Komprimierung - Belassen Sie die Dateien in ihrem ursprünglichen Format oder kompromieren Sie sie optional in einer ZIP-Datei.
- Batch-Komprimierung - Komprimieren Sie bis zu 10.000 Dateien gleichzeitig. Ideal zum schnellen komprimieren kleiner Inhaltsordner.
- Dateien für E-Mail komprimieren - E-Mail-Anhänge automatisch komprimieren, wenn sie von Outlook oder Lotus Notes gesendet werden.
- Gleiches Dateiformat - Komprimierte Dateien behalten dasselbe Format mit derselben Dateierweiterung. Zum Öffnen der komprimierten Dateien wird NXPowerLite nicht benötigt.
- Massenbereitstellung - NXPowerLite Desktop ist als Windows Installer .MSI-Datei zur einfachen Installation auf mehreren Desktops verfügbar, ohne dass eine Benutzerinteraktion erforderlich ist.
