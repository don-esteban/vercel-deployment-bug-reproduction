## Vergleich Zoom One Editionen

| Funktionen | Zoom One Pro | Zoom One Business |
| :--- | :--- | :--- |
| Meetings bis zu x TN | 100 | 300 |
| Anzahl Lizenzen | 1 bis 9 | 10 oder mehr |
| Unbegrenzte Anzahl Meetings| ✓  | ✓  |
| Dauer Meeting in Std. | 30 | 30 |
| Zoom Whiteboards |  3  |  ∞  |
| Cloud-Speicher für Aufzeichnungen | 5 GB | 5 GB |
| Benutzerverwaltung |  ✓  |  ✓  |
| Zentrales Admin-Portal |  –  |  ✓  |
| Transkripte |  –  |  ✓  |
| Branding |  –  |  ✓  |
| Domänenverwaltung |  –  |  ✓  |
| Dolmetscher Funktion |  –  |  ✓  |
| Eigene E-Mail-Domäne |  –  |  ✓  |
| Kundensupport | Ticket, Live-Chat | Ticket, Live-Chat, Telefon |
| Premier-Support (optional) | ✓  | ✓  |

Im Gegensatz zur kostenlosen Zoom One Basic Edition ist die Dauer eines Meetings bei Zoom One Pro und Zoom One Business nicht auf 40 Minuten begrenzt.
Weitere für Unternehmen wichtige Funktionen wie Benutzerverwaltung und Reportfunktionen sind erst ab der Zoom One Pro Edition enthalten.
