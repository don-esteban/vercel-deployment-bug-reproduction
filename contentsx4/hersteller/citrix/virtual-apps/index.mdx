<Intro>

Auf diesen Seiten finden Sie Informationen zur Lizenzierung und Kosten von Citrix Virtual Apps *(formerly Citrix XenApp)*.
Virtual Apps ermöglicht die sichere Bereitstellung von virtualisierten Anwendungen auf jedem Gerät und in jedem Netzwerk.
Diese Informationen richten sich hauptsächlich an IT-Einkäufer und Administratoren.
Fordern Sie mit einem Klick ein kostenloses und unverbindliches Sofortangebot an.

### Standard

  Bereitstellung Server basierter, virtualisierter Anwendungen (außer Linuxanwendungen), unabhängig vom Endgerät *(ehemals XenApp Advanced)*.

<Products bkey="citrix" tags={["virtualapps_std"]}/>



### Advanced

  Wie **Standard Edition**, zusätzlich Bereitstellung von Anwendungen (auch Linux) über eine Desktop-VM *(ehemals XenApp Enterprise)*.
  Optimiert für Skype und MS-Teams.

<Products bkey="citrix" tags={["virtualapps_adv"]}/>

### Premium

  Wie **Advanced Edition**, zusätzlich **Citrix Casting** (nahtloses Verschieben von virtuellen Sitzungen auf andere Geräte), PLUS erweiterte Verwaltungs- und Analysewerkzeuge *(ehemals XenApp Platinum)*.

<Products bkey="citrix" tags={["virtualapps_pre"]}/>

## Funktionen

**Citrix Virtual Apps** ermöglicht die Bereitstellung virtualisierter Anwendungen (SaaS - Software as a Service). Der Benutzer kann Anwendungen unabhängig von seinem System (Mac, Windows, Linux, Android, iOS) ausführen.
Anwendungen können in nur wenigen Schritten mehreren Benutzern bereitgestellt werden.
Unabhängig der Bereitstellung, verhalten sich Anwendungen und Desktops fast wie native Anwendungen auf den Geräten der Mitarbeiter und sehen auf Windows, Mac oder mobilen Geräten gleich aus.
Diese Umgebung bietet sich dort an, wo der/die Benutzer immer die gleiche(n) Anwendung(en) nutzen.<br />


<ArrowList>

- **Einfachere Administration** <br />
  Anwendungen müssen nicht mehr auf einzelnen Endgeräten installiert und aktualisiert werden. Sie installieren eine neue Anwendung oder ein Update einer Anwendung zentral auf dem Server und alle Arbeitsplätze auch an verteilten Standorten können diese Version zeitgleich nutzen.
- **Verbesserte Sicherheit** <br />
  Alle Anwender arbeiten mit einer einheitlichen, aktuellen Version der Anwendung, die der Server bereitstellt. Sensible Unternehmensdaten werden zentral verwaltet und gesichert.
- **Flexibilität** <br />
  Sie können von jedem Ort und mit fast jedem Endgerät auf Ihre zentral bereitgestellten Anwendungen zugreifen. Durch die Clientseitige Anwendungsvirtualisierung ist es auch möglich Offline mit den Anwendungen zu arbeiten. Dies ist zum Beispiel für mobile Mitarbeiter oder Mitarbeiter im Homeoffice praktisch.
- **Leistung** <br />
  Virtualisierte Anwendungen benötigen nur eine geringe Bandbreite, da lediglich geringe Datenmengen über das Netzwerk übertragen werden müssen.
- **Voraussetzungen** <br />
  Citrix Virtual Apps setzt auf Windows-Server auf, Anwendungen werden als SBC (Server Based Computing) bereitgestellt.

</ArrowList>
