<Intro>

Auf diesen Seiten finden Sie Informationen zur Lizenzierung und Kosten von Citrix Hypervisor und Virtual Apps and Desktops Editionen.
Citrix ist einer der führenden Anbieter im Bereich der Virtualisierung von Netzwerken, Servern, Desktops und Anwendungen.
Diese Informationen richten sich hauptsächlich an IT-Einkäufer und Administratoren.
Fordern Sie mit einem Klick ein kostenloses und unverbindliches Sofortangebot an.

## Produkte


### Hypervisor

<Info type="light">

<ArrowItem>

  [Citrix Hypervisor](/hersteller/citrix/hypervisor/) <br />

  Eigenständige Systemplattform (Bare Metal Hypervisor) zur Virtualisierung auf Basis des Xen Hypervisors. Optimiert für den Betrieb von Windows- und Linux-Servern (ersetzt die Anwendung *[Citrix XenServer](/hersteller/citrix/xenserver/)*). Lizenzierung: Subscription, **per Socket**.


</ArrowItem>

</Info>


### Virtual Apps and Desktops

<Info type="light">

Citrix bietet verschiedene Virtualisierungslösungen in verschiedenen Editionen an ([Editionenvergleich](#vergleich-virtual-apps-and-desktops-editionen)). Sie können Applikations-/Desktopvirtualisierung als "Hosted Shared Desktop" *([Citrix Virtual Apps](/hersteller/citrix/virtual-apps/))*, Desktopvirtualisierung als "Hosted Virtual Desktop" aka VDI *([Citrix Virtual Desktops](/hersteller/citrix/virtual-desktops/))* oder beide zusammen in einem Bundle lizenzieren *([Citrix Virtual Apps and Desktops](/hersteller/citrix/virtual-apps-and-desktops/))*.

</Info>

<ArrowList>

- [Citrix Virtual Apps](/hersteller/citrix/virtual-apps/) <br />
  Anwendungsvirtualisierung, stellt virtualisierte Anwendungen oder Desktops zur Nutzung über ein Netzwerk bereit (ersetzt *[Citrix XenApp](/hersteller/citrix/xenapp/)*).  Lizenzierung: Subscription, pro Zugriff (Concurrent). Erhältlich als *Standard, Advanced* und *Premium*.

- [Citrix Virtual Desktops](/hersteller/citrix/virtual-desktops/) <br />
  Stellt virtualisierte Desktops-PCs über ein Netzwerk bereit. VDI - Virtual Desktop Infrastructure (ersetzt *[XenDesktop VDI](/hersteller/citrix/xendesktop/)*). Lizenzierung: Subscription, pro User/Device oder Concurrent. Nur als *Standard* erhältlich.

- [Citrix Virtual Apps and Desktops](/hersteller/citrix/virtual-apps-and-desktops/) <br />
  Paket aus Virtual Apps und Virtual Desktops mit zusätzlichen Sicherheits-, Performance-Monitoring- und Optimierungsfunktionen (ersetzt *[XenDesktop Enterprise](/hersteller/citrix/xendesktop/)*). Lizenzierung: Subscription, pro User/Device. Erhältlich als *Advanced* und *Premium*.

</ArrowList>

### Workspace

<Info type="light">

<ArrowItem>

  [Citrix Workspace](/hersteller/citrix/workspace/) <br />

  Flexible Komplettlösung für die Virtualisierung der IT als Bundles. Eine intelligente, einheitliche und sichere Arbeitsplattform, die die wichtigsten Details organisiert, verwaltet und automatisiert. Lizenzierung: Subscription.

</ArrowItem>

</Info>

### Funktionen

#### Serverbasierte virtuelle Anwendungen
Ermöglicht den sicheren Zugriff auf Windows-, Web- oder Linux\*-Apps auf jedem Gerätetyp, einschließlich iOS, Android, Mac und Windows für den On-Demand-Zugriff von überall. ***\*Umfängliche Linux-Unterstützung nur in den Editionen ADVANCED und PREMIUM verfügbar***

#### Serverbasierte virtuelle Desktops
Hoch skalierbare virtuelle Desktops für mehrere Benutzer, Windows- und Linux-Hosted-Shared-Desktops

#### VDI - Virtual Desktop Infrastructure
Desktop-Betriebssysteme wie Windows 10, Windows 7 und Linux\* werden im Rechenzentrum gehostet und den Benutzern bei Bedarf sicher bereitgestellt.    ***\*Umfängliche Linux-Unterstützung nur in den Editionen ADVANCED und PREMIUM verfügbar***

#### Single Session Server VDI
Stellt einen Desktop von einem Serverbetriebssystem für einen einzelnen Benutzer bereit, unterstützt von Windows Server 2019 und Windows Server 2016.

#### PC-Fernzugriff
Remote-PC-Zugriff ermöglicht Benutzern den Zugriff auf ihren Büro-PC. Bieten Sie sicheren Remote-Zugriff auf physische PCs und Laptops, indem Sie die volle HDX-Erfahrung ohne SSL-VPNs oder die Notwendigkeit, Desktops in das Rechenzentrum zu migrieren, nutzen.

#### VM-gehostete Anwendungen
Mit VM-gehosteten Anwendungen kann die App auf zentralen Desktop-VMs gehostet und nahtlos für Benutzer bereitgestellt werden.

## Citrix Universal Subscription

  Das neue Lizenzmodell *Universal Subscription* richtet sich an Benutzer, die Citrix sowohl on-prem als auch in der Cloud betreiben. Das vereinfachte Lizenzmodell macht keinen Unterschied zwischen den verschiedenen Installationen (Cloud, on-premise, hybrid) und erlaubt eine flexible Nutzung.

<Info>

  Bei Universal Concurrent Lizenzen beträgt die Mindestbestellmenge **100**, bei Universal User/Device sind es **250**.  
  Preise und weitere Informationen auf Anfrage.

</Info>
