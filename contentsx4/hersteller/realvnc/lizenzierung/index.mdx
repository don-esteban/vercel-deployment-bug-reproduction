## Kauf und Lizenzierung

<Info>

Mit Release 7.x gibt es einige Änderungen bei der Lizenzierung:

- VNC 7.x kann weiterhin offline lizenziert werden, verwendet aber eine Offline-Lizenzdatei oder ein Token statt einem 25-stelligen Schlüssel.
  Die Offline-Lizenz kann wie die Offline-Lizenz für VNC 6.x über die Bereitstellungsseite des VNC Connect-Portals oder über uns bezogen werden.
- Zusätzlich zur Lizenz pro Gerät wurde eine Lizenz pro Benutzer eingeführt, die zu On-Demand (jetzt *per User*) berechtigt.
- VNC Server wird auf die gleiche Weise lizenziert wie die Lizenzierung pro Gerät.
- VNC Viewer muss lizenziert sein, um eine Verbindung zu einem VNC-Server herzustellen, der mit einem Abonnement pro Benutzer lizenziert ist.
- Lizenzen pro Benutzer sind nur verfügbar, wenn VNC Viewer 7.x und VNC Server 7.x verwendet werden, VNC 6.x muss dafür upgegraded werden.

</Info>

### Subscription

RealVNC Business wird als Subscriptions angeboten. Die Laufzeit beträgt mindestens ein Jahr.
Multiyear Option (3 Jahre) ist möglich, eine längere Laufzeit muss explizit beim Hersteller angefragt werden.

### Lizenzierung

Bei kommerzieller Nutzung benötigen Sie immer eine Lizenz. Dazu zählen auch Vereine, Hilfsorganisationen oder ehrenamtliche Angestellte. Lizenziert wird pro Endgerät (Device) oder Benutzer/Techniker (User).

**Devices** bzw. Endgeräte sind PCs oder Mobiltelefone. Allerdings zählen nicht nur physische Geräte, sondern auch Betriebssysteminstanzen oder monitorloser Server auf Linux-Plattformen. Ein Endgerät kann also auch ein virtualisierter Desktop sein.

**User** bzw. Techniker ist ein benannter Benutzer, dem eine Lizenz zugewiesen wird. Dieser Techniker kann je nach Edition, bis zu 3, 20, 50 bzw. der maximalen Anzahl an Enterprise Lizenzen, gleichzeitige Sitzungen (Remotezugriffe) aktiv haben.

Eine zentralisierte Bereitstellung der Lizenzen erhalten Sie nur mit [VNC Enterprise per Device](/hersteller/realvnc/enterprise/).

### Nachkauf und Restlaufzeiten

Beim Nachkauf können verschiedene Restlaufzeiten entstehen. Auf Anfrage passen wir Restlaufzeiten an vorhandene Lizenzen an. Bei Interesse erstelle ich Ihnen gerne ein Angebot.
