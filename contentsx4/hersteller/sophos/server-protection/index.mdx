<Intro>

 Sophos Server Protection for Virtualization, Windows and Linux (SVRWLV) enthält Antivirus für lokale Server mit Live Protection. Unterstützt physische und virtuelle Windows/Linux-Server.

</Intro>

<TOC />

 [Sophos Server Protection Enterprise](#server-protection-enterprise---renewal-preise) enthält zusätzlich Ransomware File Protection mit CryptoGuard.

## Lizenzierung

  Die Sophos Server Protection Lizenzierung erfolgt pro Server, egal ob physisch oder virtuell. Wird über die Sophos Enterprise Console verwaltet. Die zentrale Verwaltung über [Sophos Central](/hersteller/sophos/intercept-x-for-server/) ist mit Sophos Server Protection nicht möglich. Dafür gibt es hier bereits in den SMB Preisstaffeln bis 10 User Mengenrabatte.



## Server Protection - Renewal Preise

<Info>

 Achtung: Die Server Protection Produkte können nicht mehr erworben werden. [Alternativen finden Sie unten](#intercept-x-for-server-editionen).

</Info>



## Server Protection Enterprise - Renewal Preise
