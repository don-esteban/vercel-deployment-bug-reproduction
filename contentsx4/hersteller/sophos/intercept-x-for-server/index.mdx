## Intercept X for Server Editionen


<ArrowList>

- [Intercept X Essentials for Server (SVRCIXE)](/hersteller/sophos/intercept-x-for-server/intercept-x-essentials-server/)

  Intercept X Essentials for Server (vormals Central Server Protection - SVRC) Für kleine Unternehmen, mit reduzierten Kontroll- und Verwaltungsfunktionen. Bietet die wichtigsten Server-Schutz-Mechanismen wie Anti-Malware-Scanner, Antivirus, Data Loss Prevention und mehr.

- [Intercept X Advanced for Server (SVRCIXA)](/hersteller/sophos/intercept-x-for-server/intercept-x-advanced-server/)

  Intercept X Advanced for Server (SVRCIXA) bietet zusätzliche Komponenten wie Active Adversary Mitigations, Ransomware File Protection (CryptoGuard), Disk und Boot Record Protection (WipeGuard), Man-in-the-Browser Protection (Safe Browsing) und Enhanced Application Lockdown.

- [Intercept X Advanced for Server with XDR (SVRCIXAEDR)](/hersteller/sophos/intercept-x-for-server/intercept-x-advanced-server-with-edr/)

  Intercept X Advanced for Server with XDR (SVRCIXAEDR) bietet außerdem Cross Estate Threat Searching, Threat Indicators and Prioritization, Deep Learning Malware Analysis, Advanced On-demand SophosLabs Threat Intelligence und Forensic Data Export.


</ArrowList>
