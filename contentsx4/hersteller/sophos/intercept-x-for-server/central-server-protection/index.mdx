<Intro>

 Auf dieser Seite finden Sie Preise und Informationen zur Lizenzierung von Sophos Central Server Protection. Lizenziert wird pro Server als Subscription/Abonnement. Neben wichtigen Security-Komponenten wie Anti-Malware/Anti-Virus-Scanner, HIPS und Data Loss Prevention ist auch die zentrale Verwaltungskonsole Sophos Central enthalten.

 Achtung: Die Sophos on premise Lizenzen Central Endpoint Protection (CEP) und Central Server Protection (SVRC) werden ab 1. Juli 2021 nicht mehr an Neukunden verkauft. Verlängerungen sind noch bis September 2021 möglich. Anschließend wird ein Wechsel auf die neuen Produkte Central Intercept X Essentials (CIXE), bzw. Central Intercept X Essentials for Server angeboten. Diese sind ab Juli 2021 verfügbar und dann auch wie gewohnt in unserer Preisliste zu finden.



</Intro>

## Central Server Protection - Preise



<Info>

Central Server Protection ist seit 1.7.2021 nicht mehr verfügbar. Kunden werden während der Laufzeit auf [Sophos Central Intercept X Advanced for Server](/hersteller/sophos/intercept-x-for-server/intercept-x-advanced-server/) migriert und können bei der Verlängerung auch auf den günstigeren, direkten Nachfolger [Central Intercept X Essentials for Server (SVRCIXE)](/hersteller/sophos/intercept-x-for-server/intercept-x-essentials-server/) wechseln.

</Info>
