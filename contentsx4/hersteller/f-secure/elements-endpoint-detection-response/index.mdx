<Intro>

WithSecure Elements Endpoint Detection & Response (EDR) bietet schnelle und effiziente Erkennung von Bedrohungen für Endgeräte. Mit Elements Endpoint Detection & Response erhalten Sie eine umfassende Endgeräte/Netzwerk-Überwachung und entdecken auch komplexe Angriffe auf Ihre wertvollsten Daten.

 **Hinweis:** Bis Juni 2022 war das Produkt unter dem Namen **F-Secure Elements Endpoint Detection & Response** oder **F-Secure Rapid Detection and Response (RDR)** bekannt.


</Intro>

## Preise

<GetInTouch>

Ein Angebot für Elements Endpoint Detection and Response erstellt Ihnen unser F-Secure Lizenzierungs-Spezialist gerne auf Anfrage.

## Lizenzierung


WithSecure Elements Endpoint Detection & Response ist als jährliche Subscription erhältlich. Der Aufbau ist modular und kann an die eigene Umgebung angepasst werden. Kunden der Vorgängerversion F-Secure Rapid Detection & Response (RDR) können Ihre Subscription mit WithSecure Elements verlängern.

Behörden (Government) und Bildungseinrichtungen (Education) erhalten vergünstigte Konditionen.



## Elements EDR Editionen

WithSecure Elements Endpoint Detection & Response ist auch als Bundle zusammen mit den [Elements Endpoint Protection](/hersteller/f-secure/elements-endpoint-protection/) erhältlich.

- Computer Protection
- Computer Protection Premium
- Server Protection
- Server Protection Premium



## Vorteile mit Elements Endpoint Detection & Response


-  Sofortiger Einblick in potentiell schädliche Anwendungen und Cloud Dienste
-  Automatische Identifizierung fortgeschrittener Bedrohungen mit Risikobewertung für eine bessere Priorisierung
-  Zeigt alle Attacken im Überblick mit den betroffenen Hosts und Entdeckungen
-  Schnelles Stoppen von Angriffen durch automatische Reaktion
-  Einfache, zentrale Verwaltung über die Cloud
