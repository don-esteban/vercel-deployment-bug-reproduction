<Intro>

 WithSecure Business Suite ist eine lokale Sicherheitslösung für Unternehmen und bietet Antivirenschutz für Gateways, Server und Endpoints mit zusätzlichen Funktionen für die Sicherheitsverwaltung. Die [Premium Edition](#premium-preise) bietet zusätzlich Patch Management und Anwendungskontrolle, sowie Web-Inhaltskontrolle. Einen [Vergleich der Editionen](#vergleich-business-suites) finden Sie unten.

 **Hinweis:** Bis Juni 2022 war das Produkt unter dem Namen **F-Secure Business Suite** bekannt.

## Lizenzierung

Die WithSecure Business Suites sind ausschließlich als Subscription Lizenz für ein, zwei oder drei Jahre erhältlich. Die Lizenzierung basiert auf der Anzahl der von WithSecure Business Suite verwalteten Clients, Server und Desktops/Laptops (virtuell und physisch). In einem Cluster zählt jeder Node als eigenes Gerät. Höchstens 20% der Geräte, auf denen die Software installiert ist dürfen Server sein, d.h. Sie benötigen für jeden verwalteten Server mindestens 4 andere lizenzierte Geräte.

Behörden (Government) und Bildungseinrichtungen (Education) erhalten vergünstigte Konditionen.

[WithSecure Elements Endpoint Protection](/hersteller/f-secure/elements-endpoint-protection/) bietet die Funktionalität als Dienst aus der Cloud, ohne eigene Serverinstallation.



## Funktionen


 - [Client Security](/hersteller/f-secure/client-security/) - Windows Security für Laptops und Desktop-PCs
 - [Server Security](/hersteller/f-secure/server-security/) - Umfassende Sicherheit für Server
 - [Email and Server Security](/hersteller/f-secure/email-server-security/) - Schutz des E-Mail Verkehrs
 - Linux Security - Schutz für Linux Server und Endpoints
 - Policy Manager - Zentrale Verwaltung
 - Sicherheit für virtuelle Umgebungen - Scanning und Reputation Server
 - Internet Gatekeeper - Umfassende Gateway Protection
