<Intro>

Signature Manager Exchange ermöglicht alle E-Mail Signaturen, die über [Exchange](/hersteller/microsoft/exchange/) verarbeitet werden, zentral zu verwalten. Signaturen können mit allen Geräte genutzt werden, beispielsweise mit Smartphones. Auch mit automatisierten E-Mails möglich.


</Intro>

<TOC />



## Exclaimer Signature Manager Exchange Preise

- Als Kaufpakete ab 10 Benutzer verfügbar.
- Lizenziert wird pro Benutzer, geteilte Postfächer müssen nicht mehrfach lizenziert werden.
- Für eine Nachlizenzierung muss ein aktiver Supportvertrag vorliegen.
- Mehr Informationen zur Lizenzierung finden Sie auf unserer [Exclaimer-Hauptseite](/hersteller/exclaimer/#lizenzierung).


<Info>

**Achtung:** Exclaimer hat das Lizenzmodell geändert. Wir veröffentlichen hier Preise, sobald uns diese vorliegen.
Bis dahin erhalten Sie ein Angebot auf Anfrage.

</Info>
