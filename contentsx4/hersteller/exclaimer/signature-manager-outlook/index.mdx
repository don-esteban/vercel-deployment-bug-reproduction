<Intro>

Signature Manager Outlook ermöglicht es E-Mail-Signaturen direkt an den [Outlook-Client](/hersteller/microsoft/office/einzelprodukte/outlook-2021/) des Endnutzers (Outlook oder OWA) zu senden. Die Signatur kann die Kontaktdaten des Unternehmens, personalisierten Absenderinformationen, Layouts, Firmenlogo, Disclaimer, Werbebannerfunktionalität, Social-Media-Links, usw. enthalten. Auf Basis von Regeln wird diese Signatur den Benutzern mit dynamischen Daten vom Active Directory (AD) zugeordnet. Die Nutzer können die gewünschte Signatur manuell auswählen, beispielsweise in einer anderen Sprache oder für unterschiedliche Aufgabengebiete (Vertrieb, Support etc.).

</Intro>

## Exclaimer Signature Manager Outlook Preise

- Als Kaufpakete ab 10 Benutzer verfügbar.
