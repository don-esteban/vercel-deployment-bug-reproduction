<Intro>

Red Hat Kubernetes Engine (ehemals Red Hat OpenShift Container Engine) enthält Red Hat Enterprise Linux, Kubernetes Orchestrierung, Red Hat OpenShift Virtualization und Cluster Services. Unter der Kubernetes-Umgebung können containerbasierte Anwendungen erstellt, bereitgestellt und verwaltet werden. Die Managementplattform kann auf physischen, virtuellen oder in einer Cloud-Infrastruktur (privat, public oder hybrid) laufen.


</Intro>

## Preise

OpenShift Kubernetes Engine wird nach Cores oder Sockets lizenziert. Erhältlich mit [Support (SLA) Subscription (Standard oder Premium)](/hersteller/red-hat/lizenzierung/#vergleich-von-standard-und-premium-support) Eine Core-Subscription deckt 2 Cores/4 vCPU ab. Socket-Subscriptions sind nur für physische Bare-Metal-Server. Mehr unter [OpenShift Kauf- und Lizenzierungsguide](/hersteller/red-hat/openshift-lizenzierung/).
