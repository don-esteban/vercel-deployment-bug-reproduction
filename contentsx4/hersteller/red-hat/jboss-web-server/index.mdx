<Intro>

Red Hat JBoss Web Server ist ein Web-Anwendungsserver auf Basis von Apache und Tomcat. Damit lassen sich auf hohen Anforderungen, Sicherheit und Stabilität für gehostete Websites sowie Webanwendungen sicherstellen.

</Intro>
