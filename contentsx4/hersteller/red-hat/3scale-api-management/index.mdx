<Intro>

Red Hat 3scale API Management vereinfacht die Verwaltung Ihrer APIs. Mit der Infrastrukturplattform können Sie Ihre APIs teilen, sichern, verteilen, kontrollieren und monetarisieren, die bspw. für Performance, Kundenkontrolle und zukünftiges Wachstum erstellt wurden.

3scale-Komponenten lassen sich lokal, in Cloud-Umgebungen oder in einer hybriden Umgebung integrieren.


</Intro>

## Funktionen

Red Hat 3scale API Management bietet Managementfunktionen und lässt sich beispielsweise in OpenShift oder Fuse integrieren und entsprechend konfigurieren. Die Hauptfunktionen haben wir kurz zusammengefasst. Mehr Funktionen finden Sie im [Red Hat 3Scale API Management Benutzerhandbuch](https://www.redhat.com/en/resources/api-owners-manual-ebook).


- Unlimitierte Administration
- Single Sign-On
- API Lifecycle Support
- Hybrid Cloud-Support für alle Komponenten
- Vollständige (Zugriffs-)Kontrolle durch Richtlinien
- Authentifizierungsfunktionen für mehr Sicherheit
- Zentrale API-Managementplattform
- API Analytics
- Hohe Verfügbarkeit und Leistung
- Integration mit Red Hat Fuse und Red Hat OpenShift
- Red Hat Technical Support Team Zugriff
- Red Hat Customer Portal Zugriff
