## Funktionen mit Oxygen XML Editor

Oxygen XML Editor enthält alle Funktionen von [Oxygen XML Author](/hersteller/syncro-soft/oxygenxml-author/) & [Oxygen XML Developer](/hersteller/syncro-soft/oxygenxml-developer/)


- [Oxygen XML Author](/hersteller/syncro-soft/oxygenxml-author/) - Zugriff auf Standard XML basierte Frameworks zum entwickeln von XML Schemas und um XML- EPUB-Dateien zu editieren sowie validieren
- [Oxygen XML Developer](/hersteller/syncro-soft/oxygenxml-developer/) - Sammlung von Tools um XML Schemas und Transfomation Pipelines zu entwicklen




## Lizenzierung

Oxygen XML wird entweder nach Named Usern oder [Floating Usern](/glossar/floating-user/) lizenziert.



### Named User

Bei Named User wird die Lizenz auf einen benannten Benutzer lizenziert. Jeder weitere Nutzer benötigt eine zusätzliche Lizenz.



### Floating User

Wenn mehrere Entwickler zu unterschiedlichen Zeiten arbeiten, ist es von Vorteil die Lizenz für [Floating User](/glossar/floating-user/) zu erwerben. Es wird die maximale, gleichzeitige Nutzung der Software lizenziert, dh. wenn Sie 10 Entwickler im Team haben, welche die Software nutzen aber nur 3 Entwickler **gleichzeitig** diese nutzen, benötigen sie nur 3 Floating User Lizenzen.
