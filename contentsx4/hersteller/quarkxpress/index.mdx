## Lizenzierung

QuarkXPress wird als Einzellizenz oder als Volumenlizenz pro User lizenziert. Mit der Software erwerben Sie eine zeitlich unbegrenzte Kauflizenz. Bildungseinrichtungen erhalten vergünstigte Konditionen (Education).



### Einzellizenz (ESD)

Jeder Einzellizenz (ESD) enthält einen eigenen, separaten Lizenzschlüssel. Die Installation erfolgt manuell auf jedem Arbeitsplatz einzeln. Eine Lizernz berechtigt zum Einsatz auf einem Windows oder Mac Rechner. Die Software ist als Download (ESD - Electronic Software Distribution) verfügbar.



### Subscription

Die Subscriptions bieten gegenüber den Einzellizenzen eine umfangreichere Lizenzverwaltung mit webbasiertem Verwaltungsportal. Enthalten sind die kostenlosen Upgrades auf neue Produktversionen und Updates für die bestehende Version. Die Software ist als Download (ESD - Electronic Software Distribution) verfügbar.



### Advantage

**Advantage**, das Wartungsprogramm von Quark garantiert kostenlose Updates, Upgrades und unbegrenzten Support. Neulizenzen beinhalten mindestens 1 Jahr Advantage (Wartung). Diese gibt es mit 1, 2 oder 3 Jahren Laufzeit. Nach Ablauf der Wartung ist es weiterhin möglich technischen Quark-Support beziehen, dieser wird dann allerdings kostenpflichtig. Um weiterhin die Vorteile der Wartung zu erhalten, ist eine Verlängerung vor Ablauf nötig.

## Funktionen mit QuarkXPress 2024


 -  **Importieren von verschiedenen Dateitypen**
 Mit QuarkXPress lassen sich [Word](/hersteller/microsoft/office/einzelprodukte/word-2021/)-, [Excel](/hersteller/microsoft/office/einzelprodukte/excel-2021/)-, PDF-, Illustrator-, JPG-, TIF- und EPS-Dateien importieren.

 -  **Entwickeln von mobilen Applikationen**
 Gestaltung von eigenen Applikationen beispielsweise für das iPad oder Android.

 -  **Flex-Layouts**
 Ermöglicht die Erstellung von Flex-Layouts für das Webdesign, z.B. für Banner-Anzeigen, Landing-Pages, Micro-Sites.

 -  **Import-Filter**
 Um einen Umstieg von InDesign CS5 bis [InDesign CC](/hersteller/adobe/creative-cloud-einzelprodukte/indesign-cc/) zu ermöglichen, ist die Bearbeitung von IDML-Dateien mit QuarkXPress möglich.
