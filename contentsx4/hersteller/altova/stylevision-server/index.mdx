<Intro>

Altova StyleVision Server stellt XML-, SQL- und XBRL-Daten anhand von StyleVision-Stylesheets und -Design-Elementen in HTML-, RTF-, PDF- oder Microsoft Word-Dateien dar. Ein Berichtsdesign kann Elemente mit dynamischem Content wie Tabellen, Diagramme, Bilder, Barcodes, Kopf- und Fußzeilen und vieles mehr enthalten.



</Intro>

## Lizenzierung

StyleVision Server wird nach der Anzahl der Prozessor-Kerne des Servers, als Abonnement-Lizenz (Subscription) für ein, zwei oder drei Jahre lizenziert.

Die StyleVision Client Version [StyleVision](/hersteller/altova/stylevision/) ist in den Editionen Professional und Enterprise erhältlich.

## Integration mit FlowForce Server

StyleVision Server lässt sich nahtlos mit dem [FlowForce Server](/hersteller/altova/flowforce-server/), zur Automatisierung von Workflows, integrieren.

Mit FlowForce Server können durch Zeit- oder Ereignistrigger Aufträge zur Validierung von XML-Daten, zur Ausführung von XQuery- und XSLT-Operationen, zur Validierung von XBRL-Instanzdokumenten und XBRL-Taxonomien usw. gestartet werden.


## Funktionen

<ArrowList>

- XSLT-Stylesheet-Design
- Erstellung von Berichten in mehreren Formaten
- Transformierung von XML in PDF, HTML und Word
- Datenbankberichte
- Darstellung von XBRL-Daten in Berichten
- Erstellen von Unternehmensformularen für Altova Authentic

</ArrowList>
