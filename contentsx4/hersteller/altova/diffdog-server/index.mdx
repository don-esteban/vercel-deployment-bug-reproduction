<Intro>

Altova DiffDog Server automatisiert den Vergleich großer Datenmengen und generiert Berichte zum Vergleich von Dateien, Verzeichnissen und URLs.



</Intro>

<TOC />



## Lizenzierung

DiffDog Server wird nach der Anzahl der Prozessor-Kerne des Servers, als Abonnement-Lizenz (Subscription) für ein, zwei oder drei Jahre, lizenziert.

Die Client-Version [DiffDog](/hersteller/altova/diffdog/) ist in den Editionen [DiffDog Enterprise](/hersteller/altova/diffdog/#preise---enterprise) und [DiffDog Professional](/hersteller/altova/diffdog/#preise---professional) erhältlich.

## Funktionen

<ArrowList>

- Vergleich/Zusammenführung
- Datei-/Verzeichnisvergleich
- Verzeichnissynchronisierung
- 3-Weg-Dateivergleich
- Vergleich von XML, Word und Datenbanken
- OOXML-Vergleich
- Vergleich von XML-Dateien in der Grid-Ansicht
- Vergleich von ZIP-Archiven
- Generierung von Vergleichsberichten
- XSLT-Code für XML-Schemaunterschiede
- Speichern der Vergleichseinstellungen für die Wiederverwendung
- Automatisierte Generierung von Vergleichsberichten

</ArrowList>

## Integration mit FlowForce Server

DiffDog Server lässt sich nahtlos mit dem [FlowForce Server](/hersteller/altova/flowforce-server/), zur Automatisierung von Workflows, integrieren.

Mit [FlowForce Server](/hersteller/altova/flowforce-server/) können durch Zeit- oder Ereignistrigger Aufträge zur Validierung von XML-Daten, zur Ausführung von XQuery- und XSLT-Operationen, zur Validierung von XBRL-Instanzdokumenten und XBRL-Taxonomien usw. gestartet werden.
