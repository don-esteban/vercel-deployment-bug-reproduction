<Intro>

Altova FlowForce Server ist ein konfigurierbarer Workflow-Prozessor und ermöglicht die Automatisierung der Datenintegration. Automatisiert XML- und XBRL-Verarbeitungsabläufe, Datentransformationen, die Generierung von Berichten und Dokumenten und andere Aufgaben auf eigenen Servern, Virtual Machines oder Arbeitsrechnern.


In den Editionen [FlowForce Server Advanced](#preise---server-advanced) und [FlowForce Server](#preise---server) verfügbar.


</Intro>

<TOC />



## Lizenzierung

Es gibt zwei unterschiedliche Editionen: [FlowForce Server Advanced](#preise---server-advanced) und [FlowForce Server](#preise---server).

Die Advanced Edition bietet zusätzlich vollständige Unterstützung von AS2 für sichere Datenübertragung.

FlowForce Server wird nach der Anzahl der Prozessor-Kerne des Servers lizenziert.

Beide Editionen werden als Abonnement-Lizenz (Subscription) für ein, zwei oder drei Jahre lizenziert. Für die Advanced Edition sind mindestens 2 Prozessor-Kerne nötig.

## Funktionen

<ArrowList>

- Automatisierte Ausführung von Raptor XML/XBRL Server für XML- und XBRL-Verarbeitungsoperationen
- Automatisierte Ausführung von MapForce Server zur Durchführung von Datentransformationen
- Automatisierte Ausführung von StyleVision Server zur Generierung von Berichten, Dokumenten und e-Formularen
- Automatisierte Dateisystembefehle oder andere Prozesse, die über die Befehlszeile ausgeführt werden können
- Auftragsausführung auf Basis von Zeit-, Datei- oder Remote Triggern
- Ablaufsteuerung mit Schleifen, Bedingungen, Ergebnisauswertung und Fehlerbehandlung
- Sofortige Antwort auf HTTP-Requests durch Verwendung zwischengespeicherter Auftragsergebnisse
- Aufzeichnung detaillierter Ausführungsprotokolle
- Zugriffskontrolle über Rollen und Berechtigungen
- Intuitive Weboberfläche zum Verwalten aller Server-Aktivitäten
- Bietet Unterstützung für die AS2-Spezifikation für die sichere und zuverlässige Datenübertragung

</ArrowList>
