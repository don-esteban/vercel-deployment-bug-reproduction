<Intro>

Altova Authentic Desktop ist ein Tool zum Erstellen und Bearbeiten von XML Inhalt für technisch nicht versierte Benutzer. Mit Hilfe von Authentic Unternehmensformularen können Geschäftsanwender Inhalte mit aus gebräuchlichen Büroapplikationen bekannten Methoden abfragen und erstellen, ohne sich mit der zugrunde liegenden XML- bzw. Datenbanktechnologie befassen zu müssen.


</Intro>

## Lizenzierung

<ArrowList>

- [Installed User](/hersteller/altova/#installed-user)<br />
  Lizenziert wird die Anzahl der PCs auf denen die Software installiert wird
- [Named User](/hersteller/altova/#named-user)<br />
  Der bei der Lizenzierung benannte Benutzer darf die Software auf bis zu 5 Geräten installieren. Es darf nach wie vor nur eine Lizenz gleichzeitig verwendet werden. Bei Lizenzierung mehrerer Nutzer erhält jeder einen separaten Key-Code.
- [Concurrent User](/hersteller/altova/#concurrent-user)<br />
  Hier wird die Anzahl der gleichzeitigen Zugriffe lizenziert. Die Software darf auf bis zu 10 Geräten gleichzeitig installiert werden. Für diese Lizenzierungsart wird der kostenlose [Altova LicenseServer](https://www.altova.com/de/licenseserver) zur Verwaltung benötigt.
- Virtuelle Umgebungen<br />
  Die Software darf in virtuellen Umgebungen, wie auf Terminal Servern oder Virtualisierungsservern, eingesetzt werden. Die Nutzung ausschließlich im eigenen Netzwerk muss sichergestellt sein. Auch hier wird der [Altova LicenseServer](https://www.altova.com/de/licenseserver) zur Verwaltung benötigt.
- Option Wartung<br />
  Der Support- und Wartungsvertrag (SMP) beinhaltet kostenlose Upgrades bzw. Updates auf die neueste Software-Version, sowie technischen Support von Altova.

</ArrowList>

## Funktionen

<ArrowList>

 - XML-Editierung & Validierung
 - WYSIWYG-Oberfläche wie bei einem Textverarbeitungsprogramm
 - Validierung & Syntaxprüfung
 - Spezifische kontextsensitive Eingabehilfen
 - Absolute Positionierung in Design-Layouts
 - Rechtschreibprüfung in mehreren Sprachen
 - XML-Industriestandard-Vorlagen
 - Unterstützung für Projektmanagement
 - Darstellung von benutzerdefinierten HTML-Elementen/Blöcken
 - Darstellung von Variablen
 - Anzeige und indirekte Bearbeitung von Diagrammen
 - RichEdit-Symbolleiste für Formatierung wie in Word
 - Barcode-Unterstützung
 - Zuweisen und Überprüfen von digitalen XML-Signaturen
 - Unterstützung beim Editieren von PXF (Portable XML Form)-Dokumenten
 - XSL / XPath 1.0/2.0 Unterstützung
 - Vordefinierte Industriestandardvorlagen (DITA, DocBook, NCAXML, NewsML, NITF, RIXML, DAISY, TEI, P3P, XMLResume)
 - Umfassende Datenbankunterstützung (Firebird, DB2, Access, Oracle, MySQL, SQL, etc.)
 - Entwickler-Tools (Java/COM-API, FTP-Zugriff, etc.)
 - Bereitstellung unter Windows
 - Integration mit Visuals Studio und Eclipse

</ArrowList>
