<Intro>

Authentic Browser-Plugin ist ein erweiterter WYSIWYG XML- und Datenbank-Editor für technisch nicht versierte Benutzer. Authentic Browser wird auf einem Server installiert und kann über eine URL im Netzwerk erreicht werden.

Authentic gibt es auch als [Authentic Desktop Edition](/hersteller/altova/authentic-desktop/), benötigt keinen Server.


</Intro>

<TOC />



## Lizenzierung

Authentic Browser-Plugin wird auf einem Server ausgebracht und wird pro zugreifendem Nutzer (**[Named User](/hersteller/altova/#named-user)**) lizenziert.

<ArrowList>

- Der bei der Lizenzierung benannte Benutzer darf die Software auf bis zu 5 Geräten nutzen.
- Es darf nach wie vor nur eine Lizenz gleichzeitig verwendet werden.
- Bei Lizenzierung mehrerer Nutzer erhält jeder einen separaten Key-Code.
- Die Lizenz hat eine Laufzeit von einem Jahr, mehrjährige Lizenzen gibt es nicht.
- Ausschließlich als Abonnement-Lizenz (Subscription) verfügbar

</ArrowList>

## Funktionen

<ArrowList>

- XML-Editierung & Validierung
- WYSIWYG-Oberfläche wie bei einem Textverarbeitungsprogramm
- Validierung & Syntaxprüfung
- Spezifische kontextsensitive Eingabehilfen
- Absolute Positionierung in Design-Layouts
- Rechtschreibprüfung in mehreren Sprachen
- XML-Industriestandard-Vorlagen
- Unterstützung für Projektmanagement
- Darstellung von benutzerdefinierten HTML-Elementen/Blöcken
- Darstellung von Variablen
- Anzeige und indirekte Bearbeitung von Diagrammen
- RichEdit-Symbolleiste für Formatierung wie in Word
- Barcode-Unterstützung
- Zuweisen und Überprüfen von digitalen XML-Signaturen
- Unterstützung beim Editieren von PXF (Portable XML Form)-Dokumenten
- XSL / XPath 1.0/2.0 Unterstützung
- Vordefinierte Industriestandardvorlagen (DITA, DocBook, NCAXML, NewsML, NITF, RIXML, DAISY, TEI, P3P, XMLResume)
- Umfassende Datenbankunterstützung (Firebird, DB2, Access, Oracle, MySQL, SQL, etc)
- Entwickler-Tools (Java/COM-API, FTP-Zugriff, etc)
- Bereitstellung im Browser

</ArrowList>
