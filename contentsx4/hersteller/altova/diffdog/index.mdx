<Intro>

Mit Altova DiffDog können Text (Word-Dokumente) oder Quellcodedateien (XML, JSON) verglichen und zusammengeführt werden. DiffDog bietet Funktionen zum Vergleichen und Synchronisieren von Verzeichnissen, Datenbank-Schemas/-Inhalten und vielem mehr.


In den Editionen [DiffDog Enterprise](#preise---enterprise) und [DiffDog Professional](#preise---professional) verfügbar. Siehe dazu auch [Vergleich der DiffDog-Editionen](#vergleich-der-editionen).

</Intro>

## Lizenzierung

DiffDog ist in den Editionen Professional und Enterprise erhältlich. Einen [Funktions-Vergleich](#vergleich-der-editionen) finden Sie unten in der Tabelle.

<ArrowList>

- [Installed User](/hersteller/altova/#installed-user)<br />
  Lizenziert wird die Anzahl der PCs auf denen die Software installiert wird
- [Named User](/hersteller/altova/#named-user)<br />
  Der bei der Lizenzierung benannte Benutzer darf die Software auf bis zu 5 Geräten installieren. Es darf nach wie vor nur eine Lizenz gleichzeitig verwendet werden. Bei Lizenzierung mehrerer Nutzer erhält jeder einen separaten Key-Code.
- [Concurrent User](/hersteller/altova/#concurrent-user)<br />
  Hier wird die Anzahl der gleichzeitigen Zugriffe lizenziert. Die Software darf auf bis zu 10 Geräten gleichzeitig installiert werden. Für diese Lizenzierungsart wird der kostenlose [Altova LicenseServer](https://www.altova.com/de/licenseserver) zur Verwaltung benötigt.
- Virtuelle Umgebungen<br />
  Die Software darf in virtuellen Umgebungen, wie auf Terminal Servern oder Virtualisierungsservern, eingesetzt werden. Die Nutzung ausschließlich im eigenen Netzwerk muss sichergestellt sein. Auch hier wird der [Altova LicenseServer](https://www.altova.com/de/licenseserver) zur Verwaltung benötigt.
- Option Wartung<br />
  Der Support- und Wartungsvertrag (SMP) beinhaltet kostenlose Upgrades bzw. Updates auf die neueste Software-Version, sowie technischen Support von Altova.

</ArrowList>

<Info>

DiffDog ist auch als [DiffDog Server-Version](/hersteller/altova/diffdog-server/) zur automatisierten die Generierung von Vergleichsberichten erhältlich.<br />
Hierfür wird die [DiffDog Server Lizenz](/hersteller/altova/diffdog-server/#lizenzierung) benötigt.

</Info>

## Funktionen


- Vergleich/Zusammenführung
- Datei-/Verzeichnisvergleich
- Verzeichnissynchronisierung
- 3-Weg-Dateivergleich
- Vergleich von XML, Word und Datenbanken
- OOXML-Vergleich
- Vergleich von XML-Dateien in der Grid-Ansicht
- Vergleich von ZIP-Archiven
- Generierung von Vergleichsberichten
- XSLT-Code für XML-Schemaunterschiede
- Speichern der Vergleichseinstellungen für die Wiederverwendung
- DiffDog Server für die automatisierte Generierung von Vergleichsberichten
