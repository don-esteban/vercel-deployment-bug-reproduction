## Lizenzierung

RaptorXML Server wird nach der Anzahl der Prozessor-Kerne des Servers lizenziert.

Es gibt zwei unterschiedliche Editionen: Mit und ohne die XBRL Unterstützung, die jeweils als Abonnement-Lizenz (Subscription) für ein, zwei oder drei Jahre lizenziert werden können.

## Integration mit FlowForce Server

RaptorXML Server lässt sich nahtlos mit dem [FlowForce Server](/hersteller/altova/flowforce-server/), zur Automatisierung von Workflows, integrieren.

Mit FlowForce Server können durch Zeit- oder Ereignistrigger Aufträge zur Validierung von XML-Daten, zur Ausführung von XQuery- und XSLT-Operationen, zur Validierung von XBRL-Instanzdokumenten und XBRL-Taxonomien usw. gestartet werden.
