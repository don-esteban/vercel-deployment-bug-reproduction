<Intro>

MobileTogether bietet ein benutzerfreundliches Framework zur Entwicklung beliebiger datenzentrierter Apps. Der MobileTogether Server verarbeitet die Daten von nativen iOS-, Android-, Windows- und Browser-basierten Apps und bildet als Backend-Server die Schnittstelle zum Endanwender.

MobileTogether ist in den Editionen [MobileTogether Server Advanced](#preise---server-advanced) und [MobileTogether Server](#preise---server) verfügbar. Siehe dazu auch: [Vergleich MobileTogether Editionen](#vergleich-der-editionen).


</Intro>

## Lizenzierung

MobileTogether Server wird nach der Anzahl der Prozessor-Kerne des Servers lizenziert.

Die Advanced Edition bietet zusätzliche Funktionen zur Ausführung von Apps als Dienst.

Beide Editionen sind als Abonnement-Lizenz (Subscription) für ein, zwei oder drei Jahre erhältlich.

## Funktionen

<ArrowList>

- Hochleistungsdatenverarbeitung für Desktop und mobile Apps
- Umfassende Cache-Optionen
- Editor für Rollen und Berechtigungen
- Detaillierte Sicherheitsoptionen
- Sofortige Bereitstellung von Unternehmens-Apps
- Datenverarbeitung für App Store-Apps
- Flexible Bereitstellung vor Ort oder in der Cloud
- Kostengünstiges, skalierbares Preismodell für Firmen jeder Größe

</ArrowList>

## Integration mit FlowForce Server

MobileTogether Server lässt sich nahtlos mit dem [FlowForce Server](/hersteller/altova/flowforce-server/), zur Automatisierung von Workflows, integrieren.

Mit FlowForce Server können durch Zeit- oder Ereignistrigger Aufträge zur Validierung von XML-Daten, zur Ausführung von XQuery- und XSLT-Operationen, zur Validierung von XBRL-Instanzdokumenten und XBRL-Taxonomien usw. gestartet werden.
