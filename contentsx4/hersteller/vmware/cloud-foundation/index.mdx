<Intro>

VMware Cloud Foundation ist die Hybrid-Cloud-Lösung für Unternehmen, die die komplette Plattform von VMware umfasst.
Die folgenden Informationen richten sich an IT-Spezialisten und Administratoren, die sich mit dem Kauf von VMware Cloud Foundation, Hybrid-Cloud-Lösungen und weiteren VMware Lizenzen beschäftigen. Mit nur wenigen Klicks können Sie Sofortangebote abrufen.

</Intro>

<TOC />



**Programmpaket bestehend aus:** vSphere Enterprise Plus, vCenter Server Standard, ESXi, Tanzu, Aria Suite Enterprise, inklusive Production Support. Details unter [Funktionen](#funktionen).

## Lizenzierung

VMware Cloud Foundation wird [pro Core lizenziert](/hersteller/vmware/kauf-und-lizenzierung/#lizenzierung-nach-prozessorkernen). Jeder Prozessorkern erfordert eine Lizenz. Minimum 16 Cores pro CPU.
