vCenter Server Essential ist eine eingeschränkte vCenter Edition zur Verwaltung von bis zu drei ESXi Hosts mit jeweils zwei Prozessoren.

## vCenter Server Essentials Preise

vCenter Server 8 Essentials ist in [vSphere Essentials Kits](/hersteller/vmware/vsphere/essentials-kit/) enthalten und kann nicht separat erworben werden. Essentials Kits sind unteilbare Bundles bestehend aus ESXi und vCenter. Mit vCenter Server Essentials können nur die im Kit enthaltenen vSphere ESXi Hosts verwaltet werden. Das sind maximal drei Hosts je zwei Prozessoren.

## vCenter Server Essentials Alternativen

vCenter Server Essentials ist für den Einsatz in kleinen Organisationen konzipiert, die höchstens drei vSphere ESXi Hosts benötigen. [vCenter Server Foundation](/hersteller/vmware/vcenter-server/foundation/) unterstützt bis zu vier Hosts und [vCenter Server Standard](/hersteller/vmware/vcenter-server/standard/) bis zu 2.000 [vSphere ESXi Hosts](/hersteller/vmware/vsphere/).
