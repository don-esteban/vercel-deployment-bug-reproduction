<Intro>

VMware vCenter Server ist die zentrale Verwaltungslösung für vSphere-Umgebungen. Es gibt die Editionen Foundation, Standard und Essentials. Die aktuelle Version ist vCenter Server 8.0.

Die folgenden Informationen richten sich an IT-Spezialisten und Administratoren, die sich mit dem Kauf von vSphere und vCenter Lizenzen beschäftigen. Mit nur wenigen Klicks können Sie Sofortangebote abrufen.

## Editionen

- [vCenter Server Foundation](/hersteller/vmware/vcenter-server/foundation/)<br />
  vCenter Server Foundation ist für den Einsatz in kleineren Rechenzentren konzipiert und kann bis zu vier ESXi-Hosts verwalten.
- [vCenter Server Standard](/hersteller/vmware/vcenter-server/standard/)<br />
  vCenter Server Standard bietet den vollen Funktionsumfang von vCenter und kann bis zu 2.000 ESXi-Hosts verwalten. Beispiele für erweiterte Funktionen im Vergleich zu Foundation sind: vRealize Orchestrator (Automatisierung), Enhanced Linked Mode (ELM), vCenter Server High Availability (VCHA) und vCenter Server File-Based Backup/Restore.
- [vCenter Server Essentials](/hersteller/vmware/vcenter-server/essentials/)<br />
  vCenter Server Essentials ist Bestandteil der vSphere Essential Kits (Bundles) und nicht separat erhältlich.


## Lizenzierung

vCenter Server wird [pro Instanz lizenziert](/hersteller/vmware/kauf-und-lizenzierung/#lizenzierung-nach-instanzen). Jede Instanz erfordert eine Lizenz. Die zugrunde liegende Hardware spielt keine Rolle.

## vCenter Server Appliance

vCenter Server kann auf physischen und virtuellen Windows- und Linux-Betriebssystemen installiert werden. Die Software inklusive Installer steht als Download zur Verfügung. Installationen werden mit Lizenzschlüsseln aktiviert.

vCenter Server ist auch als vorkonfigurierte, virtuelle Linux Appliance erhältlich. Beim Deployment werden nur wenige Parameter abgefragt. vCenter Appliance ist **keine** weitere vCenter Edition. Sie ist in der vCenter Standard-Lizenz enthalten.

## vCenter Server Heartbeat

vCenter Server Heartbeat war eine Hochverfügbarkeitslösung für vCenter Server und ist seit 2014 End of Life (EOL). Es gibt kein Nachfolgeprodukt. Es wird empfohlen, vCenter in einer virtuellen Maschine auszuführen und mit vCenter Server High Availability (VCHA) und anderen vSphere-Funktionen abzusichern.

## Kostenlose vCenter Server Tools

### vCenter Converter

VMware vCenter Converter konvertiert physische in virtuelle Maschinen. Alle gängigen Formate von physischen und virtuellen Maschinen werden unterstützt. Die kostenpflichtige Version VMware vCenter Converter Enterprise ist EOL und wird nicht mehr angeboten. Die Funktionalität ist nun in der kostenlosen Free-Edition enthalten.
