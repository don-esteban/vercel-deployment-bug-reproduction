<Intro>

VMware Workstation 17 Pro richtet sich an Anwender, die mehrere Betriebssysteminstanzen auf einem Windows oder Linux Rechner ausführen möchten. Typische Einsatzgebiete sind das Testen von Software, Konfigurationen und virtualisierten Netzwerken. Fehlersimulation, Software-/Web-Entwicklung, Security Tests (Kali Linux) und Docker/Kubernetes.

</Intro>

<TOC />



## Preise

VMware Workstation Plus wird pro Host [lizenziert](/hersteller/vmware/workstation/#lizenzierung). VMware stellt die Software als Download zur Verfügung und liefert einen Lizenzschlüssel zur Aktivierung aus (ESD). Schulen, Universitäten und andere anerkannte Bildungseinrichtungen können zu einem reduzierten Preis kaufen (Academic).

<Info>

**Bitte beachten Sie bei Ihrer Bestellung:** Das erste Jahr Support ist Pflicht und muss beim Kauf von Neulizenzen oder Updates mitbestellt werden.

</Info>

## Funktionsumfang

Unterstützt alle gängigen Windows Versionen und Linux Distributionen. Workstation Plus richtet sich an anspruchsvolle Anwender. Wenn Sie lediglich eine Gast-Betriebssystem-Instanz ausführen müssen, reicht Ihnen eventuell die einfachere Edition [VMware Workstation 17 Player](/hersteller/vmware/workstation/player/). Für Mac-Anwender steht [VMware Fusion](/hersteller/vmware/fusion/) mit einem ähnlichen Funktionsumfang zur Verfügung.
