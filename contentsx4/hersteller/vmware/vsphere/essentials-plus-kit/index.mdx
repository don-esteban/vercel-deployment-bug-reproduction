<Intro>

Auf dieser Seite finden Sie Preise und die Möglichkeit, mit wenigen Klicks Sofortangebote für vSphere Essentials Plus Kit anzufordern. VMware vSphere Essentials Plus Kit ist ein Komplettpaket für kleinere Datacenter, die physische Server auf bis zu drei vSphere Hosts konsolidieren möchten. Diese Seite richtet sich an Administratoren und IT-Experten, die den Kauf von vSphere Standard Lizenzen planen. Mit dem Modul für Sofortangebote (in den Warenkorb legen) können Sie Preise verschiedener Konfigurationen per E-Mail anfordern und weiteren Überlegungen und Verhandlungen zugrunde legen.

</Intro>

<TOC />



**Programmpaket bestehend aus:** vSphere Essentials Plus (vCenter Server Essentials, vSphere ESXi) inklusive Production Support. Lizenzierung: Mindestens 16 Cores pro CPU. Maximal 3 Hosts mit bis zu 96 Cores. Details unter [Funktionen](#funktionen).

## Lizenzierung

VMware vSphere Essentials Plus (VVEP) wird [nach Hosts lizenziert](/hersteller/vmware/kauf-und-lizenzierung/#lizenzierung-nach-hosts). *Im Paketpreis sind 3 Hosts mit bis zu 96 Cores enthalten.*
Es gelten folgende Bedingungen:

<ArrowList>

- Mindestens 16 Cores pro CPU.
- Maximal 3 Hosts mit bis zu 96 Cores.
- Die vSphere Lizenzen können nur mit dem im Kit enthaltenen vCenter Server Essentials verwaltet werden.

</ArrowList>

## Funktionen
