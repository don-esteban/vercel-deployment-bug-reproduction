## Editionen

<ArrowList>

- [vSphere 8 Essentials Kit](/hersteller/vmware/vsphere/essentials-kit/essentials/)<br />
Serverkonsolidierung für kleinere Organisationen. Das Zusammenfassen mehrerer Server zu wenigen großen Servern (max. 3 Hosts mit je zwei CPUs) vereinfacht den Rechenzentrumsbetrieb und senkt die Kosten.

- [vSphere 8 Essentials Plus Kit](/hersteller/vmware/vsphere/essentials-kit/essentials-plus/)<br />
Serverkonsolidierung und Business Continuity für kleine Unternehmen. Wie Essentials Kit, jedoch zusätzlich vMotion und X-Switch vMotion, Hochverfügbarkeit und Data Protection and Replication.

</ArrowList>

## Lizenzierung

vSphere Essentials Kits enthalten vSphere/ESXi und [vCenter Server Essentials](/hersteller/vmware/vcenter-server/essentials/). vSphere wird [nach Hosts lizenziert](/hersteller/vmware/kauf-und-lizenzierung/#lizenzierung-nach-hosts). Es gelten folgende Regeln:

- Eine Essentials Kit Lizenz umfasst drei Hosts (ESXi).
- Jeder Host darf maximal zwei CPUs enthalten.
- Jede CPU darf maximal 32 Cores enthalten.
- Der im Kit enthaltene [vCenter Server Essentials](/hersteller/vmware/vcenter-server/essentials/) kann nur die drei Hosts aus dem Kit verwalten.
- Die vSphere Lizenzen können nur mit dem im Kit enthaltenen vCenter Server Essentials verwaltet werden.
- Ein Essentials Kit ist eine untrennbare Einheit, bestehend aus vSphere und vCenter Server. Das Bundle ist unteilbar und kann nicht erweitert werden.
- Für steigende Anforderungen, wenn das erworbene Essential Kit nicht mehr ausreicht, gibt es Tradeups auf [Acceleration Kits](/hersteller/vmware/vsphere/acceleration-kit/).
