<Intro>

vSphere Acceleration Kits sind für den Einstieg in die Virtualisierung von Rechenzentren vorgesehen. Sie enthalten vSphere und vCenter Server in einem kostengünstigen Paket. Jede Organisation darf nur ein Acceleration Kit erwerben.

Auf dieser Seite finden Sie Informationen zum Kauf von vSphere Acceleration Kits und können mit wenigen Mausklicks Sofortangebote anfordern.

</Intro>

## Editionen

- [vSphere 8 Standard Acceleration Kit](/hersteller/vmware/vsphere/acceleration-kit/standard/)<br />
Paket zum Einstieg in Serverkonsolidierung und Business Continuity. Es gibt zwei Varianten. Die erste Variante enthält Lizenzen für sechs Prozessoren und einen [vCenter Server Standard](/hersteller/vmware/vcenter-server/standard/). Die zweite Variante enthält Lizenzen für acht Prozessoren und einen [vCenter Server Foundation](/hersteller/vmware/vcenter-server/foundation/).

- [vSphere 8 Enterprise Plus Acceleration Kit](/hersteller/vmware/vsphere/acceleration-kit/enterprise-plus/)<br />
Paket zum Einstieg in die Datacenter-Virtualisierung. Enthält Lizenzen für sechs Prozessoren und einen vCenter Server Standard. Enthält auch App-Performance und Container-Anwendungen.

## Lizenzierung

vSphere Acceleration Kits enthalten Lizenzen für vSphere und vCenter Server. Im Gegensatz zu [Essential Kits](/hersteller/vmware/vsphere/essentials-kit/) bestehen Acceleration Kits aus einzelnen Lizenzen. vSphere wird [nach CPUs der ESXi-Hosts lizenziert](/hersteller/vmware/kauf-und-lizenzierung/#lizenzierung-nach-prozessoren). vCenter Server wird [pro Instanz lizenziert](/hersteller/vmware/kauf-und-lizenzierung/#lizenzierung-nach-instanzen).


<Info>

**Tipp:** Acceleration Kits haben gegenüber den einzelnen Lizenzen weder lizenzrechtliche noch funktionale Einschränkungen. Sie bieten jedoch einen Preisvorteil. Daher sollten Sie beim Einstige in die Virtualisierung den Kauf eines Acceleration Kits erwägen.

</Info>

Die Kits sind für den Einstieg in die Virtualisierung von Rechenzentren vorgesehen. Pro Organisation ist nur **ein Acceleration Kit** zulässig. Wer mehr Lizenzen benötigt, kauft je nach Bedarf einzelne vSphere- und vCenter-Lizenzen hinzu.
