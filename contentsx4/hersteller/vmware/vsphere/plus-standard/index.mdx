<Intro>

vSphere+ Standard entspricht vSphere Standard plus einer Vielzahl von VMware Cloud Services, die über die VMware Cloud Console (VMC) bereitgestellt werden. vCenter Server Standard ist ebenfalls enthalten.

Diese Seite richtet sich an Administratoren und IT-Experten, die den Kauf von vSphere+ Standard Lizenzen planen. Mit unseren Sofortangeboten (in den Warenkorb legen) können Sie per E-Mail Preise für verschiedene Konfigurationen anfordern und diese als Grundlage für weitere Überlegungen und Verhandlungen nutzen.

</Intro>

<Info>

Bitte beachten Sie beim Kostenvergleich: vSphere+ Subscriptions enthalten bereits [vCenter Server Standard](/hersteller/vmware/vcenter-server/standard/). Sie dürfen so viele vCenter Server Standard betreiben, wie Sie für das Verwalten Ihrer vSphere+ Infrastruktur benötigen.

</Info>

## Lizenzen

VMware vSphere+ 8 Standard wird als [Subscription](/hersteller/vmware/kauf-und-lizenzierung/#subscriptions) angeboten, [nach Prozessorkernen der ESXi-Hosts lizenziert](/hersteller/vmware/kauf-und-lizenzierung/#lizenzierung-nach-prozessorkernen) und über [Download und Lizenz-Keys](/hersteller/vmware/kauf-und-lizenzierung/#download-und-lizenzschlüssel) bereitgestellt. [Production Support 24x7](/hersteller/vmware/kauf-und-lizenzierung/#support-and-subscription-services-sns) ist enthalten.
