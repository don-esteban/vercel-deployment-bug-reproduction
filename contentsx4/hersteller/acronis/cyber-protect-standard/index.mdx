<Intro>

  Die Acronis Cyber Protect Standard Produkte bieten die Standard Backup Funktionalität und mehrschichtigen Schutz vor Cyber Angriffen. Es gibt unterschiedliche Editionen für den Schutz von PC, Server (Windows/Linux), Windows Server Essentials und virtuelle Maschinen, sowie Microsoft 365.


</Intro>

<TOC />



## Acronis Cyber Protect Standard Produkte


<Info>

  Die [Cyber Protect Advanced Editionen](/hersteller/acronis/cyber-protect-advanced/) bieten gegenüber Cyber Protect Standard erweiterte Funktionen wie erweiterte Verwaltung, Deduplizierung, Unterstützung für Bandlaufwerke oder Cluster Unterstützung.


</Info>



- **[Cyber Protect Standard Workstation](/hersteller/acronis/cyber-protect-standard/workstation/)** - Backup und Wiederherstellung, sowie Anti-Virus und Anti-Malware für Windows PCs und Macs.

- **[Cyber Protect Standard Server](/hersteller/acronis/cyber-protect-standard/server/)** - Continuous Data Protection für physische Windows- und Linux-Server oder Cloud-Workloads inkl. unterstützter Applikationen.

- **[Cyber Protect Standard Windows Server Essentials](/hersteller/acronis/cyber-protect-standard/windows-server-essentials/)** - Kostengünstige Edition für komplette Windows Server Essentials Systeme inkl. Applikationen.

- **[Cyber Protect Standard Virtual Host](/hersteller/acronis/cyber-protect-standard/virtual-host/)** - Umfassende Data Protection für komplette VMware ESX(i)- oder HyperV-Hosts für unbegrenzte Anzahl von VMs sowie unterstützte Applikationen.


## Kostenloser Cloud-Speicherplatz

  Die Acronis Cyber Protect und Cyber Backup Subscriptions enthalten immer kostenlosen Speicherplatz in der Acronis Cloud. Die Größe des Cloud-Speichers hängt von der Lizenzart und Anzahl ab:

 - Cyber Protect Standard Workstation: 50 GB Cloud-Speicher
 - Cyber Protect Standard Windows Server Essentials: 150 GB Cloud-Speicher
 - Cyber Protect Standard Server: 250 GB Cloud-Speicher
 - Cyber Protect Standard Virtual Host: 250 GB Cloud-Speicher
