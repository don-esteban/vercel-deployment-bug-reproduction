<Intro>

  Mit Acronis Cyber Files Cloud erhalten Sie vollständige Enterprise File Sync & Share von jedem Gerät, gehostet in der Acronis Cyber Cloud.

  Die Funktionalität lässt sich mit Cyber Files Cloud Advanced um elektronische Signaturen und Notarization (Beurkundung) erweitern und in die [Acronis Cyber Protect Cloud](/hersteller/acronis/cyber-protect-cloud/) integrieren.


</Intro>

## Acronis Cyber Files Cloud Lizenzierung

  Lizenziert wird entweder nach **Nutzern (User)** oder der benötigten **Speichermenge in GB**.


### User

  Lizenziert werden die Nutzer, die den Dienst verwenden. Die lokale Sicherung ist immer enthalten. Wenn Sie die Daten zusätzlich in der Cloud sichern möchten, kann für den Nutzer der Acronis Cloud Speicher dazu lizenziert werden.

### Speicherplatz (GB)

  Hier wird die benötigte Speichermenge in der Acronis Cloud nach GB lizenziert. Eine zusätzliche, lokale Sicherung kann mit hybrid Storage dazu gebucht werden.

### Cyber Files Cloud Advanced

  Acronis Cyber Files Cloud Advanced ist ein **Add-On**, das eine Cyber Files Cloud User Lizenz voraussetzt. Die Advanced Editionen können nur mit der Workload Lizenzierung genutzt werden und bieten Funktionen der [Acronis Cyber Notary Cloud](/hersteller/acronis/cyber-notary-cloud/) (u.a. elektronische Signaturen).

## Funktionen mit Acronis Cyber Files Cloud


 -  Ermöglicht Nutzung und Teilen von Inhalten von jedem Gerät und Standort mit Internet.
 -  Ermöglicht Anwendern das sichere Erstellen, Bearbeiten und Freigeben von Inhalten über eigene Smartphones, Tablets, PCs und Macs.
 -  Gewährleistet die Produktivität Ihrer mobilen Anwender mit einer vollständigen Unterstützung der Microsoft Office-Mobilgeräte-Apps, sowie komfortablen Funktionen zur PDF-Bearbeitung/-Kommentierung.
 -  Unterstützt eine vollständige Ende-zu-Ende-Verschlüsselung, unternehmensgerechte Audit-Trails sowie ausgefeilte Richtlinienkontrollen für Inhalte, Benutzer und Geräte.
 -  Ermöglichen Sie eine einfache Remote-Zusammenarbeit von internen und externen Beteiligten, ein intuitives Benutzererlebnis sowie die Einhaltung umfassender Sicherheit- und Compliance-Vorgaben
