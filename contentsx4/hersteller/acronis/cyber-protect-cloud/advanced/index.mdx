<Intro>

  Acronis Cyber Protect Cloud Advanced Pakete sind **Add-Ons** und erweitern die Funktionalität der [Acronis Protect Cloud](/hersteller/acronis/cyber-protect-cloud/) in bestimmten Bereichen: Backup, Disaster Recovery, File Sync & Share und vielen weiteren (Verwaltung, Security, Email).

  Die Advanced Pakete werden immer nach Workloads lizenziert und können daher nur zusammen mit der Cyber Protect Cloud Workload Lizenzierung genutzt werden.


</Intro>

<TOC />



## Lizenzierung

  Die Acronis Cyber Protect Cloud enthält immer die Security und Anti-Malware Funktionalität, sowie die zentrale Verwaltung in der Acronis Cloud.
  Diese kann bei Bedarf mit den **Cyber Protect Cloud Advanced Paketen** erweitert werden. Die Advanced Versionen sind nur mit der Lizenzierung nach **Workloads** verfügbar.
  Die Preise für das Acronis Hosting beziehen sich immer auf die Region G1.


<Info>

**Kontinuierliche Datensicherung (CDP)**

Seit November 2021 ist Continouus Data Protection (CDP) in allen Cyber Protect Cloud Advanced Backup Lizenzen enthalten.
Der Acronis Agent kann Änderungen in Echtzeit überwachen und diese mit RPO-Werten nahe Null sichern, sodass keine Daten verloren gehen (auch nicht zwischen zwei regulären Backups)

</Info>

## Cyber Protect Advanced Funktionen

- **Advanced Backup** - Erweiterte Backup Funktionalität: u.a. Continuous Data Protection, Cluster Support
- **Advanced Disaster Recovery** - Test Failover in der Acronis Cloud, Deployment ohne VPN, eigene DNS-Konfiguration
- **Advanced Security** - URL Filter, Exploit Prevention, uvm.
- **Advanced Management** - u.a. Automatisches Patch Management, Software Inventar
- **Advanced File Sync & Share** - Lokale Repositories (NAS, SharePoint), Automatisches Backup der synchronisierten Dateien, Templates
- **Advanced Disaster Recovery** - Advanced Disaster Recovery
- **Advanced Email Security** - Anti-Malware und Anti-Phishing mit automatischer Überprüfung von Anhängen
