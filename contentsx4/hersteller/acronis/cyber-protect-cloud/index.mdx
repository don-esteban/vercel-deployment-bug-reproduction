<Intro>

  Die Acronis Cyber Protect Cloud vereint professionelle Backup-Funktionen mit KI-basiertem Anti-Malware Schutz der nächsten Generation und unternehmensgerechten Endpoint Protection Management-Fähigkeiten in einer einzigen Lösung.

  Die Cyber Protect Cloud besteht aus [Acronis Cyber Backup](/hersteller/acronis/cyber-backup-standard/) mit Endpoint Protection, optional erweiterbar mit [Advanced Paketen](#cyber-protect-cloud-advanced). Die Funktionen aus der [Acronis Cyber Files Cloud](/hersteller/acronis/cyber-files-cloud/) und [Cyber Notary Cloud](/hersteller/acronis/cyber-notary-cloud/) können in der Acronis Protect Cloud integriert werden.

</Intro>

## Lizenzierung

  Die Acronis Cyber Protect Cloud enthält immer die **Security und Anti-Malware Funktionalität**, sowie die **zentrale Verwaltung** in der Acronis Cloud. Diese kann bei Bedarf mit den [Advanced Paketen](#cyber-protect-cloud-advanced) erweitert werden.

  Lizenziert wird das Backup, entweder nach **Workloads** oder nach der **Speicherplatzgröße in GB**.
  Die Preise für das Acronis Hosting beziehen sich immer auf die europäischen Rechenzentren.



### Workload

  Hier werden die **zu sichernden Workloads** lizenziert. Acronis unterscheidet bei Workloads zwischen Server, virtuelle Maschine (VM), Workstation, Hosting Server, Microsoft 365 Konto, Google Workspace, Mobilgerät und Webseite.

  Die lokale Sicherung ist immer enthalten. Wenn Sie zusätzlich in der Cloud sichern möchten, kann der benötigte Speicherplatz in der Cloud (Acronis, Google, Azure) zusätzlich lizenziert werden.


<Info>

 **Achtung:** Bei der VM Lizenzierung benötigen der Hosting Server und die VMs jeweils eigene Lizenzen.

</Info>



### Speicherplatz (GB)

  Hier wird die benötigte Speichermenge in GB lizenziert. Die Daten können in der Cloud (Acronis, Google oder Azure), lokal oder an beiden Speicherorten (hybrid) gesichert werden.



### Cyber Protect Cloud Advanced

<Info>

**Kontinuierliche Datensicherung (CDP)**

Seit November 2021 ist Continouus Data Protection (CDP) in allen Cyber Protect Cloud Advanced Backup Lizenzen enthalten.
Der Acronis Agent kann Änderungen in Echtzeit überwachen und diese mit RPO-Werten nahe Null sichern, sodass keine Daten verloren gehen (auch nicht zwischen zwei regulären Backups)

</Info>

  Acronis Cyber Protect Cloud Advanced Lizenzen sind **Add-Ons** und erweitern die Funktionalität der Acronis Protect Cloud in bestimmten Bereichen (Verwaltung, Backup, Security, etc).

  Die Advanced Editionen werden nach Workloads lizenziert und können daher nur zusammen mit der Workload Lizenzierung genutzt werden.

  Weitere Informationen finden Sie in der [Acronis Protect Cloud Funktionsübersicht](#funktionen).

## Funktionen

  Die Acronis Cyber Protect Cloud bietet folgende Funktionalität:

- **Security** - Anti-Virus/Malware/Ransomware mit signaturbasierter Erkennung und KI-Unterstützter Analyse aus der Cloud
- **Verwaltung** - Gruppenverwaltung von Geräten, Remote Verbingungen, Hardware Inventar
- **Backup** - Die lizenzierten Workloads können vielfältig gesichert werden (File, Image, VM).


### Weitere Dienste

  Für die Acronis Cyber Protect Cloud gibt es weitere, kostenpflichtige Dienste, die hinzu gebucht werden können:

- **File Sync and Share** - [Cyber Files Cloud](/hersteller/acronis/cyber-files-cloud/)
- **Notary - eSignature** - [Cyber Notary Cloud](/hersteller/acronis/cyber-notary-cloud/)


### Cyber Protect Advanced

  Die Funktionalität einzelner Bereiche kann mit den [Cyber Protect Cloud Advanced Paketen](/hersteller/acronis/cyber-protect-cloud/advanced/) erweitert werden:

- **Advanced Backup** - Erweiterte Backup Funktionalität: u.a. Continuous Data Protection, Cluster Support
- **Advanced Disaster Recovery** - Test Failover in der Acronis Cloud, Deployment ohne VPN, eigene DNS-Konfiguration
- **Advanced Security** - URL Filter, Exploit Prevention, uvm.
- **Advanced Management** - u.a. Automatisches Patch Management, Software Inventar
- **Advanced File Sync & Share** - Lokale Repositories (NAS, SharePoint), Automatisches Backup der synchronisierten Dateien, Templates
- **Advanced Disaster Recovery** - Minutenschnelle Wiederherstellung an jeden beliebigen Speicherort.
- **Advanced Email Security** - Anti-Malware und Anti-Phishing mit automatischer Überprüfung von Anhängen


### Zukünfigte Erweiterungen

  Die Acronis Cyber Protect Cloud wird stetig weiter entwickelt und enthält zusätzliche Funktionen. Momentan sind die folgenden Erweiterungen geplant, das geplante Release-Quartal finden Sie in Klammern dahinter:

- **Advanced Security + EDR (Q4 2021)** - Erweiterte Sicherheit mit automatischer Endpoint Detection and Response (EDR)
- **Advanced Data Loss Prevention (Q3 2021)** - Erweiterte Sicherheit vor dem unbeabsichtigten Verlust von Daten
- **Advanced Automation (Q4 2021)** - Verbesserte Automatisierung durch Integration von unterschiedlichen Zahlungssystemen und Abrechnunsmöglichkeiten


## Vorteile mit der Acronis Cyber Protect Cloud

- **Cyber Security** - Bleiben Sie modernen Cyber-Bedrohungen immer einen Schritt voraus – mit einer preisgekrönten Antiransomware Protection, die um Antimalware-Funktionen erweitert wurde und die bereits vorhandene Antivirus-Lösungen im Unternehmen ergänzen oder ersetzen kann. Verhindern Sie Datenlecks über lokale Kanäle.
- **Backup und Recovery** - Minimieren Sie Datenverluste in der Infrastruktur und bei Workloads mit der besten Backup & Recovery-Technologie auf dem Markt – über verbrauchsbasierte oder pro-Workload-Lizenzierungsmodelle verfügbar.
- **Disaster Recovery** - Ermöglicht mit persönlichen Geräten eine sichere Verbindung zum Unternehmensnetzwerk herzustellen. Oder testen Sie Failover-Abläufe in einer isolierten Netzwerkumgebung.
- **File Sync & Share** - Steigern Sie die Produktivität, mit der Möglichkeit, Unternehmensinhalte jederzeit, von überall und mit jedem Gerät zu erstellen und sicher zu teilen. Es wird nur der Storage berechnet – und nur der, den Sie tatsächlich verwenden. Nutzen Sie eine Beglaubigungs- und E-Signatur-Technologie zur Datenvalidierung.
- **Verwaltung** - Zentralisieren Sie die Verwaltung von Backup und Security und stellen Sie Endpunkte in wenigen Minuten bereit. Behalten Sie mit den integrierten Monitoring- und Berichtsfunktionalitäten alles im Überblick.
