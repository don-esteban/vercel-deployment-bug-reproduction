<Intro>

  Die Acronis Cyber Notary Cloud bietet Funktionen für elekronische Signaturen und Beurkundung. Jede Datei wird über eine Blockchain Technologie abgesichert und kann nicht mehr verändert werden.

  Cyber Notary Cloud ist als Standalone Lizenz erhältlich, die Funktionalität lässt sich auch in der [Acronis Cyber Protect Cloud](/hersteller/acronis/cyber-protect-cloud/) integrieren.


</Intro>

## Acronis Cyber Notary Cloud Lizenzierung

  Lizenziert wird nach **Workloads**:


### Workload

  Jede signierte oder beurkundete Datei benötigt eine eigene Lizenz.
  Ein Workload ist je nach Lizenzierung eine einzelne Datei (pro File) oder ein Template (pro Template).
  Wenn Sie die Daten zusätzlich in der Cloud sichern möchten, kann für den Nutzer der Acronis Cloud Speicher dazu lizenziert werden.
