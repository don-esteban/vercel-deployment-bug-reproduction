## Produkte

Windows-Software für effektive und schnelle Datensicherung und Wiederherstellung.

## Lizenzierung

<ArrowList>

### Einzelprodukte

  - Lizenziert wird nach Anzahl zu sichernder physischer oder virtueller Systeme (Workstation oder Server)
  - Jedes System (Server oder Workstation) benötigt eine entsprechende Lizenz
  - Workstation-Lizenz darf nicht auf Server installiert werden
  - Lizenzierung berechtigt zu wichtigen Updates, Upgrades und Support.
  - Macrium Reflect Einzellizenzen sind unbefristet (perpetual).


### Bundles

  - Lizenzierung erfolgt in Packs (á 5, 10, 50, etc.)
  - Sie erhalten einen einzigen Volumenlizenzschlüssel
  - Zentrale Verwaltung erfolgt über [Macrium Site Manager](/hersteller/macrium/#macrium-site-manager) (inklusive)

</ArrowList>


### Techniker-Produkte

<ArrowList>

  [Macrium Reflect Technicians License](/hersteller/macrium/technicians/)

  - Nur als Subscription erhältlich
  - Ermöglicht portable Installation
  - Enthält eine Macrium Server Lizenz (perpetual)


  [Macrium SiteDeploy](/hersteller/macrium/sitedeploy/)

  - Nur als Subscription erhältlich
  - *Single Site:* Lizenzierung des Managers plus Anzahl der Endpoints
  - *System Builder:* Lizenzierung des Managers plus Anzahl der Deployments

</ArrowList>

#### Macrium Image Guardian

Macrium Image Guardian **(MIG)** ist ein Tool (in Macrium Reflect Produkten inklusive), welches den Zugriff auf von Macrium erstellte Backup- und Image-Dateien zum Schutz vor Ransomware einschränkt.
MIG gewährt nur Macrium-Imaging-Tools Schreibzugriff. Unbefugte Änderungen – wie Verschlüsselung durch Ransomware – sind ausgeschlossen.
Ein weiterer Vorteil von MIG ist, dass es Ransomware-Schutz für Backups auf vernetzten Rechnern bieten kann, ohne global installiert werden zu müssen.


#### Macrium Site Manager

Macrium Site Manager ist die zentrale Verwaltungskonsole und Nachfolger von Macrium Central Management Console (CMC) 6.3.
Der Site Manager ist kostenlos und verwaltet Einzelprodukte und Macrium Agent Licenses (MAL) der Bundles.

Die Installation braucht nicht auf einem Windows Server erfolgen, er kann auch auf Workstation installiert werden und lässt sich in AD integrieren.

Agenten aus den Bundles können nicht manuell installiert, sondern nur über den Site Manager ausgebracht werden.
Bei MAL wird ein Volumenlizenzschlüssel in der Konsole eingegeben, um alle Agenten des Bundles zu administrieren.
Bei Einzellizenzen muss jeder Key separat eingetragen werden.

Mit dem Site Manager lassen sich die meisten Backup-Aufgaben zentral erledigen: Auf welchen Geräten sind noch keine Agenten aufgespielt? Arbeiten alle Agenten zuverlässig? Das Ausbringen von Agenten auf neuen PCs oder Servern kann hier zentral überblickt und gesteuert werden.

Für den Macrium Site Manager 8 wird kein separat buchbarer Support angeboten. Hilfe bekommt man über den Support der kostenpflichtigen MAL Bundles.

*Macrium Reflect Server Plus und Technicians License, sowie Deployment Kit, werden nicht unterstützt.*


#### Macrium viBoot

Macrium viBoot ermöglicht das sofortige Erstellen, Starten und Verwalten von virtuellen Maschinen unter Microsoft Hyper-V und VirtualBox, indem eine oder mehrere Macrium Reflect-Image-Dateien als Basis für das Speichersubsystem der virtuellen Maschine verwendet werden.

Mit viBoot können Sie zumindest in Images booten, die Sie mit Macrium Reflect erstellt haben.
Diese lassen sich dann überprüfen oder Sie können selektiv Daten wiederherstellen, die auf einem bootfähigen Image gespeichert sind.

Auf Unternehmensebene können Sie eine komplette Netzwerkumgebung in wenigen Minuten wiederherstellen.




## Macrium Reflect Download

Macrium stellt eine 30-Tage-Version mit vollem Funktionsumfang (Workstation, ohne Macrium ReDeploy) zum ausgiebigen Testen zur Verfügung.
Während der Testphase kann auch der Support in Anspruch genommen werden.
Nach Ablauf der Testversion können zuvor erstellte Sicherungsdateien wiederhergestellt werden, alle anderen Funktionen sind gesperrt.
