<Intro>

Speziell für den IT-Techniker-Teams. Wie *Macrium Reflect Technicians License*, Lizenzierung für je 5 Techniker. Erweiterte Bereitstellung von Golden Images.

</Intro>

<TOC />


Die zentrale Verwaltung über den [Site Manager](/hersteller/macrium/#macrium-site-manager) ist **nicht** möglich.

<Info>

Dieses Produkt wird nicht weiter vertrieben. Das Nachfolge-Produkt ist [Macrium SiteDeploy](/hersteller/macrium/sitedeploy/). Erstellte Images können weiterhin verteilt werden.

</Info>

## Lizenzierung

<ArrowList>

Das Deployment Kit ist ausschließlich als Abonnement-Lizenz (Subscription) erhältlich.

- Als Subscription für 1 Jahr verfügbar. Andere Laufzeiten erhalten Sie auf Anfrage.
- Wird vom Site Manager **nicht** unterstützt.
- Enthält Macrium Reflect Server Lizenz (perpetual)


## Unterschied Deployment Kit und Technician

- [Technicians Licence](/hersteller/macrium/technicians/) ist für einen einzelnen Techniker. Die Technikerlizenz deckt die Wiederherstellung auf demselben Computer oder dessen Ersatz ab.
- [Deployment Kit](/hersteller/macrium/deployment-kit/) lizenziert bis zu 5 Techniker und deckt eine großflächige Bereitstellung ab, dazu zählen u.a Wiederherstellung eines Golden Image auf mehreren Computern.

</ArrowList>
