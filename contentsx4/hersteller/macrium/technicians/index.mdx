## Lizenzierung

<ArrowList>

Die Macrium Reflect Technicians License ist ausschließlich als Abonnement (Subscription) erhältlich.

- Als Subscription für 1 Jahr verfügbar. Andere Laufzeiten erhalten Sie auf Anfrage
- Wird vom Site Manager **nicht** unterstützt
- Enthält eine Macrium Reflect Server Lizenz, die auch nach Ablauf der Subscription genutzt werden darf

</ArrowList>


## Funktionen

Werkzeug speziell für den IT-Techniker. Ermöglicht Macrium Reflect von USB-Stick (Portable Application Support) zu starten oder als ISO zu booten. Berechtigt zu Arbeiten an unbegrenzter Anzahl von Servern und Workstations.
Enthält eine unbefristete Einzelinstallation von *Macrium Reflect Server*.
