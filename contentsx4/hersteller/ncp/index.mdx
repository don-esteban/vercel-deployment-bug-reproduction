<Intro>

NCP Engineering bietet mit NCP Secure eine VPN Remote Access Lösung für Android oder Windows- und Mac-Clients. NCP Secure funktioniert mit den gängigen IPsec VPN Gateways und verfügt über eine IPv6-fähige dynamische Personal Firewall.

</Intro>

## Lizenzierung

**NCP Secure** wird, je nach Produkt, als Einzellizenz oder als Volumenlizenz angeboten. Die Software enthält eine dauerhafte Lizenz und wird auf einem einzelnen Gerät installiert. Wenn auf einem Computer mehrere Betriebssysteme (virtuelle Umgebungen) installiert sind, wird für jede installierte virtuelle Umgebung eine Lizenz benötigt. Lizenziert wird generell nach Clients (Windows, macOS oder Android). Um Sonderkonditionen für akademische Einrichtungen und Bildungseinrichtungen zu erhalten, wird ein Nachweis (Student/Schüler/Beschäftigter der Einrichtung) des Lizenznehmers benötigt.


### Einzellizenz

[NCP Secure Entry für Windows](/hersteller/ncp/windows/) und [NCP Secure Entry für Mac](/hersteller/ncp/mac/) wird als Einzellizenz angeboten.

Mit jeder Einzellizenz erhält man einen separaten Lizenzschlüssel. Die Installation der Software erfolgt manuell. Der Einzellizenzpreis richtet sich nach der Bestellmenge.


### Volumenlizenz

[NCP Secure Android](/hersteller/ncp/android/) wird als Volumenlizenz angeboten.

Volumenlizenzen sind ab einer Bestellmenge von 5 Lizenzen erhältlich. Sie erhalten einen zentralen Lizenzschlüssel, mit dem alle erworbenen Lizenzen freigeschaltet werden. Weitere Bestellungen unter Verwendung des gleichen Lizenzschlüssels werden kumuliert.


#### Beispiel zum Kauf

**Ersteinkauf:** Beim Kauf von 20 Lizenzen erhalten Sie den Preis der Staffel 5-24 User und einen einzigen Volumenlizenzschüssel zur Freischaltung.

**Nachbestellung von 8 Lizenzen:** Bei der Bestellung von 8 zusätzlichen Lizenzen, erhalten Sie diese in der günstigeren Preisstaffel 25-49 User. Der bestehende Volumenlizenz-Key wird um die 8 zusätzlichen Lizenzen erweitert und besitzen nun ingesamt 28 Lizenzen.

**Weitere Nachbestellung von 3 Lizenzen:** Mit einem Volumenlizenzvertrag können Sie auch weniger als 5 Lizenzen kaufen. Die 3 Lizenzen erhalten Sie in der Staffel 25-49 User und der vorhandene Volumenlizenz-Key wird erweitert.
