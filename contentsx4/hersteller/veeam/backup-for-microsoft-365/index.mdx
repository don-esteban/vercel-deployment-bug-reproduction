<Intro>

  Veeam Backup for Microsoft 365 (ehemals Veeam Backup for Microsoft Office 365) ermöglicht sicheres Backup und Wiederherstellung Ihrer Microsoft 365 Daten (Exchange Online, SharePoint Online und OneDrive for Business, Teams, etc.).
  Integration in Veeam Data Platform.
  Aktuell ist Version 7.


</Intro>

<TOC />

Im Bundle [VDP Advanced](/hersteller/veeam/data-platform-advanced/) und [VDP Premium](/hersteller/veeam/data-platform-premium/) inklusive oder als eigenständige Anwendung lauffähig.

**NEU:** Auch als Backup-as-a-Service (BaaS) erhältlich: Cirrus for Microsoft 365

## Funktionen

### NEU in Version 7

Veeam Backup for Microsoft 365 schließt eine Lücke bei Microsofts Online-Diensten, da MS diese nicht verpflichtend sichert.

<ArrowList>

- Unveränderbare Backup-Kopien (Immutability).
- Echte, überprüfbare Unveränderbarkeit oder Objektsperre von Backup-Sets möglich
- 3-2-1-Backup-Methode mit zwei Kopien eines Backups möglich
- Schnelle und unkomplizierte Wiederherstellung durch primäre Backups auf performante Objektspeicher (nicht gesperrt)
- Sicherheit gegen Ransomware durch Immutability.
- Integration in Veeam Data Platform Dashboard
- Ermöglicht nun auch Daten von JEDEM Objektspeicher auf JEDEN Objektspeicher zu sichern
- Unterstützung durch Self-Service-Portal zur Wiederherstellung
- Erweiterte Verwaltungs- und Automatisierungsfunktionen

Siehe dazu auch: [Was ist neu in Veeam Backup für M365 V7](/hersteller/veeam/veeam_backup_m365_whats_new.pdf)

</ArrowList>
