## Lizenzierung

Intel System Studio Composer Edition wird entweder nach **Nutzern** oder nach **Netzwerk-Zugriffen (floating)** lizenziert. Die (dauerhaften) Lizenzen können ausschließlich mit Wartung erworben werden. Die Wartung beinhaltet Software Updates, Support und günstigere Konditionen für ein Upgrade, auf eine höherwertige Edition.

Mehr dazu finden Sie in unserer [Intel-Seite](/hersteller/intel/#lizenzierung).
