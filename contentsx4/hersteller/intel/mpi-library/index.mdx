Intel MPI Library wird entweder nach **Nutzern** oder nach **Netzwerk-Zugriffen (floating)** lizenziert. Die Lizenzen können ausschließlich mit Wartung erworben werden. Die Wartung beinhaltet Software Updates, Support und günstigere Konditionen für ein Upgrade, auf eine höherwertige Edition.

## Funktionen von Intel MPI Library

- Interconnect‐ Unterstützung für Myrinet MX‐ und QLogic PSM‐Schnittstellen
- Unterstützung von hybrider Parallelisierung
- Programming Library (DAPL) oder die Open Fabrics Association (OFA)‐Methodik
- Anwendungs‐Tuning
- Interconnect Independence
- Schnelles On‐Node Messaging
- Verringerung des Speicherbedarfs durch dynamische Verbindungen
