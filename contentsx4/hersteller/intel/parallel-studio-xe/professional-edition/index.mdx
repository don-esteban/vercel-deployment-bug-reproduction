## Lizenzierung

Intel Parallel Studio XE Professional Edition wird entweder nach **Nutzern** oder nach **Netzwerk-Zugriffen (floating)** lizenziert. Die (dauerhaften) Lizenzen können ausschließlich mit Wartung erworben werden. Die Wartung beinhaltet Software Updates, Support und günstigere Konditionen für ein Upgrade, auf eine höherwertige Edition.

Mehr dazu finden Sie in unser [Intel-Überssichtseite](/hersteller/intel/#lizenzierung).





## Funktionen

- Alle Funktionen von der [Composer Edition](/hersteller/intel/parallel-studio-xe/composer-edition/#funktionen).

- Intel Inspector (C, C++, Fortran)
- Intel Advisor (C, C++, Fortran)
- Vollständige Bibliotheken, Analyse Tools und APIs
