<Intro>

Intel Parallel Studio XE ist eine Entwickler-Suite, welches unter anderem Intel Data Analytics Acceleration, Intel VTune Amplifier (ab der Professional Edition), Intel MPI Library (Cluster Edition) und weitere Tools für C++, Fortan, Python sowie Bibliotheken beinhaltet. Parallel Studio XE ist in 3 Editionen erhältlich.

</Intro>

</Info>


## Intel Parallel Studio XE Produktübersicht

<ArrowList>

- [Intel Parallel Studio XE Composer Edition](/hersteller/intel/parallel-studio-xe/composer-edition/) <br />
  Enthält Compiler für C++ und Fortran, Performance Librarys, Standardbasiertes Parallelprogramming Modell und optimiert Python.
  [Parallel Studio XE Composer Edition](/hersteller/intel/parallel-studio-xe/composer-edition/) wird ersetzt durch: [Intel oneAPI Base & HPC Toolkit](/hersteller/intel/oneapi/hpc-toolkit/) *Single-Node*.

- [Intel Parallel Studio XE Professional Edition](/hersteller/intel/parallel-studio-xe/professional-edition/) <br />
  Enthält Composer Edition. Inklusive Analyse-Tools zum Entwerfen, Verbessern und Korrigieren von parallelem Code.
  [Parallel Studio XE Professional Edition](/hersteller/intel/parallel-studio-xe/professional-edition/) wird ersetzt durch: [Intel oneAPI Base & HPC Toolkit](/hersteller/intel/oneapi/hpc-toolkit/) *Single-Node*.

- [Intel Parallel Studio XE Cluster Edition](/hersteller/intel/parallel-studio-xe/cluster-edition/) <br />
  Enthält Professional Edition. Inklusive MPI Library, MPI Tuning und Trace Analyzer. [Parallel Studio XE Cluster Edition](/hersteller/intel/parallel-studio-xe/cluster-edition/) wird ersetzt durch: [Intel oneAPI Base & HPC Toolkit](/hersteller/intel/oneapi/hpc-toolkit/) *Multi-Node*.

</ArrowList>
