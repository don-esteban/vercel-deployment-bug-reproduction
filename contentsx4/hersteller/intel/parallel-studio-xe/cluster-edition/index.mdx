## Lizenzierung

Intel Parallel Studio XE Cluster Edition wird entweder nach **Nutzern** oder nach **Netzwerk-Zugriffen (floating)** lizenziert. Die (dauerhaften) Lizenzen können ausschließlich mit Wartung erworben werden. Die Wartung beinhaltet Software Updates, Support und günstigere Konditionen für ein Upgrade, auf eine höherwertige Edition.

Mehr dazu finden Sie in unserer [Intel-Seite](/hersteller/intel/#lizenzierung).



## Funktionen von Intel Parallel Studio XE Cluster Edition

- Alle Funktionen von der [Professional Edition](/hersteller/intel/parallel-studio-xe/professional-edition/#funktionen).
- Intel MPI Library (C, C++, Fortran)
- Intel Trace Analyzer and Collector (C, C++, Fortran)
- Intel Cluster Checker
