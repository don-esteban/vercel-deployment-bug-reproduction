## Lizenzierung

Intel Media Server wird entweder nach **Nutzern** oder nach **Netzwerk-Zugriffen (floating)** lizenziert. Die Lizenzen können ausschließlich mit Wartung erworben werden. Die Wartung beinhaltet Software Updates, Support und günstigere Konditionen für ein Upgrade, auf eine höherwertige Edition.

## Funktionen von Intel Media Server Studio

### Essentials Edition

- Bietet Zugriff auf Codecs die Intel Quick Sync-Video-fähig und hardwarebeschleunigt sind (HEVC, AVC und mehr)
- Entwicklungsumgebung für heterogene Anwendungen
- Intel Media SDK, Intel SDK for OpenCL Applications, Laufzeiten, Grafiktreiber
- Intel Premier Support mit direktem Zugang zu technischen Experten von Intel


### Professional Edition

- HEVC-, MPEG-2- und Audiocodecs für Unternehmen
- Essentials Edition
- Professionelle Leistungs- und QualitätsAnalyzer (Intel VTune Verstärker, Video Quality Caliper)
- Premium Telecine Interlace Reverser und vieles mehr
