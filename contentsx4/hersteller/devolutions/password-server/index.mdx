<Intro>


  Auf diesen Seiten finden Sie Informationen zur Lizenzierung und Kosten von Devolutions Server *(ehemals Devolutions Password Server)*.
  Der Server bietet zentrale, rollenbasierte Zugriffskontrolle mit Verwaltung, Absicherung und Überwachung privilegierter Konten im lokalen Netzwerk.
  Diese Informationen richten sich hauptsächlich an IT-Einkäufer und Administratoren.
  Fordern Sie mit einem Klick ein kostenloses und unverbindliches Sofortangebot an.


</Intro>

## Lizenzierung

<ArrowList>

- Lizenziert wird **pro User** im Pack (Team oder Enterprise)
- CAL (Client Access License) als **Named User**
- CAL sind Voraussetzung für den Zugriff über die Webanwendung oder Devolutions Web Login auf Devolutions Server
- **Eine Devolutions *Launcher-Lizenz* beinhaltet eine Launcher-Lizenz und eine Nutzer-CAL**
- Weitere Informationen zu den Lizenzierungsarten finden Sie auf der [Devolutions Übersichtsseite](/hersteller/devolutions/#lizenzierungsarten).

</ArrowList>

## Funktionen

<ArrowList>

 - Gemeinsamer On-Premises Tresor
 - Privilegierte Sitzungsverwaltung
 - 2FA - Zwei-Faktor Authentifizierung
 - Single Sign-On (SSO)
 - Verbindungen mit 1-Klick starten
 - Active Directory und Office 365 / Microsoft 365 Integration
 - Rollenbasierte Zugriffskontrolle
 - Detaillierte Zugriffsberichte

</ArrowList>

<Info>

[Remote Desktop Manager](/hersteller/devolutions/remote-desktop-manager/) lässt sich mit Devolutions Server kombinieren und zentral verwalten.


Mit Devolutions Server können Sie den Zugriff auf privilegierte Konten kontrollieren und Sitzungen über eine sichere Lösung verwalten, die lokal bereitgestellt werden kann.
In Kombination mit [Devolutions Remote Desktop Manager](/hersteller/devolutions/remote-desktop-manager/) wird Devolutions Server zum einzigen Portal jeder IT-Abteilung, die die Speicherung von Passwörtern und Anmeldeinformationen mit einer robusten und effizienten Lösung für die Verwaltung von Remoteverbindungen integriert.


</Info>
