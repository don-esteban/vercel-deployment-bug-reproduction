## Lizenzierung

  Devolutions Enterprise Produkte werden **Pro User** lizenziert, einzeln oder im Pack.

<ArrowList>

 - [Remote Desktop Manager Enterprise](/hersteller/devolutions/remote-desktop-manager/) gibt es nur als Subscription (Abonnement), Lizenz *Pro User*, erhältlich als Einzellizenz (Single User) oder im Paket (Enterprise) bis 50 User. Jeder User braucht eine eigene Lizenz, unabhängig der Anzahl aktiver Nutzer im Zugriff.

 - [Devolution Server Enterprise](/hersteller/devolutions/password-server/)	gibt es nur noch als Subscription (Abonnement). Lizenz als [Enterprise Edition](/hersteller/devolutions/password-server/#enterprise-edition) (bis 50 User) oder als [Team Edition](/hersteller/devolutions/password-server/#team-edition) (bis 15 User).
 Zusätzlich sind [Client Access License (CAL)](/hersteller/devolutions/password-server/#cal) als *Named User* erhältlich.

</ArrowList>


### Lizenzierungsarten


<ArrowList>

- **Dauerlizenz/Perpetual License:** <br />
  Die gekauften Lizenzen sind dauerhaft gültige (perpetual) Lizenzen. Die Software kann somit auch nach Ablauf der aktiven Wartung weiterhin auf dem letzten lizenzierten Versionsstand genutzt werden.
  Nach Ablauf der Wartung können Sie innerhalb von 30 Tagen wieder in die Wartung einsteigen. Danach ist nur eine Neulizenzierung möglich.

- **Subscription:** <br />
 Die Abo-Lizenzen von [Remote Desktop Manager Enterprise](/hersteller/devolutions/remote-desktop-manager/) enthalten immer den **Standard Support** und enthalten somit Software Updates und Zugang zum Support Forum. Die Software kann nach Ablauf der Laufzeit nicht mehr genutzt werden. Um die Software weiterhin nutzen zu können, muss die Subscription verlängert werden.

- **Nutzerlizenz (Named User):** <br />
 Die Software kann auf einem oder mehreren Computern des Lizenznehmers installiert und kopiert werden, aber darf aber nur von einem benannten Endnutzer verwendet werden.

- **Enterprise Lizenz:** <br />
 Pack mit Lizenzen für 50 User.

- **Standortlizenz:** *(nicht mehr erhältlich)* <br />
 Mit der Standortlizenz kann die Software auf mehreren Computern des Lizenznehmers installiert und kopiert werden, die sich an einem einzigen physischen Standort oder in einer einzigen Geschäftsstelle befinden oder hiermit aus der Ferne verbunden sind. Die Lizenz kann von einer unbegrenzten Anzahl von Endnutzern verwendet werden, die sich hauptsächlich in dieser Geschäftsstelle oder Niederlassung des Lizenznehmers befinden, dort den Mittelpunkt ihrer Beschäftigung haben oder dorthin abgestellt (Dienstleister) sind.

</ArrowList>


## Merkmale

<ArrowList>

   - **Enterprise Funktionen:** Beispielsweise Rollenbasierte Zugriffskontrolle, Zwei-Faktor-Authentifizierung, Revision und Berichte, Multi Concurrent Sessions, Unbeaufsichtigter Zugriff
   - **Kompatibel mit vielen Tools:** Kompatibel mit Apple Remote Desktop, Citrix, Dameware, FTP, Hyper-V, LogMeIn, Microsoft Remote Assistance, Microsoft Remote Desktop, Radmin, SSH Shell, TeamViewer, Telnet, Terminal Services, VMware, VNC, SCP, X Windows
   - **Plattformunabhängig:** Die kostenlose Client-Software für Devolutions Server lässt sich unter Windows, Mac, iOS oder Android installieren.
   - **Einfache Integration:** Einfache Installation und Bedienung der Benutzeroberfläche. Keine Vorkenntnisse nötig.

</ArrowList>
