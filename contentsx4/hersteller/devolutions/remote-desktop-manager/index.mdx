<Intro>

Auf diesen Seiten finden Sie Informationen zur Lizenzierung und Kosten von Remote Desktop Manager (RDM).
RDM vereint die Verwaltung verschiedener Verbindungstechnologien, Maschinendaten, Zugangsdaten und Zugriffsberechtigungen zur Remotesteuerung auf einer Plattform. RDM unterstützt plattformübergreifende Sitzungen und mehr als 40 Remote-Verbindungsprotokolle und -Verbindungen.
Diese Informationen richten sich hauptsächlich an IT-Einkäufer und Administratoren.
Fordern Sie mit einem Klick ein kostenloses und unverbindliches Sofortangebot an.


 </Intro>

## Lizenzierung

<ArrowList>

 - Lizenziert wird einzeln **Pro User** oder als **Enterprise** (50 User).
 - Als Subscription für 1 oder 3 Jahre verfügbar.
 - Weitere Informationen zu den Lizenzierungsarten finden Sie auf der [Devolutions Übersichtsseite](/hersteller/devolutions/#lizenzierungsarten).

</ArrowList>


## Funktionen

<ArrowList>

 - Remote Verbindungsverwaltung
 - Sicherer Zugriff auf kritische Assets
 - Gemeinsame Datenbank und Repositories
 - Revisionskontrolle und Berichte von Aktivitäten von Benutzern in Echtzeit
 - Einfache Verwaltung wie mobiler Zugriff, Offline Zugriff und durch die Konsole
 - Benutzerverwaltung
 - Integration mit verschiedene Benutzer-DB, einschließlich Devolutions Server, Devolutions Hub Business, Microsoft SQL Azure und Microsoft SQL Server
 - Vereinfachte Bereitstellung von Benutzereinstellungen
 - Zwei-Faktor Authentifizierung
 - Rollenbasierte Zugriffskontrolle
 - Umfangreiche Audits und Reports
 - Doppelte Verschlüsselung der Datenbank mit personalisierter Passphrase oder Zertifikat
 - Persönlicher und zentraler Passwort-Tresor
 - Integration in bestehende Passwort-Manager, wie 1Password, AuthAnvil, Dashlane, KeePass, LastPass und andere
 - Importiert direkt Passwörter anderen Anwendungen, einschließlich 1Password, KeePass, LastPass



 Und viele neue Funktionen in [Version 2023.2](https://blog.devolutions.net/2023/06/whats-new-in-remote-desktop-manager-20232/) hinzugekommen (externer Link).

</ArrowList>


### Besondere Merkmale

 **Devolutions Remote Desktop Manager** ist mit mehreren relevanten Tools und Technologien kompatibel und läßt sich mit diesen kombinieren.
 **RDM** arbeitet in Kombination mit Citrix, Dameware, FTP, Hyper-V, LogMeIn, Microsoft Remote Assistance, Microsoft Remote Desktop, PC Anywhere, Radmin, SSH, Virtual Box, Team Viewer, Telnet, Terminaldienste, Virtual PC, VMware, VNC, X Windows und mehr.
Wenn es eine Befehlszeilenschnittstelle besitzt, kann **RDM** diese aufrufen.
