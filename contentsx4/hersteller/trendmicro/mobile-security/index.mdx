<Intro>

Trend Micro Mobile Security schützt ihre mobilen Endgeräte wie iOS, Android oder Windows Phone. Damit lassen sich ihre Daten verschlüsseln, per Remote sperren oder sogar auch löschen. Enthält auch Anti-Virus Schutz.

Kann auch als Add-On für die Apex One verwendet werden. Mobile Security ist auch Bestandteil der Smart Protection Complete Suite.


</Intro>
