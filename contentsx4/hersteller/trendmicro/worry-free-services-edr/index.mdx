## Preise

Worry-Free Services + EDR ist als Subscription inklusive Standard Support verfügbar. Lizenziert wird nach Geräten. Premium Support (24/7) kann optional angefragt werden. Mindestlizenzierung: 5 Seats/Lizenzen. Verlängerungen müssen vor Ablauf veranlasst werden. Mehr dazu unter [Lizenzierung](/hersteller/trendmicro/lizenzierung/).

<Info>

**Achtung:** Die Lizenzierung hat sich geändert. Worry Free wird nach Geräten (Server, Clients) lizenziert.

</Info>

## Funktionen

- Zentrales Management<br/>
Web-basierte Managementkonsole aus der Cloud, automatische Updates/Upgrades
- Schutz von PCs, Laptops, Windows-Server, Mac-Clients und -Server, sowie mobile Endgeräte
- Anti-Virus und Web Security <br/>
Ransomware-Schutz, Schützt gegen Viren, Spyware und andere Malware, Erweiterte URL-Filterung blockiert unangemessene Internetseiten, Web-Bedrohungsschutz
- Datenschutz <br/>
USB-Gerätekontrolle, Firewall, Data Loss Prevention (DLP)
- Active Directory Support
- Detection and Response (EDR)<br/>
Automatisierte Erkennung und Überwachung, sowie Beseitigung von Bedrohungen, innerhalb und außerhalb ihres Netzwerkes

<Info>

Wenn Sie Office 365 Schutz benötigen, gibt es [Worry-Free XDR](/hersteller/trendmicro/worry-free-xdr/). Worry-Free gibt es auch als On-Prem Edition, [Worry-Free Security](/hersteller/trendmicro/#worry-free).


</Info>
