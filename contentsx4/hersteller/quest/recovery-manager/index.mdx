<Intro>

Quest Recovery Manager bietet Disaster Recovery für Active Directory(AD), Azure AD, Office 365 und Exchange. So vermeiden Sie Datenverluste und stellen einen unterbrechungsfreien Betrieb sicher.

</Intro>

<TOC />



## Recovery Manager Editionen

<ArrowList>

- [Recovery Manager Active Directory](/hersteller/quest/recovery-manager/active-directory/)<br />
Beschleunigte AD Sicherung und Wiederherstellung auf Objekt- und Attributebene. Ermöglicht die Migration von Active Directory und Azure Active Directory.

- [Recovery Manager for Exchange](/hersteller/quest/recovery-manager/exchange/)<br />
Vereinfacht und beschleunigt die Exchange-Wiederherstellung und -Erkennung. Bietet Disaster Recovery für Exchange, einzelne Postfächer oder .PST-Dateien.

</ArrowList>


## Funktionen mit Recovery Manager

- Umfassende Wiederherstellung
- Granulare Online-Wiederherstellung
- Berichterstellung zu Unterschieden für Active Directory und Azure AD
- Wiederherstellung bei Hybrid AD und Azure AD
