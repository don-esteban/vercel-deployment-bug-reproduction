<Intro>

Recovery Manager for Active Directory bietet eine effektive Notfallwiederherstellung für Active Directory (AD), um Datenverlust zu vermeiden und einen unterbrechungsfreien Betrieb zu gewährleisten.

</Intro>

<TOC />


## Preise

Recovery Manager for Active Directory ist als Kauflizenz oder Subscription jeweils inklusive 1 Jahr [Standard Maintenance](/hersteller/quest/#standard-support) oder [24x7 Support](/hersteller/quest/#24x7-support) verfügbar. Optional (auf Anfrage) gibt es auch [Premier Support](/hersteller/quest/#premier-support). Lizenziert wird nach **Managed Person**, **Enabled User Account** und **Enabled AD LDS User**. Bei Neukauf gilt eine Mindestbestellmenge von 50 Lizenzen. Mehr dazu unter [Lizenzierung](/hersteller/quest/recovery-manager/active-directory/#lizenzierung).

## Lizenzierung

Der Recovery Manager für Active Directory kann auf unterschiedliche Art lizenziert werden. Dabei stehen die folgenden Lizenzgrundlagen zur Auswahl:

- **Managed Person** <br />
Jede verwaltete physische Person muss lizenziert werden.

- **Enabled User Account** <br />
Jeder aktive Nutzer muss lizenziert werden.

- **Enabled AD LDS User** <br />
Jeder Active Directory Lightweight Directory Services (AD LDS) Nutzer muss lizenziert werden.


## Funktionen

**Recovery Manager Active Directory (AD):**

- AD Verwaltung und Zustandsvalidierung
- Automatisierte Wiederherstellung der Gesamtstruktur
- Vergleichsberichte
- Wiederherstellung bei Hybrid AD und Azure AD
- Fehlertoleranz der Konsole für Gesamtstrukturwiederherstellung
- Wiederherstellungsplan
- Integration mit IT Security Search
- Wiederherstellung ohne Offline-Schaltung des AD



**Recovery Manager Active Directory Forest Edition:**

- Alle Funktionen von Recovery Manager für Active Directory (AD)
- Virtuelles Testlabor
- Einfache Wiederherstellung der Konfiguration
- Einsicht in die Active Directory Gesamtstruktur.
