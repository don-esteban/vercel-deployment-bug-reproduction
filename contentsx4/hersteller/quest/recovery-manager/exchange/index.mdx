<Intro>

Der Recovery Manager for Exchange sichert Microsoft Exchange Server, Postfächer und .PST-Dateien und stellt mit schneller Wiederherstellung einen unterbrechungsfreien Betrieb sicher.

</Intro>

## Preise

Recovery Manager for Exchange ist als Kauflizenz oder Subscription jeweils inklusive 1 Jahr [Standard Maintenance](/hersteller/quest/#standard-support) oder [24x7 Support](/hersteller/quest/#24x7-support) verfügbar. Lizenziert wird nach **Managed Person**, **Enabled User Account**, **Mailbox** oder **PST**. Bei Neukauf gilt eine Mindestbestellmenge von 50 Lizenzen. Mehr dazu unter [Lizenzierung](/hersteller/quest/recovery-manager/exchange/#lizenzierung).

## Lizenzierung

Der Recovery Manager für Exchange kann auf unterschiedliche Art lizenziert werden. Dabei stehen die folgenden Lizenzgrundlagen zur Auswahl:

- **Managed Person** <br />
Jede verwaltete physische Person muss lizenziert werden.

- **Enabled User Account** <br />
Jeder aktive Nutzer muss lizenziert werden.

- **Mailbox only** <br />
Sichert ausschließlich das Postfach. Nur für lokale Exchange Server.

- **PST only** <br />
Sicher nur die PST Datei, nicht das Postfach. Nur für lokale Exchange Server.

## Funktionen

Recovery Manager für Exchange, hilft dabei E-Mails schnell und problemlos wiederherzustellen. Außerdem unterstützt es beim Vergleich von Online-Postfächern mit entsprechenden Sicherungsdateien, um Änderungen zu identifizieren und aktuell zu halten.

- Sammelsuche
- Intelligente Suche
- Migrationsunterstützung
- Sicherungssoftware
- Exportoptionen
- Vergleichsberichte
- Aufgabenautomatisierung
