DevExpress Universal ist ein umfangreiches Komplettpaket für Visual Studio Entwickler.
Diese Informationen richten sich hauptsächlich an IT-Einkäufer und Administratoren.

## Preise

 Ab 2 Lizenzen erhalten Sie Mengenrabatte. Gerne erstelle ich Ihnen ein Angebot auf Anfrage.


<Products bkey="devexpress" tags={["universal"]}/>

## Lizenzierung

DevExpress Lizenzen sind als Subscriptions für ein Jahr erhältlich. Die Subscriptions können bei Ablauf mit einer günstigeren Lizenz um ein weiteres Jahr verlängert werden (Renewal).

Jeder Entwickler im Unternehmen **benötigt eine Lizenz**. Beim Kauf mehrerer Lizenzen sind Mengenrabatte möglich.

 - [Reporting](/hersteller/devexpress/reporting/)
 - [Office File API](/hersteller/devexpress/office-file-api/)

</ArrowList>


 Weitere enthaltene Module:

 <ArrowList>

- **CodeRush for Visual Studio**  <br />
Unterstützung durch Codevisualisierung und intuitiver Fehlerkorrektur
- **XPO - ORM Library**  <br />
Erstellt eine Brücke zwischen Objekten unter .NET und relationalen DBs
- **TestCafe**  <br />
Plattformübergreifende IDE um Web-Apps über eine leichtgewichtige Benutzeroberfläche zu testen
- **Dashboards**  <br />
Tool zur Datenvisualisierung unter .NET,Angular, React and Vue
- Quellcode
- Technischer Support

</ArrowList>
