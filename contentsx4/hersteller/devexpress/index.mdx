Auf diesen Seiten finden Sie Informationen zu Lizenzierung und Kosten der DevExpress Produkte.
DevExpress (Developer Express Inc.) entwickelt Werkzeuge, Bibliotheken und UI-komponenten für MS Visual Studio und Embarcadero (Delphi-, C++ Builder) Entwickler.
Tools für WinForms-, ASP.NET-, MVC-, Blazor-, ASP.NET Core-, WPF-, VCL-, Xamarin- und JavaScript.
Diese Informationen richten sich hauptsächlich an IT-Einkäufer und Administratoren.
Fordern Sie mit einem Klick ein kostenloses und unverbindliches Sofortangebot an.

</Intro>

## Produkte & Preise


### MS Visual Studio

<Info type="light">

Werkzeuge, Bibliotheken und UI-Komponenten für Softwareentwickler unter MS Visual Studio. <br />
Siehe dazu: [Vergleich der Produktpakete für Visual Studio](#für-ms-visual-studio)

</Info>

<ArrowList>

- [Universal](/hersteller/devexpress/universal/) <br />
  DevExpress Universal ist das komplette Entwicklungspaket. Das Bundle besteht aus "DXperience" plus Enterprise & Server-Tools (Reporting, Business Intelligence Dashboard, Office File API) und weitere.

- [DXperience](/hersteller/devexpress/dxperience/) <br />
  DevExpress DXperience ist ein Bundle bestehend aus Desktop Controls (WinForms, WPF, Windows 10 Apps) zusammen mit Web Controls (ASP.NET, Bootstrap WebForms, HTML/JS for Angular, etc.), sowie Framework und Produktivitätswerkzeuge (XPO - ORM Library, CodeRush).

- [DevExtreme](/hersteller/devexpress/devextreme/) <br />
  DevExpress DevExtreme ist eine JavaScript Component Suite zur Entwicklung von "responsive" Web-Anwendungen, enthält eine umfassende Sammlung von leistungsstarken UI-Komponenten.

- [Reporting](/hersteller/devexpress/reporting/) <br />
  DevExpress Reporting ist eine leistungsstarke Lösung für Entwickler, die komplexe Datenberichte in Desktop- und Webanwendungen integrieren wollen. Ermöglicht die Gestaltung dynamischer Berichte für verschiedene Plattformen.

- [Office File API](/hersteller/devexpress/office-file-api/) <br />
  DevExpress Office File API ist eine Programmier-Schnittstelle zur Erstellung und Bearbeitung von Dokumenten und Dateien im MS-Office-Format (XLSX, DOCX, PDF und weitere).

- [ASP.NET](/hersteller/devexpress/asp-net/) <br />
  DevExpress ASP.NET Unterstützt die Entwicklung von .NET Anwendungen mit mehr als 110 UI Werkzeugen für WebForms (einschließlich Reporting).

- [WPF](/hersteller/devexpress/wpf/) <br />
  DevExpress WPF ist eine Erweiterung für MS Visual Studio und bietet Zugriff auf mehr als 130 Komponenten und UI Bibliotheken zum Erstellen von Microsoft Office-Anwendungen und Datenanalyse-Software.

- [WinForms](/hersteller/devexpress/winforms/) <br />
  DevExpress WinForms ist eine Erweiterung für MS Visual Studio, die 190 leistungsstarke Steuerelemente und UI-Bibliotheken für die Windows Forms-Plattform enthält (jetzt auch mit Windows 11 Support).

</ArrowList>



### Embarcadero VCL (Delphi-/C++-Builder)

<Info type="light">

Werkzeuge, Bibliotheken und UI-Komponenten für Softwareentwickler unter Delphi-/C++-Builder. <br />
Siehe dazu: [Vergleich der Produktpakete für VCL Embarcadero](#für-embarcadero-vcl-delphi-c-builder)

</Info>

<ArrowList>

- [QuantumPack](/hersteller/devexpress/quantumpack/) <br />
DevExpress QuantumPack bietet Komponenten zur Einbindung flexibler Raster-, Druck-, Layout- und Skinning-Funktionen in Ihre Delphi- und C++Builder Anwendungen.

- [GridPack](/hersteller/devexpress/gridpack/) <br />
DevExpress GridPack beinhaltet **DevExpress QuantumPack** PLUS Pivot Grid und Vertical Grid Steuerelemente.

- [NavigationPack](/hersteller/devexpress/navigationpack/) <br />
DevExpress NavigationPack bietet Steuerelemente für die Navigation (Multifunktionsleisten, Symbolleisten sowie Layout- und Skinning-Funktionen) an.

- [VCL](/hersteller/devexpress/vcl/) <br />
DevExpress VCL ist das kompletteste Entwicklungspaket für Delphi und C++-Builder. Das Bundle beinhaltet alle VCL-Produkte in einem Paket.

</ArrowList>
