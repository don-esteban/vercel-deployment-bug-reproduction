## Lizenzierung

DevExpress Lizenzen sind als Subscriptions für ein Jahr erhältlich. Die Subscriptions können bei Ablauf mit einer günstigeren Lizenz um ein weiteres Jahr verlängert werden (Renewal).

Jeder Entwickler im Unternehmen **benötigt eine Lizenz**. Beim Kauf mehrerer Lizenzen sind Mengenrabatte möglich.



## Funktionen

DevExpress ASP.NET enthält mehr als 110 AJAX-Webformular-Steuerelemente (inklusive Reporting), serverseitige MVC-Erweiterungen oder clientseitige Steuerelemente mit allem, was Sie zum Erstellen von Office-inspirierten Anwendungen benötigen.
Weitere über 70 leistungsstarke ASP.NET Core-UI-Steuerelemente , ASP.NET Bootstrap-Steuerelemente, Blazor-Komponenten (einschließlich Data Grid und Pivot Grid) sind zur Unterstützung serverseitiger (Razor-Komponenten) und clientseitiger (Blazor) Plattformen für eine "responsive Web"-Entwicklung enthalten.


<ArrowList>

- **ASP.NET WebForm Controls** <br />
Entwicklung interaktiver Schnittstellen für das Web, auch auf Touch-Devices
- **ASP.NET MVC Extensions**  <br />

- **ASP.NET Core**  <br />
Grundfunktionen bei Entwicklungen unter .NET
- **ASP.NET Bootstrap Controls**  <br />
Einbindung von Bootstrap CSS für responsive Design

Einbindung von Javascript
- **Reporting**  <br />
Werkzeug zur erweiterten Berichterstellung
