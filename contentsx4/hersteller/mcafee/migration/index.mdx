McAfee hat die traditionellen Business-Produkte mit Ausnahme der CEB bereits 2021 eingestellt. Hier finden Sie eine Übersicht und Vergleichstabelle der alternativen Trellix Produkte und deren Funktionen, um Ihnen den Umstieg zu erleichtern. Gerne können Sie mich auch kontaktieren.

</Intro>

## Wie kann ich migrieren?

Trellix bietet mit dem Trellix Endpoint Upgrade Assistant ein Tool, das Sie beim Upgrade auf die neueste Version von Endpoint Security unterstützt. Dieses Tool bietet auch Unterstützung bei der Migration von anderen Trellix Produkten. Der Trellix Endpoint Upgrade Assistant ist Bestandteil von Trellix ePO 5.10.x (und höher) und kann von der McAfee Produkt Download Website heruntergeladen werden. Auf folgender Webseite wird der Umstieg bzw. Upgrade beschrieben, ebenso die Übernahme vorhandener Sicherheitsrichtlinien: [Upgrades von älteren McAfee-Produkten auf Endpoint Security](https://docs.trellix.com/de-DE/bundle/endpoint-security-10.7.x-installation-guide-windows/page/GUID-BE61F0F4-A0FA-4D74-8C9C-DEAB469A2ED4.html#GUID-BE61F0F4-A0FA-4D74-8C9C-DEAB469A2ED4)


## Welche Produkte sind abgekündigt?

Folgende Produkte sind abgekündigt:

- ETP: Endpoint Threat Protection	 - zum 14.1.2022 eingestellt.
- ETD: Endpoint Threat Defense (AddOn) - zum 14.1.2022 eingestellt.
- EPA: Endpoint Protection Advanced Suite	- zum 14.1.2022 eingestellt.
- CTP: Complete Endpoint Threat Protection - zum 14.1.2022 eingestellt.
- TSH: Endpoint Protection Essential for SMB (TSH) - zum 31.12.2022 eingestellt.
- CWS: Cloud Workload Security (CWS) - Kauflizenzen eingestellt. Support dieser läuft bis 2025.*

Der Support wird nach dem angegebenen Zeitpunkt eingestellt, Bestellungen sind nicht mehr möglich.

_*Cloud Workload Security (CWS) Produkte gibt es weiterhin, diese werden aber nur noch als Subscriptions und nicht mehr als Kauflizenzen (perpetual) vertrieben._

## Welche Produkte bieten sich als Alternative an?

Als Alternative bieten sich die Trellix Enterprise Security Produkte [MVISION](/hersteller/mcafee/mvision/) oder [CEB](/hersteller/mcafee/complete-endpoint-protection-business/) an. Eine Gegenüberstellung der Funktionen der abgekündigten Produkte mit denen der aktuellen alternativen Produkte finden Sie auf dieser Seite in unten folgender Vergleichstabelle.


Siehe dazu auch [Vergleich Trellix Endpoint Security Produkte](/hersteller/mcafee/vergleich-mcafee-endpoint-security-loesungen/). Gerne beraten wir Sie zu den Produkten.
