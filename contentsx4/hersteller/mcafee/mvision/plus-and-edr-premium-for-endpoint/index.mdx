<Intro>

MVISION Protect Plus and EDR Premium for Endpoint (MV7) ist ein Bundle aus MVISION Protect Plus (MV2) und MVISION EDR Premium (MV5). Umfasst Endpoint und Mobile Protection sowie Detection and Response mit bis zu 90 Tage zurückreichender Bedrohungsanalyse.


</Intro>

## Preise

Trellix MVISION Protect Plus and EDR Premium for Endpoint (MV7) ist als Subscription (Abonnement) verfügbar. Lizenziert wird pro Nutzer (Installation auf 5 Endgeräten). Jede virtuelle Instanz oder Server wird als Nutzer gezählt. Die Mindestbestellmenge liegt bei 5 Lizenzen. Business-Support (Updates, Upgrades, Telefon-Support) ist enthalten. Weitere Informationen finden Sie auf unserer [Lizenzierungsseite](/hersteller/mcafee/lizenzierung/).

## Funktionen

Enthalten ist die Managementkonsole **ePolicy Orchestrator (ePO)** für On-Premises, SaaS und für AWS. Zusätzlich enthalten ist proaktive bzw. adaptive Bedrohungserkennung in Echtzeit.
Die Daten von Endgeräte können bis zu 90 Tage zurück auf Bedrohungen überprüft werden, unterstützt durch Assistenten.


- **[Endpoint Security (ENS)](/hersteller/mcafee/endpoint-security/)** <br/>
Für Windows (Adaptive Threat Protection (ATP)), macOS, Linux.
- **[MVISION Endpoint](/hersteller/mcafee/vergleich-mcafee-endpoint-security-loesungen/#trellix-mvision-endpoint)** <br/>
Erweitert die Virenschutz-, Firewall- und Exploit-Schutz-Kontrollen von Windows 10 und bietet so eine solide Endgerätesicherheit.
- **[MVISION Mobile](/hersteller/mcafee/vergleich-mcafee-endpoint-security-loesungen/#mvision-mobile)** <br/>
Schutz der mobilen Geräte unter iOS und Android.
- **[Device Control](/hersteller/mcafee/vergleich-mcafee-endpoint-security-loesungen/#device-control)** <br/>
Vollständige Kontrolle, für welche Geräte der Zugang gesperrt bzw. gewährt werden soll.
- **Application Control** <br/>
PC-Geräteschutz zentral verwalten und Zugang für das Deployment von Strategic Innovation Alliance (SIA).
- **[MVISION ePO](/hersteller/mcafee/epolicy-orchestrator-epo-faq/)** <br/>
Zentrale Verwaltung und Übersicht als Software-as-a-Service-Lösung (SaaS).
- **[MVISION EDR Premium](/hersteller/mcafee/mvision/edr-premium/)** <br/>
Erweitert die Virenschutzfunktionen für Endgeräte.
- **MITRE ATT&CK** <br/>
Eine weltweit zugängliche Wissensdatenbank der Taktiken und Techniken von potenziellen Angriffen, die auf realen Beobachtungen basieren.
- **Ad-hoc-Abfragen auf Endpunkten** <br/>
Die Daten von Endgeräte können bis zu 90 Tage zurück auf Bedrohungen überprüft werden, unterstützt durch Assistenten.
- **KI-unterstützte Untersuchung** <br/>
Analyse auf schädliche Verhaltensmuster und Bedrohungen, sammelt relevante Daten und Nachweise.
