<Intro>

Trellix MVISION Protect Standard (MV1) ist das MVISION Basispaket für Windows-, macOS- und Linux-Umgebungen. Neben den Basisfunktionen zum Schutz Ihrer Endgeräte sind eine zentrale Managementkonsole sowie Device Control und Data Exchange Layer (DXL) enthalten.

</Intro>

<TOC />


## Preise

Trellix MVISION Protect Standard (MV1) ist als Subscription (Abonnement) verfügbar. Lizenziert wird pro Nutzer (Installation auf 5 Endgeräten). Jede virtuelle Instanz oder Server wird als Nutzer gezählt. Die Mindestbestellmenge liegt bei 5 Lizenzen. Business-Support (Updates, Upgrades, Telefon-Support) ist enthalten. Weitere Informationen finden Sie auf unserer [Lizenzierungsseite](/hersteller/mcafee/lizenzierung/).

## Funktionen


- **[MVISION ePO (SaaS)](/hersteller/mcafee/vergleich-mcafee-endpoint-security-loesungen/#trellix-epolicy-orchestrator-cloud-epo-cloud)** <br/>
Zentrale Verwaltung und Übersicht als Software-as-a-Service-Lösung (SaaS).
- **[Trellix ePolicy Orchestrator (ePO)](/hersteller/mcafee/vergleich-mcafee-endpoint-security-loesungen/#trellix-epolicy-orchestrator-epo)** <br/>
Zentrale Verwaltung und Übersicht für On-Premises und Amazon Web Services (AWS).
- **[MVISION Endpoint](/hersteller/mcafee/vergleich-mcafee-endpoint-security-loesungen/#trellix-mvision-endpoint)** <br/>
Erweitert die Virenschutz-, Firewall- und Exploit-Schutz-Kontrollen von Windows 10 und bietet so eine solide Endgerätesicherheit.
- **[Endpoint Security (ENS)](/hersteller/mcafee/endpoint-security/)** <br/>
Echtzeitanalyse und -schutz für Windows (Adaptive Threat Protection (ATP)), macOS, Linux.
- **Data Exchange Layer (DXL)** <br/>
Sicherer Echtzeitzugang zu neuen Daten und Ressourcen schonende, verzögerungsfreie Interaktionen mit anderen Produkten (Third-Party).
