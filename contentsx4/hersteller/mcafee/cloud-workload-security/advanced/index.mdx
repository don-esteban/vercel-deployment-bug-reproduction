## Preise

Cloud Workload Security (CWS) Advanced ist als Subscription verfügbar. Lizenziert wird pro Betriebssystem-Instanz (OS-Instance). Subscriptions enthalten immer 1 Jahr Business-Support (Updates, Upgrades, Telefon-Support). Um weiterhin die Software nutzen zu können, muss die Subscription vor Ablauf verlängert werden.
