<Intro>

Trellix Cloud Workload Security (CWS) bietet umfassenden Schutz für virtuelle und physische IT, Netzwerke, Server und hybride IT-Strukturen. CWS automatisiert die Erkennung und Abwehr elastischer Workloads und Container, beseitigt Schwachstellen, bietet erweiterte Bedrohungsabwehr und vereinfacht das Multi-Cloud-Management. Auf dieser Seite finden Sie die einzelnen Cloud Workload Security Editionen und weiterführende Information.

</Intro>

<TOC />

## Editionen

<ArrowList>

- [Trellix Cloud Workload Security Basic](/hersteller/mcafee/cloud-workload-security/basic/) <br/>
Enthält die Verwaltungskonsole ePO, Endpoint Security (ENS) für Server, Firewall Module und VirusScan Enterprise.
- [Trellix Cloud Workload Security Essentials](/hersteller/mcafee/cloud-workload-security/essentials/) <br/>
Wie Cloud Workload Security Basic, jedoch mit erweiterte Netzwerkssicherheit und zusätzliche KI Funktionen.
- [Trellix Cloud Workload Security Advanced](/hersteller/mcafee/cloud-workload-security/advanced/): <br/>
Wie Cloud Workload Security Essentials, jedoch mit Audit-Protokollierung, Datei- und Ordnerschutz sowie Kontrollfunktionen über Anwendungen.

</ArrowList>

<Info>

Der Support der perpetual Lizenzen läuft noch bis Ende 2025. Neubestellungen sind nicht mehr möglich. Danach empfiehlt es sich auf die Subscription umzusteigen. Gerne können Sie mich bei Fragen kontaktieren.

</Info>
