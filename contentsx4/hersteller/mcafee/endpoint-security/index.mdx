Trellix Endpoint Security (ENS) bzw. McAfee Endpoint Security ist der zentrale Endgeräteschutz für Desktops, Notebooks und Server. Als Modul ist ENS in den meisten Trellix Produkten enthalten. ## Was ist ENS?

Endpoint Security oder auch ENS, ist ein moderner Ansatz zur Absicherung Ihrer Endgeräte, egal ob unter Windows, MacOS, Linux. Trellix (McAfee) Endpoint Security fasst ältere Einzelprodukte in den Modulen Bedrohungsschutz, Webkontrolle und Firewall zusammen. Die gemeinsame Architektur bietet den Vorteil, dass die Schutzmodule besser interagieren, um die Sicherheit zu verbessern.

 **Funktionen:**
 - Signaturlosen Schutz vor Viren, Exploit-Schutz, Firewall
 - beobachtet und regiert auf verdächtige Kommunikationsmuster, Verhaltensanalyse durch selbstlernende Algorithmen
 - schützt vor Bedrohungen durch unsicheren Web-Seiten und schädlichen Downloads

## Wie wird ENS lizenziert?

Trellix Endpoint Security (ENS) (bzw. ein Produkt mit ENS) wird pro Knoten (Node) oder Benutzer lizenziert. Ein Node ist ein Endpunkt (physikalische oder virtuelle Recheneinheit), der mit ENS geschützt wird. Dazu gehören Server, Notebooks, virtuelle Desktops, Datenbankserver oder Storage Devices. Bei der Benutzerlizenzierung können pro Benutzer 5 Endgeräte geschützt werden. Mehr zur Lizenzierung finden Sie auf unserer [Trellix Lizenzierungsseite](/hersteller/mcafee/lizenzierung/).


## Wie wird ENS installiert und aktualisiert?

Trellix Endpoint Security (ENS) kann vor Ort auf dem zu schützenden Gerät installiert werden. Zunächst muss die Vorarbversion deinstalliert werden, danach kann die neue Produktversion installiert werden.

Der einfachere Weg führt jedoch über den Trellix ePO (ePolicy Orchestrator), sprich zentral verteilen, konfigurieren, aktualisieren und überwachen. Vor der Bereitstellung muss jedoch ein manuelles Upgrade vom Tool **Trellix Agent** durchgeführt werden. Trellix ePO ist in den meisten Trellix Produkten bereits enthalten und muss nicht separat gekauft werden. Dazu mehr in unserem [Trellix Produktvergleich](/hersteller/mcafee/vergleich-mcafee-endpoint-security-loesungen/).
