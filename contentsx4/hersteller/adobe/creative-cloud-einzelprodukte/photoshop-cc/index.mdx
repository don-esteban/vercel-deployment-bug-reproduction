## Lizenzierung

Adobe PhotoShop CC ist eine Abonnement-Lizenz im [Adobe Value Incentive Plan (VIP)](/hersteller/adobe/lizenzierung/#adobe-value-incentive-plan-vip).

Die Software wird pro Benutzer lizenziert. Behörden und Bildungseinrichtungen (Education) erhalten Sonderkonditionen.

### Value Incentive Plan (VIP)

 Der Adobe Value Incentive Plan (VIP) ist das Lizenzprogramm für Unternehmen, Behörden und Bildungseinrichtungen. Der Preis für eine Lizenz ist abhängig von der Rabattstaffel. Level 1-9 ist der Preis pro Lizenz bis zu einer Lizenzanzahl von 9. Preise für Bildungseinrichtungen erhalten Sie auf Anfrage.


## Alternative Adobe Creative Cloud

Wenn Sie mehrere Produkte aus der Creative Cloud einsetzen möchten, bieten sich die Unternehmenslösungen [Creative Cloud for Teams](/hersteller/adobe/creative-cloud-for-teams/) und [Creative Cloud for Enterprise](/hersteller/adobe/creative-cloud-for-enterprise/) an.
Mit diesen Bundles erhalten Sie alle Produkte aus der Creative Cloud zum günstigen Bundle-Preis.
