<Intro>

Mit InstallShield können MSI(X)- und EXE-Installer sowie komplette MSIX-Projekte erstellt, modifiziert oder migriert werden. UWP- und WSA-Pakete für Windows-Anwendungen werden unterstützt. Neben dem Support für AWS Cloud werden auch Cloud Builds auf Cloud License Server (CLS) unterstützt.

</Intro>

## Preise

InstallShield ist als Subscription für 3 Jahre verfügbar. Lizenziert wird nach [**Node-Locked**](/hersteller/flexera/#lizenzierung) und ist an einen benannten Nutzer gebunden.
