<Intro>

Mit InstallAnywhere können Multiplattform-Installationen aus einer einzigen Projektdatei erstellt werden. Es werden physische, cloud-basierte und virtuelle Umgebungen, sowie Docker-Container unterstützt. Neu mit InstallAnywhere 2023 ist die Unterstützung von macOS Sonoma sowie File Association in Windows.


</Intro>

## Lizenzierung

InstallAnywhere ist als Subscription für 3 Jahre verfügbar. Lizenziert wird nach [**Node-Locked**](/hersteller/flexera/#lizenzierung) und ist an einen benannten Nutzer gebunden.

## Funktionen

- Erstellt Java-basierte Installationen für Multiplattformanwendungen, einschließlich Windows 11, macOS Big Sur, Sonoma und für Apple M1
- Silent- (Hintergrund-), Konsolen- und Benutzeroberflächeninstallationen
- Verwaltet Regeln und Abhängigkeiten (Automatisiert Prozesse)
- Installiert Abhängigkeiten von Linux-Packagemanagern
- Erfüllt die aktuellen Zertifikatanforderungen von Microsoft
- Installiert Webapps in Tomcat und WebSphere
- Erstellt Installationen, die sich mit IBM DB2, [Microsoft SQL](/hersteller/microsoft/sql-server/), MySQL und [Oracle](/hersteller/oracle/) verbinden lassen
- Definieren von Apps für Dateizuordnungen in Windows
- Aktion zur Aktivierung der 8.3-Namenskonvention in Windows
- Neues Dienstprogramm zur Automatisierung der Notarisierung von Installationsprogrammen in macOS
- Möglichkeit zur Definition von Pflichtfeldern im Fenster „Benutzereingabe abrufen“
- Unterstützung von [Red Hat Enterprise Linux 8 (RHEL)](/hersteller/red-hat/), macOS und Amazon Corretto OpenJDK
