**VM Backup** wird als [Perpetual](#vm-backup-perpetual-preise) (Lizenzierung per Host) in drei Editionen *(Standard, Unlimited und Unlimited Plus)* oder als [Subscription](#vm-backup-subscription-preise) (Lizenzierung per VM, nur als *Unlimited Plus*) angeboten.

Einen Vergleich der Editionen finden Sie unter: [Vergleich VM Backup Editionen](#vm-backup-editionenvergleich).





## VM Backup Perpetual Preise

Sicherung von Hyper-V oder VMware VM *(früher Altaro VM Backup for Mixed Environments)*.
Preise inklusive 1 Jahr SMA. **Lizenzierung pro Host**.

<Info>

VM Backup SMA verlängert sich automatisch um die vereinbarte ursprüngliche Laufzeit, wenn diese nicht 3 Monate vor Ablauf gekündigt wird.

</Info>


### Standard Edition

Unterstützt *je Host* maximal fünf (5) zu sichernde VMs pro Lizenz.

## VM Backup Subscription Preise

Für Hyper-V oder VMware, je zu sichernde VM eine Lizenz, unabhängig vom Host. 
Funktionsumfang ist identisch mit VM Backup Unlimited Edition (siehe auch [Vergleich VM Backup Editionen](#vm-backup-editionenvergleich)).
Mindestmenge bei Erstbestellung: fünf (5) Lizenzen.

## Produktfamilie Informationen

Sie können VM Backup als Dauerlizenz/Kauflizenz oder als Subscription lizenzieren.

  **[VM Backup Perpetual](#vm-backup-perpetual-preise/)**: <br />
  Sichert VMware und Hyper-V VMs und Host. Kauflizenz als Standard, Unlimited und Unlimited Plus Edition. Lizenzierung pro Host.

  **[VM Backup Subscription](#vm-backup-subscription-preise/)**:  <br />
  Sichert VMware, Hyper-V VMs, Applikationen und Host. Subscription (Unlimited Plus Edition). Lizenzierung pro VM.


Hornetsecuritys VM Backup ist eine leistungsstarke und einfach zu bedienende Backup- und Replikationslösung
für virtuelle Maschinen (VMs) von [Microsoft Hyper-V](/hersteller/microsoft/windows-server/) und [VMware vSphere](/hersteller/vmware/) sowie physische Windows-Server.
Sichert [Microsoft Exchange Server](/hersteller/microsoft/exchange/)- und [Microsoft SQL Server](/hersteller/microsoft/sql-server/)-VMs und andere Anwendungen, die mit dem Microsoft VSS Writer kompatibel sind.
