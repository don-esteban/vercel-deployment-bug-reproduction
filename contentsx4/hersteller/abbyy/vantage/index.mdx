<Intro>

ABBYY Vantage ist mehr als ein OCR-Server. Die neue KI-Engine erkennt und versteht Dokumenteninhalte. Sie extrahiert selbstständig Felder und Informationen aus unstrukturierten Dokumenten. So können bspw. unterschiedlichste Rechnungen oder Frachtpapiere gescannt und automatisch weiterverarbeitet werden. Beispielsweise durch Zuführung der extrahierten Daten an Ihr Buchhaltungs- und ERP-System.

</Intro>

## Deployment

ABBYY Vantage wird entweder selbst installiert (lokal, virtuell, Cloud) oder ohne eigene Installation als Managed Service aus der ABBYY Cloud bezogen. Eine Lizenz (Subscription) gilt für beide Nutzungsmöglichkeiten. Sie können nach dem Kauf entscheiden, ob Sie selbst installieren oder den Cloud Service nutzen.

## Was sind Skills?

Skills sind digitale Mitarbeiter. Sie können Dokumente lesen und relevante Informationen extrahieren. Wie menschliche Mitarbeiter werden Skills mit der Zeit immer besser und effizienter. Sie lernen bei Ihrer täglichen Arbeit.

ABBYY beschreibt Skills so:

> Dank Vantage-Skills können intelligente Automatisierungsplattformen, digitale Mitarbeiter und Automatisierungsroboter Geschäftsdokumente und -inhalte auf menschenähnliche Weise intelligent verarbeiten – um sie zu verstehen, Bedeutungen herauszulesen und Entscheidungen zu treffen.

> Ein Skill bietet einen No-Code-Entwurfsansatz, der es Laienentwicklern ermöglicht, Dokumentenmodelle anzulernen und Dokumentenprozesse für alle Dokumentarten zu automatisieren.

## Was sind Core Cognitive Skills?

ABBYY bietet einige vortrainierte Skill. Bspw. einen Skill für die automatische Verarbeitung von Rechnungen. Sie müssen jedoch keine vortrainierten Skills kaufen. ABBYY Vantage Cognitive Skills enthält alles, um eigene Skills für Ihre individuellen Dokumente zu entwickeln.

Angenommen, Sie erhalten von Ihren Monteuren Reparaturberichte in Papierform. Core Cognitive Skills ermöglicht Ihnen einen eigenen, speziellen Skill zu entwickeln, der diese Dokumente automatisch liest (OCR Skill ist in Core Cognitive Skills enthalten) und die Daten in Ihre IT-Systeme einpflegt.
