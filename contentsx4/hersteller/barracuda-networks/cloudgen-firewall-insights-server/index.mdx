<Intro>

CloudGen Firewall Insights Server konsolidiert Informationen zu Sicherheit, Datenfluss der Applikationen und Konnektivität der Firewalls, unabhängig der Betriebsart (Hardware, Virtual, Cloud).

Wird als VM mit Unterstützung für VMware, Hyper-V und KVM bereitgestellt. Eine angepasste Version für Proxmox ist auch erhältlich.

</Intro>

<TOC />



## Preise

CloudGen Firewall Insights Server wird als Subscription (Abonnement) zur Miete angeboten. Insights Server Base-Lizenz wird benötigt. Diese wird Ihnen auf Anfrage, bei Buchung, kostenlos gestellt. Support und Wartung sind inklusive.  Mindestlaufzeit bei Neukauf sind 12 Monate. Andere Laufzeiten (mind. aber 12 Monate) können angefragt werden. Verlängerungen (Renewals) müssen vor Ablauf der Subscription veranlasst werden. Eine nachträgliche Buchung ist nicht immer möglich.

## Funktionen


- Konsolidiert Informationen aller im Unternehmensnetzwerk vorhandenen Cloudgen Firewalls
- Bietet erweiterte Bericht- und Analysefunktionen
- Statusübersicht über Details, Datenverkehr, Webaktivitäten von WAN-Verbindungen
- Übersichtliche Dashboards
