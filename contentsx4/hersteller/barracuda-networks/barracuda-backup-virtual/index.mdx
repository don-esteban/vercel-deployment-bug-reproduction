<Intro>

Barracuda Backup Virtual, virtuelle Appliance für VMware ESXi und Microsoft Hyper-V. Datensicherung in die Barracuda Cloud wird unterstützt. Schnelle (granulare) Wiederherstellung. Integraler Bestandteil von Barracuda Backup ist eine Cloud- und Site-to-Site-Replizierung. Vorhandene Computer- und Storage-Infrastruktur kann genutzt werden.

Auch erhältlich als Backup Hardware Appliance.

</Intro>

<TOC />



## Preise


Barracuda Backup Virtual wird als Subscription (Abonnement) zur Miete angeboten. Dazu gibt es Support- und Add-on-Subscriptions (Abonnements). Lizenziert als Per-Socket-Subscription für virtuelle Hosts. Lizenziert als Per-Server-Subscription für physische Server. Verlängerungen (Renewals) müssen vor Ablauf der Subscription veranlasst werden. Eine nachträgliche Buchung ist nicht immer möglich.

## Support

Patches, Updates sowie Upgrades sind bereits in der Base Subscription inklusive. Dazu kann man Premium Support (dedizierter Ansprechpartner) buchen.


### Premium Support

Mit [Premium Support](https://de.barracuda.com/support/plans-and-packages) haben Sie einen dedizierten Ansprechpartner und Supportfälle werden priorisiert.


## Funktionen


- Integrierte Inline-Deduplizierung und Kompression
- Integrierte Cloud-Replikation
- Site-to-Site-Replikation
- LiveBoot und Cloud LiveBoot
- Granulare Wiederherstellung von Dateien und Nachrichten
- Einheitliche Cloud Management Plattform
- Keine Gebühren per Feature oder per Applikation
- Cloud-to-Cloud Backup (optional)
- Keine zusätzlichen Agenten benötigt
- Unbegrenzter Barracuda Cloud Storage
- Erhöhte Sicherheit durch Verschlüsselung der Daten und der Kommunikation
- Flexible Bereitstellung
