<Intro>

Web Security Gateway Virtual, virtuelle Appliance zur Filterung von Content, Malware, Social Media und Bedrohungen, beispielsweise durch gefährliche Anwendungen bzw. Apps. Durchsetzung von Richtlinien für Web-Inhalte und Zugangsregeln für Benutzer im Netzwerk, aber auch für Benutzer von außen. Nur als Subscription verfügbar.

Auch als Hardware Appliance erhältlich.

## Add-ons

Für manche Ausführungen sind Add-ons verfügbar. Die Add-ons zum Kauf finden Sie unter dem jeweiligen Gateway-Modell. Gerne können Sie mich bei Fragen kontaktieren.


### Advanced Threat Protection

Advanced Threat Protection (ATP) kombiniert Verhaltens-, Heuristik- und Sandboxing-Technologien, um vor gezielten Zero-Hour Attacken zu schützen. Dateien werden automatisch in Echtzeit gescannt und auffällige Dateien werden in einer Sandbox-Umgebung auf ihre Aktivität überprüft.


### Extended Malware Protection

Extended Malware Protection für dynamische Bedrohungsanalyse in Echtzeit, Sandboxing und URL-Schutz um zu verhindern, dass E-Mail Nutzer von Malware beeinträchtigt werden.


### Additional Core License

Abhängig von der Modellnummer sind sie auch durch die Anzahl der CPU-Kerne begrenzt, die verwendet werden können. Sie müssen Ihrem Web Security Gateway die richtige CPU-Anzahl zuweisen. Wenn Sie mehr CPU-Kerne zuweisen, als von der Lizenz abgedeckt werden, wird der Lizenzstatus als abgelaufen angezeigt. Dafür gibt es die Additional Core License. Eine Lizenz deckt ein Kern (Core) ab. Falls Sie [Premium Support](#premium-support) haben, benötigen Sie auch für die Additional Core Lizenz die Additional Core Premium Support Subscription.


## Funktionen


- Web-Anwendungen und Social Media <br />
Kontrolle über Web 2.0-Sites und Webanwendungen, einschließlich Social-Media-Plattformen. Das Scannen und Prüfen von SSL-verschlüsseltem Traffic auf bestimmte Kategorien und Domänen ermöglicht eine granulare Durchsetzung von Richtlinien.
- AntiViren-Schutz <br />
Scannt durchgehend Kommunikation und eingehende Dateien. Zudem dekomprimiert die Lösung Archive, um vollständigen Schutz zu gewährleisten. Wird automatisch aktualisiert, um immer den aktuellsten Schutz vor Viren in E-Mails zu gewährleisten.
- Application Control <br />
Analyse durch Deep-Packet-Inspection der Ports und Protokolle.
- Inhaltsfilterung sowie -sperrungen durch Richtlinien, auch nach Zeiträumen
- Zentrale, Cloud-basierte Verwaltung <br />
Integriert im webbasierte Management-Portal [Barracuda Cloud Control (BCC)](https://campus.barracuda.com/product/cloudcontrol/doc/69960173/overview). Überwachung der Infrastruktur und Durchsetzung von Richtlinien.
- Umfassende Reportings, Warnmeldungen und Erkennung verdächtiger Aktivitäten
- Benutzerfreundliche Konfiguration
