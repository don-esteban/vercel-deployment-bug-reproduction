<Intro>

Message Archiver Virtual, virtuelle Appliance für E-Mail-Archivierung nach eDiscovery und Compliance-Richtlinien. Das **Outlook Add-In** ermöglicht Benutzern eine kombinierte Mailbox- und Archivsuche. Barracuda PST Enterprise bietet Administratoren die Möglichkeit, PST-Dateien zu entdecken und ggf. zu eliminieren.

Auch als Hardware Appliance verfügbar.

</Intro>

## Add-ons

Für manche Ausführungen sind Add-ons verfügbar. Die Add-ons zum Kauf finden Sie unter dem jeweiligen Gateway-Modell. Gerne können Sie mich bei Fragen kontaktieren.


### Advanced Threat Protection

[Advanced Threat Protection (ATP)](https://de.barracuda.com/solutions/advanced-threat-protection) kombiniert Verhaltens-, Heuristik- und Sandboxing-Technologien um vor gezielten Zero-Hour Attacken zu schützen. Dateien werden automatisch in Echtzeit gescannt und auffällige Dateien werden in einer Sandbox-Umgebung auf ihre Aktivität überprüft.


### Extended Malware Protection

Extended Malware Protection für dynamische Bedrohungsanalyse in Echtzeit, Sandboxing und URL-Schutz um zu verhindern, dass E-Mail Nutzer von Malware beeinträchtigt werden.

### Barracuda Enterprise PST - Account

[Barracuda PST Enterprise for Message Archiver](https://campus.barracuda.com/product/messagearchiver/doc/44436716/barracuda-pst-enterprise-deployment/) erleichtert Unternehmen das Suchen, Archivieren und Managen der Daten in PST-Dateien.​​ Weitere [Infos zu PST-Dateien](https://app.barracuda.com/resources/Barracuda_PST_Enterprise_WP_Guide_to_PST_Files_US)

### Mirrored Cloud Storage

Mit [Cloud Storage](https://campus.barracuda.com/product/messagearchiver/doc/41107099/understanding-cloud-storage/) können Sie Daten vom Barracuda Message Archiver in die Barracuda Cloud spiegeln. Sobald Sie Ihren Barracuda Message Archiver mit einem Barracuda Cloud Control Konto verknüpft haben, können Sie alle neuen und vorhandenen E-Mails spiegeln und diese E-Mails dann sowohl über die Barracuda Message Archiver -Weboberfläche als auch über Barracuda Cloud Control suchen, exportieren oder weiterleiten.


## Funktionen


- E-Mail-Archivierung von jedem standardmäßigen E-Mail-Server oder -Service, wie [Office 365](/hersteller/microsoft/office-365/), [Exchange-](/hersteller/microsoft/exchange/), [Domino-Server](/hersteller/hcl/lotus-domino-notes/), G-Suite und andere über IMAP oder POP3.
- Instant-Messaging (IM)-Archivierung von [Microsoft Teams](/hersteller/microsoft/teams/) oder [Skype for Business](/hersteller/microsoft/skype-for-business/).
- Datenimport von [Exchange](/hersteller/microsoft/exchange/) und [Office 365](/hersteller/microsoft/office-365/), andere über PST-Dateien.
- Archivierung von Termin- und Kontaktdaten, so wie Aufgaben und Notizen aus [Exchange](/hersteller/microsoft/exchange/) oder [Office 365](/hersteller/microsoft/office-365/).
- Einstellbare E-Mail-Aufbewahrungsrichtlinien und gesetzlicher Aufbewahrungsfristen
- Tagging von Nachrichten, diese können zur einfachen Identifizierung mit Tags versehen werden.
- Archiv der Daten in Branchenstandardformaten. Unternehmen können ihre archivierten Daten bei Bedarf einfach in das PST- oder EML-Format extrahieren und exportieren.
- Zentrale, Cloud-basierte Verwaltung - Integriert im webbasierte Management-Portal [Barracuda Cloud Control (BCC)](https://campus.barracuda.com/product/cloudcontrol/doc/69960173/overview). Überwachung der Infrastruktur und Durchsetzung von Richtlinien.
- Umfassende Reportings, Warnmeldungen und Erkennung verdächtiger Aktivitäten
- Benutzermanagement - lässt sich in LDAP-Services wie Active Directory und Azure AD integrieren.
- Sicherheit durch rollenbasierte Zugriffskontrollen.
- Integrierte Sicherheitsfunktionen, wie manipulationssicherer Speicher oder Verschlüsselung.
- Weitere Informationen finden Sie unter [Barracuda Message Archiver Produktmerkmale](https://de.barracuda.com/products/email-protection/message-archiver/features#paranav-navbar).
