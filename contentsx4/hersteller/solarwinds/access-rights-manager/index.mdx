## Lizenzierung

<ArrowList>

- **Mindestanforderung: ** [Microsoft Windows Server 2012 oder höher](/hersteller/microsoft/windows-server/) und [SQL Server 2012 oder höher](/hersteller/microsoft/sql-server/).
- Lizenziert wird nach **Active Directory Benutzern**.
- Als **Kauflizenz (perpetual)** oder **Subscription** verfügbar.
- Neulizenzen beinhalten bereits 1 Jahr Wartung und Support.
- Jahreslizenzen (Wartung) beinhalten kostenlose Updates und Upgrades, sowie kostenlosen Support.

</ArrowList>


## Funktionen

<ArrowList>

- Überwacht die Konfiguration von Zugriffsrechten und beschleunigt die Änderung von Berechtigungen
- Erkennt ungewöhnliche Aktivitäten bei privilegierten Konten.
- Einfachere Active Directory Bereitstellung
- Automatisierte [Microsoft SharePoint](/hersteller/microsoft/sharepoint/) Bereitstellung
- Werkzeuge zur Berichterstellung aus Active Directory
- IP-Anfrageformular (IP Request Form) zur Optimierung und Automatisierung von IP-Adressanforderungen
- Erstellen von Sicherheitsaudit-Berichten für Compliance mit HIPAA, DSGVO, PCI DSS und weiteren Vorschriften.

</ArrowList>
