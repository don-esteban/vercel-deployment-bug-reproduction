## Lizenzierung

<ArrowList>

- Mindestanforderung: Windows Server 2012 R2 oder höher.
- Lizenziert wird nach Technikern/Administratoren/Benutzer (Technician Licence), ohne weitere Gebühren für Clients.
- Nur als Kauflizenz (perpetual) verfügbar.
- Neulizenzen beinhalten bereits 1 Jahr Wartung und Support.
- Jahreslizenzen (Wartung) beinhalten kostenlose Updates und Upgrades, sowie kostenlosen Support.

</ArrowList>



## Funktionen

<ArrowList>

- Benutzerdefiniertes (eigenes) Branding
- Massenbereitstellung von Agenten auf den Geräten
- Integration mit Help Desk Software
- Benutzerfreundliche Remotesteuerung für Windows, Linux und macOS (Mac OS X)
- Durchführen von Dateiübertragungen in Remotesitzungen
- Detaillierte Berichterstattung möglich, Screenshots von Remotecomputern mit einem Klick in Remotesitzungen
- Leistungsstarke Remotesteuerungs- und Desktopfreigabefunktionen, Verschlüsselungsprotokolle, zweistufige Authentifizierung und Berechtigungen auf mehreren Ebenen
- Remoteverbindung mit inaktiven oder ausgeschalteten Systemen bei unterstützter Hardware
- Live-Chat mit Clients
- Interaktive Smartcard-Anmeldung und Smartcard-Remoteauthentifizierung
- Einfache Verwaltung von Benutzerzugriff und Berechtigungen, Zentralisierte Administration und Kontenverwaltung, Einfache Lizenzierung und schnelle Bereitstellung
- Active Directory Authentifizierung und einmalige Anmeldung

</ArrowList>
