## Lizenzierung

<ArrowList>

- *Mindestanforderung:* [SolarWinds Network Performance Monitor (NPM)](/hersteller/solarwinds/network-performance-monitor/) unter [Windows Server 2016 oder höher](/hersteller/microsoft/windows-server/).
- Lizenziert wird nach der jeweiligen [SolarWinds Network Performance Monitor Lizenz](/hersteller/solarwinds/network-performance-monitor/).
- Nur als Kauflizenz (perpetual) verfügbar.
- Neulizenzen beinhalten bereits 1 Jahr Wartung und Support.
- Jahreslizenzen (Wartung) beinhalten kostenlose Updates und Upgrades, sowie kostenlosen Support.

</ArrowList>


## Funktionen

<ArrowList>

- Überwachung der Bandbreite
- Warnungen zum Anwendungsdatenverkehr
- Netzwerkverkehrsanalyse
- Unterstützung für VMware vSphere Distributed Switch
- Leistungsanalyse-Dashboard
- Erweiterte NBAR2-Anwendungserkennung
- Durch das Cisco NetFlow Protokoll lassen sich Daten u.a. aus Routern auslesen

</ArrowList>
