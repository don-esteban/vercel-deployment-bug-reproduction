<Intro>

SolarWinds bietet professionelle Werkzeuge zur Verwaltung und Überwachung von lokalen, hybriden oder cloudbasierten Datenbanken, Netzwerken, Systemen und ganzen IT-Infrastrukturen.
Insbesondere in den Bereichen IT-Security, Netzwerk-, System- und Database-Management und IT Helpdesk werden individuelle Lösungen angeboten.

</Intro>

## Produkte

<ArrowList>

- [Database Performance Analyzer (DPA)](/hersteller/solarwinds/database-performance-analyzer/)<br />
 Agentenfreie SQL-Datenbank-Verwaltungssoftware zur Plattform übergreifenden Überwachung, Analyse und Optimierung.
 *Kauflizenz oder Abonnement*

- [Dameware](/hersteller/solarwinds/dameware/) <br />
  Remote-Verwaltung und -Steuerung von Windows-PCs und Windows-Servern, auch von iOS und Android aus. *Kauflizenz oder Abonnement*

- [Access Rights Manager (ARM)](/hersteller/solarwinds/access-rights-manager/)  <br />
  Verwaltung von Zugriffsberechtigungen über die gesamte IT-Infrastruktur hinweg. Für Windows- und SQL-Server. *Kauflizenz oder Abonnement*

- [Server & Application Monitor (SAM)](/hersteller/solarwinds/server-application-monitor/) <br />
  Tool zur Überwachung von Anwendungen und Diensten in lokaler, Cloud- oder hybriden IT-Umgebung. Für Windows-Server, Stand-alone oder als Modul für NPM. *Kauflizenz oder Abonnement*

- [Kiwi CatTools](/hersteller/solarwinds/kiwi-cattools/) <br />
  Verwaltung von Netzwerkkonfiguration und Backups vom Desktop aus. Für Windows Server und Windows Desktop (ab Win10). *Kauflizenz*

- [Kiwi Syslog Server](/hersteller/solarwinds/kiwi-syslog-server/) <br />
  Verwaltung von Syslog und SNMP Daten. Automatisierte Reaktion auf Ereignissen, scriptfähig. *Kauflizenz*

- [Network Performance Monitor (NPM)](/hersteller/solarwinds/network-performance-monitor/) <br />
  Netzwerkverwaltung und -überwachung für Windows Server von lokalen, Hybrid- und Cloud-Diensten, mit zentraler Webkonsole. *Kauflizenz oder Abonnement*

- [Network Configuration Manager (NCM)](/hersteller/solarwinds/network-configuration-monitor/) <br />
  Zentrale Verwaltung für Windows Server von Netzwerkgeräten verschiedener Hersteller. Stand-alone oder als Modul für NPM. *Kauflizenz oder Abonnement*

- [IP Adress Manager (IPAM)](/hersteller/solarwinds/ip-address-manager/) <br />
  IP-Adressenverwaltung für Windows Server. *Kauflizenz*

 - [Network Topology Mapper](/hersteller/solarwinds/network-topology-mapper/) <br />
   Automatische Abbildung des Netzwerks als Netzwerkdiagramm. Für Windows Server. *Kauflizenz*

- [Netflow Traffic Analyzer (NTA)](/hersteller/solarwinds/netflow-traffic-analyzer/) <br />
  Detailliertere Einblicke in die Netzwerkauslastung. Als Modul für NPM. Für Windows Server. *Kauflizenz oder Abonnement*

</ArrowList>
