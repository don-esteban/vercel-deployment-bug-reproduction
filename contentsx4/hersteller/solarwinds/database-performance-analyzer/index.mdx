<Intro>

Database Performance Analyzer (DPA) ist eine agentenfreie Datenbank-Verwaltungssoftware zur Plattform übergreifende Überwachung, Analyse und Optimierung von physischen, virtuellen und cloudbasierten Datenbanken, inklusive Azure.
DPA bietet umfangreiche Analysemöglichkeiten mit auf Machine Learning basierender Anomalieerkennung in Echtzeit und rückblickend.

Mit einem Tool überwachen Sie MS SQL Server, Oracle, MySQL, IBM DB2, SAP ASE und Cloud (Azure) Datenbanken.

</Intro>

## Lizenzierung


SolarWinds DPA erfordert je eine Lizenz pro überwachter Datenbankinstanz.


<ArrowList>


- Lizenziert wird pro überwachte Datenbankinstanz, d.h. für jede DB-Instanz muss eine eigene Lizenz erworben werden.
- Anforderungen: [Windows Server 2016 oder aktueller](/hersteller/microsoft/windows-server/) oder Linux-Server (zB: [RHEL](/hersteller/red-hat/enterprise-linux-server/) oder [SLES](/hersteller/suse/linux-enterprise-server/)), SQL-DB als Repository.
- SQL-Datenbank als Repository: [SQL Server](/hersteller/microsoft/sql-server/), [Oracle DB Standard](/hersteller/oracle/database-standard-edition/) oder [Oracle DB Enterprise](/hersteller/oracle/database-enterprise-edition/), MySQL.
- Als Kauflizenz (perpetual) und Subscription (Abonnement)
- Neulizenzen beinhalten bereits 1 Jahr Wartung.
- Jahreslizenzen (Wartung) beinhalten kostenlose Updates und Upgrades, sowie kostenlosen Support.
- Es lassen sich Lizenzen verschiedener Gruppen kombinieren.

</ArrowList>

Beispiel-Kombination: <br />
Es sollen 5 *Oracle EE* Datenbank-Instanzen und eine *Azure SQL DB*-Instanz über DPA verwaltet werden.

Eine mögliche Kombination mit Subscription Lizenzierung pro Jahr wäre: <br />
5 Lizenzen der *Category 1* (7.611,95 €) PLUS 1 Lizenz *DPA for Azure SQL DB* (115,26 €), ergibt 7727,21 € (statt 6 x Category 1 mit 9.134,34 €).



## Funktionen

<ArrowList>

- Diagnose und auf ML (Machine Learning) basierte Anomalieerkennung
- Server- und Leistungsüberwachung, mit Ereignisprotokollierung
- Echtzeit- und Verlaufsüberwachung
- Analyse und Reporting von Deadlocks und Blockierungen
- Alarmfunktion und Benachrichtigung
- Zugriffskontrollen und Benutzerberechtigungen
- Optimierung und Fehlerbehebung der DB
- Automatisierte Verwaltung über API
- Dashboard mit Datenvisualisierung

</ArrowList>
