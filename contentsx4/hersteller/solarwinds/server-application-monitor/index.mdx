<Intro>

Server and Application Monitor (SAM) dient als Software zur zentralen Überwachung lokaler, hybrider oder Cloud-basierter Anwendungsumgebung und Infrastruktur.
Überprüft Anwendungsverfügbarkeit und Leistungsbereitschaft, insbesondere detaillierte Überwachung von Microsoft Active Directory, Exchange, IIS und SQL
Server.

## Lizenzierung

<ArrowList>

- *Mindestanforderung:* Vierkernprozessor, [Windows Server 2016 oder aktueller](/hersteller/microsoft/windows-server/).
- *Unterstützung:* [SQL Server 2019](/hersteller/microsoft/sql-server/), Cloud: Amazon RDS, [Azure](/hersteller/microsoft/azure/) SQL-Datenbank.
- Lizenziert wird nach Nodes. (Bspw. Anwendungen, Systeme, Hypervisoren, IaaS, PaaS und SaaS)
- Als Kauflizenz (perpetual) und Subscription (Abonnement)
- Neulizenzen beinhalten bereits 1 Jahr Wartung und Support. In Subscriptions ist diese immer enthalten.
- Jahreslizenzen (Wartung) beinhalten kostenlose Updates und Upgrades, sowie kostenlosen Support.
- Wir listen hier nur die Node-Lizenzen auf bis 1000 Nodes. Höhere Preisstaffeln oder AL(Monitor)-Lizenzen müssen angefragt werden.

</ArrowList>


## Funktionen

<ArrowList>

- Server- und Anwendungsüberwachung bspw. von Apache- und Cisco-Tools, Microsoft Active Directory, [Exchange Server](/hersteller/microsoft/exchange/), [SQL Server](/hersteller/microsoft/sql-server/) und Cloud Umgebungen wie [Azure](/hersteller/microsoft/azure/), Amazon Web Service (AWS) und Container wie [Mirantis-Docker](/hersteller/mirantis-docker/).
- Anpassbare Leistungs- und Verfügbarkeitsberichte und Warnungen mit benutzer-, oder vordefinierten Vorlagen.
- Mehr als 1.200 Überwachungsvorlagen für Anwendungen, Systeme und Infrastruktur.
- Stapelübergreifende IT-Datenkorrelation - Beschleunigen Sie die Ermittlung von zugrunde liegenden Ursachen, indem Sie Netzwerk-leistungsmessdaten per Drag & Drop auf einer gemeinsamen Zeitleiste anzeigen und so Ihre gesamten Netzwerkdaten sofort visuell korrelieren.
- Dynamische Karten - SAM erkennt automatisch die Beziehungen zwischen Anwendungen und Servern auf Basis der aktiven Anwendungskommunikation und erfasst Statistiken zu Netzwerkverbindungen wie etwa Paketverluste und Latenz.
- Integrierte Virtualisierungsüberwachung - Neben den physischen Servern können Sie auch die Integrität und Verfügbarkeit von virtuellen Hosts und Gastmaschinen wie VMware und Hyper-V überwachen. Die Integration in SolarWinds Virtualization Manager liefert umfassende Einblicke für die Behandlung von Virtualisierungs-problemen.
- Integration mit [Network Performance Monitor](/hersteller/solarwinds/network-performance-monitor/), Virtualization Manager, Storage Resource Monitor und Web Performance Monitor möglich.
- Ermöglicht die Erstellung benutzerdefinierte Systemleistungsberichte und Warnungen bei entsprechenden Ereignissen.

</ArrowList>
