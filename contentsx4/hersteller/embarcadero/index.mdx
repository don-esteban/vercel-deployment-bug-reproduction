## Kauf und Lizenzierung

### Kauflizenz

Embarcadero Entwicklungsumgebungen werden generell als Kauflizenz/Dauerlizenz angeboten. Neulizenzen beinhalten eine [Update-Subscription](/hersteller/embarcadero/#update-subscriptions) (Wartung und Support). Die gängigste Laufzeit ist ein Jahr. Sie können jedoch auch andere Laufzeiten anfragen.

### Subscription

Nur Embarcadero C++ Builder wird zusätzlich zur Kauflizenz, auch als Abonnement (Subscription) bereitgestellt. Embarcadero nennt dies Pro Term License. Die Laufzeit beträgt 1 Jahr. Die Subscription beinhaltet bereits eine [Update-Subscription](/hersteller/embarcadero/#update-subscriptions) (Wartung und Support).

### Akademische Lizenzen

Akademische Lizenzen sind 1-Jahres-Lizenzen bzw. Abonnements (Subscription) mit vergünstigten Konditionen. Die Subscription beinhaltet bereits eine [Update-Subscription](/hersteller/embarcadero/#update-subscriptions) (Wartung und Support). Akademische Lizenzen entsprechen dem Funktionsumfang einer Enterprise Edition. Ein Angebot erhalten Sie auf Anfrage, halten Sie Ihren Nachweis über einer schulischen Tätigkeit bereit.

### Lizenzierung

Embarcadero Entwicklungsumgebungen werden nach Usern Lizenziert. Es wird zwischen [Named User](#named-user), [Network Named User](#network-named-user) und [Concurrent User](#concurrent-user) unterschieden.

#### Named User

Eine Named User-Lizenz ist auf eine individuell benannte Person in einem Unternehmen registriert. Die Lizenz kann von einem beliebigen Arbeitsplatz genutzt werden (max. drei Installation). Andere Personen dürfen die Lizenz nicht nutzen. Ein Lizenzserver ist hierfür nicht nötig.

#### Network Named User

Die Lizenz ist, wie die Named User Lizenz, an einen benannten Nutzer gebunden. Diese kann ebenfalls von einem beliebigen Arbeitsplatz genutzt werden (max. drei Installation). Die Lizenzen werden über einen zentralen Lizenzserver verwaltet, mit dem unter anderem Berichte über die Aktivitäten einsehbar sind.

#### Concurrent User

Mit einer Concurrent User Lizenz können mehrere Personen die Software nutzen. Die Anwendung kann jedoch nur so oft gleichzeitig geöffnet werden, wie Concurrent User lizenziert sind. Dieses Lizenzierungsmodell setzt einen Appwave- oder FLEXnet Lizenzserver voraus.

#### Lizenzmodelle im Vergleich

|Named User|Network Named User|Concurrent User|
|:---|:---|:---|
|Lizenz ist an einen bestimmten Benutzer gebunden|Lizenz ist an einen bestimmten Benutzer gebunden|Lizenzen sind bis zu einem vorgegebenen Maximum von beliebigen Nutzern nutzbar|
|Das Teilen von Lizenzen mit anderen Nutzern ist untersagt, auch wenn die Zugriffe nicht zeitgleich erfolgen|Das Teilen von Lizenzen mit anderen Nutzern ist untersagt, auch wenn die Zugriffe nicht zeitgleich erfolgen|Lizenzen können problemlos mit anderen Nutzern geteilt werden (solange das Maximum nicht erreicht ist)|
|Keine zentrale Verwaltung|Zentrale Verwaltung|Zentrale Verwaltung|
|Benötigt keinen zentralen Lizenzserver|Benötigt zentralen Lizenzserver|Benötigt Appwave- oder FLEXnet Lizenzserver|


### Update Subscriptions

Update Subscription ist bei Embarcadero die Bezeichnung für Support und Wartung. Damit erhalten Sie immer aktuelle Produkt-Upgrades, Updates und können bei Bedarf auf den Support zugreifen. Es gibt Standard- und Premium-Update-Subscription.

Neulizenzen beinhalten 1 Jahr Standard-Update Subscription. Eine Verlängerung für 1 Jahr kostet ca. 1/5 des Neukaufpreises und ist günstiger als alle 4-5 Jahre ein neues Produkt zu kaufen. Bei mehrjähriger Wartungsverlängerung erhalten Sie sogar mehr Rabatte. Premium-Update Subscription gibt es auf Anfrage.

Wir empfehlen die Wartungsverlängerung ca. 2-3 Wochen vor Ablauf der Subscription zu bestellen. Sie können bis zu 365 Tage ihre Subscription rückwirkend buchen, aber es entstehen zusätzliche Kosten. Nach der Kulanzzeit bleibt Ihnen nur ein Neukauf, ein Upgrade gibt es nicht.

Die Standard-Update Subscription bietet Ihnen:

-  **Updates und Hotfixes**
-  **Laufende Wartung - Updates und Hotfixes für verschiedene frühere Releases**
-  **Alle Major-Updates (z.B. 10.3 Rio auf 11 Alexandria)**
-  **3 Support-Anfragen für Entwickler im Jahr**
-  **Exklusive Bonus-Features, Add-ons und technische Inhalte**
-  **Technischer Support beim Hersteller**

Die Premium-Update Subscription bietet Ihnen:

- **Alle Inhalte aus Standard-Update Subscription**
- **Höhere Priorität und erweiterter technischer Support**
- **Exklusive technische Deep-Dive-Webinare von Top-Experten**
- **Regelmäßige Online-Meetings mit dem Produktteam und individuelle Unternehmens-Roadmap-Sessions**
- **Kostenloser zeitlich begrenzter Zugriff auf TestRail Enterprise**
