## Funktionen

- **Plattformübergreifendes C++** <br />
Modernes C++ mit optimierten Spracherweiterungen und Cross-Kompilation für Windows und iOS.
- **HTML 5 Unterstützung** <br />
Generieren Sie HTML5 über schnell kompilierbare ISAPI- und Apache-Webmodule, die eine Unternehmensdatenbankkonnektivität unterstützen.
- **Debugging** <br />
Ermitteln und beheben Sie Fehler schneller mit integriertem, plattformübergreifendem, nativem Debugging.
- **Unterstützung verschiedener OS** <br/>
C++ Builder unterstützt in der aktuellen Version die Entwicklung nativer Anwendungen für Windows 11, iOS 15.

## Downgrade - Zugriff auf ältere C++ Builder Versionen

Mit C++ Builder 12 Athens erhalten Sie auch Zugriff auf die Vorgängerversionen bis C++ Builder 6, wenn diese bis 180 Tage nach Kauf angefordert werden. Dazu benötigen Sie ein Embarcadero Developer Network (EDN)-Konto mit registrierter 11.x Seriennummer. Mehr dazu finden Sie in der [Anleitung auf der Embarcadero Seite](https://www.embarcadero.com/de/products/cbuilder/previous-versions).
