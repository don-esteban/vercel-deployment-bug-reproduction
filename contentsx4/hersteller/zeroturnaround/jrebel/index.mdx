<Intro>

Mit dem Entwicklungstool JRebel können veränderte Programmiercodes in vielen Sprachen sofort, ohne Neustart (Rebuild/Restart) übernommen werden. Der Code kann direkt getestet werden.

Außerdem können alle Anwendungen, Server und die Umgebung verwaltet und Aktivitäten verfolgt werden. Fehlerhafte Veröffentlichungen können sofort rückgängig gemacht werden.

</Intro>

## Lizenzierung

JRebel wird als Subscription (Abonnement) angeboten. Abonnements enthalten immer einen Anspruch auf Support und Updates.

Lizenziert wird nach Usern, Named-User oder Managed-User. Named-User eignen sich für einzelne Benutzer, es gibt keine zentrale Lizenzverwaltung. Managed-User eignen sich für Teams, diese werden über eine zentrale Lizenzmanagement Konsole verwaltet. Lesen Sie zum besseren Verständnis den Abschnitt [Lizenzierung](/hersteller/zeroturnaround/#lizenzierung) auf unserer ZeroTurnaround-Hauptseite.
