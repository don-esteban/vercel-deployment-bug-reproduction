## Lizenzierung

<ArrowList>

- Lizenzierung erfolgt pro Remote-Computer, auf den zugegriffen wird (Radmin Server).
- Auf einen lizenzierten Remote-Computer, dürfen Sie von beliebig vielen anderen Geräten aus zugreifen.
- Bis zu 5 gleichzeitige Zugriffe auf einen Radmin Server sind möglich. Brauchen Sie mehr simultane Zugriffsmöglichkeiten? Gerne können Sie mich kontaktieren.
- Als Einzellizenz, Lizenz-Paket (50, 100) oder Volumenlizenz (auf Anfrage, ab 200) erhältlich
- Mindestbestellmenge für Volumenlizenzen: ab 200 Lizenzen
- Lizenzen sind dauerhaft gültig (perpetual license)
- Kostenlose Updates, Hotfixes und Patches
- Technischer Support inklusive

</ArrowList>


## Funktionen

Radmin dient zur Fernsteuerung und Fernwartung von Computern.
Weitere Werkzeuge ermöglichen darüber hinaus Lizenzen über das Netzwerk zu verwalten und bereitzustellen.
Dieses Remote-Tool ist für System-, Netzwerkadministratoren und Techniker. <br />

*Radmin Server* ist die gleichnamige Software zur Bereitstellung der Fernsteuerung, diese muss lizenziert werden. *Radmin Viewer* ist kostenlos und ermöglicht den Zugriff auf den fernzusteuernden Computer mit *Radmin Server*.


<ArrowList>

- Unterstützung mehrerer gleichzeitiger Verbindungen
- Unterstützung mehrerer Monitore
- Unterstützung Windows 10/11, Windows 8, Windows 7, Windows Vista und Windows XP
- Geringe Hardwarevoraussetzungen, hohe Geschwindigkeit
- Verschlüsselte Verbindungen (AES256-Bit)
- Hardware-Remote-Control mit Intel AMT-Unterstützung
- Sichere "Drag and Drop"-Dateiübertragung mit "Delta Copy"-Funktion
- Text- und Voice-Chat
- Radmin Deployment Tools - kostenloses Bereitstellungs-Werkzeug für den Administrator
- Radmin VPN - kostenlos, ermöglicht den Zugriff auf den Radmin Server hinter einer Firewall

</ArrowList>
