## WS_FTP Server

FTP-Server mit benutzerfreundlicher Verwaltung für [Microsoft Windows](/hersteller/microsoft/), das
speziell auf die Bedürfnisse kleinerer und mittlerer Organisationen ausgerichtet ist.
WS_FTP Server lässt sich einfach in Microsoft Active Directory integrieren.
Lizenziert wird traditionell On-premises (eigene Installation) und Dauerlizenzen/Kauflizenzen.


Es gibt drei Editionen:


<ArrowList>

- [WS_FTP Server Basic](/hersteller/progress/ws_ftp/server/#ws_ftp-server-basic)<br />
*WS_FTP Server Basic* mit Microsoft AD Authentication und fünf WS_FTP Pro Clients inklusive. Protokolle: FTP/SSL/FTPS

- [WS_FTP Server Secure](/hersteller/progress/ws_ftp/server/#ws_ftp-server-secure)<br />
*WS_FTP Server Secure* wie Basic, erweitert durch SSH/SFTP-Protokoll, FIPS-140-2 Kryptografie und Lizenz für 25 Ad-Hoc User. Auch mit Failover erhältlich.

- [WS_FTP Server Premium](/hersteller/progress/ws_ftp/server/#ws_ftp-server-premium)<br />
*WS_FTP Server Premium* wie Secure, erweitert durch Web Transfer Modul, LDAP Integration, HTTPS und SCP2 (Secure Copy) Protokoll. Auch mit Failover erhältlich.

</ArrowList>





## WS_FTP Professional

WS_FTP Professional ist ein Secure File Transfer Client mit benutzerfreundlicher Verwaltung und bietet sichere Dateitransaktionen über FTP, SSH/SFTP und SSL/FTPS.
Mit Kennwortauthentifizierung, Verzeichnissynchronisierung, Dateikomprimierung.


<ArrowList>

- [WS_FTP Professional Single User](/hersteller/progress/ws_ftp/professional/#single-user)<br />
WS_FTP Professional für Einzelbenutzer (ohne Support).

- [WS_FTP Professional Single User plus Support](/hersteller/progress/ws_ftp/professional/#single-user-inklusive-support)<br />
Wie "Single User" + 1 Jahr E-Mail-Support

- [WS_FTP Professional Multi User inkl. Support](/hersteller/progress/ws_ftp/professional/#multi-user-inklusive-support)<br />
WS_FTP Professional für Mehrbenutzer + 1 Jahr E-Mail-Support + Zentraler Lizenzverwaltung und Preisstaffelung

</ArrowList>


## Lizenzierung

<ArrowList>

- WS_FTP Server Lizenz gilt pro Server-Instanz (physisch/virtuell)
- WS_FTP Professional Lizenz gilt pro Benutzer
- Die Lizenzierung basiert auf der Produktlizenz als Dauerlizenz (perpetual) und einer zeitlich befristeten Wartungslizenz.
- Neulizenzen sind inklusive Wartungslizenz (Ausnahme: [WS_FTP Pro Single User](/hersteller/progress/ws_ftp/professional/#single-user)).

</ArrowList>

## Support

<ArrowList>

- Technischer Support kann nur bei aktiver Wartungslizenz erfolgen.
- Die Wartungslizenz ermöglicht Zugriff auf technische Unterstützung (Support), Softwareupdates und Fehlerkorrekturen.
- Wartungslizenzen (Supportlizenzen) sind zeitlich befristet und müssen bei Ablauf erneuert werden.
- Abgelaufene Wartungslizenzen lassen sich nachträglich gegen eine zusätzliche Gebühr wieder aktivieren.

</ArrowList>
