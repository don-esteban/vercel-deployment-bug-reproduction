<Intro>

AvePoint bietet Lösungen zum Schutz, sowie zur Migration und Verwaltung von Daten in Cloud-Umgebungen, wie Microsoft 365, Salesforce und Google Workspace.
AvePoint SaaS (Software as a Service) vereinfacht die tägliche Arbeit und erweitert die Cloud Lösungen mit zusätzlichen Funktionen, unter anderem für Sicherheit, Zusammenarbeit, Backups oder Policies.

</Intro>

<TOC />



## AvePoint Cloud Dienste

AvePoint bietet eine Vielzahl von Lösungen. Hier finden Sie einen Auszug der wichtigsten Dienste:

- **[Cloud Backup](/hersteller/avepoint/cloud-backup/)**: <br />
  Backup Lösung in der AvePoint Cloud. Sorgt für besseren Schutz vor Ransomware-Angriffen und Nutzerfehlern.
- **[Fly Migrator](/hersteller/avepoint/fly-migrator/):** <br />
  Die schnellste Lösung für Microsoft 365 Migrationen, egal ob lokal, aus einer andere Cloud oder einem anderen Microsoft Tenant.
- **[Policies & Insights](/hersteller/avepoint/policies-insights/):** <br />
  Bietet zusätzliche Sicherheitsfunktionen, wie Richtlinien und Berichterstellung.


## Lizenzierung und Preise

<Info>

Ein [Microsoft 365](/hersteller/microsoft/microsoft-365/) oder [Office 365](/hersteller/microsoft/office-365/) Konto wird als Basis benötigt.

</Info>

Die AvePoint Dienste sind als **monatliche Abonnements**, auf Basis der Anzahl an Microsoft 365 Konten, erhältlich.

Die Preisgebung erfolgt individuell von AvePoint und muss daher angefragt werden.
