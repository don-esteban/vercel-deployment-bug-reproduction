<Intro>

IBM Spectrum Protect Base Edition ist eine erweiterbare Lösung für Backup, Restore und Archivierung. Sie richtet sich an mittelständische Unternehmen. Enthalten ist der Backupserver. Erweitert wird mit Einzelkomponenten wie Databases, ERP, Mail, SAN, Snapshot und Virtual Environments. Die Vorgängerversion heißt Tivoli Storage Manager Base Edition.

</Intro>
