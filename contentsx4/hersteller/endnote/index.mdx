## Lizenzierung

EndNote wird pro Benutzer lizenziert und kann als Einzel- oder Volumenlizenz (ab 5 Benutzern) erworben werden. Jede EndNote-Lizenz umfasst eine Windows- und eine Macintosh-Version. Eine Lizenz berechtigt einen Benutzer, die Software auf bis zu 3 Geräten zu installieren. Zum Beispiel auf einem Arbeitsplatzrechner, einem Notebook und einem Heim-PC. Nur der lizenzierte Benutzer darf mit der Software arbeiten. Die Software darf jedoch nur auf einem Gerät gleichzeitig genutzt werden und nicht mit anderen Geräten zusammen.

### Einzellizenz

Jede Einzellizenz erhält einen eigenen Lizenzschlüssel und muss mit diesem freigeschaltet werden. Es gibt keine Mengenrabatte.

### Volumenlizenz

Volumenlizenzen sind ab einer Bestellmenge von 5 Lizenzen verfügbar. Mit einem einzigen Lizenzschlüssel können alle erworbenen Lizenzen freigeschaltet werden. Bei einem bestehenden Vertrag können Nachbestellungen auf die ursprüngliche Lizenz registriert werden, Sie behalten also die ursprüngliche Rabattierung.




## Neu mit EndNote 21

<Info>

Mit der Version X9.3. wurde das Datenbankformat geändert, um die 64-bit Systemen von Microsoft und Apple zu unterstützen.
Alle älteren Versionen (X9.2., X8, X7) können keine Daten aus EndNote X9.3 oder höher öffnen.

- Cite While You Write in Google Docs  <br />
Mit Cite While You Write (CWYW) können Sie Ihre EndNote-Literaturangaben in ein Microsoft Word- oder Apple Pages-Dokument einfügen. CWYW erstellt automatisch textintegrierte Zitate und Referenzlisten und passt deren Format an Ihre Bedürfnisse an. In EndNote 21 wird CWYW erweitert, um mit Google Docs kompatibel zu sein.

- Datenwiederherstellungsfunktion  <br />
Wenn etwas schief geht, kann EndNote 21 Ihre Bibliothek aus der Cloud wiederherstellen. Einzelne Referenzen können mit früheren Versionen Ihrer Referenzen verglichen und wiederhergestellt werden.

- 7000+ Zitierstile  <br />
Erstellen Sie Ihre Bibliographie in einem der mitgelieferten Stile oder erstellen Sie Ihren eigenen benutzerdefinierten Stil. Auch neue, moderne Arten von Referenzen wie Blogs und Online-Videos werden unterstützt.

- Neue EndNote Web Oberfläche  <br />
Mit EndNote haben Sie über die Cloud jederzeit und überall Zugriff auf Ihre Daten. Wechseln Sie nahtlos zwischen der Online-, Desktop- und iPad-Anwendung. Das neue EndNote Web ist für EndNote 21-Kunden drei Jahre lang ab dem Aktivierungsdatum verfügbar.  Synchronisieren Sie Ihre gesamte Bibliothek, einschließlich PDFs, Notizen und Anmerkungen, sicher zwischen all Ihren Geräten.
