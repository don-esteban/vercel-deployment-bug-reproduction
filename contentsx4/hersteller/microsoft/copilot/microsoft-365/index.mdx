<Intro>

Microsoft Copilot für Microsoft 365 ist die kombiniert die Leistungs- und Kombinationsfähigkeit von Künstlicher Intelligenz mit Ihren Unternehmensdaten.
Die Lizenz richtet sich an Unternehmen, die Microsoft 365 Dienste nutzen und mit den Copilot KI-Funktionen erweitern, vereinfachen und automatisieren möchten.
Über unseren Warenkorb erhalten Sie mit wenigen Klicks ein Sofortangebot.


</Intro>

## Lizenzüberblick

Microsoft Copilot for Microsoft 365 ist ein [Add-On](#voraussetzungen) und wird als benutzerbasiertes Abonnement ([User Subscription License - USL](/hersteller/microsoft/microsoft-365/lizenzierung/#user-subscription-license-usl)) gekauft. Die Laufzeit mit Preisschutz beträgt ein Jahr.
Für Bildungseinrichtungen und gemeinnützige Organisationen gibt es momentan (Stand Januar 2024) keine Sonderkonditionen.

<ArrowList>

- [Microsoft 365 Lizenzierung](/hersteller/microsoft/microsoft-365/lizenzierung/)<br />
  Ausführliche Informationen zu Kauf und Lizenzierung der Microsoft 365 Pläne.

- [CSP Subscription](/hersteller/microsoft/csp/)<br />
  Abonnement-Lizenz. Die Laufzeit mit Preisschutz beträgt wahlweise ein Jahr oder einen Monat. Keine Mindestbestellmenge.

- [NCE](/hersteller/microsoft/microsoft-365/lizenzierung/#new-commerce-experience-nce)<br />
  New Commerce Experience - CSP Subscription für Unternehmen und Behörden.

</ArrowList>



## Copilot Funktionen

Hier finden Sie einige Copilot for Microsoft 365 Funktionen in den Microsoft 365 Diensten, die Ihnen bei der täglichen Arbeit helfen können:

<ArrowList>

- [Microsoft Teams](/hersteller/microsoft/teams/)<br />
  Einfaches Erstellen von Meetings, automatische Chatzusammenfassungen und schnelleres Auffinden von Informationen
- [Microsoft Outlook](/hersteller/microsoft/office/einzelprodukte/outlook-2021/)<br />
  Schnelles Erstellen von E-Mails, automatische Zusammenfassungen von Konversationen und langen E-Mails.
- [Microsoft Word](/hersteller/microsoft/office/einzelprodukte/word-2021/)<br />
  Starten von Entwürfen, Hinzufügen zu bestehenden Dokumenten, Neuschreiben von Texten und Erstellung von Zusammenfassungen.
- [Microsoft Excel](/hersteller/microsoft/office/einzelprodukte/excel-2021/)<br />
  Bessere Einsichten in Daten, Automatisch generierte Funktionen und vieles mehr.
- [Microsoft PowerPoint](/hersteller/microsoft/office/einzelprodukte/powerpoint-2021/)<br />
  Automatisches Erstellen von ansehlichen Präsentationen, Organisation und Zusammenfassungen.
- Microsoft OneNote
  Zusammenfassung der Notes, Erstellen von To-Do-Listen, Planerstellung und mehr.

</ArrowList>
