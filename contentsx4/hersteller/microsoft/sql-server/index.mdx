<ArrowList>

- **[SQL Server 2022 Standard Core](/hersteller/microsoft/sql-server/standard-2022/#core-lizenzierung)**<br />

## SQL Server 2022

<Info>

SQL Server 2022 Release Date war der 16.11.2022. Weitere Infos zu den Neuerungen finden Sie in unserem [Blogeintrag zum SQL Server 2022 Release](/blog/sql-server-2022-release/)

</Info>

SQL Server 2022 lässt sich auf Linux, [Windows](/hersteller/microsoft/windows-server/) und Kubernetes bereitstellen.
Hier finden Sie einen kurzen Auszug der wichtigsten neuen Funktionen mit SQL Server 2022.

<ArrowList>

- **Business Continuity mit Azure**<br />
  Mit dem neuen Link-Feature in der verwalteten Azure SQL-Instanz, können Sie eine vollständig verwaltete Notfallwiederherstellung Ihres lokalen Servers in der Cloud sicherstellen. Replizieren Sie kontinuierlich Daten in die Cloud und aus der Cloud.
- **Analyse von On-Premises-Betriebsdaten**<br />
  Beseitigen Sie die Hindernisse zwischen der operativen und analytischen Datenspeicherung.
  Mit Azure Synapse Link können Sie Spark und SQL-Runtimes in der Cloud nutzen und alle Ihre Daten nahezu in Echtzeit analysieren.
- **Transparenz für den gesamten Datenbestand**<br />
  Durch die Integration des SQL Servers mit Azure Purview verwalten und steuern Sie Ihren gesamten Datenbestand. Damit vermeiden Sie Probleme, wie Datensilos.
- **Mehr Sicherheit**<br />
  Verwenden Sie einen unveränderlichen Ledger, um Daten vor Manipulationen zu schützen.
- **Verbesserte Performance und Verfügbarkeit**<br />
  Zahlreiche Verbesserungen bei Leistung und Verfügbarkeit ermöglichen schnellere Abfragen und sichere Geschäftskontinuität. Unter anderem mit schnellerer Abfrageleistung und -optimierung ohne Code-Änderung und reibungsloser Ausführung von Multi-Write-Umgebungen über mehrere Standorte.

</ArrowList>

<Info>

Eine vollständige Übersicht finden Sie in unserem [Blogbeitrag zum SQL Server 2022 Release](/blog/sql-server-2022-release/), auf der [Microsoft SQL Server 2022 Seite](https://www.microsoft.com/de-de/sql-server/sql-server-2022) oder im [Microsoft SQL Server 2022 Datasheet](https://download.microsoft.com/download/d/1/1/d115ab99-0b39-48e2-987f-dfee5731437c/SQL_Server_2022_datasheet.pdf).
