<Intro>

Microsoft bietet für den SQL Server 2012 kostenpflichtige erweiterte Sicherheitsupdates (ESU - Extended Security Update). Mit den ESU kann der Support um bis zu drei Jahre verlängert werden.
Supportende des SQL Server 2012 war bereits am 12. Juli 2022.
Die Extended Security Updates sind ab sofort auch im CSP-Lizenzprogramm verfügbar. Für die ESU-Nutzung wird eine aktive Software Assurance oder Subscription benötigt.

</Intro>

<TOC />



## SQL Server 2012 Extended Security Updates

<ArrowList>

- Benötigt aktive Software-Assurance oder Subscription für Server und CALs.
- Mit ESU kann der Support um bis zu 3 Jahre verlängert werden (bis maximal 12.07.2025).
- Die Laufzeit des ersten ESU Jahres beginnt immer ab 11.10.2023 für ein Jahr (bis 08.10.2024).
- ESU sind kummulativ, d.h. für ESU in Jahr 2 werden auch ESU für Jahr 1 benötigt.

</ArrowList>

## Lizenzüberblick

SQL Server 2012 ESU werden in Lizenz-Paketen für 2 Prozessoren angeboten. Jeder Prozessor muss mit mindestens 8 Cores lizenziert werden. Für jeden physische Server gilt eine Mindestlizenzierung von 16 Cores.
