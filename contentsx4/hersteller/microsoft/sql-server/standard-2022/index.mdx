<Intro>

  Microsoft SQL Server 2022 Standard wurde am 16. November 2022 released. Die Standard Edition enthält grundlegende Datenbanken für Verwaltung und Business Intelligence. Die Virtualisierungrechte sind gegenüber der SQL Server Enterprise Edition eingeschränkt.
  Wie beim Vorgänger SQL Server Standard 2019, kann wahlweise nach Server-/CAL oder nach physischen, bzw. virtuellen Cores lizenziert werden. Beachten Sie bitte, dass bei der Core-Lizenzierung SA oder eine Subscription für die Virtualisierung benötigt wird! Nutzen Sie unsere Sofortangebote zur Beschaffungsplanung!


</Intro>

## Core Lizenzierung

<Info>

Mindestlizenzierung von 4 Cores pro Server (entspricht 2 Lizenzen). Für die Lizenzierung virtueller Kerne wird eine CSP Subscription oder aktive Software Assurance benötigt. Zugriffe müssen nicht lizenziert werden.

</Info>

## Lizenzüberblick

<Info>

**Achtung:** Mit der Einführung des SQL Server 2022 wird bei der Core-Lizenzierung für die Nutzung in virtuellen Umgebungen eine aktive Software Assurance oder CSP Subscription benötigt!
SQL Server Standard Core Lizenzen ohne SA dürfen nicht mehr für die Verwendung in einzelnen VMs lizenziert werden. Siehe: [SQL Server Lizenzierung](/hersteller/microsoft/sql-server/lizenzierung/)

</Info>

- [CSP Subscription](/hersteller/microsoft/csp/)<br />
  Abonnementlizenz (Subscription). Keine Mindestbestellmenge.
  Enthält eine perpetual SQL Server 2022 Lizenz und [zusätzliche Vorteile und Rechte](/hersteller/microsoft/sql-server/lizenzierung/#vorteile-mit-csp-subscription-und-sa).

- [Open Value](/hersteller/microsoft/open-value/)<br />
  Volumenlizenz mit 3 Jahren Software Assurance (SA). Mindestbestellmenge: 5 Lizenzen. Enthält mit aktiver SA [zusätzliche Vorteile und Rechte](/hersteller/microsoft/sql-server/lizenzierung/#vorteile-mit-csp-subscription-und-sa).
