## Besonderheiten der OEM-Lizenz

<Info>

**Achtung:** Mit Einführung des SQL Server 2022 wird bei der Core-Lizenzierung für die Lizenzierung von virtuellen Maschinen immer eine aktive Software Assurance oder CSP Subscription benötigt! Siehe: [SQL Server Lizenzierung](/hersteller/microsoft/sql-server/lizenzierung/)

</Info>

Die SQL Server OEM Lizenz enthält lediglich das [Downgrade-Recht](/blog/microsoft-downgrade-rechte/) zum Einsatz der Vorgängerversionen.
Ein Nachteil ist dass, im Gegensatz zur Volumenlizenz, kein Re-Imaging Recht enthalten ist. Die Installation von einem einzigen Installationsträger und das verteilen auf mehreren Geräten sind somit den OEM-Lizenzen nicht erlaubt. Auch der Datenträger für die Nutzung älterer Versionen ist nicht enthalten, für die Installation wird daher ein Datenträger und Key der alten Version vorausgesetzt.
