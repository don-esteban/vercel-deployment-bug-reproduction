<Intro>

 Microsoft Project ist eine lokale oder cloudbasierte Projektmanagement Lösung. Die lokalen Lösungen sind Project Standard 2021, für einfachere Anforderungen ohne Integration mit SharePoint/Teams, und Project Professional 2021, mit allen Funktionen und einer Zugriffslizenz für den Project Server mit unternehmensweiten Planungsaufgaben für größere Teams.
 Die Cloud-basierten Project Pläne, ehemals Project Online, sind eine interessante Alternative, wenn Sie flexibel sein möchten, sowie Kosten und Aufwand für den Unterhalt einer eigenen Server Infrastruktur. Es ist auch möglich, je nach Projektaufwand die lokale und die cloudbasierte Lösung gemeinsam zu nutzen.


</Intro>

## Project 2021

<ArrowList>

- **[Project Standard 2021](/hersteller/microsoft/project/standard-2021/)**<br />
 Project Standard bietet grundlegenden Funktion für Projektplanung und Projektverwaltung. Dazu gehören, Projektzeitpläne, Kosten planen, Vorgänge verwalten, Berichte erstellen und Business Intelligence nutzen. Es fehlen unter anderem die Integrationsmöglichkeiten mit Skype, SharePoint und einem Project Server.

- **[Project Professional 2021](/hersteller/microsoft/project/professional-2021/)**<br />
 Project Professional arbeitet zusätzlich mit Skype/Teams, MS SharePoint und dem Project Server zusammen. Für den Zugriff auf den Server ist eine CAL enthalten. Weitere Vorteile sind Ressourcenverwaltung, Arbeitszeittabellen und mehr.

</ArrowList>


## Project Server

<ArrowList>

- **[Project Server Subscription Edition](/hersteller/microsoft/project/server-subscription-edition/)**<br />
 Der neue Project Server ist ausschließlich als Subscription erhältlich, da die Weiterentwicklung mit Hilfe der Microsoft Cloud erfolgt. Die aktuelle perpetual Version erhalten Sie weiterhin mit [Project Server 2019](/hersteller/microsoft/project/server-2019/).

- **[Project Server 2019](/hersteller/microsoft/project/server-2019/)**<br />
 Die perpetual Lizenz ist weiterhin als Project Server 2019 verfügbar. Der Server unterstützt Zusammenarbeitsmöglichkeiten, anspruchsvolle BI-/Analysemethoden und erweiterte gruppenorientiertes Projektmanagement (mit Project Standard und Professional) zu konzernweitem EPM/Enterprise Project Management (mit Project Professional und Server). Benötigt [SharePoint](/hersteller/microsoft/sharepoint/), [SQL Server](/hersteller/microsoft/sql-server/) und [Windows Server](/hersteller/microsoft/windows-server/) inkl. Zugriffslizenzen.

- **[Project Server 2019 CAL](/hersteller/microsoft/project/server-2019/cal/)**<br />
 Die Zugriffslizenz (CAL) für den Project Server 2019.

</ArrowList>


## Project Online

<ArrowList>

- **[Project Plan 1](/hersteller/microsoft/project/plan-1/)**<br />
 Project Plan 1, auch Project für das Web, enthält grundlegende Funktionen für den informellen Projektmanager. Der lokal installierbare Desktop-Client ist nicht enthalten.

- **[Project Plan 3](/hersteller/microsoft/project/plan-3/)**<br />
 Project Plan 3 enthält alles aus Project Plan 1 und zusätzliche Funktionen für Projektleiter und Projektmanager. Damit lassen sich Projekte planen, erstellen und terminieren. Zusätzlich ist auch der Desktop Client mit enthalten.

- **[Project Plan 5](/hersteller/microsoft/project/plan-5/)**<br />
 Project Plan 5 enthält alle Funktionen von Project Plan 3. Plan 5 eignet sich für die unternehmensweite Projektsteuerung und enthält zusätzlichen Funktionen für Portfolio Management, wie Enterprise Ressource Planing (ERP Management).

- **[Project Online Essentials](/hersteller/microsoft/project/online-essentials/) (Add-On)**<br />
 Project Online Essentials ist ein Add-On für den Project Plan 3 und Project Plan 5. Dieser richtet sich an Projektmitarbeiter, die Zugriff auf den Project Server benötigen, um über den WebBrowser an Projekten mitzuwirken und deren Arbeitszeittabellen mitzuteilen. Projekte selbst können nicht erstellt werden. Der lokal installierbare Desktop-Client ist ebenfalls nicht enthalten.

</ArrowList>
