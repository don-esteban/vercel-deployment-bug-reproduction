<Intro>

 Project Plan 3, ehemals Project Online Professional, umfasst die grundlegenden Funktionen von Plan 1, sowie zusätzliche Funktionen für das Projektmanagement. Neben dem Webinterface, kann die Software auch mit einem Desktop Client genutzt werden.
 Neu mit der Umstellung von Project ist Project für das Web, kurz Project, womit das cloudbasierte arbeiten sowie Projektmanagement ermöglicht wird.
 Wenn Sie zusätzlich ERP-Management Funktionen benötigen, gibt es den Project Plan 5. Über unseren Warenkorb erhalten Sie mit wenigen Klicks ein Sofortangebot.

## Funktionen

 Project Plan 3 enthält alle Funktionen von [Project Plan 1](/hersteller/microsoft/project/plan-1/), wie Termin- und Zeitplanung. Neben dem Webinterface kann Project auch über den Desktop Client ausgeführt werden. Zusätzlich enthalten sind:

<ArrowList>

 - **Umfassende Benutzeroberfläche über Desktopclient**<br />
  Der Project Professional Client ist immer auf der aktuellsten Version und kann auf bis zu 5 PCs installiert werden.
 - **Project Web App**<br />
  Web Interface für Projektmanagement. Zugriff jederzeit von jedem Gerät über den Browser.
 - **Planung und Kostenberechnung von Projekten**<br />
  Automatisierte Planungswerkzeuge für effizienteres Arbeiten. Darüber hinaus vereinfachen mehrere Zeitachsen die Visualisierung komplexer Zeitpläne.
 - **Ressourcenmanagement**<br />
  Werkzeuge für das Ressourcenmanagement helfen Projektteams zusammenzustellen, erforderliche Ressourcen anzufordern und effizientere Zeitpläne aufzustellen. Für die Verwaltung der Ressourcen ist jedoch der [Project Plan 5](/hersteller/microsoft/project/plan-5/) erforderlich.
 - **Verbesserte Zusammenarbeit**<br />
  Durch die nahtlose Integration in Teams können sich Ihre Projektmitarbeiter einfacher abstimmen.

</ArrowList>
