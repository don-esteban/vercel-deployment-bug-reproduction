<Intro>

Project Plan 1, auch Project für das Web genannt, ist seit Januar 2020 über CSP verfügbar. Der kleinste Project Plan richtet sich an informelle Projektmanager bzw. Projektmitarbeiter, welche nur die grundlegenden Project [Funktionen](#funktionen), wie beispielsweise Rasterfunktionen benötigen. Project Plan 1 kann von jedem Gerät aus, über den Webbrowser für gemeinsame Projekte genutzt werden. Ein Project Desktop Client ist hier nicht enthalten.
Über unseren Warenkorb erhalten Sie mit wenigen Klicks ein Sofortangebot.

</Intro>

## Lizenzüberblick

Project Online Plan 1 wird als benutzerbasierte Abonnement-Lizenz ([User Subscription License - USL](/hersteller/microsoft/exchange/lizenzierung/#user-subscription-license-usl)) gekauft.

<ArrowList>

- [Microsoft 365 Lizenzierung](/hersteller/microsoft/microsoft-365/lizenzierung/)<br />
  Ausführliche Informationen zu Kauf und Lizenzierung der Online Pläne.

- [CSP Subscription](/hersteller/microsoft/csp/)<br />
  Abonnement-Lizenz. Die Laufzeit mit Preisschutz beträgt wahlweise ein Jahr oder einen Monat. Keine Mindestbestellmenge. Keine Sonderkonditionen für Bildungseinrichtungen.

- [NCE](/hersteller/microsoft/microsoft-365/lizenzierung/#new-commerce-experience-nce)<br />
  New Commerce Experience - CSP Subscription für Unternehmen und Behörden.

- [Open Value](/hersteller/microsoft/open-value/)<br />
  Einjährige Abonnement-Lizenz mit Software Assurance. Mindestbestellmenge: 5 Lizenzen.

</ArrowList>


## Funktionen

 Project Plan 1 enthält grundlegende Project Funktionen:

<ArrowList>

 - **Project Home**<br />
  Übersichtliches Dashboard (Startseite) mit den wichtigsten Informationen.
 - **Benutzeroberfläche über den Webbrowser**<br />
  Über den Webbrowser kann von jedem Ort aus gearbeitet werden.
 - **Raster**<br />
  Auflistung der Vorgänge als Tabelle ohne Zeitachse.
 - **Board**<br />
  Gruppierung der Vorgänge nach Status, Bucket und Datum.
 - **Co-authoring**<br />
  Zusammenarbeit an einem Projektplan nach Office 365 in Echtzeit.
 - **Collaboration**<br />
  Integration und Zusammenarbeit in Teams möglich.
 - **Funktionen für PowerBI**<br />Wenn PowerBI installiert ist, können vorgefertigte Berichte über Aufgaben, Ressourcen und Projekte erstellt werden.

</ArrowList>

<Info>

 Einen Desktop Client, sowie Funktionen zur Projekterstellung, -planung oder Kostenberechnung, erhalten Sie mit [Project Plan 3](/hersteller/microsoft/project/plan-3/) oder [Project Plan 5](/hersteller/microsoft/project/plan-5/). Plan 5 enthält zusätzliche Funktionen zum ERP-Management.

</Info>
