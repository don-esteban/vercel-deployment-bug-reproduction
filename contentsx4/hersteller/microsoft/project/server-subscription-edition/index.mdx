<Intro>

Die Project Server Subscription Edition (auch Project Server 2022 Version), ist seit November 2021 der offizielle Nachfolger des Project Server 2019. Microsoft hat die Project Lizenzierung mit Version 2022 geändert:
Die Server-Lizenz für die Subscription Edition ist ausschließlich als Abonnement verfügbar. Die Subscription Edition enthält eine perpetual Project Server 2019 Lizenz, d.h. nach Ablauf der Subscription können Sie den Project Server weiterhin nutzen.
Für den Zugriff werden Project Server Subscription Edition CALs (Device and User) benötigt. Nutzen Sie unsere Sofortangebote zur Beschaffungsplanung!

</Intro>

## Lizenzüberblick

Project Server Subscription Edition ist ausschließlich im [Open Value](/hersteller/microsoft/open-value/) Lizenzprogramm erhältlich und benötigt aktive [Software Assurance](/hersteller/microsoft/software-assurance/) für Serverlizenz und CALs (siehe: [Microsoft Product Terms](https://www.microsoft.com/licensing/terms/productoffering/ProjectServer/OVOVS)).
Enthält eine perpetual [Project Server 2019](/hersteller/microsoft/project/server-2019/) Lizenz, die nach Ablauf der Subscription weiterhin genutzt werden kann.

Für den Zugriff werden zusätzlich [CALs](/hersteller/microsoft/project/server-2019/cal/) benötigt!

<ArrowList>

- [SharePoint Lizenzierung](/hersteller/microsoft/sharepoint/lizenzierung/)<br />
  Ausführliche Informationen zu Kauf und Lizenzierung anhand der SharePoint Server Subscription Edition.

- [CAL](/hersteller/microsoft/project/server-2019/cal/)<br />
  Jeder Zugriff auf den Project Server benötigt eine Zugriffslizenz (CAL)!

- [Open Value](/hersteller/microsoft/open-value/)<br />
  Volumenlizenz mit 3 Jahren Software Assurance. Mindestbestellmenge: 5 Lizenzen.

</ArrowList>


### Neuerungen und Änderungen mit dem Project Server 2022

Die wichtigste Änderungen, [laut der Ankündigung von Microsoft](https://techcommunity.microsoft.com/t5/exchange-team-blog/exchange-news-and-announcements-microsoft-ignite-2020-edition/ba-p/1662224), ist die Verabschiedung des Lizenzmodell Server-/CAL. Den Project Server 2022 gibt es nur noch als Subscription.
Das Abo-Modell wird zunächst über Open Value verfügbar sein, CSP wird zum späteren Zeitpunkt freigeschaltet.


### Upgrades auf Project Server 2022

Wie auch der [SharePoint Server](/hersteller/microsoft/sharepoint/server-subscription-edition/) wird die neue Version Project Server 2022 direkte Upgrade unterstützen. Das bedeutet, dass eine Migration von der Kauflizenz zur Mietlizenz ohne Probleme ablaufen soll.
Kunden mit aktiver Software Assurance können kostenlos auf die neue Version upgraden.
