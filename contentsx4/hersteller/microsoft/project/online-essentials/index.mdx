<Intro>

Project Online Essentials ist eine Zusatzlizenz (Add-on) für Project Plan 3 oder Project Plan 5.
Hiermit können Teammitglieder über den Browser Vorgänge aktualisieren, Arbeitszeiten übermitteln und mit dem Team kommunizieren. Wenn Sie nur grundsätzliche Funktionen für Project benötigen, gibt es günstigen Project Plan 1. Damit erhalten Sie Project für das Web.
Über unseren Warenkorb erhalten Sie mit wenigen Klicks ein Sofortangebot.

</Intro>

## Lizenzüberblick

Project Online Essentials ist ein Add-on für [Project Plan 3](/hersteller/microsoft/project/plan-3/) oder [Project Plan 5](/hersteller/microsoft/project/plan-5/).
Lizenziert wird als Abonnement-Lizenz ([User Subscription License - USL](/hersteller/microsoft/exchange/lizenzierung/#user-subscription-license-usl)).

<ArrowList>

- [Microsoft 365 Lizenzierung](/hersteller/microsoft/microsoft-365/lizenzierung/)<br />
  Ausführliche Informationen zu Kauf und Lizenzierung der Online Pläne.

- [CSP Subscription](/hersteller/microsoft/csp/)<br />
  Abonnement-Lizenz. Die Laufzeit mit Preisschutz beträgt wahlweise ein Jahr oder einen Monat. Keine Mindestbestellmenge. Keine Sonderkonditionen für Bildungseinrichtungen.

- [NCE](/hersteller/microsoft/microsoft-365/lizenzierung/#new-commerce-experience-nce)<br />
  New Commerce Experience - CSP Subscription für Unternehmen und Behörden.

- [Open Value](/hersteller/microsoft/open-value/)<br />
  Einjährige Abonnement-Lizenz mit Software Assurance. Mindestbestellmenge: 5 Lizenzen.

</ArrowList>

## Funktionen

Zusatzlizenz für Projektteams um über den Browser Aufgaben zu aktualisieren, Arbeitszeittabellen zu übermitteln und Dokumente freizugeben.

<ArrowList>

 -  Zugriff jederzeit von jedem Gerät über den Browser.
 -  Teammitglieder können Vorgänge auf praktisch jedem Gerät aktualisieren.
 -  Erfassen Sie den projektbezogenen und sonstigen Aufwand für Gehaltsabrechnung, Rechnungsstellung und andere geschäftliche Zwecke.
 -  Durch die nahtlose Integration mit Skype for Business, SharePoint und Yammer können sich Ihre Teams einfacher abstimmen.

</ArrowList>
