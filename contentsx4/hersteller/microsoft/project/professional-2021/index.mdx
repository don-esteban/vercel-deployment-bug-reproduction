<Intro>

Project 2021 Professional richtet sich an Projektmanagementteams. Funktional entspricht Project 2021 Professional weitgehend der letzten Version des [Project Plan 5](/hersteller/microsoft/project/plan-5/). Die neue Version ist seit 5. Oktober 2021 verfügbar.
Project Professional ist als PKC, CSP-Kauflizenz oder im Open Value Volumenlizenzprogramm mit Software Assurance verfügbar.
Über unseren Warenkorb erhalten Sie mit wenigen Klicks ein Sofortangebot.

</Intro>

## Lizenzüberblick

Project Professional 2021 wird pro Gerät lizenziert. Jedes Gerät, das auf die Software zugreift, benötigt eine Lizenz.

<ArrowList>

- [Office Lizenzierung](/hersteller/microsoft/office/lizenzierung/)<br />
  Ausführliche Informationen zu Kauf und Lizenzierung der Office Produkte.

- [PKC](/hersteller/microsoft/pkc/)<br />
  Günstigere, dauerhaft gültige Einzellizenz mit eingeschränkten Rechten.

- [ESD](/glossar/esd-electronic-software-distribution/)<br />
  Dauerhaft gültige ESD-Einzellizenz (Electronic Software Distribution) mit eingeschränkten Rechten zum Download.

- [CSP](/hersteller/microsoft/csp/)<br />
  Dauerhafte Lizenz (Kauflizenz) ohne Wartung. Keine Mindestbestellmenge. Keine Sonderkonditionen für Behörden.

- [Open Value](/hersteller/microsoft/open-value/)<br />
  Volumenlizenz mit 3 Jahren Software Assurance (SA). Mindestbestellmenge: 5 Lizenzen.

</ArrowList>


### Project Einzellizenzen

<Info>

  Die Kauflizenz erhalten Sie im [CSP](/hersteller/microsoft/csp/) oder als [PKC](/hersteller/microsoft/pkc/)/[ESD](/glossar/esd-electronic-software-distribution/) Version. Die Volumenlizenzen mit Software Assurance und Verlängerungen sind im [Open Value Volumenlizenzprogramm](/hersteller/microsoft/open-value/) verfügbar.

</Info>

 Project Professional 2021 ist als Product Key Card (PKC) und Download Version (Electronic Software Distribution - ESD) oder als CSP Lizenz auch einzeln erhältlich.
 Die Vorteile der Volumenlizenzen (u.a. [Downgrade-Recht](/blog/microsoft-downgrade-rechte/), Re-Imaging, Zweitinstallation) sind nicht enthalten. Verwendung nur auf einem Gerät.
