<Intro>

 Die Volumenlizenzprogramme von Microsoft bieten Firmen, Behörden, Bildungseinrichtungen sowie gemeinnützigen Organisationen verschiedene Möglichkeiten, die Lizenzen zu buchen und zu verwalten. Lizenzen können gekauft oder gemietet werden, auch hier mit der Möglichkeit nach einer bestimmten Laufzeit die Lizenz zu kaufen. Je nach Größe der Organisation, können auch günstigere Konditionen angeboten werden. Für weitere Informationen lesen Sie weiter.


</Intro>

## Volumenlizenzprogramme im Allgemeinen

 Die meisten Microsoft Volumenlizenzen werden im [Volume Licensing Service Center (VLSC)](https://www.microsoft.com/licensing/servicecenter/default.aspx) herunterladen. Volumenlizenzen erleichtern das Deployment in größeren Unternehmen durch Installationen mit einem **einzigen Volumenlizenz-Key** ([VLK](/glossar/vlk-volume-license-key/)). Die Aktivierung Ihrer Lizenzen können Sie lokal über den Key Management Service ([KMS](/glossar/kms-key-management-service/)) oder über einen Microsoft Dienst via Multiple Activation Key ([MAK](/glossar/mak-multiple-activation-key/)) durchführen, auch eine Kombination von beiden Aktivierungsoptionen ist möglich.

 Lizenzen, die in den Volumenlizenzprogrammen erworben wurden enthalten das **[Downgrade-Recht](/blog/microsoft-downgrade-rechte/)** auf ältere Produktversionen und das **Reimagingrecht** für die Erstellung eines einheitlichen Abbilds. Auch das **Zweitkopierecht** für die zusätzliche Installation der lizenzierten Software auf einem tragbaren Gerät ist bei Volumenlizenzen immer enthalten.

 Mit diesen (Volumen-)Lizenzverträgen können Unternehmen sowie Behörden viele Kosten einsparen, da die Lizenzerwebsmöglichkeiten flexibel sind und eine einfache Bereitstellung dieser ermöglichen. Auch ist es möglich zwischen verschiedenen Zahlungsoptionen zu wählen und andere Vorteile wie Software Assurance zu wählen.

 Microsoft Produkte in den Volumenlizenzprogrammen können optional mit einer [Software Assurance (SA)](#microsoft-software-assurance-sa) ausgestattet werden. Damit erhalten Sie unter anderem Support und das Recht auf neue Versionen der Software upzugraden, die erst nach dem Kauf der Lizenz veröffentlicht wurden. Weitere Informationen dazu, finden Sie auf unserer [Software Assurance](/hersteller/microsoft/software-assurance/) Seite.


## Open Value

 Open Value Verträge haben eine **3-jährige Laufzeit** und benötigen für den Kauf die Lizenzierung von **mindestens 5 qualifizierten PCs**. [Software Assurance](#microsoft-software-assurance-sa) ist für die gesamte Laufzeit immer enthalten. Bei Open Value Verträge erfolgt die **Bezahlung vorab oder in Raten**. Außerdem ist eine **Standardisierung der Desktopplattform** für mindestens ein Produkt (Office Professional Plus, Windows Enterprise) mit Open Value möglich. Dabei werden alle qualifizierten PCs im Unternehmen erfasst und mit den standardisierten Produkten ausgerüstet.

 Als **qualifizierte Geräte** zählen alle Desktop- oder tragbaren PCs (auch Tablet-PCs), Arbeitsstationen und ähnliche Geräte, die von einem Unternehmen oder zu dessen Nutzen verwendet werden, sowie die Systemanforderungen zur Nutzung der Plattformprodukte erfüllen. Ausgeschlossen sind Computer, die als Server arbeiten und nicht als PCs eingesetzt werden, sowie Systeme, die ausschließlich für die Ausführung von Geschäftsanwendungen vorgesehen sind.

 Bei der Standardisierung der **Professional Desktopplattform**, (Office Professional Plus, Core CAL Suite, Windows Enterprise) oder der **Enterprise Desktopplattform** (Office Professional Plus, Enterprise CAL Suite und Windows Enterprise), erhalten Sie einen Plattformrabatt. Eine genauere Übersicht finden Sie auf der [Open-Value Seite.](/hersteller/microsoft/open-value/)


### Open Value Varianten

<ArrowList>

 - **Open Value (OV):** Vorauszahlung oder Ratenkauf für mindestens 5 qualifizierte PCs im Unternehmen
 - **Open Value Company-Wide (OV CW):** Wie Open Value, es werden aber alle qualifizierten PCs im Unternehmen mit einem oder mehreren Plattformprodukten standardisiert. Der Preis basiert auf der Anzahl der lizenzierten PCs, das Nutzungsrecht für die Software ist unbefristet
 - **Open Value Subscription (OVS):** Die Open Value Subscription Variante erfordert ebenfalls die Standardisierung der Desktopplattform. Der Preis basiert auf der Anzahl der PCs und das Nutzungsrecht ist auf die Laufzeit befristet. Die Zahlung erfolgt hier in Jahresraten (Miete), Sie erhalten bis zu 50% Preisnachlass im ersten Vertragsjahr für bereits vorhandene Konzernprodukte. Ein dauerhaftes Nutzungsrecht ist durch einen „buy out“ der Software nach Vertragsende möglich. Die  Mindestvertragslaufzeit beträgt 3 Jahre, für Open Value Subscription for Government ist nur 1 Jahr Laufzeit nötig

</ArrowList>

## Open License

### Abkündigung des Open License Programms


<Info>

  Microsoft Open License wurde zum 31. Dezember 2021 abgekündigt. Firmen, Behörden und Bildungseinrichtungen, die [Office 2019](/hersteller/microsoft/office/), [Server Produkte](/hersteller/microsoft/) ([Windows Server](/hersteller/microsoft/windows-server/) / [SQL Server](/hersteller/microsoft/sql-server/) / [SharePoint](/hersteller/microsoft/sharepoint/) / [Skype](/hersteller/microsoft/skype-for-business/) und andere Dauerlizenzen im Einsatz haben,
  können diese im Open Lizenzprogramm nicht mehr erweitern.
  Bestehende [Software Assurance (SA)](/hersteller/microsoft/software-assurance/) Wartungsverträge können im [Open Value Lizenzprogramm](#open-value) verlängert werden. Gerne beraten wir Sie beim Umstieg.

  Die Kauflizenzen sind künftig über [CSP](/hersteller/microsoft/csp/) und [Open Value](/hersteller/microsoft/open-value/) erhältlich.

  Eine Übersicht der Änderungen und Alternativen finden Sie in unserem Beitrag [Open License: Abkündigung zum 31.12.2021](/blog/open-license-abschaffung-aenderung-2021-2022/)


</Info>

 Ein Open-License Vertrag kommt ab einer Erstbestellung von 5 Lizenzen (oder einer sogenannten qualifizierten Lizenz) zustande. Der Vertrag läuft ab Ausstellung genau **2 Jahre**. Während dieser Laufzeit können jederzeit neue Bestellungen, auch für einzelne Lizenz, aufgeben werden. Wenn Sie nicht auf 5 Microsoft Produkte bei Ihrer ersten Open-License Bestellung kommen, hilft Ihnen unser Microsoft Team gerne weiter.

 Open Lizenzen können mit oder ohne [Software Assurance](/hersteller/microsoft/software-assurance/) für die Vertragsdauer von 2 Jahren erworben werden. Teilweise können auch bereits erworbene OEM- und System-Builder-Produkte (max. 90 Tage nach Erwerb) über den Kauf von Software Assurance in das Microsoft Open License Volumenlizenzprogramm eingebunden werden. Eine genauere Übersicht finden Sie auf der [Open-License Seite.](/hersteller/microsoft/open-license/)

 Das Open Lizenzprogramm ist in 4 unterschiedliche Programme unterteilt:

<ArrowList>

 - **Open-NL** - No Level; Einstiegslevel ab 5 Lizenzen oder einem qualifiziertem Produkt
 - **Open-NL Government** – Lizenzen für öffentliche Einrichtungen (Behörden)
 - **Open-NL Academic** – Education, Lizenzen für Bildungseinrichtungen
 - **Open-NL Charity** – Lizenzen für gemeinnützige Organisationen

</ArrowList>

## Microsoft Products & Services Agreement (MPSA)

 Microsoft Products & Services Agreement (MPSA) ist ein **unbefristeter Vertrag, der hybride Umgebungen (lokal und Cloud) berücksichtigt**. Es spielt keine Rolle, ob Volumen-Lizenzen oder Cloud-Dienste erworben werden. Auf diesen Vertrag können Sie mit unterschiedlichen, selbst definiterten Einkaufskonten für Ihr Unternehmen und auch mit diesem verbundenen Unternehmen, wie Töchter-, bzw. Muttergesellschaften, die Beschaffung weltweit vereinheitlichen. Für die Verwaltung der Lizenzen und Cloud-Dienste ist hier nicht das VLSC, sondern das sogenannte Microsoft Volume Licensing Center (MVLC), zuständig.

 MPSA ist der direkte Nachfolger des alten Select Plus Lizenzprogramms für Unternehmen und ab einer Größe von 250 Nutzern verfügbar.


## Enterprise Agreement

 Das Enterprise Agreement ([EA](/glossar/ea-enterprise-agreement-microsoft/)) ist ein Lizenzvertrag von Microsoft und richtet sich an große Organisationen und Unternehmen. EA bietet eine große und vielfältige Auswahl an Software Assurance Services an, um den Nutzen Ihrer Software zu maximieren- und Ihre Kosten zu minimieren. Durch eine **organisationsweite Standardisierung**, die Lizenzierung unter einem **weltweit gültigen Zentralvertrag** und individuellen Kombinationsmöglichkeiten von Online-Services und Softwarelizenzen, wird Ihre Softwareverwaltung optimiert. Hierbei bleibt Ihre persönliche IT-Strategie frei und flexibel, denn Sie entscheiden individuell ob oder wann Sie die Microsoft Online Services nutzen möchten. Darüber hinaus erhalten Sie über den gesamten Zeitraum Support für Ihre Microsoft Produkte und Services.

 Für eine Teilnahme am Enterprise Agreement ist eine Mindestanzahl von **500 Desktop-PCs** für Unternehmen, oder **250 Desktop-PCs** für Behörden, mit **Standardisierung für 3 Jahre** nötig.

 Eine genauere Beschreibung der einzelnen Aspekte finden Sie auf der [Microsoft Homepage](https://www.microsoft.com/de-de/licensing/licensing-programs/enterprise?activetab=enterprise-tab%3aprimaryr2?activetab=enterprise-tab%3aprimaryr2).


### EA Varianten:

<ArrowList>

 - **Enterprise Agreement (EA):** Der Volumenlizenzvertrag für große Unternehmen setzt die Standardisierung voraus. Das EA ermöglicht die einfache Kombination von Lizenzen für den Eigenbetrieb sowie den Microsoft Online Services. Das Nutzungsrecht ist unbefristet, die Zahlung erfolgt jährlich in Raten. Inklusive Software Assurance. Die Mindestvertragslaufzeit beträgt 3 Jahre.
 - **Enterprise Agreement Subscription (EAS):** Mit EAS erhalten Sie das Enterprise Agreement als Abonnement-Lizenzmodell. Das Nutzungsrecht von EAS ist entsprechend der Vertragslaufzeit befristet, die Zahlung erfolgt jährlich in Raten. Die Preise basieren auf der Anzahl der lizenzierten PCs. Die Mindestvertragslaufzeit beträgt 3 Jahre.

</ArrowList>

## Microsoft Software Assurance (SA)

 Die Microsoft Software Assurance (SA) ist der **Wartungs- und Supportvertrag** für Produkte, die in den Microsoft Volumenlizenzverträgen gekauft wurden und hat eine Laufzeit von 2 Jahren bei Open License oder 3 Jahren bei den anderen Volumenlizenzprogrammen wie Open Value oder Enterprise Agreement.

 Software Assurance bietet neben technischen Support-Leistungen und kostenlosen Upgrades auf die aktuellen Produktversionen zahlreiche Zusatzleistungen, die während der gesamten Laufzeit zur Verfügung stehen.

 Eine Übersicht der zahlreichen Zusatzleistungen, die Sie mit der Software Assurance nutzen können finden Sie auf unserer [Microsoft Software Assurance Seite](/hersteller/microsoft/software-assurance/).
