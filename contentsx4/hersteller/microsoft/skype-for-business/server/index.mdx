<Intro>

Das Release Date für die nächste Skype for Business Server Version ist noch nicht bekannt. Microsoft hat bereits 2020 angekündigt, dass es einen Nachfolger für den Skype for Business Server 2019 geben wird.
Vermutlich wird eine Microsoft Skype for Business Server Subscription Edition, analog zu SharePoint und Project, die bisherige Lizenzierung ersetzen.

Ob der Name des Nachfolgers Skype for Business Server 2022, vNext oder Subscription Edition sein wird, ist ebenfalls noch nicht bekannt. Langfristig wird Skype for Business allerdings, wie schon bei den Online Plänen, durch die neue Kommunikationsplattform Microsoft Teams ersetzt.


</Intro>

<TOC />



## Neuerungen und Änderungen mit dem Skype for Business Server 2022

 Die wichtigste Änderungen ist die Verabschiedung des Lizenzmodell Server-/CAL. Den Skype for Business Server 2022 wird es **nur noch als Subscription** geben. Das Abo-Modell wird wahrscheinlich im CSP zur Verfügung stehen. Ob es auch im Open-Value zur Verfügung stehen wird, ist nicht bekannt.



### Upgrades auf Skype for Business Server 2022

 Wie auch der Exchange Server wird die neue Version Skype for Business Server 2022 direkte Upgrade unterstützen. Das bedeutet, dass eine Migration von der Kauflizenz zur Mietlizenz ohne Probleme ablaufen soll.



## Skype for Business Server 2022 Editionen

 Bislang ist noch nichts über die Editionen oder CALs bekannt. Skype for Business 2019 gibt es in diesen Editionen:

 - [Skype for Business Server 2019](/hersteller/microsoft/skype-for-business/server-2019/)
 - [Skype for Business Server Standard CAL 2019](/hersteller/microsoft/skype-for-business/server-2019/standard-cal/)
 - [Skype for Business Server Enterprise CAL 2019](/hersteller/microsoft/skype-for-business/server-2019/enterprise-cal/)



## Modern Workplace mit Microsoft 365 und Teams


 [Microsoft Teams](/hersteller/microsoft/teams/) ist die moderne Lösung für effizientes Arbeiten in Gruppen. Auch mit externen Mitarbeitern (Lieferanten, Kunden, Freiberufler) lässt sich auf der Kommunikationsplatform eine nahtlose Zusammenarbeit über Browser oder als Desktop App ermöglichen, egal ob Sie im Homeoffice sind oder sich unterwegs befinden. Chat, Videokonferenzen, Telefonate und auch Dateien (Word, PowerPoint und Excel, PDFs usw.) können an einem Ort untergebracht werden. Damit erhalten Sie eine solide Grundlage für eine erfolgreiche Zusammenarbeit. Teams ist in den meisten Microsoft 365 Plänen bereits enthalten. Alle Informationen finden Sie auf unserer Seite:
