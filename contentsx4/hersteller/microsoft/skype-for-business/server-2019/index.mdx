## Skype for Business Server 2019 CALs

Skype for Business CALs gibt es in den Editionen [Standard](/hersteller/microsoft/skype-for-business/server-2019/standard-cal/), [Enterprise](/hersteller/microsoft/skype-for-business/server-2019/enterprise-cal/) und [Plus](/hersteller/microsoft/skype-for-business/server-2019/plus-cal/). Die Enterprise CAL ist, ebenso wie die Plus CAL, ein Add-On zur Standard CAL (setzt diese also voraus) und erweitert die Software um weitere Funktionen.

<ArrowList>

- [Skype for Business 2019 Standard CAL](/hersteller/microsoft/skype-for-business/server-2019/standard-cal/) - Die Standard-CAL beinhaltet die Instant Messaging und Anwesenheitsfunktionen, sowie Kommunikation via Audio oder Video zwischen mehreren PCs.
- [Skype for Business 2019 Enterprise CAL](/hersteller/microsoft/skype-for-business/server-2019/enterprise-cal/) - Mit der Enterprise CAL wird die Erstellung und Moderation von Audio-, Video- oder Webkonferenzen gemeinsam mit internen oder externen Anwendern möglich.
- [Skype for Business Server Plus CAL](/hersteller/microsoft/skype-for-business/server-2019/plus-cal/) - Mit der Plus CAL erhalten Sie zusätzliche hochwertige Sprachfunktionen (VoIP).

</ArrowList>


## Lizenzüberblick

Skype for Business Server 2019 ist die Serverlizenz für eine Server Instanz. Für den Zugriff werden zusätzlich [Skype for Business Server CALs](#skype-for-business-server-2019-cals) benötigt!
Eine genaue [Übersicht der Skype for Business Server Lizenzierung finden Sie hier](/hersteller/microsoft/skype-for-business/lizenzierung/).

<ArrowList>

- [SharePoint Lizenzierung](/hersteller/microsoft/sharepoint/lizenzierung/)<br />
  Ausführliche Informationen zu Kauf und Lizenzierung.

- [Standard CAL](/hersteller/microsoft/sharepoint/server-standard-cal/)<br />
  Jeder Zugriff auf den SharePoint Server benötigt eine Zugriffslizenz (CAL)!

- [Enterprise CAL](/hersteller/microsoft/sharepoint/server-enterprise-cal/)<br />
  Upgrade für die Standard CAL mit zusätzlichen Funktionen.

- [CSP](/hersteller/microsoft/csp/)<br />
  Dauerhafte Lizenz (Kauflizenz) ohne Wartung. Keine Mindestbestellmenge. Keine Sonderkonditionen für Behörden.

- [Open Value](/hersteller/microsoft/open-value/)<br />
  Volumenlizenz mit 3 Jahren Software Assurance. Mindestbestellmenge: 5 Lizenzen.

</ArrowList>
