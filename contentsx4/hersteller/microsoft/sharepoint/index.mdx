<Intro>

 SharePoint ist die Microsoft Plattform für FileSharing. Kann als SharePoint Online Plan aus der Microsoft 365 Cloud oder als on-premises SharePoint Server mit CALs gekauft werden.
 Die aktuellste SharePoint Version erhalten Sie mit der SharePoint Server Subscription Edition. Unsere Seiten richten sich hauptsächlich an IT-Einkäufer und Virtualisierungsexperten für die Beschaffungsplanung. Mit wenigen Klicks erhalten Sie ein Angebot.


</Intro>

## SharePoint Server Subscription Edition

 Mit der Subscription Edition erhalten Sie die neueste SharePoint Version als Abonnement. Manchmal auch als SharePoint Server 2022 bezeichnet. Enthält eine dauerhaft gültige SharePoint Server 2019 Lizenz nach Ablauf der Subscription Laufzeit.
 Für den Zugriff werden CALs benötigt.

<ArrowItem>

**[SharePoint Server Subscription Edition](/hersteller/microsoft/sharepoint/server-subscription-edition/)** (SharePoint Server 2022)

</ArrowItem>


## SharePoint Server 2019

SharePoint Server 2019 ist die Basis für lokales Filesharing. Zur Lizenzierung werden eine Serverlizenz und Standard CALs für Zugriffe benötigt.
Zusätzliche Funktionen erhalten Sie mit der Enterprise CAL.

<ArrowItem>

**[SharePoint Server 2019](/hersteller/microsoft/sharepoint/server-2019/)**

</ArrowItem>


## SharePoint Server CALs

Eine Standard CAL wird immer für den Zugriff auf eine der beiden SharePoint Server Editionen benötigt. Zugriffe können nach Nutzern (User CAL) oder Geräten (Device CAL) lizenziert werden.
Enterprise CALs sind ein Add-on (Upgrade) und bieten erweiterte Funktionen.

<ArrowList>

- **[SharePoint Server Standard CAL](/hersteller/microsoft/sharepoint/server-standard-cal/)**<br />
  Enthält alle grundlegenden SharePoint Funktionen (Websites, Communitys, Inhalte, Suche). Wird immer für den Zugriff auf einen SharePoint Server benötigt. Zugriffslizenzen gibt es für Nutzer oder Geräte.

- **[SharePoint Server Enterprise CAL](/hersteller/microsoft/sharepoint/server-enterprise-cal/)**<br />
  Enterprise CALs stellen den vollen Funktionsumfang von SharePoint zur Verfügung. Enterprise CALs sind ein Add-on und benötigen eine vorhandene Standard CAL.

</ArrowList>


## SharePoint Online

SharePoint kann, auch ohne eigenen Server, aus der Microsoft Cloud genutzt werden.
SharePoint Online wird immer als benutzerbasiertes Abonnement lizenziert. Es gibt zwei Pläne mit unterschiedlicher Funktionalität:

<ArrowList>

- **[SharePoint Online Plan 1](/hersteller/microsoft/sharepoint/online-plan-1/)**<br />
  Bietet Standardfunktionen für Filesharing und zur Zusammenarbeit in Teams und Gruppen. Dateien können geteilt werden und der Zugriff ist über beliebige Endgeräte (Tablets, Smartphones, Windows und Mac Desktops) möglich.

- **[SharePoint Online Plan 2](/hersteller/microsoft/sharepoint/online-plan-2/)**<br />
 SharePoint Online Plan 2 enthält alle Features aus Plan 1 und zusätzlich weitere Services. U.a. sind eDiscovery, erweiterte Suche, Office Web-Viewer und ein permanentes Dokuementenarchiv enthalten.

</ArrowList>
