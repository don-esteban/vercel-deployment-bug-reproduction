## Lizenzüberblick

SharePoint Server Subscription Edition ist ausschließlich im [Open Value](/hersteller/microsoft/open-value/) Lizenzprogramm erhältlich und benötigt aktive [Software Assurance](/hersteller/microsoft/software-assurance/) für Serverlizenz und CALs (siehe: [Microsoft Product Terms](https://www.microsoft.com/licensing/terms/productoffering/SharePointServer/OVOVS#Documents)).
Enthält eine perpetual [SharePoint Server 2019](/hersteller/microsoft/sharepoint/server-2019/) Lizenz, die nach Ablauf der Subscription weiterhin genutzt werden kann.

Für den Zugriff werden zusätzlich [CALs](/hersteller/microsoft/sharepoint/server-standard-cal/) benötigt! Alternativ kann [SharePoint Online Plan 1](/hersteller/microsoft/sharepoint/online-plan-1/) wie eine [Standard CAL](/hersteller/microsoft/sharepoint/server-standard-cal/) verwendet werden.
[SharePoint Online Plan 2](/hersteller/microsoft/sharepoint/online-plan-2/) bietet die Funktionen der [Enterprise CAL.](/hersteller/microsoft/sharepoint/server-enterprise-cal/)

<ArrowList>

- [Standard CAL](/hersteller/microsoft/sharepoint/server-standard-cal/)<br />

- [Enterprise CAL](/hersteller/microsoft/sharepoint/server-enterprise-cal/)<br />

- [Open Value](/hersteller/microsoft/open-value/)<br />
  Volumenlizenz mit 3 Jahren Software Assurance. Mindestbestellmenge: 5 Lizenzen.
