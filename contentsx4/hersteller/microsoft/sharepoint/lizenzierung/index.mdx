<Intro>

Die Lizenzierung des SharePoint Servers unterscheidet sich je nach gewählter Bereitstellungsoption (SharePoint Server 2019 oder SharePoint Online Plan).
Auch die verschiedenen Lizenzprogramme unterscheiden sich bei den enthalten Rechten. Hier finden Sie eine Übersicht zur SharePoint-Lizenzierung.


</Intro>

## SharePoint Server 2019 Lizenzierung

<ArrowList>

- Lizenziert wird nach dem [Microsoft Server-/CAL-Lizenzmodell](/hersteller/microsoft/lizenzierung/#server-cal-lizenzierung)
- Jede Betriebssysteminstanz, in der ein SharePoint Server verwendet wird, benötigt eine Serverlizenz.
- Jeder Zugriff auf SharePoint Server erfordert eine (Standard-)CAL.
- Eine CAL berechtigt zum Zugriff auf alle SharePoint Server im Unternehmen.
- Zugriffe können nach Benutzern (User CAL) oder Geräten (Device CAL) lizenziert werden.
- Eine Enterprise CAL ist ein Upgrade für Standard CALs und erweitert die Funktionalität.
- CALs sind versionsgebunden und abwärtskompatibel.
- Kauflizenzen erhalten Sie im [CSP](/hersteller/microsoft/csp/).
- Volumenlizenzen mit 3 Jahren Software Assurance (SA) erhalten Sie im [Open Value](/hersteller/microsoft/open-value/).

</ArrowList>

## SharePoint Online Plan Lizenzierung

<ArrowList>

- Lizenziert wird als Abonnement-Lizenz ([User Subscription License (USL)](#user-subscription-license-usl)).
- Jeder Benutzer benötigt eine Lizenz.
- Die Abonnement Laufzeit beträgt ein Jahr.
- Abonnement-Lizenzen erhalten Sie im [CSP](/hersteller/microsoft/csp/) oder im [Open Value Volumenlizenzprogramm](/hersteller/microsoft/open-value/).
- Es sind keine Sonderkonditionen für Bildungseinrichtungen oder gemeinnützige Organisationen verfügbar. Einzige Ausnahme ist EOA.
- Lizenzierte Benutzer dürfen die Lizenz auf bis zu 5 Geräten installieren und nutzen (nur CSP).

</ArrowList>


<Info>

**Dual Access Right**

SharePoint Online Pläne enthalten das Dual Access Right und damit die [SharePoint Standard CAL](/hersteller/microsoft/sharepoint/server-standard-cal/) zum Zugriff auf on-premises SharePoint Server. Mehr Informationen in unserem Beitrag [Exchange Online ist die bessere CAL](/blog/exchange-online-ist-die-bessere-cal/).

</Info>

### User Subscription License (USL)

Die Lizenzierung von SharePoint Online ist benutzerorientiert. Pro Mitarbeiter wird ein Abonnement benötigt. Lizenztechnisch heißt das [User Subscription License (USL)](/hersteller/microsoft/office-365/lizenzierung/). Jede USL wird im Admin-Portal (Azure AD) einem Mitarbeiter zugeordnet. Die Software kann von lizenzierten Nutzer auf bis zu 5 Devices installiert und genutzt werden.
Abonnements haben normalerweise eine Laufzeit von einem Jahr. Es ist aber auch eine monatliche oder dreijährige Laufzeiten möglich. Während des Abonnements haben Sie einen Preisschutz.

### New Commerce Experience (NCE)

CSP Subscriptions für Unternehmen sind auch unter dem Namen [NCE (New Commerce Experience)](/blog/aenderungen-am-microsoft-csp-programm-nce/) bekannt.
Bei der Lizenzierung für Unternehmen hat Microsoft die ursprünglichen CSP-Einkaufskonditionen im März 2022 angepasst. Ausführliche Informationen zu den Änderungen finden Sie in [unserem Blogbeitrag zur New Commerce Experience](/blog/aenderungen-am-microsoft-csp-programm-nce/).


## Lizenzprogramme und Bezugswege

Auf unserer Seite zu den [Microsoft Lizenzprogrammen](/hersteller/microsoft/lizenzprogramme/) finden Sie eine vollständige Übersicht.


### CSP Kauflizenzen

  Kauflizenzen für gewerbliche Kunden sind im [CSP-Lizenzprogramm](/hersteller/microsoft/csp/) erhältlich. Es gibt hier keine Mindestbestellmenge und auch keine Software Assurance. Bildungseinrichtungen und gemeinnützige Organisationen erhalten Sonderkonditionen. Behörden erhalten dieselben Einkaufskonditionen, wie Unternehmen.
  Das [Downgrade-Recht](/blog/microsoft-downgrade-rechte/), zum Einsatz älterer Versionen, ist aber enthalten. Software Sie im CSP Portal herunterladen.

### CSP Subscription

  [CSP-Subscriptions](/hersteller/microsoft/csp/) sind der normale Bezugsweg für SharePoint Online Pläne. Lizenziert wird normalerweise als einjähriges Abonnement. Es gibt hier keine Mindestbestellmenge. Kostenlose Upgrades auf neue Versionen sind enthalten.
  Für kommerzielle Unternehmen werden die CSP Subscriptions auch als [NCE - New Commerce Experience](#new-commerce-experience-nce) bezeichnet.

### Open Value Volumenlizenzen

  [Open Value Volumenlizenzen](/hersteller/microsoft/open-value/) enthalten immer Software Assurance (SA) und können als Kauflizenz mit 3 Jahren Software Assurance im Voraus oder auf Raten gekauft werden. Es gibt Sonderkonditionen für Behörden, Bildungseinrichtungen und gemeinnützige Organisationen.
  Seit Ende 2021 werden Open License Kunden bei SA-Verlängerungen auf das Open Value Lizenzprogramm umgestellt. Beachten Sie bitte, dass sich hierbei die Vertragslaufzeit von 2 Jahren (Open License) auf 3 Jahre (Open Value) ändert. Gerne beraten wir Sie beim Wechsel.

### Open License Volumenlizenzen

  Open License ist seit Anfang 2022 nicht mehr verfügbar. Im Microsoft [Open Volumenlizenzen](/hersteller/microsoft/open-license/) waren Lizenzen mit optionaler, zweijähriger Wartung (SA - Software Assurance) verfügbar. SA-Wartungsverlängerungen können weiterhin im Open Value Programm durchgeführt werden. Kauflizenzen erhalten Sie jetzt im CSP. Weitere Details zum ehemaligen Lizenzprogramm finden Sie auf unserer Seite zu [Microsoft Open License](/hersteller/microsoft/open-license/).
