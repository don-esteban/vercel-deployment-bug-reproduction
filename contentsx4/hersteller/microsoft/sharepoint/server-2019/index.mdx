<Intro>

  Der SharePoint Server 2019 ermöglicht die SharePoint on premise Installation. Beim Zugriff auf den SharePoint Server wird für jeden Nutzer oder jedes Gerät eine Zugriffslizenz (Standard-CAL) benötigt.
  Die aktuelle SharePoint Version erhalten Sie ausschließlich mit der ShrePoint Server Subscription Edition. Nutzen Sie unsere Sofortangebote zur Beschaffungsplanung!

</Intro>

<Info>

  Die Funktionalität des SharePoint Servers richtet sich nach der CAL Edition: [Standard CALs](/hersteller/microsoft/sharepoint/server-standard-cal/) bieten die Basisfunktionen. Enterprise CALs enthalten zusätzliche Funktionen für Business Solutions und Business Intelligence.

</Info>


## Lizenzüberblick

SharePoint Server Standard 2019 ist die Serverlizenz für eine SharePoint Instanz. Für den Zugriff werden zusätzlich [CALs](/hersteller/microsoft/sharepoint/server-standard-cal/) benötigt!

<ArrowList>

- [SharePoint Lizenzierung](/hersteller/microsoft/sharepoint/lizenzierung/)<br />
  Ausführliche Informationen zu Kauf und Lizenzierung.

- [Standard CAL](/hersteller/microsoft/sharepoint/server-standard-cal/)<br />
  Jeder Zugriff auf den SharePoint Server benötigt eine Zugriffslizenz (CAL)!

- [Enterprise CAL](/hersteller/microsoft/sharepoint/server-enterprise-cal/)<br />
  Upgrade für die Standard CAL mit zusätzlichen Funktionen.

- [CSP](/hersteller/microsoft/csp/)<br />
  Dauerhafte Lizenz (Kauflizenz) ohne Wartung. Keine Mindestbestellmenge. Keine Sonderkonditionen für Behörden.

- [Open Value](/hersteller/microsoft/open-value/)<br />
  Volumenlizenz mit 3 Jahren Software Assurance. Mindestbestellmenge: 5 Lizenzen.

</ArrowList>
