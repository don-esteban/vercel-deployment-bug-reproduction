<Intro>

  SharePoint Online ist eine interessante Alternative zum on-premises [SharePoint Server 2019](/hersteller/microsoft/sharepoint/server-2019/) und zu den SharePoint Standard und Enterprise CALs. Oft wird übersehen, dass SharePoint Online auch CALs für den Zugriff auf lokale SharePoint Server enthält und damit auch interessant ist, wenn man den Dienst aus der Cloud gar nicht nutzen möchte.
  Erfahren Sie Vor- und Nachteile von SharePoint aus der Cloud vs. SharePoint Server 2019.

</Intro>

## SharePoint Online

 SharePoint Online wird pro Benutzer lizenziert und ist als Abonnement erhältlich. Für SharePoint Online gibt es zwei unterschiedliche Stand-Alone Pläne, außerdem ist es in vielen Microsoft 365 Plänen enthalten:

 - [SharePoint Online Plan 1:](/hersteller/microsoft/sharepoint/online-plan-1/) Teamwebsites, Arbeitsverwaltung, Inhaltsverwaltung, Standard-Suche. Enthalten ist der Plan 1 auch in [Office 365 Business Essentials](/hersteller/microsoft/microsoft-365/) und in [Office 365 Business Premium](/hersteller/microsoft/microsoft-365/).
 - [SharePoint Online Plan 2:](/hersteller/microsoft/sharepoint/online-plan-2/) Zusätzliche eDiscovery und Business Intelligence Funktionen sowie die Enterprise-Suche. Enthalten ist der Plan 2 auch in [Office 365 Enterprise E1](/hersteller/microsoft/office-365/enterprise-e1/), [Office 365 Enterprise E3](/hersteller/microsoft/office-365/enterprise-e3/) und in [Office 365 Enterprise E5](/hersteller/microsoft/office-365/enterprise-e5/).

 Auf unseren Seiten finden Sie Preise für die [Microsoft 365 Abonnements](/hersteller/microsoft/microsoft-365/) und die einzelnen [SharePoint Online Pläne.](/hersteller/microsoft/sharepoint/online-plan-1/)
 Eine Übersicht welche Pläne in den Microsoft 365 Paketen enthalten sind, finden Sie ebenfalls auf [unserer SharePoint Seite.](/hersteller/microsoft/sharepoint/#sharepoint-online-in-microsoft-365-plänen)


## SharePoint CAL und Dual Access Right

 SharePoint Online enthält SharePoint Server Subscription Edition CALs. Über das Dual Access Right ist der Zugriff auf On Prem SharePoint Server gestattet. Dadurch wird die hybride Nutzung der Online Services und eigener SharePoint Server ohne zusätzliche Kosten ermöglicht.

 - **SharePoint Standard CAL** ist in SharePoint Online Plan 1, Plan 2, Microsoft 365 Enterprise E1, E3 und E5 enthalten
 - **SharePoint Enterprise CAL** ist in SharePoint Online Plan 2, Microsoft 365 Enterprise E1, E3 und E5 enthalten. Rufen Sie mich gerne an.

 **Tipp:** Auch, wenn Sie heute noch keine Cloud-Dienste nutzen, sollten Sie vor Anschaffung neuer CALs (SharePoint, Teams, Exchange) prüfen, ob ein entsprechender Microsoft 365 Plan günstiger ist.




## SharePoint Server Subscription Edition

 Eine [SharePoint Server 2019 Lizenz](/hersteller/microsoft/sharepoint/server-subscription-edition/) ist für jede ausgeführte Server-Instanz erforderlich. Für jeden Zugriff auf den SharePoint Server wird zusätzlich eine Zugriffslizenz (CAL) benötigt.
 Hier können die zugreifenden Nutzer (User CAL) oder Geräte (Device CAL) lizenziert werden. Die CALs gibt es in 2 unterschiedlichen Editionen, wobei die Enterprise CAL ein Add-On ist und die Standard CAL voraussetzt.

## SharePoint Server Subscription Edition

 Eine [SharePoint Server Subscription Edition](/hersteller/microsoft/sharepoint/server-subscription-edition/) ist für jede ausgeführte Server-Instanz erforderlich. Für jeden Zugriff auf den SharePoint Server wird zusätzlich eine Zugriffslizenz (CAL) benötigt.


### SharePoint Standard-CAL

Die Standard-CAL stellt die Hauptfunktionen von SharePoint zur Verfügung:

 - Websites: eine einzige Infrastruktur für alle Ihre Geschäftswebsites
 - Communitys: eine integrierte Plattform für die Zusammenarbeit
 - Inhalte: ECM für alle
 - Suche: Personen, Personenkompetenzsuche, visuelle Vorschau, visuelle beste Suchergebnisse


### SharePoint Enterprise CAL

Die Enterprise CAL stellt den vollen Funktionsumfang von SharePoint zur Verfügung:
 - Websites: eine einzige Infrastruktur für alle Ihre Geschäftswebsites
 - Communitys: eine integrierte Plattform für die Zusammenarbeit
 - Inhalte: ECM für alle
 - Suche: Standardsuchfunktionen plus Entitätsextraktion, Videosuche, Empfehlungen zu Elementen
 - Business Solutions (enthält Access Services und InfoPath Services)
 - Business Intelligence für jeden (enthält Power View, PerformancePoint Services, Excel Services und Visio Services)


Für Extranet- und Internetwebsite-Szenarios mit externen Benutzern, sind keine CALs erforderlich.

 - Externe Benutzer sind Benutzer, die weder Mitarbeiter im Unternehmen noch Mitarbeiter von Partnern oder Vor-Ort-Vertragsnehmer sind.
 - SharePoint-Kunden mit Software Assurance (SA) können [Downgrade-Rechte](/blog/microsoft-downgrade-rechte/) aus Volumenlizenzen nutzen, um SharePoint 2016 Bits mit SharePoint 2019-Benutzerrechten auszuführen.

 Preise für Server-Lizenzen und Zugriffslizenzen finden Sie auf unserer Seite zum [SharePoint Server Subscription Edition](/hersteller/microsoft/sharepoint/server-subscription-edition/).
