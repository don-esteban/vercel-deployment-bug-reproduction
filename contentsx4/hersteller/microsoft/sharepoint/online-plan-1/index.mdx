<Intro>

 SharePoint Online Plan 1 bietet die Grundfunktionen für Filesharing über beliebige Endgeräte (Tablets, Smartphones, Windows und Mac Desktops).
 Kann als Standard CAL für SharePoint Server Subscription Edition und SharePoint Server 2019 genutzt werden.
 Über unseren Warenkorb erhalten Sie mit wenigen Klicks ein Sofortangebot.

## Lizenzüberblick

SharePoint Online Plan 1 wird als benutzerbasiertes Abonnement ([User Subscription License - USL](/hersteller/microsoft/exchange/lizenzierung/#user-subscription-license-usl)) gekauft. Die Laufzeit mit Preisschutz beträgt wahlweise ein Jahr oder einen Monat.
Kann als [SharePoint Server Standard CAL](/hersteller/microsoft/sharepoint/server-standard-cal/) verwendet werden.

<ArrowList>

- [SharePoint Lizenzierung](/hersteller/microsoft/sharepoint/lizenzierung/)<br />
  Ausführliche Informationen zur Lizenzierung.

- [CSP Subscription](/hersteller/microsoft/csp/)<br />
  Abonnement-Lizenz. Die Laufzeit mit Preisschutz beträgt wahlweise ein Jahr oder einen Monat. Keine Mindestbestellmenge.

- [Open Value](/hersteller/microsoft/open-value/)<br />
  Einjährige Abonnement-Lizenz mit Software Assurance. Mindestbestellmenge: 5 Lizenzen.

</ArrowList>

<Info>

 **Dual Access Right**

 Enthält die [SharePoint Server Standard CAL](/hersteller/microsoft/sharepoint/server-standard-cal/) zum Zugriff auf on-premises [SharePoint Server Subscription Edition](/hersteller/microsoft/sharepoint/server-subscription-edition/) und [SharePoint Server 2019](/hersteller/microsoft/sharepoint/server-2019/).
 Mehr Informationen in unserem Beitrag [Exchange Online ist die bessere CAL](/blog/exchange-online-ist-die-bessere-cal/).

</Info>


## Funktionen

SharePoint Online Plan 1 beinhaltet 1 TB online Speicher mit [OneDrive for Business Plan 1](/hersteller/microsoft/microsoft-365/onedrive-for-business-plan-1/), SharePoint Workflows und Suchfunktionen, Portale und Team-Webseiten, sowie einen Navigator für iOS, Android und Windows.
Erweiterte Funktionen zur Datenvisualisierung, eine verbesserte Enterprise-Suchfunktion oder optionale Speicherplatzerweiterungen erhalten Sie mit dem [SharePoint Online Plan 2](/hersteller/microsoft/sharepoint/online-plan-2/).
