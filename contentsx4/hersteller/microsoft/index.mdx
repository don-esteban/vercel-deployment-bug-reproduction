<Intro>

 Hier finden Sie alle Informationen und Preise zu Microsoft Lizenzen, Produkten und Services. Neben den modernen Microsoft 365 und Office 365 Cloud Plänen bietet Microsoft weiterhin die bekannten Legacy-Systeme (Windows, SQL Server, Office, Visual Studio, etc).
 Die Seiten richten sich hauptsächlich an IT-Einkäufer. Nutzen Sie unsere Sofortangebote zur Beschaffungsplanung.

</Intro>

## Microsoft Produktsortiment

### Azure

Cloud-Plattform für Rechenkapazität (Computing), Speicher (Storage) und IT-Dienste (Services). Typische Einsatzgebiete sind Infrastruktur-Tests, Systementwicklung, Outsourcing, Ausgleich von Spitzenlasten und Nutzung als virtuelles Rechenzentrum.
Im Gegensatz zu den [Microsoft 365 Plänen](#microsoft-365), wird abhängig von der Ressourcennutzung lizenziert.

<ArrowItem>

**[Azure Plan](/hersteller/microsoft/azure/)**

</ArrowItem>


### Copilot

Mit Microsoft Copilot können Sie die Vorteile von Künstlicher Intelligenz (KI) im Unternehmen nutzen. Verwandeln Sie Ihre Wörter und Ideen in eines der leistungsstärksten Produktivitätswerkzeuge der Welt.
Die Copilot Nutzung bietet Vorteile für alle Abteilungen im Unternehmen: Von der Geschäftsleitung über die IT, bis hin zur Buchhaltung bietet Copilot zahlreiche Möglichkeiten zur Vereinfachung und Automatisierung von Workflows oder der verbesserten Zusammenarbeit.

<ArrowItem>

**[Microsoft Copilot](/hersteller/microsoft/copilot/)**

</ArrowItem>


### CAL Suite

CAL Suites enthalten Zugriffslizenzen (Client Access License) für mehrere Microsoft Server (u.a. Windows Server, SharePoint, RDS) und bieten Preisvorteile gegenüber dem Kauf einzelner CALs.
Hier finden Sie auch eine Übersicht aller Microsoft CALs.

<ArrowItem>

**[CAL Suite](/hersteller/microsoft/cal-suite/)**

</ArrowItem>

### Enterprise Mobility + Security

  Enterprise Mobility + Security (EMS) ist eine Cloud-Plattform für den Schutz von mobilen Endgeräte.
  EMS ist zusammen mit [Microsoft Surface Business](#surface-for-business) und [Microsoft 365](#microsoft-365) Teil von Microsofts Modern Workplace Strategie.

<ArrowItem>

**[Enterprise Mobility + Security](/hersteller/microsoft/enterprise-mobility-security/)**

</ArrowItem>


### Exchange

Unified Messaging Plattform mit E-Mail Postfach, Kontakten, Kalender und Aufgabenverwaltung. Kann als [Exchange Server 2019](/hersteller/microsoft/exchange/#exchange-server-2019) on-premises oder als bereits eingerichteter [Exchange Online Plan](/hersteller/microsoft/exchange/#exchange-online-pläne) lizenziert werden.
Viele [Office 365](#office-365) und [Microsoft 365](#microsoft-365) Pläne enthalten bereits das Cloud Postfach Exchange Online.

<ArrowItem>

**[Exchange Server 2019 / Exchange Online](/hersteller/microsoft/exchange/)**

</ArrowItem>


### Intune

Intune ist ein cloud-basierter Service für Verwaltung und Kontrolle von Endgeräten. Intune ist eine zentrale Komponente der Modern Workplace Strategie: [Microsoft 365](#microsoft-365) + [Microsoft Surface Business](#surface-for-business) = Microsoft Modern Workplace.
In vielen [Microsoft 365](#microsoft-365) Plänen ist Intune bereits enthalten.

<ArrowItem>

**[Intune](/hersteller/microsoft/intune/)**

</ArrowItem>


### Microsoft Entra

Die Microsoft Entra Produkte bieten Lösung für die Verwaltung von Identitäten und Netzwerkzugriffen. Die Benutzer- und Rechte-Verwaltung Microsoft Entra ID ist in den [Microsoft 365 Enterprise Plänen](#microsoft-365) bereits enthalten.
Microsoft Entra ID ist auch unter dem alten Name Azure Active Directory (AAD) bekannt.

<ArrowItem>

**[Microsoft Entra](/hersteller/microsoft/entra/)**

</ArrowItem>


### Microsoft 365

Microsoft 365 Pläne enthalten meist die Lizenz für das [Windows Betriebssystem](#windows) und [Office 365](#office-365) für die Nutzung des Office Pakets mit E-Mail und FileSharing aus der Microsoft Cloud.
Außerdem sind die Sicherheitsfunktionen von [Enterprise Mobility + Security (EMS)](#enterprise-mobility--security) enthalten. Microsoft 365 bildet zusammen mit [Microsoft Surface](#surface-for-business) die Grundlage moderner IT-Arbeitsplätze.

<ArrowItem>

**[Microsoft 365](/hersteller/microsoft/microsoft-365/)**

</ArrowItem>


### Microsoft 365 Defender

Einheitliche Enterprise-Defense-Suites mit Antivirus und weiteren Sicherheitsfunktionen. Vormals als Advanced Threat Protection (ATP) bekannt. Defender-Suites bieten zusätzlichen Schutz für Endpunkte, Identitäten, E-Mails und Anwendungen.

<ArrowItem>

**[Microsoft 365 Defender Services](/hersteller/microsoft/microsoft-365/defender/)**

</ArrowItem>


### Office 365

Office 365 bietet die Funktionen von [Exchange](#exchange) (E-Mail), [SharePoint](#sharepoint) (Filesharing) und [Office](#office) als Dienste aus der Microsoft 365 Cloud.
Die meisten Office 365 Pläne sind mittlerweile in [Microsoft 365](#microsoft-365) integriert.

<ArrowItem>

**[Office 365](/hersteller/microsoft/office-365/)**

</ArrowItem>


### Office

Das traditionelle Office Paket ist weiterhin in unterschiedlichen Editionen oder auch einzeln verfügbar. Office 2021 enthält Word, Excel, PowerPoint, Outlook und weitere Programme. Das Nutzungsrecht ist hier zeitlich unbegrenzt. Microsoft hat mittlerweile [Office 2024](/hersteller/microsoft/office/2024/) für den Herbst angekündigt.

<ArrowItem>

**[Office 2021](/hersteller/microsoft/office/)**

</ArrowItem>


### Power Platform

Die Microsoft Power Platform ist eine integrierte Lösung für Business Intelligence mit den Komponenten ([Power BI](/hersteller/microsoft/power-platform/power-bi/)), Applikationsentwicklung ([Power Apps](/hersteller/microsoft/power-platform/power-apps/)) und Workflow Automatisierung ([Power Automate](/hersteller/microsoft/power-platform/power-automate/)).
Für die meisten [Microsoft 365 Cloud Pläne](#microsoft-365) kann die Power Platform als Add-on dazu gebucht werden.

<ArrowItem>

**[Power Platform](/hersteller/microsoft/power-platform/)**

</ArrowItem>


### Project

MS Project ist eine selbstverwaltete (Project 2021) oder cloud-basierte (Project Online) Projektmanagement Lösung. Mit dem Project Server kann die Projektmanagement Infrastruktur auch on-premises genutzt werden.
Die Project Online Pläne enthalten die selbe Funktionalität, meist mit geringerem Aufwand und Kosten.

<ArrowItem>

**[Project 2021 / Project Online](/hersteller/microsoft/project/)**

</ArrowItem>


### SharePoint

Die Plattform für Zusammenarbeit im Team und zum Austauschen von Dateien.
SharePoint Server 2019 steht zur eigenen on-premises Installation oder out-of-the-box als SharePoint Online Service zur Verfügung. In den meisten [Microsoft 365](#microsoft-365) Plänen ist SharePoint Online bereits enthalten.

<ArrowItem>

**[SharePoint Server 2021 / SharePoint Online](/hersteller/microsoft/sharepoint/)**

</ArrowItem>


### Skype for Business

Skype for Business (SfB) ist eine ältere Microsoft Plattform für Chat (Instant Messaging), Audio-/Video-Konferenzen und Telefonie. Der Skype for Business Server 2019 kann weiterhin on-premises lizenziert werden.
Skype for Business Online Pläne sind bereits abgekündigt und wurden durch die [Microsoft 365 Pläne](#microsoft-365) mit [Microsoft Teams](/hersteller/microsoft/teams/) ersetzt.

<ArrowItem>

**[Skype for Business Server 2019](/hersteller/microsoft/skype-for-business/)**

</ArrowItem>


### SQL Server

Das relationale SQL-Datenbankmanagementsystem (DBMS) von Microsoft ist eine flexible Universalplattform für die Datenauswertung. Der SQL Server 2022 kann on-premises eingesetzt werden oder als Azure SQL Database (Azure SQL-DB) als Dienst aus der [Azure Cloud](#azure) genutzt werden.

<ArrowItem>

**[SQL Server 2022](/hersteller/microsoft/sql-server/)**

</ArrowItem>


### Surface for Business

Software-Express ist auch autorisierter Handelspartner für Microsoft Surface Business-Geräte. Entdecken Sie alle Surface Modelle für Unternehmen und Bildungseinrichtungen in unserem Online Shop [Surface-World](https://surface-world.de).

Surface-World hilft Ihnen beim Einrichten eines modernen IT-Arbeitsplatzes mit Microsoft Surface for Business. Kontaktieren Sie unser Surface-World Team für attraktive Business-Preise und Beratung von unseren Produktexperten.

<ArrowItem>

**[Surface-World](https://surface-world.de)**

</ArrowItem>


### System Center

System Center 2022 ist Microsofts traditionelle on-premises Plattform zur Verwaltung und Automatisierung von Rechenzentren und IT-Infrastrukturen.
Mit der Cloud Lösung [Intune](#intune) erhalten Sie einen vergleichbaren Funktionsumfang, ohne eigene Server-Infrastruktur.

<ArrowItem>

**[System Center 2022](/hersteller/microsoft/system-center/)**

</ArrowItem>


### Visio

Visio bietet Funktionen zur Erstellung von Flowcharts, Netzwerk-Diagrammen oder UML-Charts. Kann als Visio Online Plan aus der Cloud oder als traditionelle Visio 2021 Dauerlizenz gekauft werden.

<ArrowItem>

**[Visio 2021 / Visio Online](/hersteller/microsoft/visio/)**

</ArrowItem>


### Visual Studio

Die integrierte Entwicklungsumgebung (IDE - integrated development environment) von Microsoft. Hier finden Sie auch Informationen zu weiteren Development-Services, wie [MSDN for Platforms](/hersteller/microsoft/visual-studio/msdn-for-platforms/) oder [Azure DevOps.](/hersteller/microsoft/visual-studio/azure-devops/)

<ArrowItem>

**[Visual Studio 2022](/hersteller/microsoft/visual-studio/)**

</ArrowItem>

### Windows

Windows 11 ist das Standard Desktop-Betriebssystem von Microsoft. Windows 11 Pro wird pro Gerät lizenziert. Die Windows Enterprise Edition erhalten Sie als geräte- oder nutzer-basierte Update-Lizenz.
Mit einer Windows 11 Lizenz darf meist auch die Vorgängerversion Windows 10 genutzt werden.

<ArrowItem>

**[Windows 11 / Windows 10](/hersteller/microsoft/windows/)**

</ArrowItem>


### Windows Server

Standard Server-Betriebssystem von Microsoft. Lizenziert wird nach Prozessor-Kernen. Zusätzlich werden Zugriffslizenzen (CALs) benötigt.
Hier finden Sie auch Lizenzen für den Zugriff externer Nutzer ([External Connector](/hersteller/microsoft/windows-server/external-connector/)) oder Terminalserver-Zugriff ([RDS CAL](/hersteller/microsoft/windows-server/remote-desktop-services/)). Microsoft hat den neuen [Windows Server 2025](/hersteller/microsoft/windows-server/2025/) für Herbst angekündigt.

<ArrowItem>

**[Windows Server 2022](/hersteller/microsoft/windows-server/)**

</ArrowItem>
