<Intro>

  Open License ist Microsofts Programm für den Kauf von Volumenlizenzen für kleinere und mittelständische Unternehmen (bis 500 PCs). Mit Open License erhalten Sie Zugriff auf das [Volumenlizenzportal VLSC (Volume Licensing Service Center)](https://www.microsoft.com/Licensing/servicecenter/default.aspx) zum Download und zur Verwaltung Ihrer Microsoft Produkte und Lizenzschlüssel, sowie zur Aktivierung zusätzlicher Rechte von Software Assurance.

  Das Open License Programm ist seit 2022 nicht mehr erhältlich. Kauflizenzen sind im [Cloud Solution Provider](/hersteller/microsoft/csp/) Modell, Lizenzen mit Software Assurance im [Open Value Lizenzprogramm](/hersteller/microsoft/open-value/) verfügbar.


</Intro>

## Abkündigung des Open License Programms

  Microsoft Open License wurde zum 31. Dezember 2021 abgekündigt. Firmen, Behörden und Bildungseinrichtungen, die [Office 2019](/hersteller/microsoft/office/), [Server Produkte](/hersteller/microsoft/) ([Windows Server](/hersteller/microsoft/windows-server/) / [SQL Server](/hersteller/microsoft/sql-server/) / [SharePoint](/hersteller/microsoft/sharepoint/) / [Skype](/hersteller/microsoft/skype-for-business/)) und andere Dauerlizenzen im Einsatz haben,
  können diese im Open Lizenzprogramm nicht mehr erweitern.
  Bestehende [Software Assurance (SA)](/hersteller/microsoft/software-assurance/) Wartungsverträge können im [Open Value Lizenzprogramm](/hersteller/microsoft/open-value/) verlängert werden. Gerne beraten wir Sie beim Umstieg.

  Die Kauflizenzen sind künftig über [CSP](/hersteller/microsoft/csp/) und [Open Value](/hersteller/microsoft/open-value/) erhältlich.

  Eine Übersicht der Änderungen und Alternativen finden Sie in unserem Beitrag [Open License: Abkündigung zum 31.12.2021](/blog/open-license-abschaffung-aenderung-2021-2022/)


</Info>

## Open License Voraussetzungen

  Mit einer Erstbestellung von nur **5 Lizenzen** können Sie bei Microsoft einen neuen Volumenlizenzvertrag starten. Bei manchen Produkten, wie beispielsweise [SQL Server](/hersteller/microsoft/sql-server/), ist eine Lizenz für die Erstbestellung ausreichend. Mit der Erstbestellung wird ein Vertrag mit **2-jähriger Laufzeit** geschlossen, unter dem Sie jederzeit neue Bestellungen, auch für eine einzelne Lizenz, aufgeben können.

 Wenn Sie bei Ihrer Erstbestellung **weniger als 5 Lizenzen** benötigen, dann rufen Sie mich bitte an.



## Open License Rabattstaffeln

 Microsoft Open License (oder **MOLP** für Microsoft Open License Program) Verträge gibt es in unterschiedlichen Formen, für kommerzielle Unternehmen, mit zusätzlichem Rabatt bei größerer Bestellmenge in einzelnen Produktgruppen, für Behörden oder Bildungseinrichtungen.

<ArrowList>

 -  [Open-NL](/glossar/open-nl-microsoft/) - Einstiegslelvel ab 5 Lizenzen
 -  [Open-C](/glossar/open-c-microsoft/) - Rabattstaffel ab 500 Punkten in Produktgruppen
 -  [Open-D](/glossar/open-d-microsoft/) - Staffel für Behörden
 -  [Open-E](/glossar/open-e-microsoft/) - Staffel für Bildungseinrichtungen

</ArrowList>

  Microsoft hat die Rabattstaffel **Open-C** [abgekündigt](/blog/non-profit-lizenzprogramm-open-charity-jetzt-auch-in-europa-verfuegbar/).



## Nutzungsrechte bei Open License

  Mit Volumenlizenzen erhalten Sie im Gegensatz zu [OEM](/hersteller/microsoft/oem/), [PKC](/hersteller/microsoft/pkc/) oder [FPP](/hersteller/microsoft/fpp/) Produkten zusätzliche Nutzungsrechte:

<ArrowList>

 -  **Volumenaktivierung** - Mit einem Multiple Activation Key (MAK) können Sie die Software auf beliebigen Geräten aktivieren
 -  **[Downgrade-Recht](/blog/microsoft-downgrade-rechte/)** - Open Lizenzen dürfen auf ältere Produktversionen downgegradet werden
 -  **Reimaging-Recht** - Das Recht zum Anfertigen und Verteilen eines Images der Software ist bei Open Lizenzen enthalten
 -  **Virtualisierungsrechte (produktabhängig)** - Die Bereitstellung in virtuellen Umgebungen ist produktabhängig und benötigt teilweise zusätzliche Rechte (beispielsweise VDA für Windows)

</ArrowList>

## Software Assurance (SA)

  [Software Assurance](/hersteller/microsoft/software-assurance/) enthält Wartung, Support und weitere Nutzungsrechte, die beispielsweise in virtuellen Umgebungen oder beim Deployment nützlich sind. SA ist optional und läuft immer zwei Jahre.

 - Upgrade-Recht auf die aktuellste Programmversion
 - Telefonische Unterstützung rund um die Uhr
 - Reimaging-Recht
 - Windows Virtual Desktop Access (VDA)
 - Home Use Program
 - Cold Backups
 - Services, Schulungen, Tools zur Software-Verwaltung, Roll-Out oder Migration
 - Erweiterter Hotfix Support


## Open License Artikelbezeichnungen


### License (L)

  Open License Produkte können wahlweise mit oder ohne Wartung, bei Microsoft als Software Assurance (SA) bekannt, gekauft werden: <br/> Mit der License, oder deutsch Lizenz, erwerben Sie das Recht, die zur Zeit des Lizenzerwerbs aktuelle Version eines Softwareprodukts zu nutzen. License entspricht einer Vollversion insofern, da keine Vorgängerversion des jeweiligen Produktes notwendig ist. Es handelt sich um einen **einmaligen Lizenzerwerb** ohne Anrecht auf den späteren Erwerb von Nachfolgeversionen. Nach einer License kann wiederum nur eine erneute License erworben werden, das heißt auch beim Upgrade muss eine Neulizenz gekauft werden.


### License & Software Assurance Package (L&SA)

  License & Software Assurance Package ist die Kombination von License und Software Assurance in einer Bestellnummer. Der Vorteil liegt im bequemen gemeinsamen Erwerb des Nutzungsrechts sowohl der aktuellen als auch sämtlicher nach folgender Versionen eines Softwareprodukts während der Vertragslaufzeit. Außerdem erhalten Sie [zusätzliche Nutzungsrechte](/hersteller/microsoft/software-assurance/) während der SA Laufzeit.



### Software Assurance only (SA only)

  Software Assurance only ist die Wartungsverlängerung (Renewal) der Open License Produkte. Voraussetzung ist ein laufender Open License Vertrag, der hiermit um 2 weitere Jahre verlängert wird.
