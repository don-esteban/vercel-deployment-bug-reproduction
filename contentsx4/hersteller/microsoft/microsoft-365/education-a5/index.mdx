<Intro>

Microsoft 365 Education A5 ist eine günstige Komplettlösung für Bildungseinrichtungen und bietet Lehrkräften, Schülern und Studenten zusätzliche VoIP und Telefonie Funktionen sowie mehr Sicherheit gegenüber dem [Microsoft 365 Education A3](/hersteller/microsoft/microsoft-365/education-a3/) Plan. Ähnelt dem Enterprise E5 Plan, enthält aber entsprechend der Richtlinien für Bildungseinrichtungen funktionielle Änderungen.
Enthält alle Funktionen aus Office 365 Education A5, Windows 10/11 Education A5 und Enterprise Mobility + Security. Über unseren Warenkorb erhalten Sie mit wenigen Klicks ein Sofortangebot.


</Intro>

## Lizenzüberblick

Microsoft 365 Education A5 wird als benutzerbasiertes Abonnement ([User Subscription License - USL](/hersteller/microsoft/microsoft-365/lizenzierung/#user-subscription-license-usl)) gekauft. Die Laufzeit mit Preisschutz beträgt wahlweise ein Jahr oder einen Monat.
Bei Education/Academic Plänen wird zwischen Lizenzen für Lehrkräfte und Mitarbeiter von Bildungseinrichtungen (for faculty) und Lizenzen für Schüler und Studenten (for students) unterschieden.
Der Plan ist ausschließlich für Schüler, Studenten und Lehrkräfte verfügbar. Ob Sie berechtigt sind die vergünstigte Edition zu kaufen, können Sie auf der [Microsoft Seite](https://www.microsoft.com/de-de/education/products/office?rtc=1&market=de&activetab=tab%3Aprimaryr1) überprüfen

## Funktionen

Eine vollständige Auflistung finden Sie auf unserer Seite [Vergleich aller Microsoft 365 Pläne](/hersteller/microsoft/microsoft-365/vergleich-aller-microsoft-365-plaene/).


<ArrowList>

- **Lokal installierbares Office Paket**<br />
  Word, Excel, PowerPoint, Outlook, Publisher, Access und OneNote Desktop-App zur Installation auf bis zu fünf PCs oder Macs. Dürfen jedoch nur von einem Benutzer genutzt werden.
- **Mobile Office Apps**<br />
  Word, Excel, PowerPoint, Outlook und Publisher. Native Apps für Android, iOS und Windows Phone. Erlaubt ist die Installation auf 10 mobilen Endgeräten (5 Tablets und 5 Smartphones) pro Nutzer.
- **Office Online Services**<br />
  Word Online, Excel Online, OneNote Online und PowerPoint Online. Abgespeckte Versionen zur ausschließlichen Nutzung über einen Web-Browser.
- **Erweiterte Office Pläne**<br />
  SharePoint Plan 2, Exchange Plan 2, Teams, Microsoft Forms, Graph AI, Microsoft Sway, Microsoft Shifts, Microsoft Planner, MyAnalytics
- **OneDrive for Business**<br />
  OneDrive for Business mit 1-5 TB Online Speicher. Cloud-Speicher-Lösung mit kontrolliertem, privatem oder gemeinsamem Zugriff, lokaler Synchronisation, Gruppenverwaltung und mehr. Ähnlich Box- und Dropbox.
- **Education Inhalte**<br />
  OneNote Class Notebook, Yammer Academic, Learning Tools, Education Analytics, School Data Sync Plan 2, Set up School PCs App, Intune Education.
- **Power Apps**<br />
  Eigene professionelle Apps entweder komplett selbst oder mit der Hilfe von Templates schnell und einfach selbst zu erstellen. Begrenzt auf 2000 Schritte.
- **Power Automate**<br />
  Erstellen Sie zeitsparende Workflows für alle Aspekte, erweitert robotergesteuerte Prozessautomatisierung (RBA). Begrenzt auf 2000 Schritte.
- **Compliance and Security**<br />
  Cloud App Security, Windows Defender Antivirus, Device Guard, Windows Information Protection, BitLocker and BitLocker To Go, Windows Defender Application Guard, Windows Defender Credential Guard, Customer Lockbox, Insider Risk Management.
- **Windows 10/11 Desktop Operating System Upgrade**<br />
  Upgrade von Windows Pro auf Education/Enterprise.
- **On-Premises CALs**<br />
  Die Microsoft 365 Enterprise Pläne enthalten auch die Zugriffsrechte (CALs) auf die lokalen Office Server Exchange, SharePoint und Skype. Mit diesem sogenannten Dual Access Right dürfen Sie sowohl auf die Cloud-, als auch auf die On-Premises-Variante der Software zugreifen.
- **DLP - Data Loss Prevention**<br />
  Schutz personenbezogener Daten. Data Loss bzw. Data Leak Prevention.
- **Advanced eDiscovery**<br />
  Unstrukturierte Daten in Office 365 analysieren, eine effizientere Dokumentüberprüfung vornehmen und redudante Daten reduzieren.
- **Advanced Threat Protection**<br />
  Schützen von E-Mails in Echtzeit.
- **Shared Computer Activation**<br />
  Mit der Shared Computer Activation (SCA), bzw. der Aktivierung gemeinsam genutzter Computer, können Sie Office-Applikationen (Microsoft 365 Apps for Enterprise) für mehrere Benutzer auf einem Rechner bereitstellen. Dies wird vor allem im Terminalserver-Betrieb benötigt. In manchen Fällen werden jedoch auch lokale Installationen von mehreren Mitarbeitern genutzt. Beispielsweise bei Computern, die in der Produktion stehen oder Notebooks, die sich Mitarbeiter teilen. Die SCA ist in den Enterprise Plänen enthalten. Eine Anleitung zur Aktivierung finden Sie im [Microsoft Technet.](https://learn.microsoft.com/en-us/deployoffice/overview-shared-computer-activation)
- **Azure Funktionen**<br />
  Azure Advanced Threat Protection, Azure Advanced Threat Analytics, Azure Information Protection P2, Azure Active Directory P2.
- **Microsoft Intune**<br />
  [Microsoft Intune](/hersteller/microsoft/intune/) ist ein cloudbasierter Dienst zur Verwaltung von Endgeräten (iOS, Android, Windows, macOS) und zur sicheren Nutzung von Applikationen für mobile Mitarbeiter. Intune ist ein Teil von Microsoft's Modern Workplace.

</ArrowList>
