<Intro>

Microsoft 365 Pläne können als Abonnements im CSP oder im Volumenlizenzprogramm Open Value gekauft werden.
Wir zeigen hier die Unterschiede in der Beschaffung und geben einen generellen Überblick der Unterschiede zwischen den Verträgen.
Im März 2022 wurden die Lizenzbestimmungen für Unternehmen mit der Microsoft New Commerce Experience (NCE) angepasst.
Die CSP Lizenzen für Bildungseinrichtungen (Education) und gemeinnützige Organisationen werden Anfang 2024 ebenfalls ins NCE aufgenommen.

</Intro>

## Microsoft 365 Lizenzierung

<ArrowList>

- Lizenziert wird als Abonnement-Lizenz.
- Jeder Benutzer benötigt eine [User Subscription License (USL)](#user-subscription-license-usl).
- Die Abonnement Laufzeit beträgt wahlweise einen Monat oder ein Jahr. Bei manchen Plänen sind auch 3 Jahre Laufzeit möglich.
- Während der Abonnement-Laufzeit ist ein Schutz vor Preiserhöhungen enthalten.
- Abonnement-Lizenzen erhalten Sie im [CSP](/hersteller/microsoft/csp/) oder im [Open Value Volumenlizenzprogramm](/hersteller/microsoft/open-value/).
- Es gibt Sonderkonditionen für Bildungseinrichtungen und gemeinnützige Organisationen verfügbar.
- Lizenzierte Benutzer dürfen die Lizenz auf bis zu 5 Geräten installieren und gleichzeitig nutzen (nur CSP).

</ArrowList>


### User Subscription License (USL)

Die Lizenzierung von Microsoft 365 ist benutzerorientiert. Pro Mitarbeiter wird ein Abonnement benötigt. Lizenztechnisch heißt das User Subscription License (USL). Jede USL wird im Admin-Portal (Azure AD) einem Mitarbeiter zugeordnet.
Abonnements haben normalerweise eine Laufzeit von einem Jahr. Es ist aber auch eine monatliche oder dreijährige Laufzeit möglich. Während des Abonnements haben Sie einen Preisschutz, d.h. Preiserhöhungen sind erst nach Ende der Laufzeit möglich.

Die USL wird im Admin-Portal (Azure AD) einem spezifischen Nutzer im Unternehmen zugewiesen, der damit von bis zu fünf Geräten gleichzeitig mit den Microsoft 365 Diensten arbeiten darf.
Nach der Installation der Software lokal auf einem Gerät, gilt dieses Gerät als eines der fünf Geräte, auf denen Microsoft 365 installiert werden darf. Die Software darf insgesamt auf bis zu 15 Devices (5 PCs/Macs + 5 Tablets + 5 Smartphones) installiert werden.
Diese Nutzerlizenzierung ermöglicht Ihren Mitarbeitern das mobile Arbeiten von jedem beliebigen Gerät, egal ob es sich um den privaten PC oder das eigene Tablet handelt.

Wenn Sie unbeaubsichtige Bots lizenzieren möchten und keine User (Mitarbeiter), benötigen Sie eine [Unattended Lizenz](/hersteller/microsoft/microsoft-365/add-ons/e3-unattended-license/).

### New Commerce Experience (NCE)

Die CSP Subscriptions für Unternehmen und Behörden sind auch unter dem Namen [NCE (New Commerce Experience)](/blog/aenderungen-am-microsoft-csp-programm-nce/) bekannt.
Microsoft hat die ursprünglichen CSP-Einkaufskonditionen im März 2022 gegenüber dem normalen CSP Programm angepasst. Ausführliche Informationen zu den Änderungen finden Sie in [unserem Blogbeitrag zur New Commerce Experience](/blog/aenderungen-am-microsoft-csp-programm-nce/).



## Microsoft 365 Vertragsbedingungen

Für den Erwerb der Microsoft 365 Pläne stehen zwei Optionen zur Auswahl:

-  CSP: [Cloud Solution Provider Lizenzprogramm](/hersteller/microsoft/csp/)
-  OV: [Open Value Volumenlizenzprogramm](/hersteller/microsoft/open-value/)

Für Volumenlizenzen ist eine Mindestbestellmenge von 5 Lizenzen bei der Erstbestellung nötig. Im CSP kann ab einer Lizenz bestellt werden.
Eine [Übersicht der beiden Lizenzprogramme](#lizenzprogramme-und-bezugswege) finden Sie unten.


### Laufzeit

Die [User Subscription License (USL)](#user-subscription-license-usl) ist eine Abonnement-Lizenz, das heißt die Software wird für die Laufzeit gemietet.

- CSP: Über CSP ist die normale Laufzeit für Erstbestellungen ein Jahr, es kann hier aber auch monatlich gebucht werden. In Sonderfällen können wir Lizenzen hier auch tagesgenau buchen. Rufen Sie uns bei Wünschen oder Fragen hierzu bitte an.
- OV: Im Volumenlizenzprogramm ist die Laufzeit für Microsoft 365 Subscriptions immer ein Jahr.


### Zahlung

- CSP: Beim Kauf über das CSP erfolgt die Abrechnung wahlweise monatlich oder für ein Jahr im Voraus. Nach Prüfung durch unsere Buchhaltung besteht für unsere Kunden auch die Möglichkeit weiterer Zahlungs-Konditionen (z.B. 7 Tage netto, Ratenzahlung, etc.).
- OV: Beim Kauf in den Volumenlizenzprogrammen wird die jährliche Laufzeit vorab bezahlt. Auch hier gibt es, anch Prüfung, unterschiedliche Zahlungs-Konditionen.

### Preisschutz

Der Preisschutz schützt Sie vor einer Preiserhöhung während der Laufzeit.

- CSP: CSP Lizenzen enthalten Preisschutz während der Laufzeit. Bei monatlichen Subscriptions ist dieser somit wesentlich kürzer als bei den Jahreslizenzen.
- OV: Im Volumenlizenzprogramm erhalten Sie nach dem Kauf keinen Preisschutz. Das Renewal kann teurer werden.


### Verlängerung

- CSP: Office 365 über das CSP kann wahlweise ein festes Ablaufdatum erhalten oder automatisch verlängert werden. Das Abonnement kann vor Ablauf verlängert werden.
- OV: Bei den Volumenlizenzverträgen können Sie das Abonnement vor Ablauf verlängern. Wenn Sie das Abonnement nicht verlängern möchten, läuft dieses automatisch aus.


### Update

Mit den Microsoft 365 Abonnements in beiden Lizenzprogrammen, dürfen Sie immer die aktuellste Version der Microsoft Dienste und Software einsetzen. Sie haben umgekehrt auch die Pflicht, spätestens 6 Monate nach Veröffentlichung auf die neueste Version zu wechseln.
Mittlerweile enthalten viele Pläne auch ein (eingeschränktes) Downgraderecht für das jeweilige Produkt.


### Übertragung

Grundsätzlich dürfen Microsoft 365-Lizenzen auf einen anderen Nutzer übertragen werden, wenn seit der letzten Zuweisung mindestens 90 Tage vergangen sind oder der Nutzer das Unternehmen verlassen hat.


### Kündigung

- CSP: Sie können Ihr Microsoft 365-Abonnement innerhalb der 72-Stunden Stornierungsfrist kündigen.
- OV: Die Kündigung eines Microsoft 365 Volumenlizenzvertrages ist, nach Bewilligung von Microsoft, gegen eine individuelle Gebühr abhängig von der Restlaufzeit jederzeit möglich.

Vor der Kündigung Ihres Microsoft 365-Abonnements sollten Sie Ihre Daten, wie E-Mails oder Dokumente auf Teamwebsites, herunterladen und an einem anderen Speicherort sichern. Nach der Kündigung des Abonnements kann der Administrator noch 90 Tage lang über ein Konto mit eingeschränkten Funktionen auf die mit Ihrem Microsoft 365-Konto verbundenen Daten zugreifen.

### Software Assurance (SA)

Die Volumenlizenzen im Open Value Lizenzprogramm sind immer mit SA während der Laufzeit ausgestattet. Da Sie bei Microsoft 365 ein Abonnement abschließen, sind viele SA Vorteile bereits standardmäßig enthalten. Sie erhalten kostenlose Upgrades, können auf Online-Trainings und -Schulungen zugreifen und ab 150 Lizenzen haben Sie die Möglichkeit den [Fasttrack Service](https://www.microsoft.com/de-de/fasttrack/microsoft-365?rtc=1) für das Deployment zu nutzen. Außerdem sind Dienste, wie das Home Use Program, mit Microsoft 365 nicht nötig, da die Software sowieso auf 5 Geräten installiert werden darf und Mitarbeiter somit auch zu Hause Microsoft 365 nutzen dürfen.


### Dual Access Right

Das Dual Access Right in den Microsoft 365 Abonnements bietet den Zugriff auf lokale Office Server, ohne zusätzliche Lizenzen, wenn die entsprechende Online Version erworben wurde. Sie benötigen also keine weiteren Zugriffslizenzen (CALs) für den lokalen Zugriff auf Exchange Server, wenn Ihre zugreifenden Nutzer über einen [Exchange Online Plan](/hersteller/microsoft/exchange/online-plan-1/) verfügen. Dies gilt auch für die Online Abonnements für SharePoint oder Teams.


### Kombination der Pläne

Verschiedene Microsoft 365-Pläne lassen sich untereinander kombinieren. Bitte beachten Sie, dass es abhängig vom jeweiligen Microsoft 365 Plan Lizenzbeschränkungen gibt:

Die Pläne Microsoft 365 Business Pläne sind auf jeweils 300 Benutzer beschränkt, die Microsoft 365 Enterprise-Pläne gelten für eine unbegrenzte Benutzer-Anzahl.

Diese Einschränkungen der Benutzeranzahl gelten aber nur für den jeweiligen Plan: Es ist beispielsweise möglich und auch lizenzrechtlich korrekt, dass ein Kunde 300 Lizenzen für Office 365 Business Premium, 300 Lizenzen für Office 365 Business Essential und 500 Lizenzen für Office 365 Enterprise E3 einsetzt.


## Lizenzprogramme und Bezugswege

Auf unserer Seite zu den [Microsoft Lizenzprogrammen](/hersteller/microsoft/lizenzprogramme/) finden Sie eine vollständige Übersicht.

### CSP Subscription

[CSP-Subscriptions](/hersteller/microsoft/csp/) sind der normale Bezugsweg für Microsoft 365 Pläne. Lizenziert wird normalerweise als einjähriges Abonnement. Es gibt hier keine Mindestbestellmenge. Kostenlose Upgrades auf neue Versionen sind enthalten. Bildungseinrichtungen und gemeinnützige Organisationen erhalten Sonderkonditionen.
Für kommerzielle Unternehmen werden die CSP Subscriptions auch als [NCE - New Commerce Experience](#new-commerce-experience-nce) bezeichnet.

### Open Value Volumenlizenzen

[Open Value Volumenlizenzen](/hersteller/microsoft/open-value/) werden hier als jährliches Abonnement lizenziert. Es gibt Sonderkonditionen für Behörden, Bildungseinrichtungen und gemeinnützige Organisationen.
Seit Ende 2021 werden Open License Kunden bei SA-Verlängerungen auf das Open Value Lizenzprogramm umgestellt. Beachten Sie bitte, dass sich hierbei die Vertragslaufzeit von 2 Jahren (Open License) auf 3 Jahre (Open Value) ändert. Gerne beraten wir Sie beim Wechsel.
