<Intro>

Microsoft 365 Enterprise E3 bietet, in einem einzigen Abonnement, alle Werkzeuge und Dienste, die zur Einrichtung eines modernen Arbeitsplatz benötigt werden.
Egal ob unterwegs, im Homeoffice oder im Büro. Lizenziert nach Benutzern. Microsoft 365 Enterprise E3 ist der Nachfolger des Plans Secure Productive Enterprise (SPE) E3.
Enthalten sind Windows 10/11 Enterprise E3, Office 365 Enterprise E3 und die Verwaltungs- und Sicherheits-Lösung Enterprise Mobility + Security E3.
Über unseren Warenkorb erhalten Sie mit wenigen Klicks ein Sofortangebot.

</Intro>

## Lizenzüberblick

Microsoft 365 Enterprise E3 wird als benutzerbasiertes Abonnement ([User Subscription License - USL](/hersteller/microsoft/microsoft-365/lizenzierung/#user-subscription-license-usl)) gekauft. Die Laufzeit mit Preisschutz beträgt wahlweise ein Jahr oder einen Monat.
Für Bildungseinrichtungen gibt es die günstigere Edition [Microsoft 365 Education A3](/hersteller/microsoft/microsoft-365/education-a3/).

## Funktionen

Funktional entspricht Microsoft 365 E3 den einzelnen Komponenten [Windows 10/11 Enterprise E3](/hersteller/microsoft/windows/enterprise/), [Office 365 Enterprise E3](/hersteller/microsoft/office-365/enterprise-e3/) und [Enterprise Mobility + Security E3](/hersteller/microsoft/enterprise-mobility-security/)
Eine vollständige Auflistung finden Sie auf unserer Seite [Vergleich aller Microsoft 365 Pläne](/hersteller/microsoft/microsoft-365/vergleich-aller-microsoft-365-plaene/).


<ArrowList>

- **Lokal installierbares Office Paket**<br />
  Word, Excel, PowerPoint, Outlook, Publisher, Access und OneNote Desktop-App zur Installation auf bis zu fünf PCs oder Macs. Dürfen jedoch nur von einem Benutzer genutzt werden.
- **Mobile Office Apps**<br />
  Word, Excel, PowerPoint, Outlook, Publisher und OneNote. Native Apps für Android, iOS und Windows Phone. Erlaubt ist die Installation auf 10 mobilen Endgeräten (5 Tablets und 5 Smartphones) pro Nutzer.
- **Office Online Services**<br />
  Word Online, Excel Online, OneNote Online und PowerPoint Online. Abgespeckte Versionen zur ausschließlichen Nutzung über einen Web-Browser.
- **Exchange Online Plan 2**<br />
  Enthält in etwa den Exchange Online Plan 2. Exchange-Server aus der Cloud mit unbegrenztem Speicher für Mailboxen inklusive In-Situ-Archiv. Unterstützt u.a. Outlook. Kontakte, Kalender, Anti-Spamware.
- **SharePoint Online Plan 2**<br />
  SharePoint aus der Cloud. Erstellen und Verwalten von Teamwebsites (im virtuellen privaten Intranet) und öffentlichen Web-Seiten (im Internet). Mit zusätzlichen Business Intelligence Funktionen.
- **OneDrive for Business Plan 2**<br />
  OneDrive for Business Plan 2 mit 1 TB Online Speicher. Cloud-Speicher-Lösung mit kontrolliertem, privatem oder gemeinsamem Zugriff, lokaler Synchronisation, Gruppenverwaltung und mehr.
- **Shared Computer Activation**<br />
  Mit der Shared Computer Activation (SCA), bzw. der Aktivierung gemeinsam genutzter Computer, können Sie Office-Applikationen (Office 365 ProPlus) für mehrere Benutzer auf einem Rechner bereitstellen. Dies wird vor allem im Terminalserver-Betrieb benötigt. In manchen Fällen werden jedoch auch lokale Installationen von mehreren Mitarbeitern genutzt. Beispielsweise bei Computern, die in der Produktion stehen oder Notebooks, die sich Mitarbeiter teilen. Die SCA ist in den Enterprise Plänen enthalten. Eine Anleitung zur Aktivierung finden Sie im <a href="https://learn.microsoft.com/en-us/deployoffice/overview-shared-computer-activation" target="_blank">Microsoft Technet.</a>
- **Self Service Business Intelligence**<br />
  Self-Service-Business Intelligence zum Suchen, Analysieren und Visualisieren von Daten in Excel.
- **Advanced Compliance**<br />
  Compliance-Lösungen zur Unterstützung von Funktionen zur Archivierung, Überwachung und eDiscovery, zum Durchsuchen von Postfächern und internen Websites. Eine detaillierte Beschreibung finden Sie auf der <a href="https://www.microsoft.com/de-de/security" target="_blank">Microsoft Seite für Compliance-Lösungen in Office 365</a>.
- **Windows Server CAL**<br />
  Zugriffslizenz (CAL) für Windows Server, egal ob lokale Installation oder Betrieb in der Cloud.
- **On-Premises CALs**<br />
  Die Office 365 Enterprise Pläne enthalten auch die Zugriffsrechte (CALs) auf die lokalen Office Server Exchange, SharePoint und Skype. Mit diesem sogenannten Dual Access Right dürfen Sie sowohl auf die Cloud-, als auch auf die On-Premises-Variante der Software zugreifen.
- **DLP - Data Loss Prevention**<br />
  Schutz personenbezogener Daten. Data Loss bzw. Data Leak Prevention.
- **Microsoft Stream (Video)**<br />
  Zentrales Portal zum Hochladen und Freigeben von Videos in der gesamten Organisation.
- **Microsoft Yammer**<br />
  Yammer ist eine weitere Anwendung für die Zusammenarbeit in Teams. Erinnert an soziale Netzwerke, quasi ein Facebook für Unternehmen mit erweiterten Verwaltungs- und Sicherheitsfunktionen.
- **Microsoft Planner**<br />
  Hilft bei der Organisation von Team-Aufgaben. Erstellen neuer Pläne, organisieren und zuweisen von Aufgaben, freigeben von Dateien und festlegen von Terminen.
- **Microsoft Sway**<br />
  Eine Anwendung für die Erstellung und Bereitstellung von interaktive Reports/Dashboards und online Präsentationen. Die Präsentationen und Reports können mit Kollegen und Kunden geteilt und über einen Browser genutzt werden. Sway ist ebenfalls als kostenlose App erhältlich.
- **Microsoft Fasttrack**<br />
  Ermöglicht die Zusammenarbeit mit Microsoft-Spezialisten, um die Einführung der Office 365-Umgebung zu planen und betriebsbereit zu machen. Der Service ist ab 150 Lizenzen kostenlos verfügbar. Mehr Infos zu FastTrack finden Sie auf der <a href="https://www.microsoft.com/de-de/fasttrack?rtc=1" target="_blank">Microsoft Seite</a>.
- **Power Apps**<br />
  Eigene professionelle Apps entweder komplett selbst oder mit der Hilfe von Templates schnell und einfach selbst zu erstellen. Begrenzt auf 2000 Schritte.
- **Power Automate**<br />
  Erstellen Sie zeitsparende Workflows für alle Aspekte, erweitert robotergesteuerte Prozessautomatisierung (RBA). Begrenzt auf 2000 Schritte.
- **Windows 10/11 Desktop Operating System Upgrade**<br />
  Upgrade von Windows Pro auf Windows 10/11 Enterprise.
- **Compliance and Security**<br />
  Windows Defender Antivirus, Device Guard, Windows Information Protection, BitLocker and BitLocker To Go, Windows Defender Application Guard, Windows Defender Credential Guard.
- **Azure Funktionen**<br />
  Azure Advanced Threat Analytics, Azure Information Protection P2, Azure Active Directory P1.
- **Microsoft Intune**<br />
  [Microsoft Intune](/hersteller/microsoft/intune/) ist ein cloudbasierter Dienst zur Verwaltung von Endgeräten (iOS, Android, Windows, macOS) und zur sicheren Nutzung von Applikationen für mobile Mitarbeiter. Intune ist ein Teil von Microsoft's Modern Workplace.


</ArrowList>
