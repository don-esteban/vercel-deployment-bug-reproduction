## Lizenzüberblick

Microsoft 365 F5 wird als benutzerbasiertes Abonnement ([User Subscription License - USL](/hersteller/microsoft/microsoft-365/lizenzierung/#user-subscription-license-usl)) gekauft. Die Laufzeit mit Preisschutz beträgt wahlweise ein Jahr oder einen Monat.
Das Add-on benötigt einen vorhandenen [Microsoft 365 F1](/hersteller/microsoft/microsoft-365/f1/) oder [Microsoft 365 F3](/hersteller/microsoft/microsoft-365/f3/) Plan.

- [CSP Subscription](/hersteller/microsoft/csp/)<br />
