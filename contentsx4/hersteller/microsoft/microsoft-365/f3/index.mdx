<Intro>

Microsoft 365 F3 richtet sich an Firstline-Worker, die nur eingeschränkte Microsoft Apps, ohne vollem Umfang, benötigen. Der F3 Plan enthält, wie der F1 Plan keine Desktop-Installationsrechte. Dieser umfasst Microsoft-Apps für das Web und für Mobilgeräte, bis zu einer maximalen Bildschirmgröße von 10,9 Zoll.
Bis 2020 hieß der F3 Plan Microsoft 365 F1. Im Gegensatz zum neuen F1 Plan, ist zusätzlich Windows 10/11 E3 enthalten und erlaubt Cloud-Management und Virtualisierung.
Über unseren Warenkorb erhalten Sie mit wenigen Klicks ein Sofortangebot.

</Intro>

## Lizenzüberblick

Microsoft 365 F3 wird als benutzerbasiertes Abonnement ([User Subscription License - USL](/hersteller/microsoft/microsoft-365/lizenzierung/#user-subscription-license-usl)) gekauft. Die Laufzeit mit Preisschutz beträgt wahlweise ein Jahr oder einen Monat.
Erweiterte Security und Compliance Funktionen sind über [Microsoft 365 F5 Add-Ons](/hersteller/microsoft/microsoft-365/f5/) verfügbar.

<ArrowList>

- [Microsoft 365 Lizenzierung](/hersteller/microsoft/microsoft-365/lizenzierung/)<br />
  Ausführliche Informationen zu Kauf und Lizenzierung der Microsoft 365 Pläne.

- [CSP Subscription](/hersteller/microsoft/csp/)<br />
  Abonnement-Lizenz. Die Laufzeit mit Preisschutz beträgt wahlweise ein Jahr oder einen Monat. Keine Mindestbestellmenge.

- [NCE](/hersteller/microsoft/microsoft-365/lizenzierung/#new-commerce-experience-nce)<br />

## Funktionen

Eine vollständige Auflistung finden Sie im PDF [Office 365 und Microsoft 365 Pläne im Vergleich](/hersteller/microsoft/microsoft-365-office-365-subscription-plan-enterprise.pdf).


<ArrowList>

- **Enterprise Mobile Security (EMS E3)**<br />
  Microsoft Intune, Azure Active Directoy Premium P1, Advanced Threat Analytics, Advanced Information Protection P1.
- **Microsoft Online Services**<br />
  Word Online, Excel Online, OneNote Online und PowerPoint Online. Abgespeckte Versionen zur ausschließlichen Nutzung über einen Web-Browser.
- **Microsoft Apps (eingeschränkt)**<br />
  Eingeschränkte Nutzung bis zur einer maximalen Bildschirmgröße von 10,9 Zoll. Enthält nur Teams, Yammer, Planner, SharePoint (Keine Administration, keine Mailbox, keine persönliche Seite, 2GB User/10GB geteilter Storage), Stream (nur Video anschauen).
- **Erweiterte Microsoft Apps**<br />
  Power Apps und Power Automate (2000 steps/Tag), OneDrive for Business Storage (2GB), Exchange Postfach (2 GB, keine Outlook Dekstop Integration möglich).
- **Windows 10/11 E3**<br />
  Windows Enterprise E3 enthält auch Cloud Management und Virtualisierungsrechte, aber kein [Downgrade-Recht](/blog/microsoft-downgrade-rechte/), sowie kein Long Term Servicing Channel (LTSC) und Microsoft Desktop Optimization Pack (MDOP).

</ArrowList>
