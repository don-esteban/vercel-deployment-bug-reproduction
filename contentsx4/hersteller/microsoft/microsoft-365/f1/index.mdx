<Intro>

Der neue Microsoft 365 F1 Plan ist für Firstline-Worker, die überwiegend mit den Kommunikationsplattformen Microsoft Teams sowie Yammer arbeiten und kein Exchange (Outlook) Postfach benötigen. F1 beinhaltet eingeschänkte Enterprise Mobile Security Funktionen, bspw. Azure Active Directoy Premium P1 und Intune. Es können nur Microsoft Online Services und Microsoft Apps (read only) bis zu einer maximalen Bildschirmgröße von 10,9 Zoll genutzt werden, keine Desktop Installationen.
Der ursprüngliche Microsoft 365 F1 Plan wurde am 1. April 2020 in Microsoft 365 F3 umbenannt. Über unseren Warenkorb erhalten Sie mit wenigen Klicks ein Sofortangebot.

</Intro>

## Lizenzüberblick

Microsoft 365 F1 wird als benutzerbasiertes Abonnement ([User Subscription License - USL](/hersteller/microsoft/microsoft-365/lizenzierung/#user-subscription-license-usl)) gekauft. Die Laufzeit mit Preisschutz beträgt wahlweise ein Jahr oder einen Monat.
Erweiterte Security und Compliance Funktionen sind über [Microsoft 365 F5 Add-Ons](/hersteller/microsoft/microsoft-365/f5/) verfügbar.

<ArrowList>

- [Microsoft 365 Lizenzierung](/hersteller/microsoft/microsoft-365/lizenzierung/)<br />
  Ausführliche Informationen zu Kauf und Lizenzierung der Microsoft 365 Pläne.

- [CSP Subscription](/hersteller/microsoft/csp/)<br />
  Abonnement-Lizenz. Die Laufzeit mit Preisschutz beträgt wahlweise ein Jahr oder einen Monat. Keine Mindestbestellmenge.

- [NCE](/hersteller/microsoft/microsoft-365/lizenzierung/#new-commerce-experience-nce)<br />
  New Commerce Experience - CSP Subscription für Unternehmen und Behörden.

- [Add-ons](/hersteller/microsoft/microsoft-365/add-ons/)<br />
  Für die Microsoft 365 Pläne gibt es zahlreiche Add-ons. Damit erhalten Sie beispielsweise zusätzlichen Speicherplatz oder erweiterte Funktionalität.

</ArrowList>


## Funktionen

Eine vollständige Auflistung finden Sie im PDF [Office 365 und Microsoft 365 Pläne im Vergleich](/hersteller/microsoft/microsoft-365-office-365-subscription-plan-enterprise.pdf).


<ArrowList>

- **Enterprise Mobile Security (EMS E3)**<br />
  Microsoft Intune, Azure Active Directoy Premium P1, Advanced Threat Analytics, Advanced Information Protection P1.
- **Microsoft Online Services**<br />
  Word Online, Excel Online, OneNote Online und PowerPoint Online. Abgespeckte Versionen zur ausschließlichen Nutzung über einen Web-Browser.
- **Microsoft Apps (eingeschränkt)**<br />
  Eingeschränkte Nutzung bis zur einer maximalen Bildschirmgröße von 10,9 Zoll. Enthält nur Teams, Yammer, Planner, SharePoint (Keine Administration, keine Mailbox, keine persönliche Seite, keine Forms, 2GB User/10GB geteilter Storage), Stream (nur Video anschauen). **Read-Only**.

</ArrowList>
