<Intro>

Die Microsoft 365 E3 Unattended-Lizenz wird benötigt um Unattended Bots zu lizenzieren. Diese Zusatzlizenz ermöglicht die Nutzung von Office-Anwendungen aus Dienst- oder anderen Nichtbenutzerkonten zur Prozessautomatisierung.
Voraussetzung ist, dass bereits M365 Pläne im Einsatz sind. Die E3 Unattended-Lizenz kann eine neue Option für die Arbeit mit Produktivitätsdaten und -werkzeugen bieten.
Über unseren Warenkorb erhalten Sie mit wenigen Klicks ein Sofortangebot.

### Nutzungsrechte und Lizenzierung

 Wir zeigen Ihnen einen kleinen Auszug aus den Product Terms (Stand: Mai 2020) von Microsoft, wie man die Unattended License verwendet. Die vollständige Übersicht finden Sie in den [Microsoft Online Services Terms](https://www.microsoft.com/licensing/docs/view/Licensing-Use-Rights) (OST.s26).

- Jede Microsoft 365 A3/E3 – Unattended-Lizenz erlaubt die Verwendung der M365 A3/E3-Suite in nur einer einzigen physischen oder virtuellen Betriebssystemumgebung für die robotergesteuerte Prozessautomatisierung.
- Die Neuzuweisung von Lizenzen für Bots folgt den gleichen Regeln für Nutzer und Geräte, als ob der Bot ein Nutzer wäre.
- Unattended Bots dürfen keine Aktivitäten oder Workflows im Namen eines nicht lizenzierten Nutzers oder Geräts erstellen oder replizieren.
- Bots, denen eine Microsoft 365 A3/E3 – Unattended-Lizenz zugewiesen wurde, können auf Windows Virtual Desktop (WVD) zugreifen.



## Was ist eine Microsoft 365 E3 Unattended License?

Wir erklären Ihnen die wichtigsten Begriffe zur Unattended License. Die vollständige Übersicht und Beispiele finden Sie in den [Product licensing briefs - Microsoft 365 Unattended License Overview](https://download.microsoft.com/download/3/D/4/3D42BDC2-6725-4B29-B75A-A5B04179958B/Licensing_brief_PLT_Microsoft365_Unattended_License_overview.pdf) (November 2020).

- **Robotergesteuerte Prozessautomatisierung**<br />
  Auch als RPA oder Bots bezeichnet, ist eine Anwendung oder eine Reihe von Anwendungen, die zur Erfassung von Daten und zur Manipulation von Anwendungen zur Ausführung sich wiederholender Aufgaben verwendet werden. Bots arbeiten mit jedem UI-Element von Windows 10 innerhalb einer Betriebssystemumgebung und/oder arbeiten mit jeder Office-Anwendung in einer Betriebssystemumgebung.
- **Unattended Bot**<br />
  Jeder Bot, der nicht der Definition von Attended Bot entspricht, wird als Unattended Bot betrachtet.
- **Attended Bot**<br />
  Ein Attended Bot unterstützt eine Person bei der Ausführung der Automatisierung an den lokalen und/oder Remote-Arbeitsplätzen der Person. Er arbeitet gleichzeitig mit der Person auf demselben Arbeitsplatz/denselben Arbeitsplätzen, um sich wiederholende Aufgaben zu erledigen, und wird durch explizite Aktionen dieser Person ausgelöst.
