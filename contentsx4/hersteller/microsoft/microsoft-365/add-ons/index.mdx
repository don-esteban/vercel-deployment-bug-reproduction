<Intro>

Für viele Microsoft 365 Pläne gibt es Add-ons, mit denen der Lizenz neue Funktionen hinzugefügt werden können. Hier finden Sie eine Übersicht der Add-ons und für welche Pläne diese verfügbar sind.


</Intro>

<TOC />


## Microsoft 365 Add-ons

<ArrowList>

- **[Microsoft 365 E3 Unattended License](/hersteller/microsoft/microsoft-365/add-ons/e3-unattended-license/)**<br />
  Microsoft 365 Enterprise E3 Lizenz für unbeaufsichtigte (Unattended) Bots zur Prozessautomatisierung.
- **[Microsoft 365 Teams Add-ons](/hersteller/microsoft/teams/add-ons/)<br />**
  Für die Kommunikationslösung Microsoft Teams gibt es Add-ons um beispielsweise VoIP Funktionen und/oder internationale Telefoniepläne hinzuzufügen.
- **[Microsoft 365 10 Year Audit Log Retention](/hersteller/microsoft/microsoft-365/add-ons/10-year-audit-log-retention/)**<br />
  Erweitert E5 Pläne mit Advanced Audit um eine 10-jährige Aufbewahrung der Audit-Protokolle, zur Bewältigung von strengen behördlichen oder internen Compliance-Verpflichtungen.
- **[Microsoft 365 Viva Topic](/hersteller/microsoft/microsoft-365/add-ons/viva-topic/)**<br />
  Viva Topic verwendet Microsoft AI-Technologie, Microsoft 365, Microsoft Graph, Suche und andere Komponenten und Dienste, um die Informationen aus den Daten in Wissen zu transformieren.
- **[Extra File Storage](/hersteller/microsoft/microsoft-365/add-ons/extra-file-storage/)**<br />
  Erweiterung des SharePoint Online Speicherplatzes um 1 GB.
- **[Stream (klassisch)](/hersteller/microsoft/microsoft-365/add-ons/stream/)**<br />
 Erstellen von Schulungsvideos mit zusätzlichen Funktionen, wie intelligenten kognitiven Fähigkeiten oder maschinell generierten Bildunterschriften.

</ArrowList>


## Microsoft 365 E5 Add-ons

Der [Microsoft 365 E5 Plan](/hersteller/microsoft/microsoft-365/enterprise-e5/) beinhaltet bereits alle Compliance Funktionen.

<ArrowList>

- **[Microsoft 365 E5 Compliance](/hersteller/microsoft/microsoft-365/add-ons/e5-compliance/)**<br />
  Bundle aus den Mini-Suiten M365 Information Protection Governance, eDiscovery & Audit, Insider Risk Management. Verfügbar für die Pläne [Microsoft 365 Business Basic](/hersteller/microsoft/microsoft-365/business-basic/), [Office 365 Business Standard](/hersteller/microsoft/microsoft-365/business-standard/) und [Office 365 Enterprise E3](/hersteller/microsoft/office-365/enterprise-e3/)/[A3](/hersteller/microsoft/office-365/education-a3/) + [Enterprise + Security E3](/hersteller/microsoft/enterprise-mobility-security/)

- **[Microsoft 365 E5 Insider Risk Management](/hersteller/microsoft/microsoft-365/add-ons/e5-insider-risk-management/)**<br />
  Mini-Suite zur Minimierung von Risikostellen, beispielsweise durch das Surfverhalten von Mitarbeitern. Verfügbar für die Pläne [Office 365](/hersteller/microsoft/office-365/), [SharePoint Online](/hersteller/microsoft/sharepoint/), [Exchange Online](/hersteller/microsoft/exchange/) und [OneDrive for Business](/hersteller/microsoft/microsoft-365/onedrive-for-business-plan-2/).

- **[Microsoft 365 E5 Information Protection Governance](/hersteller/microsoft/microsoft-365/add-ons/e5-information-protection-governance/)**<br />
  Mini-Suite zur Verwaltung von Aufbewahrungs-, sowie Sicherheitsrichtlinien um Ihre Daten zu schützen. Dateien und Chats werden verschlüsselt. Verfügbar für die Pläne [MS365](/hersteller/microsoft/microsoft-365/), [EMS FK/F3/E3/A3](/hersteller/microsoft/enterprise-mobility-security/) + [Office 365](/hersteller/microsoft/office-365/), [SharePoint Online](/hersteller/microsoft/sharepoint/), [Exchange Online](/hersteller/microsoft/exchange/) und [OneDrive for Business](/hersteller/microsoft/microsoft-365/onedrive-for-business-plan-2/)

- **[Microsoft 365 E5 eDiscovery & Audit](/hersteller/microsoft/microsoft-365/add-ons/e5-ediscovery-audit/)**<br />
  Mini-Suite zur erweiterten Überwachung und gezielten Suche von SharePoint Online, Exchange Online, oder OneDrive for Business Dateien. Verfügbar für die Pläne [Office 365](/hersteller/microsoft/office-365/), [SharePoint Online](/hersteller/microsoft/sharepoint/), [Exchange Online](/hersteller/microsoft/exchange/) und [OneDrive for Business](/hersteller/microsoft/microsoft-365/onedrive-for-business-plan-2/)

- **[Microsoft 365 E5 Security](/hersteller/microsoft/microsoft-365/add-ons/e5-security/)**<br />
  Beinhaltet die Funktionen für Identity and Threat Protection für Office 365, Windows und Enterprise Mobility + Security (EMS). Verfügbar für die Pläne [Office 365](/hersteller/microsoft/office-365/), [SharePoint Online](/hersteller/microsoft/sharepoint/), [Exchange Online](/hersteller/microsoft/exchange/) und [OneDrive for Business](/hersteller/microsoft/microsoft-365/onedrive-for-business-plan-2/)

</ArrowList>


## Weitere Microsoft 365 Add-ons

Die folgenden Dienste können auch zu den meisten Microsoft 365 Plänen dazu gebucht werden:

### Microsoft 365 Defender Services

Microsoft Defender bietet einheitliche Schutzsuites zur Erkennung, Verhinderung, Untersuchung und Reaktion von Schadsoftware in Endpunkten, Identitäten, E-Mails oder Anwendungen. Bestandteile sind [Microsoft Defender for Endpoint](/hersteller/microsoft/microsoft-365/defender/endpoint/), [Microsoft Defender for Office 365](/hersteller/microsoft/microsoft-365/defender/office-365/) und [Microsoft Defender for Identity](/hersteller/microsoft/microsoft-365/defender/identity/).
Die Microsoft Defender Services sind bereits Teil des [Microsoft 365 E5 Plans](/hersteller/microsoft/microsoft-365/enterprise-e5/), können aber auch einzeln gebucht werden..

<ArrowItem>

**[Microsoft 365 Defender Services](/hersteller/microsoft/microsoft-365/defender/)**

</ArrowItem>

### Enterprise Mobility + Security

Enterprise Mobility + Security (EMS) ist eine einheitlich Security-Suite mit Identitäts- und Zugriffssteuerung beim Zugriff auf Cloud-Dienste.
In vielen Microsoft 365 Plänen ist EMS bereits enthalten.

<ArrowItem>

**[Enterprise Mobility + Security (EMS)](/hersteller/microsoft/enterprise-mobility-security/)**

</ArrowItem>

### Microsoft Intune

Intune ist ein cloudbasierter Dienst zur Verwaltung von mobilen Endgeräten (iOS, Android, Windows, macOS) und zur sicheren Nutzung von Applikationen für mobile Mitarbeiter.
Intune ist in den meisten Microsoft 365 Plänen bereits enthalten.

<ArrowItem>

**[Microsoft Intune](/hersteller/microsoft/intune/)**

</ArrowItem>

### Teams Premium

Teams Premium ist eine (kostenpflichtige) Erweiterung für alle Microsoft 365 Pläne mit Teams. Die Teams Premium Edition enthält individuelle Besprechungs-Anpassungen und KI-Unterstützung, sowie weitere Teams Add-ons, wie das [Advanced Communications Add-on](/hersteller/microsoft/teams/add-ons/).

<ArrowItem>

**[Microsoft Teams](/hersteller/microsoft/teams/)**

</ArrowItem>

## Microsoft 365 Erweiterungen anderer Hersteller

Die Microsoft 365 Funktionalität kann auch mit Diensten anderer Hersteller erweitert werden. Wir zeigen hier einen Auszug an möglich Erweiterungen, beispielsweise für Cloud Backup in oder aus der Microsoft 365 Cloud.

### Veeam Backup for Microsoft 365

Sicheres Backup und Wiederherstellung Ihrer Microsoft 365 Daten (u.a. Exchange Online, SharePoint Online und OneDrive for Business, Teams).

<ArrowItem>

**[Veeam Backup for Microsoft 365](/hersteller/veeam/backup-for-microsoft-365/)**

</ArrowItem>


### Adobe Acrobat Sign Solutions

Digitaler Signaturdienst der sich nahtlos in die Microsoft 365 Cloud Dienste integriert. Ermöglicht Signaturen direkt in den Office Programmen (lokal oder Cloud).

<ArrowItem>

**[Adobe Acrobat Sign Solutions](/hersteller/adobe/acrobat/sign-solutions/)**

</ArrowItem>


### AvePoint

AvePoint bietet Lösungen zum Schutz, sowie zur Migration und Verwaltung von Daten in Microsoft 365-Umgebungen.

<ArrowItem>

**[AvePoint](/hersteller/avepoint/)**

</ArrowItem>
