## Lizenzüberblick

Office 365 Extra File Storage wird als benutzerbasiertes Abonnement ([User Subscription License - USL](/hersteller/microsoft/microsoft-365/lizenzierung/#user-subscription-license-usl)) gekauft.
Das Add-on benötigt einen vorhandenen Microsoft 365 Plan mit [SharePoint Online Plan 1](/hersteller/microsoft/sharepoint/online-plan-1/) oder [SharePoint Online Plan 2](/hersteller/microsoft/sharepoint/online-plan-2/).


<ArrowList>

- [Microsoft 365 Lizenzierung](/hersteller/microsoft/microsoft-365/lizenzierung/)<br />
  Ausführliche Informationen zu Kauf und Lizenzierung der Microsoft 365 Pläne.

- [CSP Subscription](/hersteller/microsoft/csp/)<br />
  Abonnement-Lizenz. Die Laufzeit mit Preisschutz beträgt wahlweise ein Jahr oder einen Monat. Keine Mindestbestellmenge.

- [NCE](/hersteller/microsoft/microsoft-365/lizenzierung/#new-commerce-experience-nce)<br />
  New Commerce Experience - CSP Subscription für Unternehmen und Behörden.

- [Open Value](/hersteller/microsoft/open-value/)<br />
  Einjährige Abonnement-Lizenz mit Software Assurance. Mindestbestellmenge: 5 Lizenzen.

</ArrowList>
