<Intro>

Microsoft 365 E5 Security beinhaltet die Funktionen für Identity and Threat Protection für Office 365, Windows und Enterprise Mobility + Security (EMS). Das Add-On ist für Nutzer ausgelegt, die Microsoft 365 E3/A5 im Einsatz haben und zusätzliche Security Funktionen benötigen, aber nicht den vollen Umfang eines E5 Plans.
Der Microsoft 365 Enterprise E5 Plan enthält das Add-on bereits. Über unseren Warenkorb erhalten Sie mit wenigen Klicks ein Sofortangebot.



</Intro>

## Lizenzüberblick

Das Microsoft 365 E5 Security Add-on wird als benutzerbasiertes Abonnement ([User Subscription License - USL](/hersteller/microsoft/microsoft-365/lizenzierung/#user-subscription-license-usl)) gekauft.
Das Add-on benötigt einen vorhandenen [Microsoft 365 E3 Plan](/hersteller/microsoft/microsoft-365/enterprise-e3/).
Bei Education/Academic Plänen wird zwischen Lizenzen für Lehrkräfte und Mitarbeiter von Bildungseinrichtungen (for faculty) und Lizenzen für Schüler und Studenten (for students) unterschieden.

## Funktionen

Beinhaltet Funktionen für Office 365, Windows und Enterprise Mobility + Security (EMS):

<ArrowList>

- [Office 365 Defender ATP](/hersteller/microsoft/microsoft-365/defender/) (einschließlich Office 365 ATP und Threat Intelligence)
- [Cloud App Security](/hersteller/microsoft/microsoft-365/cloud-app-security/)
- Azure ATP
- Azure Active Directly Plan 2
- Windows Defender ATP

</ArrowList>
