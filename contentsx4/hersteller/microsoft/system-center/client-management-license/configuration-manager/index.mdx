<Intro>

Der System Center 2022 Configuration Manager (SCCM) bietet umfangreiche Konfigurationsverwaltung für verwaltete Geräte. SCCM verbessert die Möglichkeiten zur Kontrolle von (Compliance-)Richtlinien im Unternehmen.
Jedes verwaltete Gerät ohne Serverbetriebsystem benötigt eine Lizenz. Lizenziert wird pro Betriebssystemumgebung (OSE) oder nach Nutzern. Die Configuration Manager CML enthält die Lizenz für den Virtual Machine Manager (SCVMM).
Nutzen Sie unsere Sofortangebote zur Beschaffungsplanung!

</Intro>

<TOC />


## Preise

<Info>

Der System Center Configuration Manager ist nicht mehr verfügbar. Nachfolger ist der [Endpoint Configuration Manager](/hersteller/microsoft/system-center/client-management-license/endpoint-configuration-manager/), ohne den System Center Virtual Machine Manager (SCVMM).


</Info>



## Lizenzüberblick

Es wird eine CML pro verwaltetem Gerät ohne Serverbetriebsystem benötigt. Lizenziert wird pro Betriebssystemumgebung (OSE) oder nach Nutzern (User).

<ArrowList>

- [System Center CML Lizenzierung](/hersteller/microsoft/system-center/client-management-license/)<br />
  Ausführliche Informationen zu Kauf und Lizenzierung der Client Management Lizenzen.

- [Open Value](/hersteller/microsoft/open-value/)<br />
  Volumenlizenz mit 3 Jahren Software Assurance (SA). Mindestbestellmenge: 5 Lizenzen.

</ArrowList>

<Info>

[Microsoft Core und Enterprise CAL Suites](/hersteller/microsoft/cal-suite/) enthalten [System Center Endpoint Protection](/hersteller/microsoft/system-center/client-management-license/endpoint-protection/) und [Microsoft Endpoint Configuration Manager (MECM)](/hersteller/microsoft/system-center/client-management-license/endpoint-configuration-manager/) zusammen mit weiteren Zugriffslizenzen (CALs).

</Info>


### Bestandteile der Configuration Manager CML

 Enthält die Client-Lizenzen für:

<ArrowList>

 - Endpoint Configuration Manager (ECM): Zur Konfigurationsverwaltung für die Microsoft-Plattform. Verbessert die Möglichkeiten der Kontrolle bei Beachtung von Unternehmen Compliance-Richtlinien.
 - System Center Virtual Machine Manager (SCVMM): Zur Bereitstellung und Verwaltung von virtuellen Maschinen auch in heterogenen Umgebungen.

</ArrowList>
