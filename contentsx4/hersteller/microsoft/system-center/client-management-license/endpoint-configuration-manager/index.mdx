<Intro>

Der Microsoft Endpoint Configuration Manager (früher auch SCCM) bietet umfangreiche Konfigurationsverwaltung für verwaltete Geräte. MECM verbessert die Möglichkeiten zur Kontrolle von (Compliance-)Richtlinien im Unternehmen.
Jedes verwaltete Gerät ohne Serverbetriebsystem benötigt eine Lizenz. Lizenziert wird pro Betriebssystemumgebung (OSE) oder nach Nutzern. Die Microsoft Endpoint Configuration Manager CML enthält die Lizenz für den Virtual Machine Manager (SCVMM).

</Intro>

## Lizenzüberblick

Es wird eine CML pro verwaltetem Gerät ohne Serverbetriebsystem benötigt. Lizenziert wird pro Betriebssystemumgebung (OSE) oder nach Nutzern (User).
