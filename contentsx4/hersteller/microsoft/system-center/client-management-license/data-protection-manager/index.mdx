<Intro>

System Center 2022 Data Protection Manager (DPM) ermöglicht die Datensicherung von Servern, private Cloud, Clients oder Applikationen auf unterschiedlichen Medien, wie Festplatte oder Bandlaufwerk (z.B. Backup von VMs unter vSphere).
Jedes verwaltete Gerät ohne Serverbetriebsystem benötigt eine Lizenz. Lizenziert wird pro Betriebssystemumgebung (OSE) oder nach Nutzern.
