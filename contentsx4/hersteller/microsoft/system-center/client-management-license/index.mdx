<Intro>

Microsoft System Center 2022 Client Management Licenses (CML) werden für verwaltete Geräte ohne Serverbetriebssystem benötigt. CML sind mit unterschiedlichen Funktionen verfügbar: Configuration Manager, Operations Manager, Service Manager, Data Protection Manger und Endpoint Protection.
Hier finden Sie eine Übersicht der CML und Informationen zum Kauf.


</Intro>

<TOC />


## Übersicht der Client Management Lizenzen

Geräte ohne Serverbetriebsystem benötigen für die Verwaltung mit System Center eine Client Management License (CML). Dazu zählen beispielsweise Windows-PCs. Anders als bei den beiden Server-Editionen können die Funktionen als CML einzeln lizenziert werden:

<ArrowList>

- **[Endpoint Configuration Manager](/hersteller/microsoft/system-center/client-management-license/endpoint-configuration-manager/)**<br />
   Konfigurationsverwaltung für die Microsoft-Plattform. Enthält den Virtual Machine Manager (VMM). Verbessert die Möglichkeiten der Kontrolle bei Beachtung von Unternehmen Compliance-Richtlinien. Ehemals System Center Configuration Manager (SCCM).

- **[Operations Manager](/hersteller/microsoft/system-center/client-management-license/operations-manager/)**<br />
    Bietet einen Überblick über die verwalteten Rechenzentren und Einblick in die Private und Public Cloud Infrastruktur. Außerdem werden Anwendungen und die Infrastruktur überwacht, um Leistung und Verfügbarkeit von kritischen Anwendungen zu gewährleisten.

- **[Endpoint Protection](/hersteller/microsoft/system-center/client-management-license/endpoint-protection/)**<br />
    Endpunktschutz mit Antivirus und Schutz vor weitere Malware. Früher bekannt als Forefront.

 - **[Service Manager](/hersteller/microsoft/system-center/client-management-license/service-manager/)**<br />
    Ermöglicht die Integration von Benutzern, Workflows und Informationen in die Prozesse im Datacenter, durch Standardisierung.

 - **[Data Protection Manager](/hersteller/microsoft/system-center/client-management-license/data-protection-manager/)**<br />
    Ermöglicht die Datensicherung von Servern, private Cloud, Clients oder Applikationen auf unterschiedlichen Medien, wie Festplatte oder Bandlaufwerk (z.B. Backup von VMs unter vSphere).

- **[Orchestrator](/hersteller/microsoft/system-center/client-management-license/orchestrator/)**<br />
    Integriert die Definition von Abläufen und die Prozessautomatisierung. Auch bekannt als Service Management Automation.


</ArrowList>


## System Center Client Management Lizenzierung

Geräte mit Desktop-Betriebssystem werden mit den Client Management-Lizenzen (ML) verwaltet. Seit System Center 2019 gibt die CML nur noch als Einzelprodukte und nicht mehr in einer Suite.


<ArrowList>

- Eine CML pro verwaltetem Gerät ohne Serverbetriebsystem
- Die Verwaltungs-Konsole muss lizenziert werden.
- Lizenziert wird nach verwalteten Betriebssystemumgebungen (OSE) oder nach Nutzern (User)
- Volumenlizenz mit 3 Jahren Software Assurance im [Open Value Lizenzprogramm](/hersteller/microsoft/open-value/)
- Ausnahme: System Center Endpoint Protection wird als Open Value Subscription (jährliche Zahlung) gebucht.
- In [System Center Server Standard](/hersteller/microsoft/system-center/standard-2022/) und [Datacenter](/hersteller/microsoft/system-center/datacenter-2022/) sind alle CML für den lizenzierten Server bereits enthalten bereits enthalten.

</ArrowList>


<Info>

[Microsoft Core CAL Suite](/hersteller/microsoft/cal-suite/core-cal-suite/) und [Enterprise CAL Suite](/hersteller/microsoft/cal-suite/enterprise-cal-suite/) enthalten die Endpoint Configuration Manager CML und System Center Endpoint Protection.

</Info>
