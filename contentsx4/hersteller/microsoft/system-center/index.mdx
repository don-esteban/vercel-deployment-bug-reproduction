<Intro>

System Center 2022 bietet Werkzeuge zur Verwaltung von physischen und virtualisierten IT-Umgebungen. Mit den integrierten und automatisierten Verwaltungslösungen reagieren Sie dynamischer auf Veränderungen und können Kosten reduzieren.
Microsoft unterscheidet bei System Center zwischen Server und Client Lizenzen. Unsere Seiten richten sich hauptsächlich an IT-Einkäufer und Virtualisierungsexperten für die Beschaffungsplanung. Mit wenigen Klicks erhalten Sie ein Angebot.

</Intro>

## Server Management Lizenzen

System Center für Serverbetriebssystemen gibt es in den beiden Editionen Standard und Datacenter. Wie beim [Windows Server](/hersteller/microsoft/windows-server/) wird hier nach Cores lizenziert. Die Editionen unterscheiden sich lediglich in den Virtualisierungsrechten.

<ArrowList>

- **[System Center 2022 Standard](/hersteller/microsoft/system-center/standard-2022/)**<br />
 Berechtigt zur Verwaltung eines Servers mit bis zu 2 virtuellen Maschinen sofern alle Kerne (min. 16) auf der Hardware lizenziert sind. Eignet sich für Umgebungen, die nicht viele virtualisierte Server einsetzen.

- **[System Center 2022 Datacenter](/hersteller/microsoft/system-center/datacenter-2022/)**<br />
 Eignet sich für virtualisierte Umgebungen mit einem Host-Server für die virtuellen Maschinen (Core-Lizenzierung, min. 16 Cores). Die Anzahl der verwalteten Betriebssysteme ist hier unbegrenzt.

</ArrowList>

## Client Management Lizenzen

Für verwaltete Geräte ohne Serverbetriebssystem werden Client Management Lizenzen (CML) benötigt. Diese werden pro Instanz des Betriebssytem (OSE) oder nach Nutzern lizenziert.

<ArrowList>

- **[System Center Client Management Licenses](/hersteller/microsoft/system-center/client-management-license/)**<br />
  Übersicht und Funktionen der Client Management Lizenzen (CML).

- **[Endpoint Configuration Manager](/hersteller/microsoft/system-center/client-management-license/endpoint-configuration-manager/)**<br />
  Der Endpoint Configuration Manager verbessert die Möglichkeiten für zentrale Konfigurationsverwaltung und Compliance im Unternehmen.

- **[Endpoint Protection](/hersteller/microsoft/system-center/client-management-license/endpoint-protection/)**<br />
  System Center Endpoint Protection bietet Malware-Schutz für Client-Betriebssysteme. Früher unter dem Namen Forefront bekannt.

- **[Service Manager](/hersteller/microsoft/system-center/client-management-license/service-manager/)**<br />
  System Center Service Manager (SCSM) ermöglicht eine verbesserte Integration von Benutzern, Workflows und Informationen in die Prozesse in Datacenter-Prozesse.

- **[Data Protection Manager](/hersteller/microsoft/system-center/client-management-license/data-protection-manager/)**<br />
  System Center Data Protection Manager (DPM) ermöglicht die Datensicherung von Servern, private Cloud, Clients oder Applikationen.

- **[Orchestrator](/hersteller/microsoft/system-center/client-management-license/orchestrator/)**<br />
  bietet Funktionen zur Prozessautomatisierung, Standardisierung und zur Verbindung mit Systemen verschiedener Anbieter.

</ArrowList>


## Core Infrastructure Server (CIS)

Die Microsoft Core Infrastructure Server (CIS) Suite ist ein Bundle aus dem aktuellen Windows Server und der Verwaltungslösung [System Center](/hersteller/microsoft/system-center/). Beide Produkte sind immer mit Software Assurance (SA) ausgestattet, sodass Sie immer die aktuellen Updates sowie Upgrades auf die neuste Version. Das Bundle bietet den Windows Server zusammen mit leistungsstarken Verwaltungstools für eine vereinfachte Administration, reduzierte Supportkosten und erhöhte Sicherheit.

## System Center 2022 Neuerungen

<ArrowList>

 - Support für Server 2022:<br />
    Unterstützung für die neuesten Versionen der Microsoft Server: [Windows Server 2022](/hersteller/microsoft/windows-server/), [Exchange Server 2019](/hersteller/microsoft/exchange/), [SharePoint Server Subscription Edition (2022)](/hersteller/microsoft/sharepoint/server-subscription-edition/) werden unterstützt
 - Erweiterte Serververwaltung:<br />
    Bereitstellung von SDN (Software-Defined Networking) in Virtual Machine Manager 2022 verbessert.
 - Erweiterte Zugriffskontrolle:<br />
    Microsoft-Teams-Integration und einfachere Warnhinweisverwaltung in [System Center Operations Manager (SCOM)](/hersteller/microsoft/system-center/client-management-license/operations-manager/) für die gesamte Organisation
 - Verbessertes Backup und Recovery:<br />
    Online-Sicherung und Item-Level-Wiederherstellung für Datei-/Ordner-Workloads in [System Center Data Protection Manager](/hersteller/microsoft/system-center/client-management-license/data-protection-manager/) verbessert.
 - Unterstütztung für hybride Umge­bungen mit Azure und Storage Spaces Direct (S2D)

</ArrowList>
