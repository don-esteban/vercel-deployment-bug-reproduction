## Client Management Lizenzen (CML)

Die System Center Serverlizenzen enthalten bereits alle Client Management Lizenzen für den lizenzierten Server. Für weitere Geräte können die System Center Client Management Lizenzen dazu gebucht werden:

 - [Endpoint Configuration Manager](/hersteller/microsoft/system-center/client-management-license/endpoint-configuration-manager/)<br />
    Konfigurationsverwaltung für die Microsoft-Plattform. Verbessert die Möglichkeiten der Kontrolle bei Beachtung von Unternehmen Compliance-Richtlinien. Ehemals System Center Configuration Manager (SCCM).
 - Virtual Machine Manager:<br />
    Bereitstellung und Verwaltung von virtuellen Maschinen auch in heterogenen Umgebungen. Bestandteil der [Endpoint Configuration Manager CML](/hersteller/microsoft/system-center/client-management-license/endpoint-configuration-manager/).
 - App Controller:<br />
    Verbindet Private und Public Cloud Umgebungen mit einer einheitlichen Oberfläche zur Bereitstellung, Verwaltung und Konfiguration neuer Dienste für Anwendungen.
 - [Service Manager](/hersteller/microsoft/system-center/client-management-license/service-manager/)<br />
    Ermöglicht die Integration von Benutzern, Workflows und Informationen in die Prozesse im Datacenter, durch Standardisierung.
 - [Operations Manager](/hersteller/microsoft/system-center/client-management-license/operations-manager/)<br />
    Bietet einen Überblick über die verwalteten Rechenzentren und Einblick in die Private und Public Cloud Infrastruktur. Außerdem werden Anwendungen und die Infrastruktur überwacht, um Leistung und Verfügbarkeit von kritischen Anwendungen zu gewährleisten.
 - [Data Protection Manager](/hersteller/microsoft/system-center/client-management-license/data-protection-manager/)<br />
    Ermöglicht die Datensicherung von Servern, private Cloud, Clients oder Applikationen auf unterschiedlichen Medien, wie Festplatte oder Bandlaufwerk (z.B. Backup von VMs unter vSphere).
 - [Orchestrator](/hersteller/microsoft/system-center/client-management-license/orchestrator/)<br />
    Integriert die Definition von Abläufen und die Prozessautomatisierung. Auch Service Management Automation.
 - [Endpoint Protection](/hersteller/microsoft/system-center/client-management-license/endpoint-protection/)<br />
    Schützt vor Viren und anderer Malware. Früher Forefront.


## Lizenzüberblick

System Center 2022 Standard wird, analog zum [Windows Server](/hersteller/microsoft/windows-server/), nach der Anzahl an Prozessorkernen (Cores) des Hosts lizenziert. Alle [Client Management Lizenzen (CML)](#client-management-lizenzen-cml) für den lizenzierten Server sind enthalten. Mindestens 8 Cores müssen pro Prozessor und 16 Cores pro Server lizenziert werden. Alle Kerne eines verwalteten Servers müssen abgedeckt sein.
