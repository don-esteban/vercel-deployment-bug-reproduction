<Intro>

 Microsoft Exchange kann als Exchange Online Plan aus der Microsoft 365 Cloud oder als on-premises Exchange Server mit CALs gekauft werden. Die aktuelle Version ist Exchange Server 2019.
 Exchange Online Pläne werden als nutzerbasierte Abonnements lizenziert. Exchange Server 2019 benötigt dagegen Serverlizenz und CALs, bietet aber eine dauerhafte Lizenzierung.
 Unsere Seiten richten sich hauptsächlich an IT-Einkäufer und Virtualisierungsexperten für die Beschaffungsplanung. Mit wenigen Klicks erhalten Sie ein Angebot.

</Intro>

## Exchange Server 2019

  Exchange Server Standard und Enterprise bieten identische Funktionalität und unterscheiden sich lediglich in der Anzahl der unterstützten Postfachdatenbanken. Für die Lizenzierung werden eine Serverlizenz und eine Standard CAL für Zugriffe benötigt. Zusätzliche Funktionen erhalten Sie mit der Enterprise CAL.

<ArrowList>

- **[Exchange Server Standard 2019](/hersteller/microsoft/exchange/server-standard-2019/)**<br />
 Exchange Server Standard bietet vollständige Funktionalität und unterstützt bis zu 5 Postfachdatenbanken.

- **[Exchange Server Enterprise 2019](/hersteller/microsoft/exchange/server-enterprise-2019/)**<br />
 Exchange Server Enterprise bietet dieselben Funktionen, unterstützt aber bis zu 100 Postfachdatenbanken.

</ArrowList>


## Exchange Server 2019 CALs

Eine Standard CAL wird immer für den Zugriff auf einen Exchange Server benötigt. Der Zugriff kann nach Nutzern (User CAL) oder Geräten (Device CAL) lizenziert werden.
Die Enterprise CAL ist ein Upgrade und bietet erweiterte Funktionen.


<ArrowList>

- **[Exchange Server Standard CAL 2019](/hersteller/microsoft/exchange/server-standard-cal/)**<br />
 Enthält alle grundlegenden Exchange Funktionen (E-Mail, Kalender, Kontakte). Wird immer für den Zugriff auf den Exchange Server benötigt. Zugriffslizenzen gibt es für Nutzer oder Geräte.

- **[Exchange Server Enterprise CAL 2019](/hersteller/microsoft/exchange/server-enterprise-cal/)**<br />
 Bietet zusätzliche Funktionalität, wie Unified Messaging, eine integrierte Archivierungfunktionalität und DLP. Enterprise CALs benötigen eine vorhandene Standard CAL.

</ArrowList>


## Exchange Online Pläne

Exchange Online bietet die Exchange-Funktionalität aus der Microsoft Cloud. Hier wird keine eigene Serverinfrakstruktur benötigt.
Lizenziert wird immer als benutzerbasiertes Abonnement. Es gibt zwei Pläne mit unterschiedlicher Funktionalität und einen eingeschränkten Kiosk Plan für Mitarbeiter ohne festen Arbeitsplatz.


<ArrowList>

- **[Exchange Online Plan 1](/hersteller/microsoft/exchange/online-plan-1/)**<br />
 Enthält Funktionen für E-Mail, Kontakte, Aufgabenverwaltung und Kalender. Mit 50 GB großen Postfächern genügt der Plan 1 den meisten Anforderungen kleinerer und mittlerer Organisationen.

- **[Exchange Online Plan 2](/hersteller/microsoft/exchange/online-plan-2/)**<br />
  Plan 2 enthält alle Funktionen aus Plan 1, ein doppelt so großes Postfach (100 GB), unlimitierten Archivierungsspeicher, Unified Messaging und zusätzliche Sicherheit mit Data Loss Prevention (DLP).

- **[Exchange Online Kiosk](/hersteller/microsoft/exchange/online-kiosk/)**<br />
 Der Kiosk Plan richtet sich an Mitarbeiter ohne fest zugewiesenen Arbeitsplatz (Kiosk-Worker). Der eingeschränkte Plan enthält ausschließlich die E-Mail Dienste, auf die über den Browser via OWA (Outlook Web Access) zugegriffen werden kann.

</ArrowList>


## Exchange Add-ons

Die Funktionalität des Exchange Server 2019 und der Exchange Online Pläne kann mit Add-ons erweitert werden:

<ArrowList>

- **[Exchange Online Archiving (EOA)](/hersteller/microsoft/exchange/online-archiving/)**<br />
Exchange Online Archiving (EOA) ist als Add-on für den on-premises Exchange Server oder die Online Pläne verfügbar.

- **[Exchange Online Protection (EOP)](/hersteller/microsoft/exchange/online-protection/)**<br />
Erweiterte Schutzfunktionen mit Exchange Online Protection (EOP) sind in den Exchange Online Plänen bereits enthalten, kann aber auch für on-premises Umgebungen lizenziert werden.

</ArrowList>

## Exchange 2022

Microsoft hat das Exchange Server 2022 Release Date auf das Jahr 2025 verschoben (siehe [Microsoft Roadmap](https://techcommunity.microsoft.com/t5/exchange-team-blog/exchange-server-roadmap-update/ba-p/3421389)).
Zu den Änderungen mit der neuen Version ist bislang wenig bekannt. Wahrscheinlich wird es, wie bereits bei [SharePoint](/hersteller/microsoft/sharepoint/server-subscription-edition/) und [Project](/hersteller/microsoft/project/server-subscription-edition/), eine Subscription Edition geben. Ob es weiterhin auch dauerhafte Lizenzen geben wird ist noch nicht bekannt.

Auf Grund der Verzögerung hat Microsoft eine Änderungen durchgeführt: Der Exchange Server 2019 unterstützt jetzt auch Installationen unter [Windows Server 2022](/hersteller/microsoft/windows-server/). Diese Funktion war eigentlich erst für die neue Exchange Server Version vorgesehen.
