<Intro>

  Exchange Server Standard 2019 unterstützt bis zu 5 Postfachdatenbanken, ansonsten ist die Funktionalität identisch mit dem Exchange Server Enterprise.
  Beim Zugriff auf Exchange Server wird für jeden Nutzer oder jedes Gerät mindestens eine Zugriffslizenz (Standard-CAL) benötigt.

## Lizenzüberblick

Exchange Server Standard 2019 ist die Exchange Serverlizenz für bis zu 5 Postfachdatenbanken. Für den Zugriff werden zusätzlich [CALs](/hersteller/microsoft/exchange/server-standard-cal/) benötigt!

<ArrowList>

- [Exchange Lizenzierung](/hersteller/microsoft/exchange/lizenzierung/)<br />
  Ausführliche Informationen zu Kauf und Lizenzierung.

- [Standard CAL](/hersteller/microsoft/exchange/server-standard-cal/)<br />
  Jeder Zugriff auf den Exchange Server benötigt eine Zugriffslizenz (CAL)!

- [Enterprise CAL](/hersteller/microsoft/exchange/server-enterprise-cal/)<br />
  Upgrade für die Standard CAL mit zusätzlichen Funktionen.

- [CSP](/hersteller/microsoft/csp/)<br />
  Dauerhafte Lizenz (Kauflizenz) ohne Wartung. Keine Mindestbestellmenge. Keine Sonderkonditionen für Behörden.

- [Open Value](/hersteller/microsoft/open-value/)<br />
  Volumenlizenz mit 3 Jahren Software Assurance. Mindestbestellmenge: 5 Lizenzen.

</ArrowList>
