<Intro>

 Die Standard CAL wird für den Zugriff auf Microsoft Exchange Server 2019 benötigt. Lizenziert wird entweder nach benannten Nutzern oder nach Geräten.
 Enthält alle grundlegenden Exchange Funktionen (E-Mail, Kalender, Kontakte). Erweiterte Funktionalität erhalten Sie mit einem Upgrade auf die Enterprise CAL.
 Nutzen Sie unsere Sofortangebote zur Beschaffungsplanung!


</Intro>

<Info>

  Die Funktionalität des Exchange Servers richtet sich nach der CAL Edition: [Standard CAL](/hersteller/microsoft/exchange/server-standard-cal/) bieten die Basisfunktionen. [Enterprise CAL](/hersteller/microsoft/exchange/server-enterprise-cal/) enthalten zusätzliche Funktionen, wie integrierte Archivierung und weitere.

</Info>


## Lizenzüberblick

Die Standard CAL wird immer für den Zugriff auf einen Exchange Server 2019 benötigt. Lizenziert wird nach benannten Benutzern ([User CAL](/glossar/user-cal-microsoft/)) oder nach Geräten ([Device CAL](/glossar/device-cal-microsoft/)).


<ArrowList>

- [Exchange Lizenzierung](/hersteller/microsoft/exchange/lizenzierung/)<br />
  Ausführliche Informationen zu Kauf und Lizenzierung.

- [Exchange Server 2019](/hersteller/microsoft/exchange/server-standard-2019/)<br />
  Jede Exchange Serverinstanz benötigt eine Lizenz.

- [Enterprise CAL](/hersteller/microsoft/exchange/server-enterprise-cal/)<br />
  Upgrade für die Standard CAL mit zusätzlichen Funktionen.

- [CSP](/hersteller/microsoft/csp/)<br />
  Dauerhafte Lizenz (Kauflizenz) ohne Wartung. Keine Mindestbestellmenge. Keine Sonderkonditionen für Behörden.

- [Open Value](/hersteller/microsoft/open-value/)<br />
  Volumenlizenz mit 3 Jahren Software Assurance. Mindestbestellmenge: 5 Lizenzen.
