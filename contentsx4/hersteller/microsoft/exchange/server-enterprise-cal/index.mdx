## Lizenzüberblick

Die Enterprise CAL ist ein Upgrade der [Exchange Server Standard CAL](/hersteller/microsoft/exchange/server-standard-cal/). Lizenziert wird nach benannten Benutzern ([User CAL](/glossar/user-cal-microsoft/)) oder nach Geräten ([Device CAL](/glossar/device-cal-microsoft/)).
Die Exchange Server Edition (Standard oder Enterprise) auf die zugegriffen wird, spielt beim Einsatz der Enterprise CAL keine Rolle. Die Education Lizenzen können mit zusätzlichen (Online-)Services gekauft werden. Damit erhalten Sie [Exchange Online Protection](/hersteller/microsoft/exchange/online-protection/) und Data Loss Prevention (DLP) aus der Microsoft 365 Cloud.

- [Open Value](/hersteller/microsoft/open-value/)<br />
  Volumenlizenz mit 3 Jahren Software Assurance. Mindestbestellmenge: 5 Lizenzen.
