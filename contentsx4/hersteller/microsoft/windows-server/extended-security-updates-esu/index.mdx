<Intro>

Microsoft bietet für den Windows Server 2012 R2 kostenpflichtige erweiterte Sicherheitsupdates (ESU - Extended Security Update). Mit den ESU kann der Support um bis zu drei Jahre verlängert werden.
Supportende des Windows Server 2012 R2 war bereits am 10. Oktober 2023.
Die Extended Security Updates sind ab sofort auch im CSP-Lizenzprogramm verfügbar. Für die ESU-Nutzung wird eine aktive Software Assurance oder Subscription benötigt.


</Intro>

<TOC />


## Microsoft Windows Server 2012 R2 Extended Security Updates

<ArrowList>

- Benötigt aktive Software-Assurance oder Subscription für Server und CALs.
- Mit ESU kann der Support um bis zu 3 Jahre verlängert werden (bis maximal 13.10.2026).
- Die Laufzeit des ersten ESU Jahres beginnt immer ab 11.10.2023 für ein Jahr (bis 08.10.2024).
- Preise für das zweite und dritte Jahr ESU werden erst kurz vor Beginn von Microsoft veröffentlicht. Die Kosten steigen um ca. 25% pro Jahr.
- ESU sind kummulativ, d.h. für ESU in Jahr 3 werden auch ESU für Jahr 1 und 2 benötigt.

</ArrowList>

## Lizenzüberblick

Windows Server 2012 ESU werden in Lizenz-Paketen für 2 oder 8 Cores angeboten. Jeder Prozessor muss mit mindestens 8 Cores lizenziert werden. Für jeden physische Server gilt eine Mindestlizenzierung von 16 Cores, auch wenn dieser nur über einen Prozessor verfügt.
Die Lizenzierung einzelner virtueller Maschinen mit Windows Server 2012 ESU ist nicht möglich. Es muss immer der komplette physische Server lizenziert werden.
Bei der Standard Edition ist nur Betrieb und Aktualisierung von bis zu zwei VMs abgedeckt. Wenn Sie mehr als 2 VMs für einen Server mit Standard Edition verwenden möchten, müssen alle physischen Kerne auf dem Server für jedes zusätzliche Paar virtueller Maschinen mit einer Windows Server Standard Lizenz und den ESU abgedeckt sein.
