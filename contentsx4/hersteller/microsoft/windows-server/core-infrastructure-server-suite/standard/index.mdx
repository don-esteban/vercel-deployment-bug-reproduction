<Intro>

  Microsoft Core Infrastructure Server (CIS) Suite Standard ist ein Bundle aus der aktuellen Version von Windows Server Standard und der Verwaltungslösung System Center Standard.
  Das Bundle bietet gegenüber der Standalone-Lizenzierung unter anderem reduzierte Supportkosten. Software Assurance (SA) für kostenlose Updates sowie Upgrades sind immer enthalten. Für den Zugriff werden CALs benötigt!
  Nutzen Sie unsere Sofortangebote zur Beschaffungsplanung!


</Intro>

## CIS Suite Standard Editionen

Die folgenden Produkte sind in der Suite enthalten, die Suite kann aber auch ohne eines der beiden Produkte bestellt werden:

<ArrowList>

  - [Suite Komplettbundle](#cis-suite-standard): Windows Server 2022 Standard Edition inkl. SA und System Center 2022 Standard Edition inklusive Software Assurance (SA).
  - [Suite ohne System Center Server](#cis-suite-standard-ohne-system-center): Für Kunden, die bereits System Center Server Standard im Einsatz haben. Enthält Windows Server 2022 Standard Edition inkl. SA.
  - [Suite ohne Windows Server](#cis-suite-standard-ohne-windows-server): Für Kunden, die bereits Windows Server Standard im Einsatz haben. Enthält System Center 2022 Standard Edition inkl. SA.

</ArrowList>

## Windows Server CALs

Jeder Zugriff auf einen Windows Server muss mit einer Windows Server CAL lizenziert werden. Eine External Connector Lizenz deckt alle Zugriffe von außerhalb des Unternehmens ab.
Für den Zugriff auf Dienste wie Terminalserver (RDS) oder Rechteverwaltung (AD RMS) sind zusätzliche CALs nötig. Ein Paket mit weiteren Microsoft CALs erhalten Sie mit den Microsoft CAL Suites.

<ArrowList>

 - [Windows Server CAL](/hersteller/microsoft/windows-server/cal/)
 - [External Connector](/hersteller/microsoft/windows-server/external-connector/)
 - [Remote Desktop Services (RDS) CAL](/hersteller/microsoft/windows-server/remote-desktop-services/)
 - [Rights Management Services (RMS) CAL](/hersteller/microsoft/windows-server/rights-management-services/)
 - [Microsoft CAL Suites](/hersteller/microsoft/cal-suite/)

</ArrowList>


## Lizenzüberblick

Enthält je nach Suite [Windows Server 2022 Standard](/hersteller/microsoft/windows-server/standard-2022/) und/oder [System Center Standard 2022](/hersteller/microsoft/system-center/standard-2022/).
Mindestens 16 Cores müssen pro Server lizenziert werden. Für den Zugriff werden zusätzlich [CALs](/hersteller/microsoft/windows-server/cal/) benötigt!
