<Intro>

Windows Server Zugriffslizenzen (CALs) werden für den Zugriff auf Windows Server 2022 oder Vorgängerversionen benötigt. Lizenziert wird nach benannten Nutzern oder nach Geräten. Mit älteren CALs darf nicht auf neue Server zugegriffen werden. Besteht keine aktive Software Assurance, muss neu gekauft werden.
Nutzen Sie unsere Sofortangebote zur Beschaffungsplanung!

</Intro>

## Windows Server CALs

Für den Zugriff auf Dienste wie Terminalserver (RDS) oder Rechteverwaltung (AD RMS) sind zusätzliche CALs nötig. Eine External Connector Lizenz deckt alle Zugriffe von außerhalb des Unternehmens ab.
Ein Paket mit weiteren Microsoft CALs erhalten Sie mit den Microsoft CAL Suites.

<ArrowList>

 - [External Connector](/hersteller/microsoft/windows-server/external-connector/)
 - [Remote Desktop Services (RDS) CAL](/hersteller/microsoft/windows-server/remote-desktop-services/)
 - [Rights Management Services (RMS) CAL](/hersteller/microsoft/windows-server/rights-management-services/)
 - [Microsoft CAL Suites](/hersteller/microsoft/cal-suite/)

</ArrowList>


<Info>

 **CALs dürfen nicht auf neuere Server zugreifen.**<br />
 Bei der Anschaffung von neuen Server-Versionen, zum Beispiel eines Windows Server 2022, können 2019er CALs oder älter nicht mehr verwendet werden.
 Umgekehrt darf man jedoch mit 2019er CALs auf ältere Server 2016 oder früher zugreifen. Mit SA oder eine Subscription können Sie die CALs kostenlos upgraden.

</Info>

## Lizenzüberblick

Windows Server 2022 CALs werden für den Zugriff auf [Windows Server 2022 Standard](/hersteller/microsoft/windows-server/standard-2022/) oder [Windows Server 2022 Datacenter](/hersteller/microsoft/windows-server/datacenter-2022/) benötigt. Lizenziert wird nach benannten Benutzern ([User CAL](/glossar/user-cal-microsoft/)) oder nach Geräten ([Device CAL](/glossar/device-cal-microsoft/)).

<ArrowList>

- [Windows Server Lizenzierung](/hersteller/microsoft/windows-server/lizenzierung/)<br />
  Ausführliche Informationen zu Windows Server Editionen, Kauf und Lizenzierung.

- [Windows Server und CALs](/hersteller/microsoft/windows-server/lizenzierung/#windows-server-2022-cals-kaufen)<br />
  Lizenzierung mit Server- und Zugriffslizenzen (CALs).

- [OEM](/hersteller/microsoft/oem/)<br />
  Günstigere, dauerhaft gültige Lizenz mit eingeschränkten Rechten.

- [CSP](/hersteller/microsoft/csp/)<br />
  Dauerhafte Lizenz (Kauflizenz) ohne Wartung. Keine Mindestbestellmenge. Keine Sonderkonditionen für Behörden.

- [CSP Subscription](/hersteller/microsoft/csp/)<br />
  Abonnementlizenz (Subscription). Keine Mindestbestellmenge.

- [Open Value](/hersteller/microsoft/open-value/)<br />
  Volumenlizenz mit 3 Jahren Software Assurance (SA). Mindestbestellmenge: 5 Lizenzen.

</ArrowList>
