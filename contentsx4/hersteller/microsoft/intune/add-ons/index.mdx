## Lizenzüberblick

Die Add-Ons werden als benutzerbasierte Abonnements ([User Subscription License - USL](/hersteller/microsoft/microsoft-365/lizenzierung/#user-subscription-license-usl)) gekauft. Die Laufzeit mit Preisschutz beträgt wahlweise ein Jahr oder einen Monat.
Benötigt einen vorhandenen [Intune Plan 1](/hersteller/microsoft/intune/) oder einen [Microsoft 365 Plan mit Intune](/hersteller/microsoft/intune/#intune-in-microsoft-365-plänen).
