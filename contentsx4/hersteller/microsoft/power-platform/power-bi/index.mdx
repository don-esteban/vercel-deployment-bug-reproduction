<Intro>

Microsoft Power BI ist eine Sammlung von Softwarediensten, Apps und Connectoren, kurz gesagt ein Analysetool, welches eine schnelle und fundierte Entscheidungsfindung ermöglicht. Dazu verarbeitet Power BI Daten aus verschiedenen Quellen und präsentiert die Ergebnisse sowie Berichte in Echtzeit.
Power BI Pro und Power BI Premium bieten die Verbindung mit hunderten Datenquellen, die Premium Edition zusätzlich die Self-Service-BI Funktionalität. Außerdem gibt es eine spezielle Version für Mobilgeräte (enthalten in Power BI Pro) und eine Version für eingebettete Anwendungen (Power BI Embedded). Zusätzlich gibt es den Power BI Berichtsserver für die lokale Berichterstellung (in Power BI Premium bereits enthalten).
Über unseren Warenkorb erhalten Sie mit wenigen Klicks ein Sofortangebot.

## Wann muss ich Power BI Pro lizenzieren?

Für den normalen Gebrauch gibt es die kostenlose Edition Power BI Desktop. Wenn auf Sie ein oder mehrere der folgenden Punkte zutreffen, wird Power BI Pro benötigt:

<ArrowList>

 - Verteilen von Inhalten an andere Benutzer sowie für die Peer-zu-Peer-Freigabe und -Zusammenarbeit
 - Veröffentlichen von Inhalten für Power BI Premium
 - Empfangen von Inhalten anderer Benutzer, sofern dem betreffenden Benutzer in Power BI Premium keine dedizierte Kapazität zugeordnet ist
 - Empfangen von Inhalten in Teams und SharePoint
 - Veröffentlichen von Inhalten in Power BI-Berichtsserver

</ArrowList>

Mehr zu den Inhalten der Editionen gibts unten in der Vergleichstabelle. Nutzen Sie auch den [Power BI Calculator](https://powerbi.microsoft.com/de-de/calculator/), um die von Ihnen benötige Anzahl an Power BI Lizenzen herauszufinden inkl. der ungefähren Kosten.
