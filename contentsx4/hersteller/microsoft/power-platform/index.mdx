<Intro>

  Die Microsoft Power Platform verbindet Power Apps, Power BI und Power Automate zu einer einheitlichen Plattform. Die Power Platform bietet eine einfache visuelle Bedienung, mit der eine effiziente Zusammenarbeit im Unternehmen ermöglicht wird.
  Die Power Platform setzt auf die Microsoft KI Entwicklung, um das Schreiben von Codes so weit wie möglich zu automatisieren. Hier finden Sie eine Übersicht der Microsoft Power Plattform Produkte.


</Intro>

## Power Platform Produkte

<ArrowList>

- **[Power BI](/hersteller/microsoft/power-platform/power-bi/)**<br />
 Mit Power BI vereinheitlichen Sie Daten aus einer Vielzahl von Quellen, um interaktive, umfassende Dashboards und Berichte zu erstellen, die wertvolle Erkenntnisse liefern und das Geschäft ankurbeln.

- **[Power Apps](/hersteller/microsoft/power-platform/power-apps/)**<br />
 Mit Power Apps kann jeder Apps in wenigen Stunden statt in Monaten entwickeln und teilen.

- **[Power Automate](/hersteller/microsoft/power-platform/power-automate/)**<br />
Mit Power Automate können Sie sich wiederholende Aufgaben sowie papierlose Prozesse optimieren. So können Sie sich auf das Wesentliche konzentrieren und die Produktivität in ihrem Unternehmen steigern.

</ArrowList>
