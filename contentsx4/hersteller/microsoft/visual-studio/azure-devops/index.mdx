<Intro>

Azure DevOps (ehemals Visual Studio Team Services - VSTS) bietet Software-Entwicklungstools zur Zusammenarbeit für das gesamte Team aus der Azure Cloud.
In den Visual Studio Abonnements ist Azure DevOps bereits enthalten. Azure DevOps kann auch direkt gehostet in der Microsoft Azure Cloud. Hier finden Sie eine Übersicht.


</Intro>

<TOC />




## Azure DevOps Preise

Azure DevOps ist in den [Visual Studio Professional](/hersteller/microsoft/visual-studio/professional-2022/) und [Enterprise Abonnements](/hersteller/microsoft/visual-studio/enterprise-2022/) bereits enthalten.
Visual Studio Professional bietet Zugriff auf die Basisfunktionen, die Enterprise Edition enthält zusätzlich erweiterte Module, wie Test Manager, Artefakte und selbstgehostete Pipelines.

<ArrowList>

 - [Visual Studio 2022 Professional](/hersteller/microsoft/visual-studio/professional-2022/)
 - [Visual Studio 2022 Enterprise](/hersteller/microsoft/visual-studio/enterprise-2022/)

</ArrowList>

## Lizenzierung und Kauf von Azure DevOps

Azure DevOps ist eine Sammlung von Azure-Diensten, die in den Visual Studio Abonnements enthalten sind. Die Azure DevOps Dienste können auch über die [Azure Subscription](/hersteller/microsoft/azure/) einzeln gebucht werden.
Die Abrechnung über Azure erfolgt monatlich, abhängig von der Azure Nutzung.

Überblick der verschiedenen Beschaffungswege von Azure DevOps:

<ArrowList>

 - Für kleinere Teams (bis zu 5 Benutzern) ist der Service kostenlos.
 - Für Nutzer, die an Open-Source Projekten beteiligt sind, ist der Service (eingeschränkt) kostenlos.
 - Nutzer, die über eine [Visual Studio](/hersteller/microsoft/visual-studio/) oder [MSDN-Lizenz](/hersteller/microsoft/visual-studio/msdn-for-platforms/) verfügen, erhalten Basic-Zugriff.
 - Nutzer, die eine [Visual Studio Enterprise](/hersteller/microsoft/visual-studio/enterprise-2022/) Lizenz besitzen, dürfen auch die Erweiterungen Test Manager und Package Management kostenlos nutzen.
 - Weitere Nutzer können bei Bedarf über [Azure](/hersteller/microsoft/azure/) hinzugebucht werden.
 - Weitere Pipelines und Dienste können bei Bedarf über Azure (kostenpflichtig) hinzugebucht werden. Wir helfen Ihnen gerne bei der Auswahl der Dienste und der Berechnung der Kosten.

</ArrowList>

## Azure DevOps Inhalt

Das kostenlose Paket für kleinere Teams enthält:

<ArrowList>

 - Azure Pipelines: 1 gehosteter Auftrag mit 1.800 Minuten pro Monat für CI/CD und 1 selbstgehosteter Auftrag
 - Azure Repos: Unbegrenzte private Git-Repositorys
 - Azure Artifacts: Paketverwaltung (5 Benutzer kostenlos)

</ArrowList>

Microsoft bietet einen [Kalkulator zur Berechnung der Kosten für Visual Studio Azure-Dienste](https://azure.microsoft.com/en-us/pricing/calculator/?service=visual-studio-team-services).


<Info>

  Jeder Benutzer mit kostenpflichtigem Tarif in Azure DevOps erhält auch eine [Azure DevOps Server-Clientzugriffslizenz (CAL)](/hersteller/microsoft/visual-studio/azure-devops-server/) dazu.


</Info>

## Funktionen

Azure DevOps bietet Software-Entwicklungstools zur Zusammenarbeit für das gesamte Team aus der Cloud:

<ArrowList>

 - **Agile Werkzeuge**<br />
  Nachverfolgung von Aufgaben und Verwaltung von Backlogs
 - **Git**<br />
  Kostenlose private Repositorys, Code Reviews, Codesuche und vieles mehr
 - **Continuous Integration**<br />
  Einfache Continuous Integration für Apps auf beliebigen Plattformen und Programmiersprachen
 - **Werkzeuge für Java-Teams**<br />
  Eclipse, IntelliJ, Android Studio-Plug-Ins sowie Maven, Ant, Gradle und Jenkins
 - **TFVC**<br />
  Ein zentrales System zur Versionskontrolle mit kostenlosen privaten Repository
 - **Testen**<br />
  Tools für manuelles, Leistungs- und automatisiertes Testen
 - **Paketverwaltung**<br />
  Erstellen, Verwalten, Schützen und Teilen Sie Ihre Softwarekomponenten.
 - **Release Management**<br />
  Schnellere und häufigere Bereitstellung mit Continuous Deployment

</ArrowList>
