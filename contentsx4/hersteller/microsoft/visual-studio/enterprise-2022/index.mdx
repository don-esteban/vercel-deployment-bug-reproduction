<Intro>

Die Visual Studio Enterprise Edition enthält alle Funktionen der Professional Edition und zusätzlich die Azure DevOps Erweiterungen Test Manager, Artefakte und selbstgehostete Pipelines.
Visual Studio 2022 Enterprise gibt es nur als Standard-Abonnement mit MSDN Subscription im Open Value Volumenlizenzprogramm oder als monatliches Cloud-Abonnement in der Azure Cloud.
Über unseren Warenkorb erhalten Sie mit wenigen Klicks ein Sofortangebot.

## Lizenzüberblick

Visual Studio 2022 Enterprise ist ausschließlich als Abonnement, mit MSDN Subscription, verfügbar.
Für VS 2022 Enterprise stehen die folgenden Lizenzierungsmöglichkeiten zur Verfügung:

<ArrowList>

 - **[Standard-Abonnement](/hersteller/microsoft/visual-studio/lizenzierung/#standard-abonnement)** (inkl. perpetual Lizenz)<br />
  Abonnement im [Open Value Volumenlizenzprogramm](/hersteller/microsoft/open-value/) inkl. MSDN Subscription mit 3 Jahren Laufzeit. Enthält eine Visual Studio perpetual Lizenz, die auch nach Ablauf des Abonnements (ohne MSDN) weiter genutzt werden kann. Preise finden Sie oben.
 - **[Cloud-Abonnement](/hersteller/microsoft/visual-studio/lizenzierung/#cloud-abonnement)**<br />
 Abonnement in der [Azure Subscription / Azure Plan](/hersteller/microsoft/azure/). Keine eigenständige Lizenz, die Abrechnung erfolgt über den [Azure Plan](/hersteller/microsoft/azure/). Ein Angebot erhalten Sie auf Anfrage.

</ArrowList>

## Funktionen

 Auf der Seite zum [Visual Studio Editionsvergleich](/hersteller/microsoft/visual-studio/editionen-vergleich/), finden Sie einen Überblick der Lizenzierungsmöglichkeiten und der Unterschiede zwischen den einzelnen Editionen.

<ArrowList>

 - **Visual Studio IDE**<br />
    Zugriff auf die neuste Visual Studio IDE Version und Erweiterte Features für Debuggen und Diagnose wie IntelliTrace, Code Map-Debugger, Time Travel Debugging. Komponententests, IntelliTest, Code Coverage und vollständige Plattformübergreifende Entwicklung. Eine genaue Auflistung finden Sie auf der [Microsoft Seite](https://visualstudio.microsoft.com/de/vs/compare/).
 - **Unterstützung für Entwicklungsplattformen**<br />
    Windows-Desktop, Universelle Windows-Apps, Web (ASP.NET), Office 365, Business Applications, Azure Stack, C++ für plattformübergreifende Entwicklung, Python, Node.js, .NET Core, Docker Tools (auch im Cloud-Abonnement enthalten).
 - **Azure DevOps und Azure DevOps Server**<br />
    Zugriff auf Features in beliebigen Azure DevOps-Organisationen. Enhält Erweiterte Features, Artefakte, eine Serverlizenz und CAL und selbstgehostete Pipelines (auch im Cloud-Abonnement enthalten).
 - **Cloud Services**<br />
    Windows Virtual Desktop, Microsoft 365 Entwicklerabo (Admin + 24 Benutzer), Azure Guthaben (50 USD pro Monat), Visual Studio App Center, Windows Store-Entwicklerkonto (zum Veröffentlichen von Apps) und zusätzlich Enterprise Mobility + Security sowie Power BI.
 - **Software für Dev/Test**<br />
    [Windows](/hersteller/microsoft/windows/), [Windows Server](/hersteller/microsoft/windows-server/), Windows Embedded, [Microsoft SQL Server](/hersteller/microsoft/sql-server/), R Server und zusätzlich Microsoft Office Professional Plus, Project, Visio, SharePoint, Exchange, Dynamics und andere Microsoft-Serversoftware.
 - **Software für den Produktionseinsatz**<br />
    Office 365 Pro Plus, Office Professional Plus 2019 (für die Nutzung auf einem Gerät).
 - **MSDN Abonnement**<br />
    Mit der MSDN Subscription erhält man Zugriff auf die Leistungen des Abonnement wie die Nutzung von Software für Dev/Test, Software für den Produktionseinsatz und Cloud Services. Desweiteren gibt es Visual Studio-Partnerangebote, erweiterten technischen Support und verschiedene Schulungs- und Weiterbildungskurse.

</ArrowList>
