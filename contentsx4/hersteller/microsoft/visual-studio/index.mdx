<Intro>

Visual Studio 2022 ist in den Editionen Visual Studio Enterprise und Visual Studio Professional als Standard- oder Cloud-Abonnement mit MSDN verfügbar. Die Visual Studio Test Professional Edition ist lediglich als Standard-Abonnement im Open Value Programm erhältlich.
Die Professional Edition erhalten Sie auch als Kauflizenz ohne MSDN. Mit den Cloud Diensten Azure DevOps und Azure DevOps Server 2022 erhalten Sie die Funktionalität zur Zusammenarbeit des lokalen Team Foundation Server (TFS) und des Visual Studio Team Systems (VSTS) in einer Lizenz.
Hier finden Sie eine Übersicht aller Editionen und Änderungen gegenüber der Vorgängerversion.

</Intro>

<TOC />


## Visual Studio 2022

<ArrowList>

- **[Visual Studio Professional 2022](/hersteller/microsoft/visual-studio/professional-2022/)**<br/>
 Entwicklungsumgebung (IDE) für einzelne Programmierer und kleine Teams. Kann als Kauflizenz oder optional als Standard- oder Cloud-Abonnement mit MSDN Subscription gekauft werden.

- **[Visual Studio Enterprise 2022](/hersteller/microsoft/visual-studio/enterprise-2022/)**<br />
 Entwicklungsumgebung (IDE) für große, verteilte Teams, lizenziert als Standard- oder Cloud-Abonnement. Bietet zusätzliche Funktionen, Tools und Services, wie Xamarin (mobile), Architektur- und Modellierungswerkzeuge und Lab-Management.

- **[Visual Studio Test Professional 2022](/hersteller/microsoft/visual-studio/test-professional-2022/)**<br />
 Diese Edition wurde speziell für Tester, Produktmanager und andere Projektbeteiligte entwickelt und bietet Zugang zu allen Funktionen zur Zusammenarbeit, für Nutzer die keine umfangreiche Entwicklungsumgebung benötigen. Nur als Standard-Abonnement im Open Value Volumenlizenzprogramm verfügbar.

</ArrowList>


## Azure DevOps

Die Azure DevOps Dienste aus der Microsoft Cloud bieten eine Vielzahl von zusätzlichen Diensten zur Entwicklung.
Viele Funktionen sind in den Visual Studio Abonnements bereits enthalten, können aber auch einzeln gekauft werden.

<ArrowList>

- **[Azure DevOps Server 2022](/hersteller/microsoft/visual-studio/azure-devops-server/)**<br />
 Der Entwicklungs-Server mit Funktionen wie Continuous Integration, Versionskontrolle, Paketverwaltung, Release-Management und mehr ist der Nachfolger des Team Foundation Server 2018 (TFS). Lizenzierbar als klassische Volumenlizenz im Open-Value Volumenlizenzprogramm. In [Visual Studio Enterprise](/hersteller/microsoft/visual-studio/enterprise-2022/) oder [Visual Studio Professional](/hersteller/microsoft/visual-studio/professional-2022/) bereits enthalten.

- **[Azure DevOps](/hersteller/microsoft/visual-studio/azure-devops/)**<br />
 Der Nachfolger der Visual Studio Team Systems (VSTS). Bietet die Funktionalität des Azure DevOps Servers als Dienste aus der Azure Cloud. Lizenziert als Abonnement.

- **[MSDN for Platforms](/hersteller/microsoft/visual-studio/msdn-for-platforms/)**<br />
  MSDN for Plaforms ist ein Abonnement für den Zugriff auf Microsoft Azure Dienste und Microsoft Software für Test-, Entwicklungs- und Evaluationszwecke.

</ArrowList>

## Neuerungen mit Visual Studio 2022

 Eine vollständige Übersicht finden Sie auf der [Microsoft Visual Studio Seite](https://learn.microsoft.com/de-de/visualstudio/ide/whats-new-visual-studio-docs?view=vs-2022). Hier ein Auszug der Neuerungen:

<ArrowList>

 - Enwicklung<br />
  Schnellere Entwicklung mit _Hot Reload_ für .NET und C++. Änderungen werden direkt angezeigt, ohne die App neu starten zu müssen.
 - Bugfixing<br />
  Leistungsstärkerer intergrierter Debugger mit intelligenten Haltepunkten
 - Benutzerfreundlichkeit<br />
  Neuer Dark- und Fokusmodus und mehr Kontrolle über die Optimierung Ihres Arbeitsbereichs.

</ArrowList>
