## Lizenzüberblick

Visual Studio 2022 Professional ist, als einzige Visual Studio Edition, wahlweise mit oder ohne MSDN Subscription verfügbar.
Damit gibt es die folgenden Lizenzierungsmöglichkeiten für VS 2022 Professional:


<ArrowList>

 - **[Standard-Abonnement](/hersteller/microsoft/visual-studio/lizenzierung/#standard-abonnement)** (inkl. perpetual Lizenz)<br />
  Abonnement im [Open Value Volumenlizenzprogramm](/hersteller/microsoft/open-value/) inkl. MSDN Subscription mit 3 Jahren Laufzeit. Enthält eine Visual Studio perpetual Lizenz, die auch nach Ablauf des Abonnements (ohne MSDN) weiter genutzt werden kann. Preise finden Sie oben.
 - **[Cloud-Abonnement](/hersteller/microsoft/visual-studio/lizenzierung/#cloud-abonnement)**<br />
 Abonnement in der [Azure Subscription / Azure Plan](/hersteller/microsoft/azure/). Keine eigenständige Lizenz, die Abrechnung erfolgt über den [Azure Plan](/hersteller/microsoft/azure/). Ein Angebot erhalten Sie auf Anfrage.
 - **[Kauflizenz](/hersteller/microsoft/visual-studio/lizenzierung/#kauflizenz-eigenständige-lizenz)**<br />
  Einzellizenz im [CSP](/hersteller/microsoft/csp/). Dauerhaft gültige Lizenz für die IDE. Ohne Support und MSDN Subscription. Preise finden Sie oben.

</ArrowList>

## Funktionen

Auf der Seite zum [Visual Studio Editionsvergleich](/hersteller/microsoft/visual-studio/editionen-vergleich/), finden Sie einen Überblick der Lizenzierungsmöglichkeiten und der Unterschiede zwischen den einzelnen Editionen.

<ArrowList>

 - **Visual Studio IDE**<br />
    Zugriff auf die neuste Visual Studio IDE Version und Basis Features für Debuggen und Diagnose wie Static Code Analysis, Leistungs- und Diagnosehub. Komponententests und Plattformübergreifende Entwicklung (eingeschränkt). Eine genaue Auflistung finden Sie auf der [Microsoft Seite](https://visualstudio.microsoft.com/de/vs/compare/).
 - **Unterstützung für Entwicklungsplattformen**<br />
    Windows-Desktop, Universelle Windows-Apps, Web (ASP.NET), Office 365, Business Applications, Azure Stack, C++ für plattformübergreifende Entwicklung, Python, Node.js, .NET Core, Docker Tools (auch im Cloud-Abonnement enthalten).
 - **Azure DevOps und Azure DevOps Server**<br />
    Zugriff auf Features in beliebigen Azure DevOps-Organisationen. Enhält Basis Features, Artefakte, eine Serverlizenz und CAL und selbstgehostete Pipelines beim Azure DevOps Server (auch im Cloud-Abonnement enthalten).
 - **Cloud Services**<br />
    Windows Virtual Desktop, Microsoft 365 Entwicklerabo (Admin + 24 Benutzer), Azure Guthaben (50 USD pro Monat), Visual Studio App Center, Windows Store-Entwicklerkonto (zum Veröffentlichen von Apps).
 - **Software für Dev/Test**<br />
    [Windows](/hersteller/microsoft/windows/), [Windows Server](/hersteller/microsoft/windows-server/), Windows Embedded, [Microsoft SQL Server](/hersteller/microsoft/sql-server/), R Server.
 - **MSDN Abonnement**<br />
    Mit der MSDN Subscription erhält man Zugriff auf die Leistungen des Abonnement wie die Nutzung von Software für Dev/Test und Cloud Services, sowie eingeschränkte Schulungskurse und technischen Support.

</ArrowList>
