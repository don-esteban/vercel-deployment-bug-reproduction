<Intro>

Die Visual Studio Test Professional Edition richtet sich an Tester, Produktmanager und andere Projektbeteiligte, die keine umfassende Entwicklungsumgebung benötigen, jedoch umfangreiche Funktionen und Werkzeuge zur Zusammenarbeit. Die Test Professional Edition bietet Zugriff auf den Testhub in Azure DevOps (Nachfolger von TFS und VSTS).
Visual Studio Test Professional 2022 ist ausschließlich im Open Value Volumenlizenzprogramm verfügbar und enthält daher immer die MSDN Subscription während der Laufzeit.
Über unseren Warenkorb erhalten Sie mit wenigen Klicks ein Sofortangebot.


</Intro>

## Lizenzüberblick


Visual Studio 2022 Test Professional ist ausschließlich als Standard-Abonnement im [Open-Value Lizenzprogramm](/hersteller/microsoft/open-value/) inkl. MSDN Subscription mit 3 Jahren Laufzeit verfügbar. Die Visual Studio Entwicklungsumgebung (IDE) ist hier nicht enthalten.
Die IDE erhalten Sie mit [Visual Studio 2022 Professional](/hersteller/microsoft/visual-studio/professional-2022/) und [Visual Studio 2022 Enterprise](/hersteller/microsoft/visual-studio/enterprise-2022/).


<ArrowList>

- [Visual Studio Lizenzierung](/hersteller/microsoft/visual-studio/lizenzierung/)<br />
Ausführliche Informationen zu Kauf und Lizenzierung der Office Produkte.

- [Standard-Abonnement](/hersteller/microsoft/visual-studio/lizenzierung/#standard-abonnement) (inkl. perpetual Lizenz)<br />
  Abonnement im [Open Value Volumenlizenzprogramm](/hersteller/microsoft/open-value/) inkl. MSDN Subscription mit 3 Jahren Laufzeit. Enthält eine Visual Studio perpetual Lizenz, die auch nach Ablauf des Abonnements (ohne MSDN) weiter genutzt werden kann. Preise finden Sie oben.

- [Open Value](/hersteller/microsoft/open-value/)<br />
Volumenlizenz mit 3 Jahren Software Assurance (SA). Mindestbestellmenge: 5 Lizenzen.

- [LTSC](/hersteller/microsoft/visual-studio/lizenzierung/#long-term-servicing-channel-ltsc)<br />
Visual Studio 2022 Lizenzen können wahlweise im Current Branch (immer auf dem neuesten Stand) oder als LTSC (Long Term Servicing Channel) Version genutzt werden. Die LTSC-Version ist bei Visual Studio keine eigenständige Lizenz.

## Funktionen

Auf der Seite zum [Visual Studio Editionsvergleich](/hersteller/microsoft/visual-studio/editionen-vergleich/), finden Sie einen Überblick der Lizenzierungsmöglichkeiten und der Unterschiede zwischen den einzelnen Editionen.

<ArrowList>

 - **Testfallverwaltung**<br/>
  Verwaltung von Tests, einschließlich Planung, Erstellung, Ausführung und Nachverfolgung von Tests, an einem zentralen Ort.
 - **Explorative Tests**<br/>
  Erstellen von einfachen Testfällen aus den Aufzeichnungen explorativer Tests, Kommentieren der Aufzeichnungen mit Anmerkungen, sowie umfassende Möglichkeiten zu Datenerfassung und Fehlerberichterstattung.
 - **Browserbasierte Tests**<br/>
  Durchführung von manuellen Tests und Akzeptanztests im Browser.
 - **Microsoft Test Manager**<br/>
  Ausführen, Aufzeichnen und Wiederholen von manuellen Tests.
 - **Fortlaufende Bereitstellung**<br/>
  Unterstützung von Release Management zur Vereinfachung des automatischen Bereitstellens und Testens von Anwendungen in mehreren Umgebungen.
 - **Azure DevOps und Azure DevOps Server**<br/>
  Zugriff auf Features in beliebigen Azure DevOps-Organisationen. Enhält Basis Features, Test Plans, eine Serverlizenz und CAL und selbstgehostete Pipelines beim Azure DevOps Server.
 - **MSDN Abonnement**<br/>
  Ein MSDN Abonnement für die Nutzung von Microsoft Windows Server, Windows Embedded und [Microsoft SQL Server](/hersteller/microsoft/sql-server/) für Test- und Evaluationszwecke ist enthalten. Enthält Zugriff auf Cloud Services wie Windows Virtual Desktop und eingeschränkte Schulungskurse sowie technischen Support.

</ArrowList>
