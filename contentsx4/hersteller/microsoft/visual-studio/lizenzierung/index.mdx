Die Visual Studio Produkte sind als Standard- und Cloud-Abonnements erhältlich. Die Professional erhalten Sie auch als Kauflizenz.
Auf dieser Seite finden Sie einen Überblick der verschiedenen Lizenzierungsarten und Editionen.

## Visual Studio 2022 Lizenzierung

Visual Studio kann je nach Edition über verschiedene Beschaffungswege erworben werden:

<ArrowList>

 - **Standard-Abonnement** im [Open Value Volumenlizenzprogramm](/hersteller/microsoft/open-value/) mit MSDN und perpetual Lizenz für die IDE.
 - **Cloud-Abonnement** als [Azure Subscription](/hersteller/microsoft/azure/). Die Buchung erfolgt direkt bei Microsoft. Die Abrechnung erfolgt über die Azure Subscription.
 - **Eigenständige Lizenz / Einmalkauf** (nur Professional) als [CSP Lizenz](/hersteller/microsoft/csp/)

</ArrowList>


### Standard-Abonnement

[Visual Studio 2022 Professional](/hersteller/microsoft/visual-studio/professional-2022/), [Visual Studio 2022 Enterprise](/hersteller/microsoft/visual-studio/enterprise-2022/) und die [Visual Studio 2022 Test Professional](/hersteller/microsoft/visual-studio/test-professional-2022/)
können als Abonnement mit 3 Jahren Laufzeit über das Volumenlizenzprogramm [Open Value (OV)](/hersteller/microsoft/open-value/) zusammen mit einer MSDN Subscription gekauft werden. Support und Upgrades während der Laufzeit sind enthalten.
Die Software (IDE) darf nach Ablauf der MSDN Subscription weiterhin genutzt werden, ohne zusätzlichen Leistungen, wie Support und kostenlose Updates. Ausnahme sind die Open Value Subscriptions (OVS - Mietlizenzen), die nach Ende der Mietdauer deinstalliert werden müssen.


### Cloud-Abonnement

[Visual Studio 2022 Professional](/hersteller/microsoft/visual-studio/professional-2022/) und [Visual Studio Enterprise](/hersteller/microsoft/visual-studio/enterprise-2022/) sind auch als Cloud Subscription verfügbar.
Die [Azure Subscription](/hersteller/microsoft/azure/) kann über uns als [Cloud Solution Provider (CSP)](/hersteller/microsoft/csp/) mit zusätzlichen Rabatten gegenüber dem Microsoft Store gebucht werden.
Das Cloud-Abonnement ist immer auf dem aktuellen Stand, kann aber nach Ablauf der Abonnement-Laufzeit ohne Verlängerung, nicht weiter genutzt werden.


### Kauflizenz (eigenständige Lizenz)

[Visual Studio Professional 2022](/hersteller/microsoft/visual-studio/professional-2022/) kann man auch als Kauflizenz (Eigenständige Lizenz) erwerben. Die Kauflizenz enthält keinen Anspruch auf Updates, sowie keine Leistungen gegenüber den Abonnements.
Außerdem ist es nicht möglich, den Azure DevOps Server zu nutzen. Das [Downgrade-Recht](/blog/microsoft-downgrade-rechte/) auf die Vorgängerversion ist dagegen enthalten. Für einzelne Entwickler, die nicht an ein Abonnement oder Vertragsabschlüsse gebunden werden möchten, ist dies eine kostengünstige Alternative.
Die Kauflizenz erhalten Sie im [CSP](/hersteller/microsoft/csp/), als Nachfolger der ehemaligen [Open Lizenzen](/hersteller/microsoft/open-license/).


### Long Term Servicing Channel (LTSC)

Visual Studio 2022 Lizenzen können wahlweise im Current Branch (immer auf dem neuesten Stand) oder als LTSC (Long Term Servicing Channel) Version genutzt werden.
Die LTSC-Version benötigt hier, im Gegensatz zu [Office](/hersteller/microsoft/office/) oder [Windows](/hersteller/microsoft/windows/), keine eigenständige Lizenz.

Die LTSC-Editionen deaktivieren automatische Updates, damit eine Version (aktuell: Visual Studio 2022 / Version 17.0) bis zu 18 Monate auf einer Nebenversion mit Sicherheits- und Fehlerbehebungen genutzt werden kann.
Sobald der Supportlebenszyklus für den installierten LTSC endet, müssen Sie zu einem LTSC-Kanal wechseln, der noch unterstützt wird. Sie können auch immer zurück in den aktuellen Kanal wechseln.
Eine genaue Übersicht zum Releaserhythmus der Visual Studio Kanäle finden Sie im [Microsoft Technet](https://learn.microsoft.com/de-de/visualstudio/productinfo/release-rhythm).

## Azure DevOps Lizenzierung

Azure DevOps ist eine Sammlung von Azure-Diensten. Die Dienste können auch über die [Azure Subscription](/hersteller/microsoft/azure/) einzeln gebucht werden und monatlich, abhängig von der Azure Nutzung, berechnet werden.
Microsoft bietet einen [Kalkulator zur Berechnung der Kosten für Visual Studio Azure-Dienste](https://azure.microsoft.com/en-us/pricing/calculator/?service=visual-studio-team-services).

Überblick der verschiedenen Beschaffungswege von Azure DevOps:

<ArrowList>

 - Für kleinere Teams (bis zu 5 Benutzern) ist der Service kostenlos.
 - Für Nutzer, die an Open-Source Projekten beteiligt sind, ist der Service (eingeschränkt) kostenlos.
 - Nutzer, die über eine [Visual Studio](/hersteller/microsoft/visual-studio/) oder [MSDN-Lizenz](/hersteller/microsoft/visual-studio/msdn-for-platforms/) verfügen, erhalten Basic-Zugriff.
 - Nutzer, die eine [Visual Studio Enterprise](/hersteller/microsoft/visual-studio/enterprise-2022/) Lizenz besitzen, dürfen auch die Erweiterungen Test Manager und Package Management kostenlos nutzen.
 - Weitere Nutzer können bei Bedarf über [Azure](/hersteller/microsoft/azure/) hinzugebucht werden.
 - Weitere Pipelines und Dienste können bei Bedarf über Azure (kostenpflichtig) hinzugebucht werden. Wir helfen Ihnen gerne bei der Auswahl der Dienste und der Berechnung der Kosten.

</ArrowList>


<Info>

  Jeder Benutzer mit kostenpflichtigem Tarif in Azure DevOps erhält auch eine [Azure DevOps Server-Clientzugriffslizenz (CAL)](/hersteller/microsoft/visual-studio/azure-devops-server/) dazu.


</Info>


## Lizenzprogramme und Bezugswege

Auf unserer Seite zu den [Microsoft Lizenzprogrammen](/hersteller/microsoft/lizenzprogramme/) finden Sie eine vollständige Übersicht.


### CSP Kauflizenzen

  Kauflizenzen für gewerbliche Kunden sind im [CSP-Lizenzprogramm](/hersteller/microsoft/csp/) erhältlich. Es gibt hier keine Mindestbestellmenge und auch keine Software Assurance. Bildungseinrichtungen und gemeinnützige Organisationen erhalten Sonderkonditionen. Behörden erhalten dieselben Einkaufskonditionen, wie Unternehmen.
  Das [Downgrade-Recht](/blog/microsoft-downgrade-rechte/), zum Einsatz älterer Versionen, ist hier auch enthalten. Die Software kann im CSP Portal heruntergeladen werden.

### Open Value Volumenlizenzen

  [Open Value Volumenlizenzen](/hersteller/microsoft/open-value/) enthalten immer Software Assurance (SA) und können als Kauflizenz mit 3 Jahren Software Assurance im Voraus oder auf Raten gekauft werden. Es gibt Sonderkonditionen für Behörden, Bildungseinrichtungen und gemeinnützige Organisationen.
  Seit Ende 2021 werden Open License Kunden bei SA-Verlängerungen auf das Open Value Lizenzprogramm umgestellt. Beachten Sie bitte, dass sich hierbei die Vertragslaufzeit von 2 Jahren (Open License) auf 3 Jahre (Open Value) ändert. Gerne beraten wir Sie beim Wechsel.

### Open License Volumenlizenzen

  Open License ist seit Anfang 2022 nicht mehr verfügbar. Im Microsoft [Open Volumenlizenzen](/hersteller/microsoft/open-license/) waren Lizenzen mit optionaler, zweijähriger Wartung (SA - Software Assurance) verfügbar. SA-Wartungsverlängerungen können weiterhin im Open Value Programm durchgeführt werden. Kauflizenzen erhalten Sie jetzt im CSP. Weitere Details zum ehemaligen Lizenzprogramm finden Sie auf unserer Seite zu [Microsoft Open License](/hersteller/microsoft/open-license/).

### Azure Subscription

  Cloud-Abonnements über eine Azure Subscription werden verbrauchsabhängig abgerechnet. Die Laufzeit beträgt hier einen Monat. Weitere Informationen finden Sie auf unserer Seite zur [Azure Subscription](/hersteller/microsoft/azure/).
