<Intro>

  Die Core CAL Suite enthält die Standard CALs für den Zugriff auf die meisten Microsoft Server Produkte, sowie System Center Lizenzen für die Verwaltung im Bundle. Lizenziert wird, wie bei anderen Microsoft Zugriffslizenzen (CALs), entweder nach Nutzern (User) oder Geräten (Devices).
  CAL Suites enthalten immer Software Assurance (SA) und sind daher versionslos. Mit aktiver SA können die CALs auch zum Zugriff auf neue Versionen der Microsoft Server, wie Windows Server 2022 genutzt werden. Wird die SA nicht verlängert, wird die Version auf dem Stand eingefroren.
  Nutzen Sie unsere Sofortangebote zur Beschaffungsplanung!


</Intro>

## Core CAL Suite Inhalt

 Mit der Core CAL Suite erhalten Sie die folgenden Zugriffslizenzen für [Microsoft Server Produkte:](/hersteller/microsoft/)

<ArrowList>

 - [Windows Server CAL](/hersteller/microsoft/windows-server/cal/)
 - [Exchange Server Standard CAL](/hersteller/microsoft/exchange/server-standard-cal/)
 - [SharePoint Server Standard CAL](/hersteller/microsoft/sharepoint/server-standard-cal/)
 - [Skype for Business Server Standard CAL](/hersteller/microsoft/skype-for-business/server-2019/standard-cal/)
 - [System Center Configuration Manager (SCCM) Client Management License](/hersteller/microsoft/system-center/client-management-license/configuration-manager/)
 - [System Center Endpoint Protection Client Management License](/hersteller/microsoft/system-center/client-management-license/endpoint-protection/)

</ArrowList>


## Lizenzüberblick

Die CAL Suite enthält alle (Standard-) CALs für die [oben genannten Microsoft Server](#core-cal-suite-inhalt). Lizenziert wird nach benannten Benutzern ([User CAL](/glossar/user-cal-microsoft/)) oder nach Geräten ([Device CAL](/glossar/device-cal-microsoft/)).
