<Intro>

  Microsoft CAL Suites sind Bundles mit Zugriffslizenzen (CALs - Client Access Licenses) für die Microsoft Server Produkte (u.a. Windows Server, SharePoint, Exchange, etc.). Die Core CAL Suite enthält Standard CALs. Mit der Enterprise CAL Suite (ECAL) erhalten Sie Enterprise CALs und weitere Funktionen.
  Für eine vollständige Übersicht listen wir hier auch die Microsoft 365 Online Pläne, die wie CALs verwendet werden können. Unsere Seiten richten sich hauptsächlich an IT-Einkäufer für die Beschaffungsplanung. Preise für einzelne CALs und Details zur Lizenzierung finden Sie auf den jeweiligen Unterseiten.


</Intro>

<TOC />


## Microsoft CAL Suites

<ArrowList>

- **[Core CAL Suite](/hersteller/microsoft/cal-suite/core-cal-suite/)**<br />
 Bundle bestehend aus Standard CALs für [Exchange](/hersteller/microsoft/exchange/server-standard-cal/), [SharePoint](/hersteller/microsoft/sharepoint/server-standard-cal/), [Skype for Business](/hersteller/microsoft/skype-for-business/server-2019/standard-cal/), inkl. [Windows Server CAL](/hersteller/microsoft/windows-server/cal/) und den System Center CML [Configuration Manager (SCCM)](/hersteller/microsoft/system-center/client-management-license/configuration-manager/) und [Endpoint Protection (Forefront)](/hersteller/microsoft/system-center/client-management-license/endpoint-protection/).

- **[Enterprise CAL (ECAL) Suite](/hersteller/microsoft/cal-suite/enterprise-cal-suite/)**<br />
 Enthält alle CALs der Core CAL Suite und zusätzlich die Enterprise Version für die Office Server Produkte (Exchange, SharePoint, Skype for Business).

- **[CAL Suite Bridge](/hersteller/microsoft/cal-suite/bridge/)**<br />
  Spezielle Lizenzen für die Einhaltung der unternehmensweiten Standardisierung (Open Value company-wide) bei der hybriden Nutzung mit Microsoft 365.


</ArrowList>


## Microsoft Server CALs


<ArrowList>

- **[Windows Server CAL](/hersteller/microsoft/windows-server/cal/)**<br />
 Die am häufigsten eingesetzte CAL. Lizenziert Zugriffe auf den [Windows Server](/hersteller/microsoft/windows-server/).

- **[Windows RDS CAL](/hersteller/microsoft/windows-server/remote-desktop-services/)**<br />
 Die Windows Remote Desktop Services (RDS) werden für den Zugriff auf Windows-Terminalserver benötigt.

- **[Windows RMS CAL](/hersteller/microsoft/windows-server/rights-management-services/)**<br />
RMS CALs lizenzieren den Zugriff auf die Rechteverwaltung mit Active Directory Rights Management Services (AD-RMS)

- **[Exchange Server CAL](/hersteller/microsoft/exchange/)**<br />
 Die Exchange CALs (Standard und Enterprise) lizenzieren den Zugriff auf [Exchange Server.](/hersteller/microsoft/exchange/)

- **[SharePoint Server CAL](/hersteller/microsoft/sharepoint/server-2019/)**<br />
 SharePoint CALs (Standard und Enterprise) enthalten Zugriff und Funktionalität des [SharePoint Servers.](/hersteller/microsoft/sharepoint/).

- **[Skype for Business Server CAL](/hersteller/microsoft/skype-for-business/server-2019/)**<br />
 Skype for Business Zugriffslizenzen sind in drei Editionen verfügbar: Standard, Enterprise und Plus CALs.

- **[Project Server CAL](/hersteller/microsoft/project/server-2019/cal/)**<br />
 Zugriffslizenzen für [Project Server](/hersteller/microsoft/project/server-2019/). In der [Project Professional Edition](/hersteller/microsoft/project/professional-2021/) enthalten.


 </ArrowList>
