## Lizenzüberblick

Die CAL Suite Bridge für EMS wird für die Standardisierung im [Open Value Lizenzprogramm](/hersteller/microsoft/open-value/) bei der hybriden Nutzung von [Core CAL Suite](/hersteller/microsoft/cal-suite/core-cal-suite/) oder [Enterprise CAL Suite](/hersteller/microsoft/cal-suite/enterprise-cal-suite/) benötigt. Lizenziert wird als monatliches Abonnement pro Benutzer.
Für jede Bridge CAL Suite wird ein [qualifizierender Enterprise Online Service](/hersteller/microsoft/cal-suite/bridge/#qualifizierende-online-services) benötigt.
Enthält alle die [oben aufgeführten CALs](#cal-suite-bridge-für-ems-inhalt) für den lizenzierten Nutzer.
