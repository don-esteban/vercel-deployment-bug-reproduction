<Intro>

  Die Enterprise CAL Suite enthält die Enterprise CALs für den Zugriff auf die meisten Microsoft Server Produkte, sowie weitere Lizenzen für die Verwaltung (u.a. AD Rights Management und System Center) im Bundle. Eine genaue Auflistung der Inhalte finden Sie unten.
  CAL Suites sind versionslos und können direkt bei Release neuer Versionen der Microsoft Server, wie Windows Server 2022 oder Exchange Server 2022, eingesetzt werden. Wird die SA nicht verlängert, wird die Version auf dem Stand eingefroren.
  Nutzen Sie unsere Sofortangebote zur Beschaffungsplanung!


</Intro>

## Enterprise CAL Suite Inhalt

 Mit der Enterprise CAL Suite erhalten Sie die folgenden Zugriffslizenzen für [Microsoft Server Produkte:](/hersteller/microsoft/)

<ArrowList>

 - Alle Komponenten der [Core CAL Suite](/hersteller/microsoft/cal-suite/core-cal-suite/) (u.a. Windows Server CAL)
 - [Exchange Server Enterprise CAL](/hersteller/microsoft/exchange/server-standard-cal/) (inkl. DLP und Exchange Online Protection)
 - [Exchange Online with Archiving for Exchange Server](/hersteller/microsoft/exchange/online-archiving/)
 - [SharePoint Server Enterprise CAL](/hersteller/microsoft/sharepoint/server-enterprise-cal/)
 - [Skype for Business Server Enterprise CAL](/hersteller/microsoft/skype-for-business/server-2019/enterprise-cal/)
 - [Windows Server Rights Management Services (RMS) CAL](/hersteller/microsoft/windows-server/rights-management-services/)
 - Advanced Threat Analytics Client Management License

</ArrowList>


## Lizenzüberblick

Die CAL Suite enthält alle (Enterprise-) CALs für die [oben genannten Microsoft Server](#enterprise-cal-suite-inhalt). Lizenziert wird nach benannten Benutzern ([User CAL](/glossar/user-cal-microsoft/)) oder nach Geräten ([Device CAL](/glossar/device-cal-microsoft/)).


<ArrowList>

- [Microsoft Lizenzierung](/hersteller/microsoft/lizenzierung/)<br />
  Ausführliche Informationen zu Kauf und Lizenzierung.

- [CAL Suite Bridge Lizenzierung](/hersteller/microsoft/cal-suite/bridge/#cal-suite-bridge-lizenzierung)<br />
  Übersicht zur Lizenzierung der CAL Suite Bridges.

- [Open Value](/hersteller/microsoft/open-value/)<br />
  Volumenlizenz mit 3 Jahren Software Assurance (SA). Mindestbestellmenge: 5 Lizenzen. Keine Sonderkonditionen für Bildungseinrichtungen und gemeinnützige Organisationen.

</ArrowList>
