<Intro>

Windows 11 bietet benutzerfreundliche Tools zur Optimierung des Desktops und visuelle Verbesserungen und wurde überraschend kurzfristig am 5. Oktober 2021 released. Die meisten Windows 11 Lizenzen berechtigen auch zur Nutzung der Vorgängerversion Windows 10 bis zum Supportende am 14. Oktober 2025.
Für Unternehmen ist Windows in den Editionen Windows 10/11 Pro (Gerätebasiert) oder Enterprise / Education (Nutzer- oder Gerätebasiert) verfügbar. Für die Lizenzierung virtueller Instanzen gibt es Windows VDA (Virtual Desktop Access) Lizenzen.


</Intro>

<TOC />



## Windows Editionen

Windows Pro ist als perpetual Geräte-Lizenz im CSP oder als OEM erhältlich. Die OEM Version dient auch als [Upgrade-Grundlage](/hersteller/microsoft/windows/update-voraussetzungen/).
Windows Enterprise kann als Gerätelizenz im Open Value Lizenzprogramm oder als Nutzerlizenz mitzusätzlichen Funktionen im CSP gekauft werden.

<ArrowList>

 - **[Windows 11 Pro](/hersteller/microsoft/windows/pro/)**<br />
    Gerätebasierte Lizenzierung / OEM-Version / Kauflizenz

 - **[Windows 10/11 Enterprise](/hersteller/microsoft/windows/enterprise/)**<br />
    Nutzer- oder Gerätebasierte Lizenzierung / Volumenlizenz / Education Abonnements

 - **[Windows 10/11 Education](/hersteller/microsoft/windows/education/)**<br />
    Gerätebasierte Lizenzierung / Volumenlizenz

</ArrowList>

## Weitere Windows Lizenzen

<ArrowList>

  Die Windows LTSC-Version (Long Term Servicing Channel) erhalten keine automatischen Updates. Für Windows 11 ist noch keine LTSC-Version verfügbar. Bis dahin erhalten Sie weiterhin die Windows 10 LTSC Edition.

</ArrowList>



## Windows 11 Upgrade oder neue Version?

Microsoft wirbt damit, dass Windows 11 ein kostenloses Upgrade ist. Tatsächlich betrifft dies nur Kunden, die kürzlich einen neuen PC im Einsatz oder bereits einen PC mit Windows 10 Installation nutzen.

Kunden, die bereits Windows 10 nutzen, können kostenlos Windows 11 auf upgraden.
Ob Ihre Hardware die nötigen Voraussetzungen für das Upgrade erfüllt, erfahren Sie über den [Windows PC Health Check von der Microsoft Seite](https://support.microsoft.com/en-us/windows/check-your-device-performance-and-health-in-windows-security-59d8499d-b6fd-6930-7667-ebf8ae10e08d).


## Neue Funktionen mit Windows 11

Mit der Veröffentlichung von Windows 10 hatte Microsoft eigentlich darauf bestanden in Zukunft keine Änderungen an Versionsnummern oder Namen durchzuführen. Die Änderungen bestehen daher vor allem aus optischen Verbesserungen der Bedienbarkeit.
Windows 11 erhält nur noch einmal jährlich Feature Updates, allerdings lassen sich die Updates genauso planen, vorbereiten und bereitstellen wie in Windows 10.
Die Windows 11 Editionen Home und Pro erhalten 24 Monate Bugfixes und Sicherheitsupdates, für Windows 11 Enterprise und Education beträgt dieser Zeitraum 36 Monate.

<ArrowList>

 - **Überarbeitetes Startmenü mit zentraler Steuerung**
 - **Verbesserte Suchfunktion**
 - **Rückkehr der Widgets und der Windows-Anmeldemelodie**
 - **Überarbeiteter Microsoft Store mit Amazon App Store integration**
 - **Cortana ist aus dem Windows 11 Setup entfernt**
 - **Teams ist in der Task-Leiste integriert**
 - **Optimierte Touch-, Pen- und Sprachbedienung**

</ArrowList>
