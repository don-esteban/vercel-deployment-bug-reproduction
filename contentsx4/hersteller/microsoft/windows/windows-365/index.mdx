Windows 365 Release war der 2. August 2021. Windows 365, der "Cloud PC" ist mit dem Betriebssystem Windows 10 oder Windows 11 verfügbar.
Das komplette Betriebssystem, sowie Anwendungen, Daten und Einstellungen werden in die Microsoft Cloud verlagert, wo es auf verschiedenen Endgeräten, unabhängig vom Standort nutzbar ist.

## Windows 365 Editionen

Windows 365 ist eine Abonnement/Subscription Lizenz und ausschließlich für Unternehmen verfügbar. Der Cloud Service von Microsoft und kann nur über [CSP](/hersteller/microsoft/csp/) lizenziert werden.
Windows 365 bietet Unternehmen mehr Flexibilität, sodass Mitarbeiter unabhängig von ihrem Standort und Gerät arbeiten können. Alle Informationen werden in der Cloud gespeichert und abgesichert, statt auf dem lokalen Gerät. IT-Administratoren können die Subscriptions zentral über den Microsoft Endpoint Manager verwalten.

Es gibt zwei Editionen: Windows 365 Business und Windows 365 Enterprise.

<ArrowList>

   - **Windows 365 Enterprise** benötigt Windows 10 E3 oder E5 bzw. Windows 11 E3 oder Windows 11 E5.
   - **Windows 365 Business** hat keine Voraussetzungen. Mit einer Windows Pro Lizenz (Nachweis erforderlich) erhalten Kunden Windows 365 Business zu einem reduzierten Preis (with Hybrid Benefits).

</ArrowList>

## Support und Updates

Windows 365 ist, wie alle anderen Microsoft Cloud Dienste (bspw. Microsoft 365) auch, immer auf den neusten Stand und erhält monatlich neue Feature Updates sowie Sicherheitspatches. Windows 365 ist ständig mit der Cloud verbunden. Das bedeutet, dass der **(Versions-)Zustand nicht eingefroren** werden kann und somit keine LTSC Möglichkeit gibt, dafür gibt es [Windows 10 LTSC](/hersteller/microsoft/windows/enterprise/ltsc/) bzw. [Windows 11 LTSC](/hersteller/microsoft/windows/).


## Für wen eignet sich Windows 365?

Windows 365 eignet sich für kleine und mittelständische Unternehmen, die Ihre Rechenleistung und den Speicherplatz auslagern möchten. Man benötigt lediglich einen PC, einen HTML 5 Browser und eine Internetverbindung. Insbesondere ist es für Unternehmen geeignet, die keine Kenntnisse über Virtualisierungen haben. Mit Windows 365 wird die Bereitstellung, Aktualisierung und Verwaltung des Betriebssystems für IT-Abteilungen vereinfacht. Die Verwaltung der Cloud-PCs erfolgt, wie auch bei lokalen PCs, über den Microsoft Endpoint Manager.

Da Windows 365 ständig eine Cloud Verbindung hat und aktuelle Feature Updates erhält, sollte es nicht für kritische Unternehmensbereiche eingesetzt werden.


<Info>

**Hinweis:** Für größere Unternehmen, die bereits Erfahrungen mit Virtualisierungen haben, entwickelt [Microsoft Azure Virtual Desktop](/hersteller/microsoft/azure/azure-virtual-desktop/) weiter. Mit der flexiblen VDI-Plattform (Virtual Desktop Infrastructure) in der Cloud, lassen sich die virtuellen Desktops und Remote-Apps sicher und mit maximaler Kontrolle bereitstellt. Sobald dazu weitere Informationen bekannt sind, werden wir diese veröffentlichen.

</Info>
