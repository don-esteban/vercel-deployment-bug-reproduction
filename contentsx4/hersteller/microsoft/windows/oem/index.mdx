## Lizenzüberblick

Microsoft Windows OEM	Lizenzen werden pro Gerät lizenziert und sind ausschließlich für die [Windows Pro Edition](/hersteller/microsoft/windows/pro/) verfügbar.
Für die [Windows Enterprise Edition](/hersteller/microsoft/windows/enterprise/) gibt es keine OEM Variante, die OEM Version kann hier aber als [Update-Grundlage](/hersteller/microsoft/windows/update-voraussetzungen/) genutzt werden.

## Besonderheiten der OEM-Lizenz

Die Windows OEM Lizenz enthält lediglich das [Downgrade-Recht](/blog/microsoft-downgrade-rechte/) zum Einsatz der Vorgängerversionen.
Ein Nachteil ist, das im Gegensatz zur Volumenlizenz, kein Re-Imaging Recht enthalten ist. Die Installation von einem einzigen Installationsträger und das verteilen auf mehreren Geräten sind somit den OEM-Lizenzen nicht erlaubt.
Auch der Datenträger für die Nutzung älterer Versionen ist nicht enthalten, für die Installation wird daher ein Datenträger und Key der alten Version vorausgesetzt.
