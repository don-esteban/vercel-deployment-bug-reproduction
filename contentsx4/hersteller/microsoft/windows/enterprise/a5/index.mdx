<Intro>

Windows 10/11 Enterprise A5 ist die Windows 11 Nutzerlizenz für Bildungseinrichtungen, die eine ganzheitliche, über die Cloud bereitgestellte Lösung für Endpunktsicherheit nutzen möchten.
Windows Enterprise A5 enthält alle Funktionen aus Windows Enterprise A3 und zusätzlich den Microsoft Defender for Endpoint. Über unseren Warenkorb erhalten Sie mit wenigen Klicks ein Sofortangebot.


</Intro>

## Lizenzüberblick

Windows 10/11 Enterprise A5 wird als benutzerbasiertes Abonnement ([User Subscription License - USL](/hersteller/microsoft/microsoft-365/lizenzierung/#user-subscription-license-usl)) gekauft und ist ausschließlich als Upgradelizenz verfügbar. Die Laufzeit mit Preisschutz beträgt wahlweise ein Jahr oder einen Monat.
Es wird eine [Windows Upgrade-Grundlage](/hersteller/microsoft/windows/update-voraussetzungen/) benötigt! Bei Education/Academic Plänen wird zwischen Lizenzen für Lehrkräfte und Mitarbeiter von Bildungseinrichtungen (for faculty) und Lizenzen für Schüler und Studenten (for students) unterschieden.

Der Plan ist ausschließlich für Schüler, Studenten und Lehrkräfte verfügbar. Ob Sie berechtigt sind die vergünstigte Edition zu kaufen, können Sie auf der [Microsoft Seite](https://www.microsoft.com/de-de/education/products/office?rtc=1&market=de&activetab=tab%3Aprimaryr1) überprüfen

## Funktionen

Windows 11 Enterprise A5 enthält den [Microsoft Defender für Endpunkt](/hersteller/microsoft/microsoft-365/defender/endpoint/), eine Komplettlösung für Endpunktsicherheit.
Ansonsten erhalten Sie dieselben Vorteile wie mit [Windows 11 Enerprise A3](/hersteller/microsoft/windows/enterprise/a3/) gegenüber [Windows 11 Pro](/hersteller/microsoft/windows/pro/):

<ArrowList>

-  Vielseitige Optionen für die Bereitstellung des Betriebssystems und kontrollierte Updates
-  Umfassende Geräte- und App-Verwaltung
-  Universelles Drucken für serverlose Druckverwaltung
-  Erweiterten Schutz vor den neuesten Sicherheitsbedrohungen

</ArrowList>

## Microsoft 365 Enterprise A5

Windows 11 Enterprise A5 ist auch in [Microsoft 365 Education A5](/hersteller/microsoft/microsoft-365/) enthalten. Das Bundle besteht aus [Office 365 Education A5](/hersteller/microsoft/office-365/education-a5/), [Enterprise Mobility + Security](/hersteller/microsoft/enterprise-mobility-security/) und der Windows Enterprise E5 Lizenz.
