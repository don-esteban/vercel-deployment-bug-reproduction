<Intro>

Die Windows 11 Enterprise Edition ist seit 5. Oktober 2021 erhältlich. Windows 11 Enterprise ist ausschließlich als Update-Lizenz verfügbar.
Windows 11 Enterprise ist im Open Value Volumenlizenzprogramm inklusive Software Assurance pro Gerät (per Device) oder als Windows 11 Enterprise E3 / E5 im CSP pro Benutzer lizenziert werden.
Für den Zugriff von Nicht-Windows-PCs kann Virtual Desktop Access (VDA) lizenziert werden.  Hier finden Sie eine Übersicht der Windows Enterprise Lizenzen.


</Intro>

<TOC />


## Windows 10/11 Enterprise Editionen

Windows Enterprise E3 und Windows Enterprise E5 sind die nutzerbasierten Enterprise Editionen. Alternativ kann auch mit Volumenlizenz nach Geräten lizenziert werden.

<ArrowList>

- [Windows 11 Enterprise per Device](/hersteller/microsoft/windows/enterprise/per-device/) - Gerätebasierte Open Value Volumenlizenz.
- [Windows 11 Enterprise E3](/hersteller/microsoft/windows/enterprise/e3/) - CSP-Subscription mit erweiterter Sicherheit und Verwaltung.
- [Windows 11 Enterprise E3 VDA](/hersteller/microsoft/windows/enterprise/e3/) - CSP-Subscription für Virtual Desktop Access (VDA) mit erweiterter Sicherheit und Verwaltung.
- [Windows 11 Enterprise E5](/hersteller/microsoft/windows/enterprise/e5/)  - CSP-Subscription mit allen Funktionen aus E3 und zusätzlich [Defender for Endpoint](/hersteller/microsoft/microsoft-365/defender/endpoint/)

</ArrowList>


## Windows 10/11 Enterprise Academic Editionen

Bildungseinrichtungen erhalten mit [Windows 11 Education](/hersteller/microsoft/windows/education/) eine vergünstigte Enterprise Edition.
Die Nutzerbasierte Lizenzierung erfolgt mit den Enterprise Academic Abonnements Windows 11 Enterprise A3 / A5.


<ArrowList>

- [Windows 11 Education](/hersteller/microsoft/windows/education/) - Gerätebasierte Open Value Volumenlizenz.
- [Windows 11 Enterprise A3](/hersteller/microsoft/windows/enterprise/a3/) - Nutzerbasierte CSP Subscription mit erweiterter Sicherheit und Verwaltung.
- [Windows 11 Enterprise A5](/hersteller/microsoft/windows/enterprise/a5/) - Nutzerbasierte CSP Subscription mit allen Funktionen aus E3 und zusätzlich [Defender for Endpoint](/hersteller/microsoft/microsoft-365/defender/endpoint/)

</ArrowList>


### Windows 10/11 Enterprise LTSC

Windows Enterprise LTSC (Long Term Servicing Channel) erhalten keine automatischen Updates und bleiben damit immer auf dem Versionsstand beim Kauf.
Die Windows 11 Enterprise LTSC-Version erscheint voraussichtlich erst 2024. Bis dahin erhalten Sie weiterhin die Windows 10 Enterprise LTSC Edition.
