<Intro>

Windows 10/11 Enterprise E5 ist die Nutzerlizenz für größere Unternehmen, die eine ganzheitliche, über die Cloud bereitgestellte Lösung für Endpunktsicherheit benötigen.
Windows 10/11 Enterprise E5 enthält alle Funktionen aus Enterprise E3 und zusätzliche Sicherheit mit dem Microsoft Defender for Endpoint. Über unseren Warenkorb erhalten Sie mit wenigen Klicks ein Sofortangebot.


</Intro>

## Lizenzüberblick

Windows 10/11 Enterprise E5 wird als benutzerbasiertes Abonnement ([User Subscription License - USL](/hersteller/microsoft/microsoft-365/lizenzierung/#user-subscription-license-usl)) gekauft und ist ausschließlich als Upgradelizenz verfügbar. Die Laufzeit mit Preisschutz beträgt wahlweise ein Jahr oder einen Monat.
Es wird eine Windows 10/11 Pro Lizenz oder Windows 10 Enterprise als [Upgrade-Grundlage](/hersteller/microsoft/windows/update-voraussetzungen/) benötigt! Für Bildungseinrichtungen gibt es die günstigere Edition [Windows Enterprise A5](/hersteller/microsoft/windows/enterprise/a5/).

- [Windows Lizenzierung](/hersteller/microsoft/windows/lizenzierung/)<br />
  Ausführliche Informationen zu Windows Kauf und Lizenzierung.

- [CSP Subscription](/hersteller/microsoft/csp/)<br />
  Abonnement-Lizenz. Die Laufzeit mit Preisschutz beträgt wahlweise ein Jahr oder einen Monat. Bereitstellung auf bis zu fünf Geräten pro Nutzer. Keine Mindestbestellmenge.

</ArrowList>


## Microsoft 365 Enterprise E5

Windows 11 Enterprise E5 ist auch in [Microsoft 365 Enterprise E5](/hersteller/microsoft/microsoft-365/enterprise-e5/) enthalten. Das Bundle besteht aus [Office 365 Enterprise E5](/hersteller/microsoft/office-365/enterprise-e5/), [Enterprise Mobility + Security](/hersteller/microsoft/enterprise-mobility-security/) und der Windows Enterprise E5 Lizenz.


## Funktionen

Windows 11 Enterprise E5 enthält den [Microsoft Defender für Endpunkt](/hersteller/microsoft/microsoft-365/defender/endpoint/), eine Komplettlösung für Endpunktsicherheit.
Ansonsten erhalten Sie dieselben Vorteile wie mit [Windows 11 Enerprise E3](/hersteller/microsoft/windows/enterprise/e3/) gegenüber [Windows 11 Pro](/hersteller/microsoft/windows/pro/):

<ArrowList>

-  Vielseitige Optionen für die Bereitstellung des Betriebssystems und kontrollierte Updates
-  Umfassende Geräte- und App-Verwaltung
-  Universelles Drucken für serverlose Druckverwaltung
-  Erweiterten Schutz vor den neuesten Sicherheitsbedrohungen

</ArrowList>
