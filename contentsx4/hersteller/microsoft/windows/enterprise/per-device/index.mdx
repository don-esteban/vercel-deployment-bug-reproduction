## Lizenzüberblick

Windows 10/11 Enterprise per Device wird pro Gerät lizenziert und ist ausschließlich als Upgradelizenz im Open Value Volumenlizenzprogramm verfügbar.
Auf unserer Seite finden Sie eine [Übersicht der Windows Upgrade-Grundlagen und Update Voraussetzungen.](/hersteller/microsoft/windows/update-voraussetzungen/)
Die Enterprise Lizenz für Bildungseinrichtungen heißt [Windows Education](/hersteller/microsoft/windows/education/). 

<ArrowList>

- [Windows Lizenzierung](/hersteller/microsoft/windows/lizenzierung/)<br />
  Ausführliche Informationen zu Windows Kauf und Lizenzierung.

- [Open Value](/hersteller/microsoft/open-value/)<br />
  Volumenlizenz mit 3 Jahren Software Assurance (SA). Mindestbestellmenge: 5 Lizenzen.

</ArrowList>

### Microsoft 365 Enterprise E3

Windows Enterprise E3 ist auch in [Microsoft 365 Enterprise E3](/hersteller/microsoft/microsoft-365/enterprise-e3/) enthalten. Das Bundle besteht aus [Office 365 Education A3](/hersteller/microsoft/office-365/enterprise-e3/), [Enterprise Mobility + Security](/hersteller/microsoft/enterprise-mobility-security/) und der Windows Enterprise Lizenz.
