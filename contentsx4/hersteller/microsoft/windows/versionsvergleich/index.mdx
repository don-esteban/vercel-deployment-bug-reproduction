## Windows LTSC Versionen

Die aktuelle LTSC Version ist: [Windows 10 Enterprise 2021](/hersteller/microsoft/windows/enterprise/ltsc/)

Die LTSC Version (Long Term Servicing Channel) beinhaltet eingeschränkte Windows Funktionen und liefert nur Sicherheits-Updates.
Es gibt hier keine Funktionsupdates, die gekaufte Version ist damit praktisch eingefroren.

Windows LTSC Versionen werden immer auf dem Stand des letzten Windows Updates veröffentlicht. Windows 10 Enterprise 2021 entspricht damit dem Funktionsstand des **Windows 10 21H2 Updates**.
Eine Windows 11 LTSC Version erscheint voraussichtlich erst 2024.
