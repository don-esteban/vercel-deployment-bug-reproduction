<Intro>

Microsoft Entra ID P2 ist eine Lösung zur Identitätsverwaltung aus der Microsoft Azure Cloud. Azure Active Directory (AAD) wurde im August 2023 in Microsoft Entra ID umbenannt.
Microsoft Entra ID P2 bietet erweiterten Schutz für alle Benutzer und Administratoren mit Funktionen wie Identity Protection und Privileged Identity Management und wird als Subscription pro verwaltetem Nutzer lizeniert.
Über unseren Warenkorb erhalten Sie mit wenigen Klicks ein Sofortangebot.


</Intro>
