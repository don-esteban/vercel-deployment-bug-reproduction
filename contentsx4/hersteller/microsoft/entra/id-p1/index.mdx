<Intro>

Microsoft Entra ID P1 ist eine Lösung zur Identitätsverwaltung aus der Microsoft Azure Cloud. Azure Active Directory (AAD) wurde im August 2023 in Microsoft Entra ID umbenannt.
Microsoft Entra ID P1 wird als Subscription pro verwaltetem Nutzer lizenziert. Über unseren Warenkorb erhalten Sie mit wenigen Klicks ein Sofortangebot.


</Intro>
