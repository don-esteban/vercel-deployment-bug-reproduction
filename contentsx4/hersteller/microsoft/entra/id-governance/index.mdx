<Intro>

Microsoft Entra ID Governance ist eine erweiterte Suite von Identity Governance-Funktionen zum Schützen, Überwachen und Prüfen des Zugriffs auf wichtige Ressourcen.
Bietet unter anderem Machine Learning-unterstützte Zugriffszertifizierungen und -prüfungen, Lebenszyklus-Workflows und Berechtigungsverwaltung mit Verified ID.

Microsoft Entra ID Governance ist ein Add-on und benötigt einen vorhanden Entra ID P1 oder P2 Plan. Lizenziert wird als jährliche Subscription pro verwaltetem Nutzer. Über unseren Warenkorb erhalten Sie mit wenigen Klicks ein Sofortangebot.



</Intro>
