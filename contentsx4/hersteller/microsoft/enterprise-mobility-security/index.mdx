<Intro>

Enterprise Mobility + Security (EMS) verbessert die Sicherheit beim Zugriff auf Cloud-Dienste und für mobile Mitarbeiter. Dazu erhalten Sie eine Identitäts- und Zugriffssteuerung für den Cloud Zugriff mit der Sie Ihre Unternehmensdaten besser schützen können. Außerdem ist die Virtualisierung von Desktops enthalten.
Es gibt die Editionen EMS E3 und EMS E5. Einen Vergleich der beiden Editionen finden Sie in unserer Tabelle unten.


</Intro>

<TOC />



## Enterprise Mobility + Security Editionen

<ArrowList>

- **[Enterprise Mobility + Security E3](/hersteller/microsoft/enterprise-mobility-security/e3/)**<br />
  Verbessert die Sicherheit beim Zugriff auf Cloud-Dienste und für mobile Mitarbeiter. Ermöglicht die Desktop-Virtualisierung.

- **[Enterprise Mobility + Security E5](/hersteller/microsoft/enterprise-mobility-security/e5/)**<br />
 Enthält die Funktionen aus [EMS E3](/hersteller/microsoft/enterprise-mobility-security/e3/) und zusätzlich intelligente Klassifizierung und Verschlüsselung von Dateien, sowie Kontrolle und Schutz für Ihre Cloud-Apps.

</ArrowList>

## Enterprise Mobility + Security Editionsvergleich

Enterprise Mobility + Security ist in zwei Varianten erhältlich: Wie bei den [Windows 10](/hersteller/microsoft/windows/)/[Windows 11](/hersteller/microsoft/windows/) und [Office 365](/hersteller/microsoft/office-365/) bzw. [Microsoft 365](/hersteller/microsoft/microsoft-365/) Abonnements, gibt es die [EMS E3](/hersteller/microsoft/enterprise-mobility-security/e3/) und [EMS E5](/hersteller/microsoft/enterprise-mobility-security/e5/).

Beide Editionen bieten die Identitäts- und Zugriffsverwaltung mit Single Sign-On und mehrstufiger Authentifizierung, die Verwaltung mobiler Geräte und Apps zum Schutz von Unternehmensdaten auf jedem Gerät, Verschlüsselung und Cloud-basierte Dateinachverfolgung, sowie Advanced Threat Protection zum Schutz vor fortschrittlichen und gezielten Angriffen. Für den Zugriff auf den Windows Server ist eine [Windows Server CAL](/hersteller/microsoft/windows-server/cal/) für den lizenzierten Nutzer bereits enthalten.

Die höherwertige Edition Enterprise Mobility + Security E5 enthält den [Azure Active Directory Premium Plan 2] mit zusätzlichem risikobasiertem, bedingtem Zugriff und individueller Identitätsverwaltung, sowie Azure Information Protection Premium Plan 2 für eine intelligente Klassifizierung und Verschlüsselung von Dateien, die innerhalb und außerhalb der Organisation freigegeben werden. Außerdem erhalten Sie mit [Microsoft Cloud App Security](/hersteller/microsoft/microsoft-365/cloud-app-security/) verbesserte Sichtbarkeit, Kontrolle und Schutz für Ihre Cloud-Apps.
