## Lizenzüberblick

Lizenziert wird als benutzerbasiertes Abonnement ([User Subscription License - USL](/hersteller/microsoft/microsoft-365/lizenzierung/#user-subscription-license-usl)). Jeder Nutzer kann die Software auf 5 Geräten installieren und nutzen. Die Laufzeit mit Preisschutz beträgt wahlweise ein Jahr oder einen Monat.
Auch als vergünstigtes Add-On für bestehende [Microsoft 365](/hersteller/microsoft/microsoft-365/) und [Office 365](/hersteller/microsoft/office-365/) Pläne erhältlich.
Bei Education/Academic Plänen wird zwischen Lizenzen für Lehrkräfte und Mitarbeiter von Bildungseinrichtungen (for faculty) und Lizenzen für Schüler und Studenten (for students) unterschieden.

<ArrowList>

- [Microsoft 365 Lizenzierung](/hersteller/microsoft/microsoft-365/lizenzierung/)<br />
  Ausführliche Informationen zu Kauf und Lizenzierung.

- [CSP Subscription](/hersteller/microsoft/csp/)<br />
  Abonnement-Lizenz mit Preisschutz während der Laufzeit. Keine Mindestbestellmenge. Keine Sonderkonditionen für Behörden.

- [NCE](/hersteller/microsoft/microsoft-365/lizenzierung/#new-commerce-experience-nce)<br />
  New Commerce Experience - CSP Subscription für Unternehmen.

- [Open Value](/hersteller/microsoft/open-value/)<br />
  Einjährige Abonnement-Lizenz mit Software Assurance. Mindestbestellmenge: 5 Lizenzen.

</ArrowList>


<Info>

Sie benötigen nur eine günstige Lösung zur Sicherung der mobilen Endgeräte Ihrer Mitarbeiter? <br />
Darauf bietet [Microsoft Intune](/hersteller/microsoft/intune/) eine einfache Antwort. Mit Intune lassen sich Endgeräte einfach in der Cloud verwalten. Außerdem kann Intune auch nach Geräten lizenziert werden kann.

</Info>


## Funktionen

<ArrowList>

- [Microsoft Azure Active Directory Premium P1](/hersteller/microsoft/azure/active-directory/) - Sicheres Single Sign-On für Apps in der Cloud und lokal, Mehrstufige Authentifizierung, Bedingter Zugriff, Erweiterte Sicherheitsberichte
- [Microsoft Intune](/hersteller/microsoft/intune/) - Verwaltung mobiler Geräte und Apps zum Schutz von Apps und Daten des Unternehmens auf jedem Gerät
- [Windows Server Client Access License (CAL)](/hersteller/microsoft/windows-server/cal/) - Zugriffslizenz für den [Windows Server 2022](/hersteller/microsoft/windows-server/).
- Azure Information Protection Premium P1 - Verschlüsselung für alle Dateien und Speicherorte, Cloud-basierte Dateinachverfolgung
- Microsoft Advanced Threat Analytics - Schutz vor fortschrittlichen und gezielten Angriffen durch Anwendung von verhaltensbasierten Analysen auf Benutzer und Entitäten, [Mainstream Support endet am 21. Januar 2021, Extended Support Januar 2026](https://techcommunity.microsoft.com/t5/security-compliance-and-identity/end-of-mainstream-support-for-advanced-threat-analytics-january/ba-p/1539181).

</ArrowList>


## Modern Workplace mit Microsoft 365

Enterprise Mobility + Security gibt es auch im [Microsoft 365 Bundle](/hersteller/microsoft/microsoft-365/). Hier sind auch [Office 365](/hersteller/microsoft/office-365/) bzw. [Microsoft 365 Apps](/hersteller/microsoft/microsoft-365/) und [Windows 10](/hersteller/microsoft/windows/)/[Windows 11](/hersteller/microsoft/windows/) als Nutzerabonnement enthalten.
Microsoft 365 ist als E3 oder E5 Plan verfügbar und enthält die jeweilige Enterprise Mobility + Security Edition. [Microsoft 365 E3](/hersteller/microsoft/microsoft-365/enterprise-e3/) enthält [Office 365 Enterprise E3](/hersteller/microsoft/office-365/enterprise-e3/) Plan und [Windows 10/11 Enterprise E3](/hersteller/microsoft/windows/enterprise/) zusammen mit EMS E3.
