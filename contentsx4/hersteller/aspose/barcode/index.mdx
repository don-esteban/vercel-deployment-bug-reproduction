Aspose.BarCode gibt es Kauflizenz (perpetual), die dauerhaft gültig ist, mit einem Jahr Basis-Support als Subscription.

- Als dauerhafte Kauflizenz (perpetual) verfügbar. Es ist immer 1 Jahr Subscription-Basis-Support inklusive, erweiterbar mit Paid Support.
- Andere Laufzeiten, Laufzeitanpassungen oder Wartungsverlängerung sind möglich. Gerne können Sie mich dazu kontaktieren.
- Mehr Informationen finden Sie auf unserer [Aspose Hauptseite](/hersteller/aspose/#lizenzierung).

Die Subscription kann man mit **Paid Support** erweitern.
**Basis-Support** beinhaltet kostenlose Updates, Hotfixes und Zugriff auf den allgemeinen Techniker-Support.

Paid-Support enthält den Basis-Support inkl. 24 Stunden Paid-Management-Support, höhere Priorität. Es gibt 4 verschiedene Lizenzarten, welche allgemeingültig sind:

- Developer Small Business: 1 Entwickler und 1 Bereitstellung, keine Verteilung oder Verwendung an Dritte/Öffentlichkeit möglich
- Site Small Business: 10 Entwickler und 10 Bereitstellungen, keine Verteilung oder Verwendung an Dritte/Öffentlichkeit möglich
- Developer OEM: 1 Entwickler und unbegrenzte Bereitstellungen, Verteilung oder Verwendung an Dritte/Öffentlichkeit möglich
- Site OEM: 10 Entwickler und unbegrenzte Bereitstellungen, Verteilung oder Verwendung an Dritte/Öffentlichkeit möglich

Ein individuelles Angebot für Wartungsverlängerungen erhalten Sie auf Anfrage.

## Funktionen

<ArrowList>

- **Barcode lesen und erkennen** <br/> Der Barcode-Leser unterstützt die folgenden Barcode-Symbologien:
  - 1D-Verkaufsstelle: UPC-A, UPC-E, EAN-8, EAN-13, GS1 DataBar (auch bekannt als RSS)
  - 1D-Industriesymbole: Code 39, Code 93, Code 128, GS1- 128, Codabar
  - ITF-14 2D-Symbole: QR-Code, Datenmatrix, Aztec, PDF 417


- **Barcode generieren** <br/> Generieren Sie QR-Barcodes, generieren Sie 1D-, 2D-, Post-Barcodes und viele andere Barcode-Typen:
  - PNG, JPEG, TIFF, BMP, GIF, SVG, EMF
  - 1D-Verkaufsstelle: UPC-A, UPC-E, EAN-8, EAN-13, GS1 DataBar (auch bekannt als RSS)
  - 1D-Industriesymbole: Code 39, Code 93, Code 128, GS1- 128, Codabar
  - ITF-14 2D-Symbole: QR-Code, Datenmatrix, Aztec, PDF 417


</ArrowList>
