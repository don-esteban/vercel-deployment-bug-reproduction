## Lizenzierung

<ArrowList>

Aspose.PDF gibt es Kauflizenz (perpetual), die dauerhaft gültig ist, mit einem Jahr Basis-Support als Subscription.

- Als dauerhafte Kauflizenz (perpetual) verfügbar. Es ist immer 1 Jahr Subscription-Basis-Support inklusive, erweiterbar mit Paid Support.
- Andere Laufzeiten, Laufzeitanpassungen oder Wartungsverlängerung sind möglich. Gerne können Sie mich dazu kontaktieren.
- Mehr Informationen finden Sie auf unserer [Aspose Hauptseite](/hersteller/aspose/#lizenzierung).

Die Subscription kann man mit **Paid Support** erweitern.
**Basis-Support** beinhaltet kostenlose Updates, Hotfixes und Zugriff auf den allgemeinen Techniker-Support.

Paid-Support enthält den Basis-Support inkl. 24 Stunden Paid-Management-Support, höhere Priorität. Es gibt 4 verschiedene Lizenzarten, welche allgemeingültig sind:

- Developer Small Business: 1 Entwickler und 1 Bereitstellung, keine Verteilung oder Verwendung an Dritte/Öffentlichkeit möglich
- Site Small Business: 10 Entwickler und 10 Bereitstellung, keine Verteilung oder Verwendung an Dritte/Öffentlichkeit möglich
- Developer OEM: 1 Entwickler und unbegrenzte Bereitstellung, Verteilung oder Verwendung an Dritte/Öffentlichkeit möglich
- Site OEM: 10 Entwickler und unbegrenzte Bereitstellung, Verteilung oder Verwendung an Dritte/Öffentlichkeit möglich

Unterschieden wird dann noch zwischen den jeweiligen APIs bzw. Platforms, bspw. max. Anzahl an Apps etc., genaueres finden Sie im jeweiligen Preisabschnitt. Mehr Informationen und Details zur Lizenzierung, Upgrades, Laufzeitanpassung finden Sie auf unserer Aspose Hauptseite im Abschnitt [Lizenzierung](/hersteller/aspose/#lizenzierung). Ein individuelles Angebot für Wartungsverlängerungen erhalten Sie auf Anfrage.

</ArrowList>


## Funktionen mit Aspose.PDF

<ArrowList>

- **PDF konvertieren** <br/> Konvertiert PDF-Dateien in in Microsoft Office-Formate, Bilder, Web und viele andere beliebte Dateiformate, ohne Ihren bevorzugten Browser zu verlassen.
- **Metadaten** <br/> Laden Sie einfach ihre PDF-Dateien hoch, um die Metadaten in einem beliebigen Browser anzuzeigen oder zu bearbeiten. Keine Installation oder zusätzliche Werkzeuge erforderlich.
- **PDF-Viewer** <br/> Sehen Sie von überall aus in ihrem bevorzugtem Browser PDF-Dokumente an.
- **Aspose.PDF Editor** <br/> Bearbeiten, Anzeigen und Konvertieren von PDFs von jedem Gerät aus.
- **Aspose.PDF Splitter** <br/> Teilen oder extrahieren Sie PDF-Dateien online.
- **Aspose.PDF eSign** <br/> Fügen Sie eine elektronische Signatur in ihren PDF-Dokumenten hinzu.
- **Aspose.PDF Verschlüsselung** <br/> Versehen Sie PDFs mit Passwörtern oder entfernen Sie diese.

</ArrowList>
