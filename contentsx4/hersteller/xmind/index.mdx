<Intro>

Xmind ist eine Mindmapping Software zur visuellen und professionellen Gestaltung von Gedankenprozesse und komplexen Informationen. Die Software unterstützt die Formate [Adobe PDF](/hersteller/adobe/), [Microsoft Word](/hersteller/microsoft/office/einzelprodukte/word-2021/), [Microsoft Excel](/hersteller/microsoft/office/einzelprodukte/excel-2021/), [Microsoft PowerPoint](/hersteller/microsoft/office/einzelprodukte/powerpoint-2021/), [Microsoft Project](/hersteller/microsoft/project/), FreeMind, HTML und viele mehr. Teilen können Sie Ihre Ergebnisse beispielsweise über soziale Medien, wie Facebook, Twitter und LinkedIn.



</Intro>

## Lizenzierung

Xmind kann als Einzellizenz oder als Enterprise Lösung lizenziert werden und ist abhängig von der Anzahl in verschiedenen Rabatt-Staffeln erhältlich. Gemeinnützige Organisationen, Behörden und Bildungseinrichtungen erhalten Sonderrabatte.

Die Software ist je nach Produkt als Standalone, als Netzwerklizenz oder als Subscription verfügbar.


### Standalone

Jeder lizenzierte Benutzer erhält einen eigenen Lizenzschlüssel. Das bedeutet, dass Sie bei 40 Benutzern insgesamt 40 verschiedene Lizenzschlüssel erhalten, die bei der Installation manuell eingegeben werden müssen.

Ab 50 Benutzern kann man den Lizenzschlüssel über den Hersteller konsolidieren lassen, sodass nur noch ein einheitlicher Lizenzschlüssel existiert. Die Installation erfolgt weiterhin separat für jeden Desktop bzw. Benutzer.


### Netzwerklizenz

Mit der Netzwerklizenz können die Lizenzen lokal und zentral verwaltet werden. So kann die Software beispielsweise von einem Terminalserver aus an ein Endgerät versendet und installiert werden.


### Abonnement

Xmind 2023 ist nur als Subscription erhältlich und muss jährlich erneuert werden. Die Subscription-Lizenz wird pro Benutzer lizenziert. Nur der lizenzierte Benutzer darf die Lizenz nutzen, auch bei einer mehrfachen Installation.
