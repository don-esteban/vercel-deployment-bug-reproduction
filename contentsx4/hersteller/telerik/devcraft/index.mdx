<Intro>

  Telerik DevCraft ist ein Bundle aus .NET und JavaScript Entwicklerwerkzeugen für Web, Desktop und mobile Apps. In drei unterschiedlichen Editionen (UI, Complete und Ultimate) mit unterschiedlicher Austattung und Support erhältlich.

Einen [Vergleich der DevCraft Editionen](#vergleich-der-devcraft-editionen) finden Sie unten. Einen [Vergleich der Telerik Supportoptionen](/hersteller/telerik/#telerik-support-vergleich) finden Sie auf unserer Telerik Seite.



</Intro>

## Telerik DevCraft Produkte

 - **[DevCraft UI](/hersteller/telerik/devcraft/ui/)**

 Bietet .NET und JavaScript UI Komponenten für Web, Desktop und mobile Apps. **Inklusive Lite Support**

- **[DevCraft Complete](/hersteller/telerik/devcraft/complete/)**

 Enthält zusätzlich Berichterstellung für Web und Desktop, sowie [JustMock](/hersteller/telerik/just-mock/) für Rapid Unit Testing. **Inklusive Priority Support**

- **[DevCraft Ultimate](/hersteller/telerik/devcraft/ultimate/)**

  Komplettlösung mit automatisierten Test-Werkzeugen für Entwickler und einer vollständigen Lösung zur Verwaltung von Berichten. **Inklusive Ultimate Support**
