<Intro>

  Kendo UI ist ein Bundle aus JavaScript UI Komponenten mit Bibliotheken für jQuery, Angular, React und Vue.

  Kendo UI ist nur noch in einer Standardedition verfügbar. Die Bundles mit [ASP.NET](/hersteller/telerik/kendo-ui/asp/), [JavaScript](/hersteller/telerik/kendo-ui/jsp/) und [PHP](/hersteller/telerik/kendo-ui/php/) sind nicht mehr verfügbar.
  [KendoReact](/hersteller/telerik/kendo-react/) ist auch einzeln erhältlich. Einen [Vergleich der Telerik Supportoptionen](/hersteller/telerik/#telerik-support-vergleich) finden Sie auf unserer Telerik Seite.


</Intro>

## Telerik Lizenzierung

 Kendo UI wird nach Nutzern (Entwicklern) lizenziert. Ab 2, bzw. 6 Lizenzen rutschen Sie in eine höhere Rabattstaffel und erhalten günstigere Einzelpreise.

  Die Lizenzen enthalten immer 1 Jahr Maintenance & Support, der optional verlängert werden kann. Damit erhalten Sie kostenlosen Zugriff auf neue Versionen und je nach Supportstufe (Lite/Priority/Ultimate) weitere Hilfe. Ein Angebot über Priority und Ultimate Support-Verlängerungen erhalten Sie auf Anfrage.

  Einen [Vergleich der Telerik Supportstufen](/hersteller/telerik/#telerik-support-vergleich) finden Sie auf der Telerik Hauptseite.
