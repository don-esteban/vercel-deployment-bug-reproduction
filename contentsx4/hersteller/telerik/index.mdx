<Intro>

  Telerik bietet Werkzeuge für Entwickler zum Erstellen von Web- und Desktop-Anwendungen oder mobiler Apps. Bekannte Produkte sind die Telerik und Kendo User Interface (UI) Komponenten-Bibliotheken.
  Mit Telerik DevCraft erhalten Sie ein vollständiges Bundle aus den Telerik .NET und JavaScript Produkten. Die Telerik Produkte sind eine Alternative zu den SAP Business Objects Crystal Reports Produkten.


</Intro>

## Telerik Produkte


- **[Telerik DevCraft](/hersteller/telerik/devcraft/)**

  Bundle aus .NET und JavaScript Entwicklerwerkzeugen für Web, Desktop und mobile Apps.

- **[Kendo UI](/hersteller/telerik/kendo-ui/)**

  Bundle aus JavaScript UI Komponenten mit Bibliotheken für jQuery, Angular, React und Vue.

- **[UI](/hersteller/telerik/ui/)**

  Mit Telerik User Interfaces (UI) erstellen Sie im Handumdrehen Anwendungen für einer Vielzahl von Frameworks.

- **[Test Studio](/hersteller/telerik/test-studio/)**

  Automatisiertes Testen für agile Umgebungen.

- **[JustMock](/hersteller/telerik/just-mock/)**

  C# Mocking Tool zum Erstellen von Unit Tests.

- **[Kendo React](/hersteller/telerik/kendo-react/)**

  Vollständige Bibliothek mit UI Komponenten für die Programmierung mit dem React Framework.

- **[Telerik Reporting](/hersteller/telerik/reporting/)**

  Eingebettetes Berichterstellungswerkzeug für Web und Desktop Anwendungen
