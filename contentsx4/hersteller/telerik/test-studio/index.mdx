  Das Progress Telerik Test Studio bietet automatisiertes Testen für agile Umgebungen. Als Web & Desktop oder Ultimate Edition erhältlich.



</Intro>

<TOC />



## Telerik Progress Test Studio Produkte


- **[Test Studio Web & Desktop](/hersteller/telerik/test-studio/web-desktop/)**

  Automatisiertes Testen für Web und PDF mit Standalone IDE und Test Scheduler.

- **[Test Studio Ultimate](/hersteller/telerik/test-studio/ultimate/)**

  Enthält zusätzliche Funktionen für Responsive, API, Load und Performance Testing.

- **[Test Studio Runtime Add-On](/hersteller/telerik/test-studio/add-ons/#runtime-add-on-preise)**

  Ermöglicht die automatische Erkennung und Ausführungen von Tests auf Remote Servern.

- **[Test Studio Load Testing Add-On](/hersteller/telerik/test-studio/add-ons/#load-testing-add-on-preise)**

  Load Testing mit simulierten Nutzerzahlen. In Paketen mit 100, 1000 oder 10000 virtuellen Nutzern erhältlich.
