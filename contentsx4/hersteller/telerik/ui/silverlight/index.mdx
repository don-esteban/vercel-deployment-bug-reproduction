## Telerik Lizenzierung

 Telerik UI for Silverlight wird nach Nutzern (Entwicklern) lizenziert. Ab 2, bzw. 6 Lizenzen rutschen Sie in eine höhere Rabattstaffel und erhalten günstigere Einzelpreise.

  Die Lizenzen enthalten immer 1 Jahr Priority Support, der optional verlängert werden kann. Damit erhalten Sie kostenlosen Zugriff auf neue Versionen und je nach Supportstufe (Lite/Priority/Ultimate) weitere Hilfe.

  Einen [Vergleich der Telerik Supportstufen](/hersteller/telerik/#telerik-support-vergleich) finden Sie auf der Telerik Hauptseite.
